BMC Dell KACEからデータをBMC Remedyforceにインポートする前に、次の要件を満たす必要があります。

    • JREをインストールしている。詳細は、JREのインストールを参照してください。
    • Pentaho Data Integrationツールをダウンロードしている。詳細は、Pentaho Data Integrationツールのダウンロードを参照してください。
    • BMC Communities Webサイトに登録されている。詳細は、BMC Communities Webサイトへの登録を参照してください。
    • MySQLコネクターファイルを\pdi-ce-4.x.0-stable\data-integration\libext\JDBCフォルダまたは\pdi-ce-5.x.1-stable\data-integration\libフォルダに追加します。詳細は、MySQLコネクターファイルを参照してください。
    • Dell KACE Management Centre（Dell KACEコンソール）にある［設定］>［コントロールパネル］タブ>［セキュリティの設定］>［General Security Settings（全般的なセキュリティの設定）］セクションの下で［Enable database access（データベースアクセスオプションの有効化）］オプションが選択されていることを確認します。

      • 注意: データベース名は、Dell KACE Management Centre（Dell KACEコンソール）にある［設定］>［コントロールパネル］タブ>［全般設定］>［Update Reporting User Password（Reporting Userパスワードの更新）］セクションにあります。

MySQLコネクターファイル

Dell KACE PentahoパッケージとDell KACEサーバーの間の接続を確立するには、mysql-connector-java-5.1.23-bin.jarファイルが必要です。このファイルは、BMC CommunitiesのWebサイトから入手できるDell KACEパッケージに収録されています。

mysql-connector-java-5.1.23-bin.jarファイルを追加するには

  1. ジョブファイルをBMC CommunitiesのWebサイトからダウンロードするには、次のいずれかのアクションを実行します。
    • CMDB 1.0を使用している場合は、 https://communities.bmc.com/docs/DOC-21752をクリックします。
    • CMDB 2.0を使用している場合は、 https://communities.bmc.com/docs/DOC-28433をクリックします。
  2. ダウンロードしたZIPファイルを解凍します。
  3. mysql-connector-java-5.1.23-bin.jarファイルを\pdi-ce-5.x.1-stable\data-integration\libフォルダにコピーします。

    4. Pentaho 4.3または4.4を使用している場合、mysql-connector-java-5.1.23-bin.jarファイルを\pdi-ce-4.x.0-stable\data-integration\libext\JDBCフォルダにコピーします。
