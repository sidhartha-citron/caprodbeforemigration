デフォルトでは、タスクは、それを作成したユーザーに割り当てられます。レコードの保存後に、そのレコードを他のユーザーまたはキューに割り当てることができます。

［Remedyforceコンソール］または［タスクコンソール］タブでのタスクオーナーの変更

［割り当ての詳細］セクションで、システム管理者の設定に基づいて、［オーナー］フィールドまたは［キュー］フィールド、［スタッフ］フィールドが表示されます。

［Remedyforceコンソール］または［タスクコンソール］タブでタスクオーナーを変更するには

  1. 他のユーザーまたはキューに割り当てるレコードに移動します。

    2. 詳細は、タスクレコードへの移動を参照してください。

    リストビューでレコードを選択し、［オーナーを変更］をクリックすることもできます。

  2. ［オーナー］フィールドが［割り当ての詳細］セクションに表示されている場合は、次のアクションを実行します。
    1. ［オーナー］ルックアップをクリックします。

      2. スタッフメンバーが組織に存在する場合、デフォルトではすべてのスタッフメンバーが［提案されたオーナー］ウィンドウに表示されます。スタッフメンバーが存在しない場合は、キューが表示されます。

    2. レコードをスタッフメンバーに割り当てるには、目的のスタッフメンバーのレコードをクリックします。
    3. レコードをキューに割り当てるには、［キュー］オプションを選択して目的のキュー名をクリックします。
    4. キューに属しているスタッフメンバーにレコードを割り当てるには、最初に目的のキューの行をクリックし、キューに属するスタッフメンバーのリストから目的のスタッフメンバーを選択します。
  3. ［スタッフ］および［キュー］フィールドが［割り当ての詳細］セクションに表示されている場合は、次のアクションを実行します。
    1. レコードをスタッフメンバーに割り当てるには、［スタッフ］フィールドから目的のスタッフを選択します。
    2. レコードをキューに割り当てるには、［キュー］フィールドから目的のキューを選択します。
    3. キューに属しているスタッフメンバーにレコードを割り当てるには、最初に目的のキューの行をクリックし、キューに属するスタッフメンバーのリストから目的のスタッフメンバーを選択します。

      4. ［キュー］フィールドでキューを選択すると、キューに属するスタッフメンバーが［スタッフ］フィールドルックアップに表示されます。

［タスク］タブからのタスクオーナーの変更

UI でルックアップ動作が異なるのと同様に、［タスク］タブ（Salesforceページレイアウト）でタスクオーナーを変更する手順は異なります。

［タスク］タブでタスクオーナーを変更するには

  1. 他のユーザーまたはキューに割り当てるレコードに移動します。

    2. 詳細は、タスクレコードへの移動を参照してください。

  2. ［スタッフ割り当ての詳細］セクションで、レコードが現在割り当てられているユーザー名の横にある［変更］をクリックします。
  3. ［オーナー］ドロップダウンリストで、目的のオプションを選択します。
    • レコードをユーザーに割り当てるには、［ユーザー］を選択します。
    • レコードをキューに割り当てるには、［キュー］を選択します。

      • タスクには、［顧客ポータルユーザー］オプションは必要ありません。

      スタッフメンバーが応対できない場合、または外出中の場合は、［ユーザーの検索］ウィンドウの［外出中］カラムにステータスが表示されます。［ユーザーの検索］ウィンドウの［外出中］カラムを設定するには、システム管理者に問い合わせてください。

      外出中のステータスを設定するには、［設定］>［私の個人情報］>［個人情報］の順に選択します。

      注意: Salesforce組織で、改善された［設定］ユーザーインターフェイスを有効化した場合は、［私の設定］>［個人用］>［個人情報］に移動します。改善された［設定］ユーザーインターフェイスについての詳細は、 http://help.salesforce.com/apex/HTViewHelpDoc?id=admin_setup_improved.htm&language=en_USを参照してください。

      ［BMC Remedyforce］セクションで［編集］をクリックし、［外出中］チェックボックスをオンにします。

  4. ユーザーまたはキューを選択するには、をクリックして目的のオプションを選択します。
  5. ［保存］をクリックします。

    6. 注意: スタッフおよびキュー割り当て機能を使用するには、システム管理者に問い合わせて、インシデントレイアウトをカスタマイズし、［StaffAssignmentPage］Visualforceページを追加します。また、管理者は［レコードの割り当て］ページで［キューとユーザーの割り当ての有効化］チェックボックスをオンにする必要があります。

    ［StaffAssignmentPage］Visualforceページがレイアウトに表示される場合、［スタッフ］および［キュー］フィールドが表示されます。

  6. ［スタッフ］フィールドのスタッフメンバーにレコードを割り当てるには、目的のスタッフメンバーを選択します。

    7. すべてのスタッフメンバーが表示されます。

  7. キューにレコードを割り当てるには、［キュー］フィールドで目的のキューを選択します。
  8. キューに属しているスタッフメンバーにレコードを割り当てるには、最初にキューレコードの行をクリックし、キューに属するスタッフメンバーのリストから目的のスタッフメンバーを選択します。

    9. キューを選択すると、キューに属するスタッフメンバーが［スタッフ］フィールドに表示されます。

  9. ［Apply］をクリックします。

    10. ［オーナー］フィールドには、選択したスタッフメンバーの名前が表示されます。

提案されたオーナーへのタスクの割り当て

システム管理者は、カテゴリに提案されたオーナーまたはキューを設定します。［タスク］タブ（Salesforceページレイアウト）のみで、ユーザーは提案されたスタッフ、キュー、またはキューのメンバーにタスクを割り当てることができます。

提案されたオーナーにタスクを割り当てるには

  1. 提案エキスパートに割り当てるタスクに移動します。

    2. 詳細は、タスクレコードへの移動を参照してください。

  2. ［Assign to Suggested Expert（提案エキスパートへの割り当て）］をクリックします。

    3. デフォルトでは［Assign to Suggested Expert（提案エキスパートへの割り当て）］ボタンを使用できません。タスクレイアウトをカスタマイズしてレイアウトにボタンを追加するには、システム管理者に問い合わせてください。

    デフォルトでは、提案されたスタッフメンバーが表示されます。

  3. 目的のオーナーを選択します。
  4. キューにタスクを割り当てるには、［キュー］オプションを選択して［適用］をクリックし、目的のキューを選択します。
  5. キューのメンバーにタスクを割り当てるには、［キュー］オプションを選択して［適用］をクリックします。
  6. キューのメンバーから目的のスタッフメンバーを選択します。
