このセクションでは、VisualforceページレイアウトUI でレコードに関する追加情報を表示する手順を説明します。

次のセクションで構成されています。

    • VisualforceページレイアウトUI でのユーザーおよびシステムアクションの表示
    • リンクされたインシデントの表示
    • リストビューについて

VisualforceページレイアウトUI でのユーザーおよびシステムアクションの表示

ブロードキャストレコードで実行されたユーザーおよびシステム定義アクションを表示するには、［補足情報］セクションの［履歴］タブの［表示］メニューを使用します。

Visualforceページレイアウトユーザーインターフェイスでユーザーおよびシステムアクションを表示するには

  1. ユーザーおよびシステム定義アクションを表示するレコードに移動します。

    2. 詳細は、レコードへの移動を参照してください。

  2. ［補足情報］セクションで、［履歴］タブをクリックします。
  3. ［表示］メニューをクリックします。
  4. ブロードキャストで実行されたユーザー定義アクションを表示するには、［ユーザーエントリ］を選択します。
  5. ブロードキャストで実行されたシステム定義アクションを表示するには、［システムエントリ］を選択します。
  6. ブロードキャストで実行されたすべてのアクション（ユーザー定義とシステム定義の両方）を表示するには、［すべてのエントリ］を選択します。

リンクされたインシデントの表示

ブロードキャストにリンクされたインシデントを表示するには、［補足情報］セクションの［インシデント］タブの［表示］メニューを使用します。

Visualforceページレイアウトユーザーインターフェイスでリンクされたインシデントを表示するには

  1. リンクされたインシデントを表示するブロードキャストに移動します。

    2. 詳細は、レコードへの移動を参照してください。

  2. ［補足情報］セクションで、［インシデント］タブをクリックします。
  3. ［インシデント］タブで、［表示］メニューをクリックします。
  4. すべてのリンクされたインシデントを表示するには、［すべて］を選択します。
  5. 「未完了」ステージのリンクされたインシデントを表示するには、［未完了］を選択します。
  6. 「承認済み」ステージのリンクされたインシデントを表示するには、［承認済み］を選択します。
  7. 「処理中」ステージのリンクされたインシデントを表示するには、［処理中］を選択します。
  8. 「完了」ステージのリンクされたインシデントを表示するには、［完了］を選択します。
