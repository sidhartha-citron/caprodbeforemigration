Microsoft System Center Configuration Manager（SCCM）、BMC Atrium Discovery and Dependency Mapping（BMC Atrium Discovery）、BMC BladeLogic Client Automation、BMC FootPrints Asset Core、Dell KACE™、LDAPサーバー、BMC Atrium Configuration Management Database（BMC Atrium CMDB）などの外部ソースからBMC Remedyforceにインポートできるデータ。外部ソースからデータをインポートするには、Pentaho Data Integrationツールを使用します。

使用しているConfiguration Management Database（CMDB）のバージョンによって、BMC Communitiesで提供されるパッケージが異なります。詳細は、BMC Remedyforce CMDBアーキテクチャを参照してください。
