以前解決したブロードキャストに似た問題または停止をクライアントが報告してきた場合は、解決済みブロードキャストを複製またはコピーして必要な詳細を変更し、ブロードキャストレコードを再利用することができます。

ブロードキャストを複製またはコピーするには

  1. 複製するブロードキャストレコードに移動します。

    2. 詳細は、レコードへの移動を参照してください。

  2. ［複製］をクリックします。
  3. Visualforceページレイアウトを使用している場合は、をクリックします。

    4. ［新規ブロードキャスト］フォームが表示され、複製元のブロードキャストのフィールドの値が表示されます。

  4. 必要に応じて変更を加え、［保存］をクリックします。
