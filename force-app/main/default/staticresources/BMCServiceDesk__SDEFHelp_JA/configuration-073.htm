フォルダで新しいクイックビューを作成するか、既存のクイックビューをフォルダにコピーすることで、フォルダにクイックビューを追加することができます。

フォルダにクイックビューを追加するには

  1. BMC Remedyforceホームページの［カタログ］をクリックします。
  2. ［カタログ］オプションのリストから［クイックビュー］を展開します。
  3. 既存のクイックビューをフォルダに追加するには、次のアクションを実行します。
    1. 追加するクイックビューが入っているフォルダに移動します。
    2. 目的のクイックビューを選択します。
    3. ［クエリーのコピー］をクリックします。
    4. クイックビューを貼り付けるフォルダを選択します。
    5. ［クエリーの貼り付け］をクリックします。
    6. ［クエリーの追加］ウィンドウの［クイックビュー名］フィールドに、クイックビューの名前を入力します。
    7. ［OK］をクリックします。
  4. フォルダに新しいクイックビューを作成するには、次のアクションを実行します。
    1. クイックビューを作成するフォルダに移動します。
    2. ［新規クエリー］をクリックします。
    3. クイックビューウィザードで、クイックビューを作成します。

      4. 詳細は、クイックビューの作成を参照してください。
