システム管理者はSalesforceページレイアウトでVisualforceページコンポーネントを追加または削除できます。次の表に、ページレイアウトから追加または削除できるコンポーネントを示します。

Visualforceページコンポーネント                 
適用先                         
説明
Broadcast Ticker（ブロードキャストティッカー）
デフォルトでは、このコンポーネントはすべてのオブジェクトのレイアウトに表示されます。
ページレイアウトでブロードキャストメッセージを表示します。
Stage Indicator（ステージインジケータ）
デフォルトでは、このコンポーネントはすべてのオブジェクトのレイアウトに表示されます。
レコードのステージを表示します。
Add Notes（メモの追加）
このコンポーネントは、インシデント、タスク、変更リクエスト、問題の各オブジェクトで使用できます。
レコードにメモを追加できます。システム管理者は、レイアウトのユーザーにこのコンポーネントを追加する必要があります。
レイアウトにわかりやすい名前を付けた別のセクションを作成し、［メモの追加］コンポーネントをそのセクションに追加することをお勧めします。そうしないと、レイアウトにラベルのないテキストボックスが表示されます。
Display Broadcasts（ブロードキャストの表示）
このコンポーネントは、ブロードキャストオブジェクトだけで使用できます。
ユーザーにブロードキャストを表示するオプションを設定できます。
カテゴリツリー
このコンポーネントは、インシデント、タスク、変更リクエスト、問題の各オブジェクトで使用できます。
すべてのカテゴリをカテゴリツリーに表示します。

メモ:
 ［一般的なアプリケーションの設定］ページで［優先度を変更すると、ユーザー入力のない現在の日付に基づいて、期限日時が再計算されます］チェックボックスがオフになっている場合は、このコンポーネントをオブジェクトページレイアウトに追加して、［期限日の計算］ウィンドウを表示する必要があります。
StaffandQueueAssignment
N/A
ユーザーがレコードをスタッフまたはキューに割り当てることを許可します。このコンポーネントにより、レイアウトに［スタッフ］および［キュー］フィールドが表示されます。
サービスリクエストの詳細
このコンポーネントは、インシデントオブジェクトだけで使用できます。
選択したリクエスト定義の詳細を入力できます。このコンポーネントはインシデントを保存した後に表示されます。
