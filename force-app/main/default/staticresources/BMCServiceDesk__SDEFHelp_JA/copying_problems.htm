見つかった問題の発生源とカテゴリが以前の問題と同じである場合は、既存の問題を複製し、必要な詳細を修正して問題を再利用できます。

レコードのコピーを行うオプションはVisualforceページレイアウトUI だけで提供されます。Salesforceページレイアウトを使用している場合は、［複製］オプションをクリックします。

問題を複製またはコピーするには

  1. 複製またはコピーする問題に移動します。

    2. 詳細は、問題レコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［アクション］メニューをクリックして、［複製］を選択します。
    • ［問題］タブ（Salesforceページレイアウト）を使用している場合、［複製］をクリックします。
    • VisualforceページレイアウトUI を使用している場合は、をクリックします。

      • ［新規問題］フォームが表示され、複製元の問題のフィールドの値が表示されます。

  3. ［新規問題］フォームで、必要な修正を加えます。
  4. ［保存］をクリックします。
