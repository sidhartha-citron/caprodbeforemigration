イニシエータが、すでに完了した変更リクエストとよく似た変更を要求している場合は、完了した変更リクエストを複製またはコピーし、必要な詳細を修正して変更リクエストを再利用できます。

レコードのコピーを行うオプションはVisualforceページレイアウトUI だけで提供されます。

変更リクエストを複製またはコピーするには

  1. 複製またはコピーする変更リクエストに移動します。

    2. 詳細は、変更リクエストレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［アクション］メニューをクリックして、［複製］を選択します。
    • ［変更リクエスト］タブ（Salesforceページレイアウト）を使用している場合、［複製］をクリックします。
    • VisualforceページレイアウトUI を使用している場合は、をクリックします。

      • ［新規変更リクエスト］フォームが表示され、複製した変更リクエストのフィールドの値が表示されます。

  3. 新しい［変更リクエスト］フォームで、必要な修正を加えます。
  4. ［保存］をクリックします。
