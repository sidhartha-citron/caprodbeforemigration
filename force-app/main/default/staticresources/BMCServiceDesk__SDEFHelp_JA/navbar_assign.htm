ナビゲータは、BMC Remedyforceにログオンすると左のペインに表示され、オブジェクトとそのフォームにアクセスする目的で使用できます。BMC Remedyforceには、次の事前定義済みのナビゲータが用意されています。

    • HELPDESK（ヘルプデスク）
    • SYSTEM ADMINISTRATOR（システム管理者）
    • CHANGE MANAGEMENT（変更管理）

システム管理者は、これらのいずれかのナビゲータをプロファイルに割り当てることができます。HELPDESKナビゲータを割り当てられたプロファイルでは、インシデントやタスク、ブロードキャスト、構成アイテムの作成・管理を実行できます。また、データの検索や、事前定義済みのクイックビューおよびインシデント管理のAlignabilityプロセスモデルの表示、クイックビューウィザードを使用した新規クイックビューの作成を実行できます。

CHANGE MANAGEMENTナビゲータを割り当てられたプロファイルでは、HELPDESKナビゲータで使用できるすべてのオブジェクトおよびフォームへのアクセスが可能です。また、このようなプロファイルでは、変更リクエストや変更アセスメント、サービス停止計画、承認、アグリーメント、サービスリクエストを作成・管理したり、変更スケジュールおよび変更管理のAlignabilityプロセスモデルを表示したりすることもできます。

SYSTEM ADMINISTRATORナビゲータを割り当てられたプロファイルでは、HELPDESKおよびCHANGE MANAGEMENTナビゲータで使用できるすべてのオブジェクト・フォームへのアクセスが可能です。また、このようなプロファイルでは、アプリケーションの設定とグローバル検索の設定、BMC Remedyforceの標準情報の設定、カタログコンソールでのフォルダおよびクイックビューの作成・管理も実行することができます。少なくとも1つのプロファイルがSYSTEM ADMINISTRATORナビゲータに割り当てられていることを確認してください。

ナビゲータを割り当てるには

  1. BMC Remedyforceホームページの［設定］をクリックします。
  2. ［設定］オプションのリストから［設定］を展開します。
  3. ［設定］ドロップダウンリストで、［ナビゲータの割り当て］をクリックします。
  4. 各プロファイルについて、［ナビゲータ］ドロップダウンリストから目的のナビゲータを選択します。

    5. 注意: 少なくとも1つのプロファイルがSYSTEM ADMINISTRATORナビゲータに割り当てられていることを確認してください。

  5. ［保存］をクリックします。
