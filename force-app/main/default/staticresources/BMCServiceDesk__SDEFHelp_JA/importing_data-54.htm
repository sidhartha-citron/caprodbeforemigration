BMC Remedyforceでは、BMC Atrium CMDBから各種のCIタイプをインポートするためのPentahoパッケージを使用できます。各パッケージには、KJBファイル（ジョブファイル）および関連するKTRファイルが格納されたフォルダがあります。次の表は、ジョブによってBMC Remedyforce CMDBにCIがインポートされる、各CIタイプとおよび該当するBMC Atrium CMDBクラスのフォルダ名の一覧です。

（CIタイプ）

BMC_BusinessService CIタイプに関する情報を転送します。

Combined Packages
AtriumToCMDB_ComputerSystem&Hardwareパッケージは、BMC_ComputerSystem、BMC_Cluster、BMC_ConcreteCollection、BMC_IPConnectivitySubnet、BMC_IPEndPoint、BMC_LANEndPoint、BMC_NetworkPort、BMC_Printer、BMC_Processor、BMC_Rack、BMC_VirtualSystemの各CIタイプに関する情報を転送します。
 
AtriumToCMDB_ComputerSystem&Software Server パッケージは、BMC_ComputerSystem、BMC_OperatingSystem、BMC_SoftwareServer、BMC_ApplicationService、BMC_Product、BMC_Database、BMC_BusinessService、BMC_BusinessOffering、BMC_Applicationの各CIタイプに関する情報を転送します。
