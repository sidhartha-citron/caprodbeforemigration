デフォルトでは、次のユーザーライセンスに基づくプロファイルが、［アプリケーションの設定］の［デフォルトのクライアントプロファイル］ドロップダウンリストに表示されます。

    • 顧客ポータルマネージャ
    • 顧客ポータルマネージャカスタム
    • 顧客ポータルマネージャ標準

BMC Remedyforceで新しいクライアントを作成するときのデフォルトとして設定するプロファイルは、［デフォルトのクライアントプロファイル］ドロップダウンリストから選択できます。

組織で使用するライセンス名が、顧客ポータルユーザーの［デフォルトのクライアントプロファイル］ドロップダウンリストに表示されていない場合は、必要なライセンス名をSalesforceのカスタム設定で追加して、BMC Remedyforceを設定します。

BMC Remedyforceのポータルユーザーに対応した新しいライセンス名を設定するには

  1. ［設定］>［開発］>［カスタム設定］の順に選択します。
  2. ［アクション］カラムで、［Remedyforceポータルライセンス］行の［管理］をクリックします。
  3. ［新規］をクリックします。
  4. ［名前］フィールドに、数値を入力します。
  5. ［値］フィールドに、顧客ポータルのライセンス名を入力します。

顧客ポータルのライセンス名は、Salesforceでユーザーライセンスとして参照されます。

注意: ［シーケンスID］フィールドは、空白のままにしておきます。

組織で使用するユーザーライセンス名を設定すると、そのライセンスに基づくプロファイルが、［アプリケーションの設定］の［デフォルトのクライアントプロファイル］ドロップダウンリストに表示されます。BMC Remedyforceで作成したクライアントユーザーのデフォルトとして設定するプロファイルを選択します。

パッケージのインストール時に、必要な権限がプロファイルに割り当てられていることを確認してください。
