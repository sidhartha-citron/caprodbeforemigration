BMC Atrium DiscoveryからデータをBMC Remedyforceにインポートする前に、次の要件を満たす必要があります。

    • JREをインストールしている。詳細は、JREのインストールを参照してください。
    • Pentaho Data Integrationツールをダウンロードしている。詳細は、Pentaho Data Integrationツールのダウンロードを参照してください。
    • BMC Communities Webサイトに登録されている。詳細は、BMC Communities Webサイトへの登録を参照してください。
    • KETTLE_HOME環境変数を作成して、Kitchen.batファイルの場所を保存します。詳細は、環境変数の作成を参照してください。
