類似したリリースが必要である場合、リリースを複製して、詳細を変更し、リリースレコードを再利用できます。

リリースを複製するには

  1. 複製するレコードに移動します。

    2. 詳細は、リリースレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合、［アクション］メニューから［複製］をクリックします。
    • ［リリース］タブ（Salesforceページレイアウト）を使用している場合、［複製］をクリックします。

      • ［新規リリース］フォームが表示され、複製元のリリースのフィールドの値が表示されます。

  3. 必要に応じて、新しいレコードの値を改訂します。

    4. フィールドの説明は、リリースのフィールドを参照してください。

  4. ［保存］をクリックします。
