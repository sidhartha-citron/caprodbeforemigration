正確性と完全性を期するため、BMCではテンプレートを使用してレコードを作成することをお勧めします。テンプレートを使用すると、一貫した方法で情報が取得されます。管理者は、よく発生するインシデントまたはサービスリクエストのテンプレートを定義できます。レコードへの移動は、使用しているUI によって異なります。使用するUI を決定するには、使用するUI の決定を参照してください。

テンプレートを適用してインシデントまたはサービスリクエストを作成するには

  1. 次のいずれかのタブに移動し、［新規］をクリックします。
    • ［Remedyforceコンソール］>［ビュー］>［インシデント］/［サービスリクエスト］
    • インシデントコンソール
    • インシデント（Salesforceページレイアウト）

      • 注意: ［未完了］状態のインシデントレコードを表示するには、［表示］メニューで［Remedyforce Incident View（Remedyforceインシデントビュー）］が選択されていることを確認してください。

  2. ［クライアントID］フィールドで、クライアントを選択します。
  3. （オプション）［Remedyforceコンソール］タブまたは［インシデントコンソール］タブを使用していて、インシデントがすでにクライアントに対して登録されているかどうか、クライアントまたはカテゴリに対して送信済みの他のインシデントを確認する場合、［エージェントツール］メニューの［インシデントの一致］オプションを選択することによって類似のインシデントを検索できます。

    4. レコードを保存するまで、［クライアントID］または［カテゴリ］の一致に基づいて検索結果が表示されます。レコードを保存するまで、レコードを別のレコードにリンクできません。レコードを保存した後は、システム管理者が［インシデントの一致］フィールドセットに指定したフィールドが一致するインシデントの検索に使用されます。

  4. ［テンプレート］フィールドでをクリックします。

    5. ［Remedyforceコンソール］タブまたは［インシデントコンソール］タブを使用している場合、［テンプレート］フィールドに値を入力することもできます。表示されている一致するレコードのリストから、目的のテンプレートを選択できます。

  5. ［テンプレート］ウィンドウで、目的のテンプレートを選択します。

    6. ［Remedyforceコンソール］タブまたは［インシデントコンソール］タブで、画面が更新され、選択したテンプレートからフィールドに入力されます。フォームにリッチテキストタイプのフィールドがある場合、［保存］をクリックすると、フォームにテンプレートフィールド値が表示されます。

    再発レコードを作成するには、再発を定義するテンプレートを選択します。同じテンプレート情報を使用して、すべての再発レコードが同時に作成されます。再発情報は［再発］フィールドに保存されます。［再発］フィールドにすでに情報が含まれている場合、レコードに再発テンプレートを適用することはできません。

    注意: 繰り返しレコードは、再発テンプレートを少なくとも1つのレコードに適用した場合にだけ作成されます。

  6. ［インシデント］タブを使用している場合、テンプレートを適用し、［保存］をクリックします。

    7. システム管理者が［フォームのフィールド値をテンプレートフィールド値に置換］チェックボックスをオンにしている場合、フォームに入力した値が、選択したテンプレートで指定されている値（存在する場合）に置換されます。

  7. インシデントまたはサービスリクエストに関するその他の情報を入力し、［保存］をクリックします。

    8. 詳細は、インシデントフィールドを参照してください。

  8. （オプション）［保存して新規作成］をクリックして、インシデントを保存し、新しいインシデントを作成する［新規インシデント］ページを開きます。
  9. サービスリクエストを作成している場合、表示されているポップアップウィンドウで、選択したリクエスト定義についての必要な詳細情報を入力します。

    10. リクエスト定義の詳細を入力すると、レコードがサービスリクエストとして保存されます。詳細を入力しない場合、レコードのリクエスト定義が保存されません。

    注意: お使いのブラウザでポップアップが有効になっていることを確認してください。

    システム管理者が「インシデントが作成された場合にクライアントに通知」ワークフローをアクティブにすると、レコードの作成時に、電子メールメッセージがレコードのクライアントへ送信されます。インシデントのクライアントとしてリードを選択した場合、電子メールメッセージは送信されません。

    レコードを保存すると、レコードの詳細ページが表示されます。次の追加フィールドが表示されます。

    • ［ステータスの詳細］セクション： インシデント進捗状況バーが表示されます。アイコンは、レコードに関連付けられているサービス目標トランザクションがある場合にアクティブになります。
    • ［クライアントの詳細］セクションの［名前］フィールドと［名字］フィールド： レコードに選択したクライアントの名前と名字が表示されます。これらのフィールドの値は、選択するクライアントによって異なります。これらのフィールドでは値を選択できません。レコードのクライアントを編集すると、それに合わせてこれらのフィールドの値も変わります。
    • ［最終変更者］フィールド： リリースを最後に更新したユーザーの名前が表示されます。

既存のインシデントまたはサービスリクエストレコードへのテンプレートの適用

既存のレコードにテンプレートを適用している場合、［アプリケーションの全般設定］ページの［既存レコードへのテンプレートの適用を許可］設定に基づいて、テンプレートを選択できるかどうかが決まります。このチェックボックスがオフになっている場合、［Remedyforceコンソール］タブまたは［インシデントコンソール］タブの［インシデント］フォームで［テンプレート］フィールドが無効と表示され、ユーザーが［テンプレート］フィールドでをクリックすると、エラーが表示されます。

インシデントレコードにテンプレートを適用する場合や、レコードのブロードキャストを選択する場合に想定される動作を次に示します。

    • システム管理者が［Replace form field values with template field values（フォームのフィールド値をテンプレートフィールド値に置換）］チェックボックスをオンにすると、［緊急度］以外のすべてのテンプレート値がレコードに入力されます。
    • ［Replace form field values with template field values（フォームのフィールド値をテンプレートフィールド値に置換）］チェックボックスをオフにすると、レコードの空白のフィールドだけがテンプレートから入力されます。［緊急度］フィールドの値は、テンプレートに値が存在する場合でも、テンプレートから入力されません。選択したブロードキャストから入力されます。
    • ブロードキャストの緊急度が緊急度の割り当て順序の最初に割り当てられている場合、リンクされたブロードキャストの［緊急度］フィールド値はインシデントレコードの［緊急度］フィールドに保存されます。
    • ブロードキャストオーナーがインシデントオーナーになります。

既存のインシデントまたはサービスリクエストレコードへの提案されたテンプレートの適用

インシデントの詳細フォームの［提案されたテンプレート］ボタンをクリックすると、提案されたテンプレートが表示されます。このボタンはデフォルトでは表示されません。インシデントの詳細フォームのレイアウトをカスタマイズしてボタンを表示するには、システム管理者に問い合わせてください。

電子メールメッセージから作成されたインシデントに提案されたテンプレートを使用します。［カテゴリ］フィールドまたは［説明］フィールドで保存した値に基づいてテンプレートが提案されます。

テンプレートは次のように提案されます。

    • インシデントの［カテゴリ］フィールドは、使用時や一致するテンプレートが表示される際にテンプレートの［カテゴリ］フィールドまたは［キーワード］フィールドと比較されます。
    • インシデントの［説明］フィールドまたは［カテゴリ］フィールドは、テンプレートの［キーワード］フィールドまたは［カテゴリ］フィールド（存在する場合）と比較され、一致するテンプレートが表示されます。

既存のインシデントまたはサービスリクエストレコードに提案されたテンプレートを適用するには

  1. 提案されたテンプレートを適用するインシデントレコードに移動します。

    2. 詳細は、インシデントレコードまたはサービスリクエストレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［エージェントツール］メニューから［提案されたテンプレート］を選択します。
    • ［インシデントコンソール］タブを使用している場合は、［エージェントツール］メニューから［提案されたテンプレート］を選択します。
    • ［インシデント］タブ（Salesforceページレイアウト）を使用している場合は、［提案されたテンプレート］をクリックします。
  3. テンプレートを選択します。
  4. ［保存］をクリックします。
