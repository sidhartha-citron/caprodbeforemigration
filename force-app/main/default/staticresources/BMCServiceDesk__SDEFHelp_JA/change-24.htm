変更記事からナレッジ記事を作成できます。変更リクエストから作成されたナレッジ記事は、［リンクされたナレッジ記事］セクションに表示されます。

変更リクエストからナレッジ記事を作成するには

  1. 目的の変更リクエストに移動します。

    2. 詳細は、変更リクエストレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［エージェントツール］メニューから［ナレッジ記事の作成］を選択します。

      • このオプションが［エージェントツール］メニューで使用できない場合は、システム管理者に連絡してメニューに追加するように依頼してください。

    • ［変更リクエスト］タブ（Salesforceページレイアウト）を使用している場合は、［ナレッジ記事の作成］をクリックします。

      • このボタンがフォームで使用できない場合は、システム管理者に連絡して、インシデントレイアウトをカスタマイズし、フォームに追加するように依頼してください。

  3. ［タイトル］フィールドにナレッジ記事のタイトルを入力します。

    4. 変更リクエストの［説明］フィールドの最初の30文字が［タイトル］フィールドに表示されます。別のタイトルを入力することもできます。このフィールドに入力できる最大文字数は255です。

  4. ［タイプ］フィールドで、ナレッジ記事の目的のタイプを選択し、［作成］をクリックします。

    5. 記事が［ドラフト］ステータスで保存されます。

    作成しているナレッジ記事のタイプに基づいて、変更リクエストの一部のフィールドのデータがナレッジ記事の該当フィールドに保存されます。次の表に、変更リクエストの該当フィールドと、データが保存されているナレッジ記事の対応フィールドを示します。

    変更リクエストのフィールド                   
    ナレッジ記事のフィールド
    FAQ
    説明
    質問
    使い方
    説明
    質問
    ソリューションの変更
    説明
    変更
    既知エラー
    変更理由
    根本原因
    説明
    エラー
    カテゴリ
    カテゴリ

  5. ［ナレッジ記事］ウィンドウで、［編集］をクリックします。
  6. ナレッジ記事の目的の詳細情報を入力します。

    7. 詳細は、ナレッジ記事の作成を参照してください。

  7. ［保存］をクリックします。
