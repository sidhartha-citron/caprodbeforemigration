タスクを保存後、［アクション履歴］セクション（VisualforceページレイアウトUI の場合は［補足情報］セクションの［履歴］タブ）内のタスクにユーザー定義アクションとシステム定義アクションを追加できます。

タスクレコードにアクション履歴レコードを追加するには

  1. アクション履歴を追加するタスクに移動します。

    2. 詳細は、タスクレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［アクション］メニューをクリックし、［新規アクション］を選択します。

      • または、［関連リスト］をクリックし、［アクション履歴］セクションで、［新規アクション］をクリックします。

    • ［タスクコンソール］タブを使用している場合は、［アクション］メニューをクリックし、［新規アクション］を選択します。

      • または、［関連リスト］をクリックし、［アクション履歴］セクションで、［新規アクション］をクリックします。

    • ［タスク］タブ（Salesforceページレイアウト）を使用している場合は、［アクション履歴］セクションの［新規アクション］をクリックします。
    • VisualforceページレイアウトUI を使用している場合は、［補足情報］セクションで、［履歴］タブをクリックして、メニューの［追加］を選択します。
  3. 追加する適切なアクションを選択します。

    4. ［メモ］オプションを選択すると、追加メモを入力するための新しいブラウザウィンドウが開きます。メモは、Salesforceページレイアウトの［アクション履歴］セクションに表示されます。Salesforceページレイアウトを使用しており、レイアウトに［メモ］フィールドが表示されない場合は、システム管理者に問い合わせて、ページレイアウトを編集してください。

    VisualforceページレイアウトUI では、このメモは［履歴］タブに表示されます。ある特定のアクションが［履歴］タブに追加されたときにクライアントに電子メールメッセージを送信するワークフローを作成できます。

  4. メモを追加し、［保存］をクリックします。

タスクに関連付けられたアクションの表示

［アクション履歴の表示］オプションを使用して、タスクに関連付けられたアクションを表示できます。［アクション履歴の表示］オプションでは、新規アクションの作成、タイプに基づくアクションのフィルタリング、およびアクションの印刷を行えます。

注意: ［タスク履歴］ウィンドウに、表示専用モードでアクションが表示されます。［タスク履歴］ウィンドウに表示されるアクションの編集や削除はできません。

タスクに関連付けられたアクションを表示するには

  1. アクションを表示するタスクに移動します。

    2. 詳細は、タスクレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［アクション］メニューをクリックし、［アクション履歴の表示］を選択します。

      • または、［関連リスト］をクリックし、［アクション履歴］セクションで、［アクション履歴の表示］をクリックします。

    • ［タスクコンソール］タブを使用している場合は、［アクション］メニューをクリックし、［アクション履歴の表示］を選択します。

      • または、［関連リスト］をクリックし、［アクション履歴］セクションで、［アクション履歴の表示］をクリックします。

    • ［タスク］タブ（Salesforceページレイアウト）を使用している場合は、［アクション履歴］セクションの［アクション履歴の表示］をクリックします。
  3. （オプション）アクションタイプに基づいてレコードをフィルタリングするには、リストから目的のアクションを選択します。
  4. （オプション）タスクに関連付けられたアクションを印刷するには、［印刷］をクリックします。
