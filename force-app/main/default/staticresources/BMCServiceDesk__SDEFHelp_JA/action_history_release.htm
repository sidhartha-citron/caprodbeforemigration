リリースの保存後に、［アクション履歴］セクションで、ユーザー定義アクションおよびシステム定義アクションをリリースに追加できます。

アクション履歴レコードをリリースに追加するには

  1. アクションを追加するレコードに移動します。

    2. 詳細は、リリースレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［アクション］メニューをクリックし、［新規アクション］をクリックします。

      • または、［関連リスト］をクリックし、［アクション履歴］セクションで、［新規アクション］をクリックします。

    • ［リリース］タブ（Salesforceページレイアウト）を使用している場合は、［アクション履歴］セクションの［新規アクション］をクリックします。
  3. ［アクション履歴］セクションに追加するアクションを選択します。

    4. ［メモ］オプションを選択すると、追加メモを入力するための新しいブラウザウィンドウが開きます。メモは［アクション履歴］セクションに表示されます。

    注意: ページレイアウトで［メモ］フィールドを取得するには、ページの右上隅にある［レイアウトの編集］リンクをクリックして、レイアウトを編集します。［Visualforceページ］をクリックして、［Add Notes（メモの追加）］を必要なセクションにドラッグアンドドロップします。

    わかりやすい名前を付けた別のセクションをレイアウトに作成し、［メモの追加］オプションをそのセクションに追加することをお勧めします。そうしないと、レイアウトにラベルのないテキストボックスが表示されます。

リリースに関連付けられたアクションの表示

［アクション履歴の表示］オプションを使用すると、リリースに関連付けられたアクションを表示できます。［アクション履歴の表示］オプションでは、新規アクションの作成、タイプに基づくアクションのフィルタリング、およびアクションの印刷を行えます。

リリースに関連付けられたアクションを表示するには

  1. アクションを表示するリリースレコードに移動します。

    2. 詳細は、リリースレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［アクション］メニューをクリックし、［アクション履歴の表示］を選択します。

      • または、［関連リスト］をクリックし、［アクション履歴］セクションで、［アクション履歴の表示］をクリックします。

    • ［リリース］タブ（Salesforceページレイアウト）を使用している場合は、［アクション履歴］セクションの［アクション履歴の表示］をクリックします。

      • ［リリース履歴］ウィンドウに、表示専用モードでアクションが表示されます。［リリース履歴］ウィンドウに表示されるアクションの編集や削除はできません。

  3. （オプション）アクションタイプに基づいてレコードをフィルタリングするには、リストから目的のアクションを選択します。
  4. （オプション）リリースに関連付けられたアクションを印刷するには、［印刷］をクリックします。
