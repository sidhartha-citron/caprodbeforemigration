問題を報告した同様のインシデントを検索できます。必要に応じて、そのようなインシデントを別のインシデントにリンクできます。すべてのリンクされたインシデントは、リンク元のインシデントの子インシデントとして扱われます。他のインシデントをリンクしたインシデントは親インシデントとして扱われます。

他のインシデントレコードに値を一致させるフィールドを指定できます。システム管理者がそのようなフィールドを設定します。使用できないフィールドの値を一致させる場合は、システム管理者に問い合わせてください。システム管理者は［Incident Matching（インシデントの一致）］フィールドセットにそのフィールドを追加する必要があります。

一致するインシデントを検索するには

  1. 検索するレコードと一致するインシデントレコードに移動します。

    2. 詳細は、インシデントレコードまたはサービスリクエストレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［エージェントツール］メニューから［インシデントの一致］を選択します。
    • ［インシデントコンソール］タブを使用している場合は、［エージェントツール］メニューから［インシデントの一致］を選択します。
    • ［インシデント］タブ（Salesforceページレイアウト）を使用している場合は、［インシデントの一致］をクリックします。

      • ［インシデントの一致］ボタンはデフォルトでは使用できません。インシデントレイアウトをカスタマイズして、［Incident Matching（インシデントの一致）］ボタンを追加するには、システム管理者に問い合わせてください。

  3. ［Incident Matching（インシデントの一致）］ウィンドウで、［Match By（一致）］リストをクリックし、他のレコードと一致させるデータがあるフィールドを選択し、［適用］をクリックします。

    4. 開いているインシデントレコードの指定したフィールドの値が他のインシデントレコードの値と一致しています。

  4. （オプション）検索対象のインシデントレコードに共通の英数字テキストが含まれている場合、そのテキストをテキストボックスに入力し、［検索］をクリックします。

    5. ［ルックアップ］タイプのフィールドを除くすべてのインシデントフィールドに表示されるテキストを検索できます。［Match By（一致）］リストで選択したフィールドに一致するレコードと検索するテキストが表示されます。

  5. 現在のインシデントに一致するレコードをリンクするには、次のアクションを実行します。
    1. 一致するレコードのリストから、リンクするレコードのチェックボックスをオンにします。

      2. インシデントレコードをリンク済みの場合、それらのレコードはリストに表示されません。

    2. ［Set Selections To（選択を設定）］フィールドから、リンクするインシデントに適用するステータスを選択します。

      3. たとえば、［Set Selections To（選択を設定）］フィールドの値が［完了済み］に設定される場合、インシデントにリンクする一致したレコードの［ステータス］フィールドの値が［完了］に設定されます。

      システム管理者は、ステータスの作成中やステータスレコードの編集中に［ステータス］メニューのデフォルトのリンクステータスとしてのステータス値を設定できます。システム管理者がデフォルトのリンクステータスを設定しない場合、デフォルトで［Set Selections To（選択を設定）］フィールドに［完了］ステータスがデフォルトステータスとして表示されます。

    3. ［リンク］をクリックします。

      4. ［リンクされたインシデント］の関連リストのリンクされたインシデントを表示できます。Salesforceページレイアウトのデフォルトでは、この関連リストはインシデントの詳細フォームに表示されません。インシデントのフォームのレイアウトをカスタマイズして、フォーム上の［リンクされたインシデント］の関連リストを表示するには、システム管理者に問い合わせてください。
