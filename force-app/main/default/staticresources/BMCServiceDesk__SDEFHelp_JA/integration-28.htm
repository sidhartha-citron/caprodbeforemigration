Lucy DiamondはBMC Server Automationのオペレータ、Bob HeartはBMC Remedyforceの管理者です。2人はBMC Remedyforceで、新しいサーバーのプロビジョニングをおこなうための変更リクエストを作成しようとしています。


手順1
BobはBMC Remedyforceで電子メールリスナーを作成し、定義済みのChangeRequestEmailListener Apexクラスを電子メールサービスに追加しました。


手順2
BobはBMC RemedyforceでCreateBSAChangeという名前のテンプレートを作成しました。このテンプレートには、変更リクエストフォームの［優先度］、［イニシエータのID］、［変更カテゴリ］、［説明］、［ステータス］フィールドが含まれていて、電子メールメッセージから新しい変更リクエストを作成できます。BobはBMC Remedyforceで変更リクエストを作成するためのBMC Server Automationの設定 の手順4～7に従って、複製されたワークフロー内でCreateBSAChangeテンプレートを指定しました。


手順3
LucyはBMC Server Automationコンソールでのジョブの設定の手順を実行して、BMC Server Automationコンソールでジョブを作成しました。


手順4
電子メールメッセージが生成され、BMC Remedyforceに送信され、BMC Remedyforceでは受信した電子メールメッセージから変更リクエストレコードが作成されました。
