LDAP（Lightweight Directory Access Protocol）サーバーからユーザーをインポートして、一時テーブルである［クライアントユーザーインポート］テーブルに保存できます。Salesforce組織では、このテーブルのデータを使用してユーザーと連絡先が作成されます。

LDAPサーバーとSalesforce組織間のマッピングは、Pentaho Data Integrationツールによって提供されます。サンプルファイル内のマッピングは変更可能です。［クライアントユーザーインポート］リストの全フィールドは、LDAPサーバー属性の対応するフィールドにマッピングできます。

サンプルファイルのマッピングを変更する場合は、次のフィールドが必須です。

    • BMCServiceDesk__Last_Name__c
    • BMCServiceDesk__Client_User_Name__c
    • BMCServiceDesk__EmailID__c
