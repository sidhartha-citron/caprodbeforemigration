正確性と完全性を期するため、BMCではテンプレートを使用してレコードを作成することをお勧めします。テンプレートを使用すると、一貫した方法で情報が取得されます。管理者は、よく使用するリリースのテンプレートを定義できます。

リリースを作成する前に、リリースにリンクする必要がある変更リクエストが作成されていることを確認してください。

テンプレートを適用してリリースを作成するには

  1. 次のいずれかのタブに移動し、［新規］をクリックします。
    • ［Remedyforceコンソール］>［ビュー］>［リリース］
    • ［リリース］（Salesforceページレイアウト）
  2. ［テンプレート］フィールドでをクリックします。
  3. ［テンプレート］ウィンドウで、目的のテンプレートを選択します。
  4. ［保存］をクリックします。

    5. テンプレートで利用できる値がレコードに表示されます。

  5. （オプション）リリースに関する詳細情報を入力し、［保存］をクリックします。

    6. 詳細は、リリースのフィールドを参照してください。

    リリースを保存すると、リリースの詳細ページが表示されます。次の追加フィールドが表示されます。

    • ［ステータスの詳細］セクション： リリース進捗状況バーが表示されます。
    • ［最終変更者］フィールドには、リリースを最後に更新したユーザーの名前が表示されます。

レコードが作成されると、レコードが作成された日時とユーザーの詳細を追跡する［リリース履歴］セクションに次のアクションが記録されます。
