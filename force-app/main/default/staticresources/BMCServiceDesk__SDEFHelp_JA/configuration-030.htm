ログオン資格情報が記載された電子メールメッセージをSalesforce.comから受信したら、Salesforce.comの組織にインストールされている管理パッケージのユーザーアカウントにライセンスを割り当てる必要があります。

ユーザーアカウントにライセンスを割り当てるには

  1. ［設定］>［インストール済みのパッケージ］の順に選択します。
  2. ［パッケージ名］カラムで、［BMC Remedyforce］をクリックします。
  3. ［ユーザーの追加］をクリックします。
  4. 受信した電子メールメッセージに記載されているユーザーの資格情報を選択します。
  5. ［追加］をクリックします。
  6. BMC Remedyforceにアクセスするには、右上のセクションにあるForce.comの［アプリケーションメニュー］から［BMC Remedyforce］を選択します。
