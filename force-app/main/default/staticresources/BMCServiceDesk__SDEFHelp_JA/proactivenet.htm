また、BMC RemedyforceをBMC ProactiveNet Performance Managementと統合することで、BMC ProactiveNet Performance Managementで生成されたイベントを使用して、インシデントをBMC Remedyforceで作成できるようになりました。この統合は、BMC ProactiveNet Performance Managementで生成されるイベントに基づいてインシデントを作成、更新、終了する場合に役立ちます。イベントのデータはインシデントオブジェクトのフィールドにマップされます。

BMC ProactiveNet Performance Managementからのイベントを使用した電子メールの送信を有効にするには、BMC Remedyforceで電子メールサービスを設定する必要があります。

Salesforce組織の電子メール制限についての詳細は、 http://www.salesforce.com/us/developer/docs/apexcode/Content/apex_gov_limits.htmを参照してください。

BMC RemedyforceとBMC ProactiveNet Performance Managementを統合するには、次の手順に従います。

    • BMC ProactiveNet Performance Managementでのイベント規則のカスタマイズ
    • 電子メールサービスの作成
    • アプリケーションの設定

ケースシナリオの使用については、ケースシナリオの使用を参照してください。
