BMC FootPrints Asset Coreデバイスのインシデントまたはサービスリクエストを作成する場合は、このデバイスをインシデントまたはサービスリクエストにリンクする必要があります。［CIのリンク］をクリックすると、［構成アイテムから選択］ウィンドウに［Remedyforce CMDB］および［アセットコア］タブが表示されます。

注意: ［インシデント］フォームに［CIのリンク］ボタンが表示されない場合は、システム管理者に連絡してフォームレイアウトをカスタマイズします。

［Remedyforce CMDB］および［アセットコア］タブは、システム管理者がBMC RemedyforceとBMC FootPrints Asset Coreとの統合を有効にした場合にだけ表示されます。統合が無効な場合は、［構成アイテムから選択］ウィンドウにタブは表示されません。

［Remedyforce CMDB］タブには、Remedyforce CMDBデバイスおよびBMC FootPrints Asset Coreデバイスの構成アイテムのうち、Remedyforce CMDBにとって重要なものが表示されます。必要なデバイスがこのタブに表示されていない場合は、［アセットコア］タブの［検索］フィールドでデバイスを検索できます。

［アセットコア］タブには、現在のインシデントまたはサービスリクエストに対応するクライアントのデバイスが表示されます。ただし、この操作が可能なのは、［ユーザー］オブジェクトの［ソースの固有ユーザID］フィールドに、リンクされたクライアントドメイン名およびユーザー名が入力されている場合のみです。LDAPサーバーからSalesforce組織にユーザーがインポートされた場合は、［ソースの固有ユーザID］フィールドに入力されています。詳細は、システム管理者に問い合わせてください。

［ソースの固有ユーザID］フィールドの値は次の形式で格納できます。

    • ドメイン\ユーザー名
    • ユーザー名@ドメイン

［検索］フィールドに検索条件を入力し、［検索］をクリックすると、BMC RemedyforceによってBMC FootPrints Asset Coreデータベースが照会され、BMC FootPrints Asset Coreデータベース内の一致するデバイスが表示されます。

デバイスの検索に関する次の重要な点に注意してください。

    • 検索では、大文字と小文字が区別されません。
    • 「*」（任意の複数文字列）や「?」 （任意の1文字）などのワイルドカード文字を使用できます。
    • 検索を絞り込むには、AND、OR、NOTなどの演算子を使用します。複数の演算子を使用する場合は、かっこを使用します。
    • <フィールド名>::<値>表記を使用して、特定のフィールドにある値を検索できます。次のフィールドを検索できます。name、os、fpac、ip、domain、netbios、macaddress、serialnumber、type、currentuser、user、security、vendor、po、support、dept、sku、status、contact、owner

      • 例： domain::bmc.com

デバイスの検索やインポート、インシデントやサービスリクエストへのリンクをおこなうには

  1. 目的のインシデントやサービスリクエストに移動します。

    2. 詳細は、インシデントレコードまたはサービスリクエストレコードへの移動を参照してください。

  2. ［CIのリンク］をクリックします。

    3. BMC Remedyforceから初めてBMC FootPrints Asset Coreに接続する場合は、［BMC FootPrints Asset Core認証］ダイアログボックスが表示されます。BMC FootPrints Asset Core資格情報をこのダイアログボックスに入力します。BMC FootPrints Asset Coreユーザー名が保存され、以降は接続のたびに資格情報を入力する必要はありません。

  3. 必要なデバイスが［Remedyforce CMDB］タブに表示されている場合は、このデバイスをダブルクリックしてインシデントまたはサービスリクエストにデバイスをリンクします。

    4. ［Remedyforce CMDB］タブに表示されているレコードをフィルタするには、［ビュー］メニューの次のコマンドを使用します。

    • ［クライアントにリンク］： インシデントまたはサービスリクエストのクライアントに属するすべての構成アイテムを表示します。
    • ［アカウントにリンク］： インシデントまたはサービスリクエストのアカウントに属するすべての構成アイテムを表示します。
    • ［すべて］： すべての構成アイテムを表示します。
  4. 必要なデバイスが［Remedyforce CMDB］タブに表示されていない場合は、次のアクションを実行します。
    1. ［アセットコア］タブをクリックします。
    2. BMC FootPrints Asset Coreデータベース内のデバイスを検索するには、［検索］フィールドに検索条件を入力し、［検索］をクリックします。
    3. 表示された検索結果内の必要なデバイスをダブルクリックして、このデバイスをインシデントまたはサービスリクエストにリンクします。

選択したデバイスがインシデントまたはサービスリクエストの［サービスと構成アイテムの詳細］セクションに表示されます。デバイスはコンピュータシステムクラスにインポートされます。
