ここでは、変更依頼を作成するためのBMC Server Automationの設定手順を示します。

  1. BMC Remedyforceで電子メールリスナーを作成します。電子メールリスナーの作成についての詳細は、電子メールサービスの作成を参照してください。
  2. 定義済みのChangeRequestEmailListener Apex クラスを電子メールサービスに追加します。

    3. 注意: テンプレートを使用しないで変更リクエストを作成する場合は、手順7を省略します。

  3. （オプション）新しい変更リクエストが電子メールから作成されている場合に、よく使用されるデータを変更リクエストフォームに入力するには、BMC Remedyforceでテンプレートを作成します。
  4. （オプション）新規作成されたテンプレートで、定義済みの［BSA通知］カテゴリを選択します。
  5. （オプション）［Remedyforce管理］>［アプリケーションの設定］>［アプリケーションの全般設定］ページで、［フォームのフィールド値をテンプレートフィールド値に置換］オプションを選択してテンプレートを使用します。
  6. （オプション）Visualforceページレイアウトユーザーインターフェイスを使用している場合は、［設定］>［アプリケーションの設定］>［一般］タブの順に選択し、［フォームフィールド値をテンプレートフィールド値と置換］チェックボックスをオンにします。
  7. （オプション）BMC Remedyforceのカスタム設定で、新規作成したテンプレートの名前を次のように入力します。
    1. ［Remedyforce管理］タブをクリックします。
    2. ［ホーム］ページで［アプリケーション設定］タイルをクリックし、メニューから［カスタム設定］を選択します。
    3. （オプション）Visualforceページレイアウトユーザーインターフェイスを使用している場合は、［設定］>［開発］>［カスタム設定］の順に選択します。
    4. ［アクション］カラムで、［Remedyforceの設定］の隣にある［管理］をクリックします。
    5. ［アクション］カラムで、［変更リクエストの電子メールリスナーテンプレート］の隣にある［編集］をクリックします。
    6. ［値］フィールドに、新規作成したテンプレートの名前を入力します。
    7. ［保存］をクリックします。
  8. BMC Server Automationコンソールでジョブを作成します。詳細は、BMC Server Automationコンソールでのジョブの設定を参照してください。
  9. BMC Server Automationコンソールでジョブを送信します。

設定の結果

BMC Server Automationで電子メールメッセージが生成され、BMC Remedyforceに送信されます。受信した電子メールメッセージに基づいて、BMC Remedyforceで変更リクエストが作成されます。電子メールの内容の本文が変更リクエストの［説明］フィールドにコピーされます。

注意: 受信した電子メールメッセージの電子メールアドレスがどのクライアントの電子メールアドレスとも一致しない場合は、［識別されていない送信者のクライアント］フィールドを選択する必要があります。［識別されていない送信者のクライアント］フィールドは、［Remedyforce管理］>［電子メールの設定］> ［インシデント - 電子メールの設定］ページの［電子メール送信者の識別］セクションにあります。Visualforceページレイアウトユーザーインターフェイスを使用している場合は、［設定］>［アプリケーションの設定］>［インシデント - 電子メールの設定］タブの順に選択します。ただし、監視システムのクライアントを作成する場合は、適切な電子メールアドレスを指定することをお勧めします。
