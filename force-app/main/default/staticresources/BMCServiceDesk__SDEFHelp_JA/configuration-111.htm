Salesforceでは、使用する言語を変更できます。

言語を変更するには

  1. BMC Remedyforceホームページの右上のセクションにある［設定］リンクをクリックします。
  2. 確認ダイアログボックスで［Yes］をクリックします。
  3. ［私の個人情報］セクションで、［個人情報、言語、タイムゾーン、売上目標、営業チームを編集する］をクリックします。
  4. ［編集］をクリックします。
  5. ［ロケール設定］セクションで、適切な言語、ロケール、タイムゾーンを選択します。
  6. ［保存］をクリックします。
