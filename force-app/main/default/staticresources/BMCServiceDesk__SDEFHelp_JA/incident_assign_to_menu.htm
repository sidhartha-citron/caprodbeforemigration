デフォルトでは、インシデントまたはサービスリクエストは、作成したユーザーに割り当てられます。レコードの保存後に、そのレコードを他のユーザーまたはキューに割り当てることができます。レコードを提案されたオーナーに割り当てるには、インシデントレコードまたはサービスリクエストレコードの提案されたオーナーへの割り当てを参照してください。

［Remedyforceコンソール］タブまたは［インシデントコンソール］タブでのインシデントオーナーの変更

［割り当ての詳細］セクションで、システム管理者の設定に基づいて、［オーナー］フィールドまたは［キュー］フィールド、［スタッフ］フィールドが表示されます。

［Remedyforceコンソール］タブまたは［インシデントコンソール］タブでインシデントオーナーを変更するには

  1. 他のユーザーまたはキューに割り当てるレコードに移動します。

    2. 詳細は、インシデントレコードまたはサービスリクエストレコードへの移動を参照してください。

    リストビューでレコードを選択し、［オーナーを変更］をクリックすることもできます。

  2. ［オーナー］フィールドが［割り当ての詳細］セクションに表示されている場合は、次のアクションを実行します。
    1. ［オーナー］ルックアップをクリックします。

      2. スタッフメンバーが組織に存在する場合、デフォルトではすべてのスタッフメンバーが［提案されたオーナー］ウィンドウに表示されます。スタッフメンバーが存在しない場合は、キューが表示されます。

    2. レコードをスタッフメンバーに割り当てるには、目的のスタッフメンバーのレコードをクリックします。
    3. レコードをキューに割り当てるには、［キュー］オプションを選択して目的のキュー名をクリックします。
    4. キューに属しているスタッフメンバーにレコードを割り当てるには、最初に目的のキューの行をクリックし、キューに属するスタッフメンバーのリストから目的のスタッフメンバーを選択します。
  3. ［スタッフ］および［キュー］フィールドが［割り当ての詳細］セクションに表示されている場合は、次のアクションを実行します。

    2. レコードをキューに割り当てるには、［キュー］フィールドから目的のキューを選択します。
    3. キューに属しているスタッフメンバーにレコードを割り当てるには、最初に目的のキューの行をクリックし、キューに属するスタッフメンバーのリストから目的のスタッフメンバーを選択します。

      4. ［キュー］フィールドでキューを選択すると、キューに属するスタッフメンバーが［スタッフ］フィールドルックアップに表示されます。

    10. ［オーナー］フィールドには、選択したスタッフメンバーの名前が表示されます。

複数のインシデントレコードまたはサービスリクエストレコードのオーナーの変更

複数のインシデントやサービスリクエストをスタッフメンバーやキューに割り当てることができます。

複数のインシデントレコードまたはサービスリクエストレコードのオーナーを変更するには

    1. インシデントリストから、他のユーザーまたはキューに割り当てるインシデントまたはサービスリクエストのチェックボックスをオンにします。
    2. ［Change Owner（オーナーを変更）］をクリックします。
    3. 選択したレコードをスタッフメンバーに割り当てるには、目的のスタッフメンバーのレコードをクリックします。
    4. 選択したレコードをキューに割り当てるには、［キュー］オプションを選択して目的のキューをクリックします。
    5. キューに属しているスタッフメンバーに選択したレコードを割り当てるには、最初にキューレコードをクリックし、キューに属するスタッフメンバーのリストから目的のスタッフメンバーを選択します。
  2. ［インシデント］タブ（Salesforceページレイアウト）を使用している場合は、次のアクションを実行します。
    1. ［表示］メニューから［すべて］を選択します。
    2. 割り当てを変更するインシデントやサービスリクエストを選択します。
    3. ［Change Owner（オーナーを変更）］をクリックします。
    4. ［オーナー］ドロップダウンリストで、目的のオプション（［ユーザー］または［キュー］）を選択します。

インシデントレコードまたはサービスリクエストレコードの提案されたオーナーへの割り当て

システム管理者は、目的の条件に基づいて提案されたオーナーを設定します。レコードをユーザー、キュー、または選択したキューのメンバーに割り当てることができます。

Salesforceページレイアウトでは、［提案されたオーナー］ボタンはデフォルトでは表示されません。インシデントの詳細フォームのレイアウトを編集してボタンを表示するには、システム管理者に問い合わせてください。

インシデントレコードまたはサービスリクエストレコードを提案されたオーナーに割り当てるには

  1. 提案されたオーナーに割り当てるレコードに移動します。

    2. 詳細は、インシデントレコードまたはサービスリクエストレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［エージェントツール］メニューから［提案されたオーナー］を選択します。
    • ［インシデントコンソール］タブを使用している場合は、［エージェントツール］メニューから［提案されたオーナー］を選択します。
    • ［インシデント］タブ（Salesforceページレイアウト）を使用している場合は、［提案されたオーナー］をクリックします。

      • デフォルトでは、提案されたスタッフメンバーが表示されます。提案されたスタッフメンバーが存在しない場合は、提案されたキューが表示されます。

  3. ［提案されたオーナー］ウィンドウで目的の提案されたスタッフメンバーを選択します。
  4. キューにレコードを割り当てるには、［キュー］オプションを選択して、次に目的のキューを選択します。

    5. キューをクリックすると、レコードがキューに割り当てられます。

  5. レコードをキューのメンバーに割り当てるには、目的のキューの行を選択します。

    6. キューのメンバーが表示されます。

  6. 選択したキューのメンバーから目的のスタッフメンバーを選択します。
