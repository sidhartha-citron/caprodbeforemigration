ここでは、BMC End User Experience Managementから受信した電子メールメッセージに基づいてインシデントを作成するようにBMC Remedyforceを設定する手順、およびBMC End User Experience Managementでアラートを設定する手順を示します。

BMC End User Experience Managementから受信した電子メールメッセージに基づいてインシデントを作成するようにBMC Remedyforceを設定するには

  1. BMC Remedyforceで電子メールリスナーを作成します。電子メールリスナーの作成についての詳細は、電子メールを使用したインシデントの作成およびインシデントのステータスの取得を参照してください。
  2. 定義済みのEmailConversationListener Apex クラスを電子メールサービスに追加します。

    3. 注意: テンプレートを使用しないでインシデントを作成する場合は、手順8を省略します。

  3. （オプション）新しいインシデントが電子メールから作成されている場合に、よく使用されるデータを［インシデント］フォームに入力するには、BMC Remedyforceでテンプレートを作成します。
  4. （オプション）新規作成されたテンプレートで、定義済みの［EUEMアラート］カテゴリを選択します。
  5. （オプション）［Remedyforce管理］>［アプリケーションの設定］>［アプリケーションの全般設定］ページで、［フォームのフィールド値をテンプレートフィールド値に置換］オプションを選択してテンプレートを使用します。
  6. （オプション）Visualforceページレイアウトユーザーインターフェイスを使用している場合は、［設定］>［アプリケーションの設定］>［一般］タブの順に選択し、［フォームフィールド値をテンプレートフィールド値と置換］チェックボックスをオンにします。
  7. （オプション）［Remedyforce管理］>［ワークフローと承認の設定］>［ワークフロー規則］の順に移動して、［エンドユーザー経験の管理で作成されたインシデントにテンプレートを適用］ワークフロー規則を複製して有効にします。

    8. デフォルトでは、ワークフローは非アクティブです。

  8. （オプション）Visualforceページレイアウトユーザーインターフェイスを使用している場合は、［設定］>［ワークフローと承認］>［ワークフロー規則］の順に移動して、［エンドユーザー経験の管理で作成されたインシデントにテンプレートを適用］ワークフロー規則を複製して有効にします。
  9. （オプション）複製されたワークフローを編集するには、次のアクションを実行します。
    1. ［ワークフローアクション］セクションの［ワークフローのアクションの追加］ドロップダウンリストで、［新しいフィールドの更新］アクションタイプを選択します。
    2. ［更新するフィールド］ドロップダウンリストで、［テンプレート名］を選択します。
    3. ［新しいフィールド値の指定］セクションで、［式を使用して新しい値を設定する］を選択します。
    4. 新規作成したテンプレートの名前を、二重引用符で囲んで入力します。たとえば、テンプレート名がCreateEUEMIncidentの場合は、「“CreateEUEMIncident“」と入力します。
  10. 監視対象の属性に違反した場合にアラートを設定するように、Automated Incident Management（AIM）を設定します。詳細は、BMC End User Experience Managementコンソールでアラートを設定するにはを参照してください。

BMC End User Experience Managementコンソールでアラートを設定するには

  1. BMC End User Experience Management - Real User Monitorで［管理］>［ウォッチポイント管理］>［使用可能なウォッチポイント］の順に移動して、ウォッチポイントを構成します。アラート設定の詳細は、BMC End User Experience Managementのマニュアルを参照してください。
  2. ［管理］>［しきい値および問題検出］>［インシデント検出］の順に移動して、インシデント検出規則を編集します。
  3. ［通知］タブをクリックします。
  4. （オプション）［通知グループに電子メールを送信する］を選択して、必要なグループを選択します。
  5. 送信するデータに応じて、［重要な場合通知する］セクションで［電子メール送信先］または［通知グループに電子メールを送信する］フィールドを選択します。
  6. ［電子メール送信先］フィールドに、BMC Remedyforceで設定された新規作成された電子メールリスナーの電子メールアドレスを入力します。

設定の結果

BMC End User Experience Managementで電子メールメッセージが生成され、BMC Remedyforceに送信されます。受信した電子メールメッセージに基づいて、BMC Remedyforceでインシデントが作成されます。電子メールの内容の本文がインシデントの［説明］フィールドにコピーされます。

注意: 受信した電子メールメッセージの電子メールアドレスがどのクライアントの電子メールアドレスとも一致しない場合は、［識別されていない送信者のクライアント］フィールドを選択する必要があります。［識別されていない送信者のクライアント］フィールドは、［Remedyforce管理］>［電子メールの設定］> ［インシデント - 電子メールの設定］ページの［電子メール送信者の識別］セクションにあります。（オプション）Visualforceページレイアウトユーザーインターフェイスを使用している場合は、［設定］>［アプリケーションの設定］>［インシデント - 電子メールの設定］タブの順に選択します。ただし、監視システムのクライアントを作成する場合は、適切な電子メールアドレスを指定することをお勧めします。
