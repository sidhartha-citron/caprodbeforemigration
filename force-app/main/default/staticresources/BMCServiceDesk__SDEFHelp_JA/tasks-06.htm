システムにテンプレートが定義されていない場合、この手順を使用してタスクを作成します。

テンプレートなしでタスクを作成するには

  1. 次のいずれかのタブに移動し、［新規］をクリックします。
    • ［Remedyforceコンソール］>［表示］>［タスク］
    • タスクコンソール
    • ［タスク］（Salesforceページレイアウト）
  2. ［クライアントID］フィールドで、クライアントを選択します。
  3. タスクに関するその他の情報を入力します。

    4. 詳細は、タスクフィールドを参照してください。

  4. ［保存］をクリックします。

    5. タスクを保存すると、タスクの詳細ページが表示されます。次の追加フィールドが表示されます。

    • ［ステータスの詳細］セクション： タスク進捗状況バーが表示されます。
    • ［クライアントの詳細］セクションの［名前］フィールドと［名字］フィールド： タスクに選択したクライアントの名前と名字が表示されます。これらのフィールドの値は、選択するクライアントによって異なります。これらのフィールドでは値を選択できません。レコードのクライアントを編集すると、それに合わせてこれらのフィールドの値も変わります。
    • ［クライアントの詳細］セクションの［クライアントアカウント］フィールド： 選択したクライアントのアカウントが表示されます。

      • システム管理者が「タスクが作成された場合にクライアントに通知」ワークフローをアクティブにすると、タスクの作成時に、電子メールメッセージがタスクのクライアントへ送信されます。タスクのクライアントとしてリードを選択した場合、電子メールメッセージは送信されません。

レコードが作成されると、レコードが作成された日時とユーザーの詳細を追跡する［タスク履歴］セクションに次のアクションが記録されます。

タスクフィールド

フォームに一般的なアイテムが表示されます。詳細は、Salesforceページレイアウトのフォーム上の共通のアイテムを参照してください。プロファイルに割り当てられたレイアウトまたは使用するUI に基づき、表示される［新規タスク］フォーム上で次の表で説明する一部のフィールドが利用できないことがあります。

システム管理者は、［タスク］フォームのルックアップフィールドにフィルタを適用できます。ここで、ルックアップフィルタを理解するために役立ついくつかの重要なポイントを説明します。

    • ルックアップフィールドにフィルタが適用されている場合、フィルタ条件と一致しているレコードのみがポップアップウィンドウに表示されます。設定されたフィルタがオプションの場合、［すべての結果を表示］オプションを使用してフィルタを削除できます。
    • ルックアップタイプフィールドの先行入力は、フィルタ結果の一部である結果に対してのみ機能します。たとえば、［カテゴリ］ルックアップフィールドで、結果として［ハードウェア］および［アプリケーション］が表示されるようにフィルタが適用される場合、先行入力はこれらの2つのカテゴリのみに対して動作します。また、ルックアップアイコンをクリックして、検索フィールドに入力することによって他のカテゴリを検索する場合、［すべての結果を表示］オプションを使用してフィルタを削除しない限り、検索で何も結果が返されません。

完了者

無効
現在のレコードをインアクティブとして表示するには、このチェックボックスをオンにします。
レコードはすでに無効になっているものの、レコード保持ポリシーを順守するためにレコードを保持する必要があるという理由で、レコードをインアクティブにしなければならない場合があります。たとえば、問題や停止が、すでに存在しないルーターやネットワークから削除されたルーターに関連している場合などです。

メモ:

ステータスと優先度の詳細
インパクト
タスクに割り当てるインパクトを選択します。
インパクトレベルと緊急度は、タスクの優先度の計算に使用されます。
緊急度
タスクに割り当てる緊急度を選択します。
緊急度とインパクトレベルは、タスクの優先度の計算に使用されます。
優先度
緊急度とインパクトを選択してレコードを保存すると、［優先度］フィールドに値が入力されます。優先度は、選択した緊急度とインパクトの値によって計算されます。
ステータス
タスクに割り当てるステータスを選択します。
このフィールドを使用して、タスクの進捗状況をステージを通して追跡できます。
日時の詳細
開始日
このフィールドには、タスクが作成された日時が表示されます。
期限日
緊急度とインパクトを選択してレコードを保存すると、［期限日］フィールドに値が入力されます。この期限日は優先度の期間に基づいて計算されます。
必要に応じて、計算された期限日とは異なる日時を選択できます。
詳細は、期限日の計算オプションを参照してください。
終了日
このフィールドには、［終了］または［クイック終了］をクリックした後、システム日付が表示されます。［終了］ボタンおよび［クイック終了］ボタンは、タスクの保存後に、タスクの詳細ページに表示されます。
割り当ての詳細
オーナー
このフィールドには、タスクを作成したログオンユーザーの名前が表示されます。タスクの保存後に、タスクのオーナーを変更できます。タスク割り当ての変更についての詳細は、タスクオーナーの変更を参照してください。
