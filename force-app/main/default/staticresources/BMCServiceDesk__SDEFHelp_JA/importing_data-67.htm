失敗行ファイルは、KJBファイルを保存したフォルダに保存されているテキストファイルです。必要な手順を実行して、失敗行ファイルの失敗行に記載されているエラーを削除します。その後、ここで説明する手順を使用してデータをインポートします。

サンプルジョブファイルが、BMC Communities BMC Remedyforceページに用意されています。このサンプルジョブファイルは、失敗行ファイルからSalesforce組織にデータをインポートするための、変換ファイルを作成する際にベースとして使用できます。

次の手順では、BMCから提供されたKTRファイルを使用することを前提としています。

失敗行ファイルからデータをインポートするには

  1. ［<CIタイプ名>の失敗ファイル入力］をダブルクリックします。
  2. ［選択したファイル］フィールドで、失敗ファイル名の場所を入力し、［OK］をクリックします。
  3. 適切なKTRファイルから［クラス<CIタイプ名>にアップサート］手順をコピーし、失敗行ファイルからのデータインポート用に作成した変換ファイルに貼り付けます。

    4. たとえは、オペレーティングシステムCIタイプのデータをインポートする場合は、オペレーティングシステムのKTRファイルから［クラスオペレーティングシステムCIタイプにアップサート］手順をコピーします。

  4. ［実行］をクリックします。
