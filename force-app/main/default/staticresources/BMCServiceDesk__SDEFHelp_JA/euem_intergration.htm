BMC End User Experience Managementによってアプリケーション運用が一段と強化されるため、会社の評判や収益が低下する前に、エンドユーザーのパフォーマンス問題を検出し、特定することができます。BMC End User Experience Managementを使用すると、実際のユーザーをリアルタイムで監視できるため、一貫したアプリケーションパフォーマンスを実現できます。

BMC End User Experience Managementを使用すると、次のことが可能になります。

    • "標準"パフォーマンスの傾向と基準を自動的に判別する
    • 一般的な速度低下と断続的な速度低下を区別して、詳細を調べる
    • エンドユーザーパフォーマンスのリアルタイム表示機能を強化することによってサービスレベルを高めて、コストのかかるアプリケーション速度低下と故障を回避する
    • サービスレベルと影響を受けるユーザーに基づいて、問題の迅速な検出、適切な優先度設定、問題の根本原因のすばやい特定をおこなうことによって、アプリケーションの信頼性とパフォーマンスを高める
    • 新しいアプリケーション、リリース、テクノロジ変更がエンドユーザーエクスペリエンスに与える影響を理解することで、これらの変更に関するビジネスニーズに対応できるようにする
    • 手動プロセスを自動化し、問題の特定に要する時間を短縮することによって、コスト削減とサービスの迅速な復元を実現する

BMC RemedyforceとBMC End User Experience Managementを統合すると、BMC End User Experience Managementアプリケーションから受信したアラートに基づいてインシデントを作成できるようになります。また、BMC Remedyforceでアラートを管理し、適切なスタッフメンバーに割り当てて、すべてのアクティビティの解決とレポート作成を実行できるようになるため、発生する可能性のあるパフォーマンス問題に事前に対処できます。
