システム管理者の場合、［概要］タブがデフォルトタブです。ただし、アプリケーションに慣れたら、希望のタブをデフォルトタブに設定できます。

デフォルト表示タブを設定するには

  1. ［設定］>［作成］>［アプリケーション］の順に選択します。
  2. BMC Remedyforceの［アクション］カラムで［編集］をクリックします。
  3. ［デフォルト表示タブ］リストから、設定するタブを選択します。
  4. ［保存］をクリックします。
