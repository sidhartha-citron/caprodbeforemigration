サイドバーにある次のポートレットは、表示または非表示に設定できます。

    • アクション項目： ログオンしているスタッフメンバーが、割り当てられているインシデント、タスク、ブロードキャスト、変更リクエスト、問題の数を表示できます。これにより、完了する必要がある作業量を見積もって、その日の作業スケジュールを計画することができます。また、目的のアイテムをクリックして割り当てられているアイテムに直接アクセスし、そのアイテムのリストビューを開くことができます。たとえば、ログオンしているスタッフメンバーがポートレットでインシデントをクリックすると、そのインシデントのリストビューが開き、そのスタッフメンバーに割り当てられているインシデントだけが表示されます。

      • ログオンユーザーがサービスリクエストの所有者の場合、サービスリクエストを登録すると、割り当てられたインシデントの数が1つ増加します。割り当てられた変更リクエストとタスクの数は、サービスリクエストを実行するために変更リクエストとタスクが作成された場合でも変化しません。

      テンプレートを使用してインシデントまたは変更リクエストを作成すると、割り当てられたインシデントまたは変更リクエストの数が1つ増加します。ただし、テンプレートにリンク済みのタスクが設定されている場合は、タスク数は増加しません。

    • ブロードキャスト： ログオンしているスタッフメンバーが、全ユーザーまたは自分のプロファイルに対して表示されている既存のブロードキャストを表示できます。ブロードキャストを選択すると、ポートレットにブロードキャストメッセージを表示することができます。スタッフメンバーはブロードキャストをフォローして、フィードの更新を受信することができます。
    • 検索： ログオンしているスタッフメンバーが、［グローバル検索］を開かずにグローバル検索を実行できます。
    • 世界時間： ログオンしているスタッフメンバーが、世界3か所の時刻を同時に表示できます。このポートレットの時計を使用すると、異なるタイムゾーンの対応する時刻に、タスクを割り当てることができます。また、多くのクライアントが地理的に異なる場所に所在している場合は、異なるタイムゾーンを表示できるため、そのクライアントに連絡を取る時間として適切かどうかを判断することができます。
    • 上位のリクエスト： ログオンしているスタッフメンバーが、1クリックで、［インシデント］テンプレートからインシデントを作成できます。このポートレットで［インシデント］テンプレートをクリックすると、［インシデント］テンプレートの値が各フィールドに入力された状態で、新しい［インシデント］フォームが開きます。テンプレートが自分のプロファイルまたは全ユーザーと共有されている場合は、そのテンプレートが［上位のリクエスト］ポートレットに表示されます。このポートレットには、最大で5つのテンプレートが表示されます。ポートレットに表示される［インシデント］テンプレートの順序は、［テンプレート］フォームでテンプレートを開き、このフォームで表示順序を定義することによって設定できます。詳細は、テンプレートの作成を参照してください。

      • 注意: テンプレートの表示順序に加えた変更は、ブラウザをリフレッシュした場合、またはサイドバーでポートレットを上下にドラッグしてポートレットの表示順序を変更した場合に、［上位のリクエスト］ポートレットに反映されます。

    • Chatterグループ： ログオンしているスタッフメンバーが、自分がメンバーとなっているChatterグループを表示できます。このポートレットのChatterグループをクリックすると、そのChatterグループの詳細ページが開きます。グループの詳細ページには、そのグループのメンバーで共有されているすべての投稿、コメント、リンクが表示されます。［Chatterグループ］ポートレットは、Salesforce.com組織でChatterグループを作成していない場合は、サイドバーに表示されません。Chatterグループについての詳細は、 https://na11.salesforce.com/help/doc/user_ed.jsp?section=help&target=collab_group_overview.htm&loc=help&hash=topic-titleを参照してください。Chatterグループの使用についての詳細は、Chatterフィードの使用を参照してください。

      • 注意: サイドバーのポートレットの表示順序は、サイドバーでポートレットを上下にドラッグすると変更できます。また、［次のポートレット］または［前のポートレット］をクリックすると、サイドバーのその他のポートレットを表示することができます。

サイドバーのポートレットを設定するには

  1. BMC Remedyforceホームページの右上セクションにある［追加］をクリックします。
  2. ［サイドバーの内容］ウィンドウで、サイドバーに表示するポートレットを選択します。
  3. ［OK］をクリックします。
  4. ［世界時間］ポートレットで、［設定］アイコンをクリックして、このポートレットを設定します。
  5. 各時計について、次のアクションを実行します。
    1. ［タイムゾーン］ドロップダウンリストから、時計に表示する時刻のタイムゾーンを選択します。

      2. ［タイムゾーン］ドロップダウンリストには、標準時間と夏時間のゾーンおよびコードが表示されます。夏時間が実施されている期間は、正しい時刻が反映されるように、［タイムゾーン］ドロップダウンリストから適切なタイムゾーンを選択する必要があります。

    2. （オプション）ポートレットに表示する時刻の都市名を入力します。
    3. ［OK］をクリックします。

      4. 定義した設定に基づいて、時計に時刻が表示されます。
