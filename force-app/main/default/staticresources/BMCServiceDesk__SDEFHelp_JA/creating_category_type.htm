カテゴリタイプを使用することで、関連する複数のカテゴリどうしをグループ化することができます。

カテゴリタイプを作成するには

  1. BMC Remedyforceホームページの［設定］をクリックします。
  2. ［設定］オプションのリストから［設定］を展開します。
  3. ［設定］ドロップダウンリストで、［カテゴリタイプ］をクリックします。

    4. 注意: BMC Remedyforceのタブ形式ビューでは、［Remedyforce管理］タブに［カテゴリタイプ］があります。

  4. ［カテゴリタイプ］リストビューで、［新規］をクリックします。
  5. ［カテゴリタイプ］フィールドに、現在のカテゴリタイプに割り当てる固有の識別子を入力します。
  6. ［補足情報］セクションで、［カテゴリ］タブをクリックします。
    1. ［カテゴリ］タブで、［アクション］>［追加］の順にクリックします。
    2. ［カテゴリ］ウィンドウで、このカテゴリタイプに追加するカテゴリを作成します。詳細は、カテゴリレコードの作成を参照してください。
    3. ［保存］をクリックします。

      4. ［カテゴリ］ウィンドウが閉じ、作成したカテゴリが［カテゴリ］タブに表示されます。

  7. ［保存］をクリックします。

    8. 注意: 新規フォームを開いてレコードを作成した後、レコードを保存した場合は、フォームの右上セクションにある［次のレコード］および［前のレコード］ボタンを使用して、次または前のレコードに移動することはできません。これらのボタンは新規フォームで使用できません。

  8. （オプション）現在のレコードをインアクティブとして指定するには、［インアクティブ］チェックボックスをオンにします。

    9. たとえば、セキュリティ上の理由からネットワークにおけるUSBドライブの使用を禁止したアカウントでは、今後USBドライブのカテゴリタイプが適用されることはありません。

    注意: インアクティブとしてマークされたレコードは、データベースで実行される検索の対象に含まれません。
