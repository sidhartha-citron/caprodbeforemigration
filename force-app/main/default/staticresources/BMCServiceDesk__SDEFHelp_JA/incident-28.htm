インシデントを送信して承認を求めるように承認プロセスが設定されている場合は、承認のためにインシデントを送信する必要があります。

承認のためにインシデントを送信するには

  1. 承認を送信するインシデントレコードに移動します。

    2. 詳細は、インシデントレコードまたはサービスリクエストレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［関連リスト］ボタンをクリックし、［承認履歴］セクションで［承認のために送信］をクリックます。
    • ［インシデントコンソール］タブを使用している場合は、［関連リスト］ボタンをクリックし、［承認履歴］セクションで［承認のために送信］をクリックます。
    • ［インシデント］タブ（Salesforceページレイアウト）を使用している場合は、［承認履歴］セクションの［承認のために送信］をクリックします。
    • VisualforceページレイアウトUI を使用している場合は、［補足情報］セクションで、［承認］タブをクリックして、メニューの［承認のために送信］を選択します。
  3. 確認ダイアログボックスで、［OK］をクリックします。

    4. インシデントが承認のために送信され、現在の承認ステータスが［承認履歴］セクション（VisualforceページレイアウトUI の［ステータス］カラム）に表示されます。インシデントの承認リクエストを取り消すには、［承認リクエストの取り消し］をクリックします。
