Pentaho Data Integrationツールは、さまざまなデータソースからBMC Remedyforceへのデータインポートに使用できるオープンソースの統合ツールです。ここでは、Pentaho Data Integrationツールをダウンロードする手順について説明します。詳細は、 http://wiki.pentaho.com/display/EAI/.01+Introduction+to+Spoonおよび http://wiki.pentaho.com/display/EAI/Spoon+User+GUI deを参照してください。

Javaの最新バージョンがコンピュータにインストールされている必要があります。

Pentaho Data Integrationツールをダウンロードするには

  1. http://sourceforge.net/projects/pentaho/files/latest/download?source=filesを開きます。
  2. ［ファイルのダウンロード］ウィンドウで、［保存］をクリックします。

    3. pdi-ce-5.0.1.A-stable.zipファイルが保存されます。

  3. 保存したファイルを解凍します。
  4. 解凍したpdi-ce-5.0.1.A-stableフォルダで、data-integrationフォルダをダブルクリックします。
  5. Pentaho Data IntegrationツールのUI を表示するには、Spoon.batファイルをダブルクリックします。
  6. ［Repository Connection（リポジトリの接続）］ウィンドウが表示されます。［Show this dialog at startup（開始時にこのダイアログを表示）］チェックボックスをオフにして、ウィンドウを閉じることができます。
