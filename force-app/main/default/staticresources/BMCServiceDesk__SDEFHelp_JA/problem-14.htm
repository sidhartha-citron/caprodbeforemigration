レコードの保存の前後を問わず、問題レコードに添付ファイルを追加できます。

注意: ［Remedyforceコンソール］タブで、問題レコードを保存する前に添付ファイルを追加するには、［アクション］メニューから、［ファイルを添付・表示］を選択します。

SalesforceページレイアウトUI を使用していて、同じレコードのVisualforceページレイアウトUI からレコードに添付したドキュメントが表示されていない場合は、システム管理者に問い合わせてください。

問題に添付ファイルを追加するには

  1. 添付ファイルを追加する問題に移動します。

    2. 詳細は、問題レコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブで、［アクション］メニューから、［ファイルを添付・表示］を選択します。
    • ［問題］タブ（Salesforceページレイアウト）で、［アクション履歴］セクションの［ファイルを添付］をクリックします。
  3. ［ファイルを添付・表示］ウィンドウで、［ファイルを選択］をクリックします。
  4. 追加するファイルを選択し、［開く］をクリックします。
  5. ［終了］をクリックします。

    6. 選択されたファイルがレコードに添付されます。［ファイルを添付・表示］ウィンドウで、既存の添付ファイルを表示または削除できます。

  6. VisualforceページレイアウトUI を使用している場合は、次のアクションを実行します。
    1. ［補足情報］セクションで、［ドキュメンテーション］タブをクリックします。
    2. ［］メニューから、［追加］を選択します。
    3. ファイルを添付するには、［ファイルを選択］をクリックして目的のファイルを選択します。
    4. URLを添付するには、［URL］を選択して目的のURLを指定します。
    5. ［OK］をクリックします。
    6. （オプション）添付されたドキュメントまたはURLを開くには、ドキュメントまたはURLを選択し、［］メニューから［開く］を選択します。
    7. （オプション）添付されたドキュメントまたはURLを削除するには、ドキュメントまたはURLを選択し、［］メニューから［削除］を選択します。
