スタッフが注意を払う必要があるイベントが生成されたとき、BMC Remedyforceに電子メールを送信してそのイベントのインシデントを作成する必要があります。BMC Remedyforceに電子メールを送信するには、そのようなイベントが生成されたときに電子メールを送信するアクションを作成します。アクションを作成するには、追跡するイベントのイベント規則をカスタマイズします。

BMC ProactiveNet Performance Managementでイベント規則をカスタマイズするには

  1. Operations Consoleで、［Options］をクリックします。
  2. ［Options］ウィンドウで、［Administration］をクリックします。
  3. 目的のイベント規則エントリの横の［Edit］をクリックします。
  4. ［Edit Event Rule (Page 1/3)］画面で、［Next］をクリックします。
  5. ［Edit Event Rule (Page 2/3): Select Event/Monitor/Severity Information］ページで、［Next］をクリックします。
  6. ［Edit Event Rule (Page 3/3): Action Information for Event Rule］画面で、インシデントを作成するためにBMC Remedyforceに送信する電子メールの送信元電子メールアドレスを［From］フィールドに入力します。
  7. ［To］フィールドに、BMC Remedyforceですでに設定している電子メールアドレスを入力します。
  8. ［追加］をクリックします。
  9. ［Event Text Customization］をクリックします。
  10. ［Subject］フィールドに、BMC Remedyforceに送信する電子メールの件名を入力します。
  11. ［Body］フィールドに、BMC Remedyforceで作成されるインシデントにマップするフィールドを入力します。［Body］フィールドでは次の形式を使用します。

    12. <キーワード> <区切り記号> <BMC ProactiveNet変数> <区切り記号>

    <キーワード>と<区切り記号>は、BMC Remedyforceの［Remedyforce管理］>［電子メールの設定］>［インシデント - 電子メールの設定］ページで選択したキーワードと区切り記号です。Visualforceページレイアウトユーザーインターフェイス（UI ）を使用している場合は、［設定］>［アプリケーションの設定］で［インシデント - 電子メールの設定］タブが表示されます。<BMC ProactiveNet変数>の値が、<キーワード>にマップされたインシデントフィールドにコピーされます。このマッピングは、BMC Remedyforceの［Remedyforce管理］>［電子メールの設定］>［インシデント - 電子メールの設定］>［インシデントフィールドへの電子メールの内容のマッピング］ページで指定します。たとえば、EVENTID##$EVENTID##とCATEGORY##General Information##のマッピング例では、EVENTIDとCATEGORYはキーワード、$EVENTIDはBMC ProactiveNet変数、General Informationは作成するインシデントに割り当てるカテゴリ、##は区切り記号です。

    EVENTIDとCATEGORYは必須フィールドであり、それぞれBMC RemedyforceのインシデントオブジェクトのBMCServiceDesk__Event_ID__cフィールドとCategoryフィールドにマップされています。BMCServiceDesk__Event_ID__cフィールドは固有のイベントIDを格納するフィールドで、BMC RemedyforceからBMC ProactiveNet Performance Managementのイベントへの参照として使用されます。

  12. ［Apply］をクリックします。
