［カテゴリ］タブでは、新しいカテゴリを作成し、セルフサービスクライアントが使用できるカテゴリを絞り込むことができます。セルフサービスクライアントは、使用可能なカテゴリについてのみインシデントおよびサービスリクエストを作成できます。

セルフサービスのカテゴリを定義するには

  1. BMC Remedyforceホームページの［設定］をクリックします。
  2. ［設定］オプションのリストから［セルフサービス］を展開します。
  3. ［セルフサービス］ドロップダウンリストで、［セルフサービスの設定］をクリックします。

    4. 注意: BMC Remedyforceのタブ形式ビューでは、［Remedyforce管理］タブに［セルフサービスの設定］があります。

  4. ［カテゴリ］タブをクリックします。

    5. セルフサービスクライアントが使用できるすべてのカテゴリが、［カテゴリ］ドロップダウンリストに表示されます。

  5. 新しいカテゴリを作成するには、［新規］をクリックします。また、［カテゴリ］フォームのカテゴリを編集するには、既存のカテゴリをクリックします。

    6. 詳細は、カテゴリレコードの作成を参照してください。

  6. ［アクション］>［カテゴリの管理］をクリックします。
  7. セルフサービスクライアントで使用できるカテゴリを絞り込むには、［カテゴリの管理］フォームで次のアクションを実行します。
    1. ［使用可能なカテゴリ］リストで、セルフサービスクライアントが使用できるようにするカテゴリを選択します。
    2. 選択したカテゴリを［使用可能なカテゴリ］リストから［選択したカテゴリ］リストに移動するには、右矢印をクリックします。
    3. （オプション）選択したカテゴリを［選択したカテゴリ］リストから［使用可能なカテゴリ］リストに移動し、セルフサービスクライアントがこれらのカテゴリを使用できないようにするには、左矢印をクリックします。

      4. 注意: ダブルクリックすると、そのカテゴリが［使用可能なカテゴリ］リストと［選択したカテゴリ］リストの間で移動します。

    4. セルフサービスクライアントへのカテゴリの表示順序を定義するには、目的のカテゴリを選択して上下矢印をクリックし、そのカテゴリを適切な表示順序位置に移動します。
    5. ［保存］をクリックします。
    6. 確認ダイアログボックスで、［OK］をクリックします。

      7. 使用可能に設定したカテゴリが、［カテゴリ］ドロップダウンリストに表示されます。
