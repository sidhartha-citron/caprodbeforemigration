［セルフサービスプリファレンス］フィールドは、ユーザーのセルフサービスのホームページレイアウトのプリファレンスを復元する目的で使用します。ポートレットの再配置など、セルフサービスユーザーが各自のセルフサービスホームページをカスタマイズすると、その内容は［セルフサービスプリファレンス］フィールドに保存されます。ユーザーの依頼により、デフォルトのセルフサービスホームページに復元する場合、システム管理者は［セルフサービスプリファレンス］フィールドの内容を削除することができます。このフィールドは、［ユーザーレイアウト］ページのページレイアウトに追加する必要があります。これによりシステム管理者は、セルフサービスユーザーからの依頼を受けて、デフォルトのセルフサービスホームページを復元できるようになります。

［セルフサービスプリファレンス］フィールドを［ユーザーレイアウト］に追加するには

  1. ［設定］>［カスタマイズ］>［ユーザー］>［ページレイアウト］の順に選択します。
  2. ［ユーザーページレイアウト］ページで、［編集］をクリックします。
  3. ［フィールド］セクションから、［セルフサービスプリファレンス］フィールドを［その他の情報］セクションにドラッグします。
  4. ［保存］をクリックします。
