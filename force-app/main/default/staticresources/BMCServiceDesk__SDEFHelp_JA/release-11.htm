リリースにメモを記録する場合は、［アクション履歴］セクションにこれらの情報を入力できます。

リリースにメモを追加するには

  1. メモを追加するリリースレコードに移動します。

    2. 詳細は、リリースレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［アクション］メニューをクリックし、［Add Notes（メモの追加）］をクリックします。

      • または、［アクション履歴］をクリックし、［アクション履歴］セクションで、［新しいメモ］をクリックします。

    • ［リリース］タブ（Salesforceページレイアウト）を使用している場合は、［アクション履歴］セクションの［新しいメモ］をクリックします。

      • ［新しいメモ］ボタンを使用できない場合は、システム管理者に連絡して、変更リクエストフォームのレイアウトをカスタマイズしてボタンを追加するように依頼してください。

  3. メモの詳細を入力します。
  4. （オプション）［所要時間］フィールドに、メモを追加するアクションにかかった時間を入力します。
  5. ［追加］をクリックします。
