Visualforceページレイアウト、コンソールレイアウト、またはSalesforceページレイアウトを使用して、タスクを作成したり処理したりできます。［タスクコンソール］タブは、システム管理者がコンソールレイアウトを構成した場合のみ使用できます。複数のUI タイプについての詳細は、使用するUI の決定を参照してください。

［タスクコンソール］タブの使用に関する詳細は、タスクの処理を参照してください。

次の表に、［タスクコンソール］タブで使用可能な機能を説明します。

機能                         
説明
リストビュー
デフォルトでは、すべてのレコードがリストビューに表示されます。また、要件に合ったビューを作成することもできます。詳細は、 https://help.salesforce.com/htviewhelpdoc?err=1&id=customviews.htm&siteLang=en_USを参照してください。また、選択したビューに基づいたリストでレコードをフィルタリングし、レコードを検索することもできます。

メモ:
 このレコードリストには、最大10,000件のレコードが表示されます。
ブロードキャストティッカー
をクリックして、［Chatter］上でブロードキャストを表示、フォロー、またはフォロー解除します。
Chatterフィード
Salesforce組織でChatterが有効になっている場合、レコードの関連リストの上にChatterフィードが表示されます。
レコードを選択した後、［関連リスト］をクリックします。Chatterフィードを表示または非表示にするには、［Show Feed（フィードを表示）］または［Hide Feed（フィードを非表示）］をクリックします。
ルックアップ
［タスクコンソール］タブのルックアップでは、次の機能を使用できます。
  • オブジェクトに使用可能なすべてのレコードが表示されます。たとえば、［アカウント］フィールドルックアップをクリックすると、［アカウント］オブジェクトのすべてのアクティブレコードが表示されます。
  • システム管理者はルックアップタイプフィールドにフィルタを作成して、フィルタされたレコードを表示できます。
  • ［カテゴリ］フィールドのルックアップウィンドウには、カテゴリがツリービューで表示され、レコードの作成中のカテゴリの階層の理解に役立ちます。カテゴリの上にマウスを移動すると、カテゴリの説明がツールチップとして表示されます。
カスタムアクション
システム管理者によって追加されたカスタムアクションは、関連リストの上にボタンとして表示されます。
検索
リストビュー、ルックアップウィンドウ、［<オブジェクト>のリンク］ウィンドウ、［一致するインシデント］ウィンドウでは、名前、電子メール、電話番号、テキスト、テキスト領域、ロングテキスト領域、リッチテキスト領域のフィールドを検索できます。ただし、数式、ルックアップ、ロールアップサマリー、番号、日時のフィールドは検索されません。また、検索操作では、フィールドの値の先頭文字との一致を検索します。そのため、検索するフィールド値には常に正しい先頭文字を指定するようにしてください。
たとえば、「John」を検索する場合に「ohn」と入力すると、 Johnを含むレコードは表示されません。ただし、「John」を検索した場合、Johnを含むすべてのレコード（JohnsやJohnsonなど）が表示されます。
