ここでは、インシデントを作成するためのジョブをBMC Server Automationコンソールで設定する手順を示します。

  1. BMC Server Automationコンソールで［ジョブ］をクリックします。
  2. 必要なジョブエントリを選択します。
  3. ［デフォルト通知］タブをクリックします。
  4. 送信するデータに応じて、［遵守結果の通知］セクションまたは［ジョブ実行通知］セクションの［電子メールの送信先］フィールドを選択します。
  5. ［電子メール送信先］フィールドに、BMC Remedyforceで設定された新規作成された電子メールリスナーの電子メールアドレスを入力します。
