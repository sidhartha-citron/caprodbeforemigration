デフォルトでは、変更リクエストは、作成したユーザーに割り当てられます。変更リクエストの保存後に、割り当てを他のユーザーまたはキューに変更できます。

［Remedyforceコンソール］タブでの変更リクエストオーナーの割り当ての変更

［割り当ての詳細］セクションで、システム管理者の設定に基づいて、［オーナー］フィールドまたは［キュー］フィールド、［スタッフ］フィールドが表示されます。

［Remedyforceコンソール］タブで変更リクエストオーナーの割り当てを変更するには

  1. 他のユーザーまたはキューに割り当てるレコードに移動します。

    2. 詳細は、変更リクエストレコードへの移動を参照してください。

    リストビューでレコードを選択し、［オーナーを変更］をクリックすることもできます。

  2. ［オーナー］フィールドが［割り当ての詳細］セクションに表示されている場合は、次のアクションを実行します。
    1. ［オーナー］ルックアップをクリックします。

      2. スタッフメンバーが組織に存在する場合、デフォルトではすべてのスタッフメンバーが［提案されたオーナー］ウィンドウに表示されます。スタッフメンバーが存在しない場合は、キューが表示されます。

    2. レコードをスタッフメンバーに割り当てるには、目的のスタッフメンバーのレコードをクリックします。
    3. レコードをキューに割り当てるには、［キュー］オプションを選択して目的のキュー名をクリックします。
    4. キューに属しているスタッフメンバーにレコードを割り当てるには、最初に目的のキューの行をクリックし、キューに属するスタッフメンバーのリストから目的のスタッフメンバーを選択します。
  3. ［スタッフ］および［キュー］フィールドが［割り当ての詳細］セクションに表示されている場合は、次のアクションを実行します。
    1. レコードをスタッフメンバーに割り当てるには、［スタッフ］フィールドから目的のスタッフを選択します。
    2. レコードをキューに割り当てるには、［キュー］フィールドから目的のキューを選択します。
    3. キューに属しているスタッフメンバーにレコードを割り当てるには、最初に目的のキューの行をクリックし、キューに属するスタッフメンバーのリストから目的のスタッフメンバーを選択します。

      4. ［キュー］フィールドでキューを選択すると、キューに属するスタッフメンバーが［スタッフ］フィールドルックアップに表示されます。

［変更リクエスト］タブでの変更リクエストオーナーの割り当ての変更

［スタッフ割り当ての詳細］セクションで、システム管理者の設定に基づいて、［オーナー］フィールドまたは［キュー］フィールド、［スタッフ］フィールドが表示されます。

変更リクエストの割り当てを変更するには

  1. 他のユーザーまたはキューに割り当てる変更リクエストに移動します。

    2. 詳細は、変更リクエストレコードへの移動を参照してください。

  2. ［スタッフ割り当ての詳細］セクションで、変更リクエストが割り当てられているユーザー名の横にある［変更］をクリックします。
  3. ［オーナー］ドロップダウンリストで、目的のオプションを選択します。
    • 変更リクエストをユーザーに割り当てるには［ユーザー］を選択します。
    • 変更リクエストをキューに割り当てるには［キュー］を選択します。

      • スタッフメンバーが応対できない場合、または外出中の場合は、［ユーザーの検索］ウィンドウの［外出中］カラムにステータスを表示することができます。［ユーザーの検索］ウィンドウの［外出中］カラムを設定するには、システム管理者に問い合わせてください。

      外出中のステータスを設定するには、［設定］>［私の個人情報］>［個人情報］の順に選択します。

      注意: Salesforce組織で、改善された［設定］ユーザーインターフェイスを有効化した場合は、［私の設定］>［個人用］>［個人情報］に移動します。改善された［設定］ユーザーインターフェイスについての詳細は、 http://help.salesforce.com/apex/HTViewHelpDoc?id=admin_setup_improved.htm&language=en_USを参照してください。

      ［BMC Remedyforce］セクションで［編集］をクリックし、［外出中］チェックボックスをオンにします。

  4. ユーザーまたはキューを選択するには、をクリックして目的のオプションを選択します。
  5. ［保存］をクリックします。

    6. 注意: スタッフおよびキュー割り当て機能を使用するには、システム管理者に問い合わせて変更リクエストレイアウトを編集し、［StaffAssignmentPage］Visualforceページを表示する必要があります。

    ［変更リクエスト］フォームの［割り当て先］メニューでは、未完了の変更リクエストを自分自身、他のスタッフメンバー、自分が属するプロファイルのスタッフメンバーに割り当てます。

    ［割り当て先］メニューを使用して現在のレコードをスタッフメンバーに割り当てるとき、開いた［スタッフから選択］ウィンドウに一部のスタッフメンバーが表示されない場合があります。

VisualforceページレイアウトUI からの変更リクエストオーナーの割り当ての変更

VisualforceページレイアウトUI を使用している場合は、このセクションで説明している手順を実行します。

  2. ［割り当て先］メニューをクリックします。

    説明
    スタッフ

    スタッフメンバーが応対できない場合、または外出中の場合は、［スタッフから選択］ウィンドウの［外出中］カラムにステータスが表示されます。
