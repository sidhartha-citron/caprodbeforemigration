インシデントまたはサービスリクエストの保存後に、［アクション履歴］セクション（VisualforceページレイアウトUI の［補足情報］セクションの［履歴］タブ）で、ユーザー定義アクションとシステム定義アクションをインシデントに追加できます。

アクションをインシデントまたはサービスリクエストに追加するには

  1. アクションを追加するレコードに移動します。

    2. 詳細は、インシデントレコードまたはサービスリクエストレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［アクション］メニューをクリックし、［新規アクション］を選択します。

      • または、［関連リスト］をクリックし、［アクション履歴］セクションで、［新規アクション］をクリックします。

    • ［インシデントコンソール］タブを使用している場合は、［アクション］メニューをクリックし、［新規アクション］を選択します。

      • または、［関連リスト］をクリックし、［アクション履歴］セクションで、［新規アクション］をクリックします。

    • ［インシデント］タブ（Salesforceページレイアウト）を使用している場合は、［アクション履歴］セクションの［新規アクション］をクリックします。
    • VisualforceページレイアウトUI を使用している場合は、［補足情報］セクションで、［履歴］タブをクリックして、メニューの［追加］を選択します。
  3. 追加する適切なアクションを選択します。

    4. ［メモ］オプションを選択すると、追加メモを入力するための新しいブラウザウィンドウが開きます。メモは、Salesforceページレイアウトの［アクション履歴］セクションに表示されます。

    VisualforceページレイアウトUI では、このメモは［履歴］タブに表示されます。ある特定のアクションが［履歴］タブに追加されたときにクライアントに電子メールメッセージを送信するワークフローを作成できます。

    注意: このアクションを追加した後は、［Client Note（クライアントメモ）］アクションを保存できません。［Client Note（クライアントメモ）］アクションは、セルフサービスクライアントが排他的に使用するものです。この制限は、セルフサービスクライアントだけが［Client Note（クライアントメモ）］アクションを編集できるように追加されました。

  4. 必要な詳細を追加して、［保存］をクリックします。

インシデントまたはサービスリクエストに関連付けられたアクションの表示

［アクション履歴の表示］オプションを使用して、インシデントまたはサービスリクエストに関連付けられたアクションを表示できます。［アクション履歴の表示］オプションでは、新規アクションの作成、タイプに基づくアクションのフィルタリング、およびアクションの印刷を行えます。

インシデントまたはサービスリクエストに関連付けられたアクションを表示するには

  1. 関連付けられたアクションを表示するインシデントまたはサービスリクエストに移動します。

    2. 詳細は、インシデントレコードまたはサービスリクエストレコードへの移動を参照してください。

  2. 次のいずれかのアクションを実行します。
    • ［Remedyforceコンソール］タブを使用している場合は、［アクション］メニューをクリックし、［アクション履歴の表示］を選択します。

      • または、［関連リスト］をクリックし、［アクション履歴］セクションで、［アクション履歴の表示］をクリックします。

    • ［インシデントコンソール］タブを使用している場合は、［アクション］メニューをクリックし、［アクション履歴の表示］を選択します。

      • または、［関連リスト］をクリックし、［アクション履歴］セクションで、［アクション履歴の表示］をクリックします。

    • ［インシデント］タブ（Salesforceページレイアウト）を使用している場合は、［アクション履歴］セクションの［アクション履歴の表示］をクリックします。

      • ［インシデントの履歴］ウィンドウに、表示専用モードでアクションが表示されます。［インシデントの履歴］ウィンドウに表示されるアクションの編集や削除はできません。

  3. （オプション）タイプに基づいてアクションをフィルタリングするには、リストから目的のアクションを選択します。
  4. （オプション）インシデントレコードまたはサービスリクエストレコードに関連付けられたアクションを印刷するには、［印刷］をクリックします。
