調整形式
要素の地理的位置の形式を指定します。事実上すべての地理情報システムで使用されるWorld Geodetic System 1984（WGS84）規格が優先されます。
ここで、
  • 緯度は-180.000から+180.000の間の小数値です（正の値は北緯の度数を表し、負の値は南の度数を表します）。
  • 経度は-180.000から+180.000の間の小数値です（負の値はグリニッジ子午線の東の度数を表し、負の値は西の度数を表します）。
  • 高度は、平均海水面（MSL）上の高さを指定する小数値（メートル単位）で、負の値はMSL以下を示します。
［調整形式］フィールドでは他の値も使用できますが、パートナー製品やその他のBMC製品で機能しないことがあります。地理的位置の値は、ユーザーインターフェイスで表示するために、人が読み取りやすい形式（度、分、秒、北、西、南、北など）に変換することができます。
