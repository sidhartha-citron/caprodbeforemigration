Pentaho Data Integrationツールを実行するには、Java Runtime Environment（JRE）をインストールする必要があります。ここでは、JREをWindowsコンピュータにインストールする手順について説明します。

JREをインストールするには

  1. http://www.oracle.com/technetwork/java/index.htmlを開きます。
  2. ［Software Downloads］セクションで、［Java SE］をクリックします。
  3. ［Java Platform］の［Java Downloads］セクションで、［Download JRE］をクリックします。
  4. ［Java SE Downloads］セクションで［Accept License Agreement］を選択し、［Download］列で対象の.exeファイルをクリックします。
  5. ［ファイルのダウンロード］ウィンドウで、［実行］をクリックします。
  6. ［インストール］をクリックします。
