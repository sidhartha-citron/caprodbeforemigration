Work Instructions for Service Desk Analysts

4.2.1

Check the status of the support request from time to time to ensure that it continues to progress towards completion.

4.2.2

If you find that the support request is not progressing any further (e.g. because, one day after it was registered, its Status field is still set to "Assigned"), contact the coordinator of the group to which the support request is assigned (by phone or email) and ask him/her to make sure that the support request gets dealt with.
