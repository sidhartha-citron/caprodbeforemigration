Work Instructions for Service Desk Analysts

7.8.1

Enter the number of minutes you spent on the solution verification in the Time spent (min.) field of the support request and change the value in the Status field from "Completed" to "Closed".

Note:

Do not change the status if it was set to "ClosedMail" in 7.3.2. In that case, the service management application will automatically set the Status field to "Closed" if, after 28 days, the user has not responded to the automatic email notification to indicate that the solution is not satisfactory.
