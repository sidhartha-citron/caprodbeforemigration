Work Instructions for Service Desk Analysts

1.2.1

IIf you have the user on the telephone, or if the user visited the service desk in person, ask the user how you may be of assistance.

1.2.2

Use the information that the user has provided you with (regardless of whether this information was obtained over the telephone, in an email, or otherwise) to determine if the user has contacted the service desk about a previously reported support request, or if the user wants to report a new support request.

1.2.3

If the user is submitting a new support request, continue by determining the category that the support request relates to. Also determine what the user is requesting, and ask for any additional information that could help resolve the request.

Note:

If you need more information to be able to provide sufficient information in the support request so that it can be resolved, and if you are not currently in contact with the user (e.g. if you do not have the user on the telephone, or the user is not personally at the service desk), contact the user by phone or email and request the additional information that is needed. Do not register a support request until this information has been provided.
