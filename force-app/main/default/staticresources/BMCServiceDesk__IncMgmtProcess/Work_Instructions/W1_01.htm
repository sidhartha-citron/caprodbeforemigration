Work Instructions for Service Desk Analysts

1.1.1

Open a new support request by selecting the new button on the "My Work" QuickView displayed on the dashboard.

1.1.2

Select the client in the "Client ID" field. To do this quickly, fill out the first few characters of the client's last name in the last name field and press the tab key. If the characters entered in the last field are not unique, the service management application displays a list of persons whose last name start with the entered characters. Select the client from this list.
