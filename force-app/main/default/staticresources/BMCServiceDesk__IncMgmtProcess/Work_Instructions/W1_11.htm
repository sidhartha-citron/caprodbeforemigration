Work Instructions for Service Desk Analysts

1.11.1

Select the applicable support request category in the Category field.

1.11.2

If the support request concerns (or is suspected to be caused by) a specific configuration item (CI), select it in the CI field.

1.11.3

Enter a short but meaningful description of the request in the Description field.

1.11.4

Add an action incident updated, fill out any additional information that the user can provide and that could prove useful when resolving the support request.
