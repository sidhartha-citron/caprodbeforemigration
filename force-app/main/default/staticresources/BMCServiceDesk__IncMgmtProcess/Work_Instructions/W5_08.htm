Work Instructions for Specialists

5.8.1

Specify in the Solution field what has been done to complete the support request. Also use this field to provide detailed solution instructions (e.g. if the user needs to adjust a setting), to provide the answer to a request for information, or to describe why the support request cannot be resolved if it is not possible to provide a solution or workaround.

5.8.2

Select the appropriate CI in the CI field if this has not been done yet, or if the wrong CI had been selected before

5.8.3

Enter the number of minutes you spent on the resolution of the support request in the Time spent (min.) field

5.8.4

Perform a final review of the support request to ensure that its information is accurate and complete.
