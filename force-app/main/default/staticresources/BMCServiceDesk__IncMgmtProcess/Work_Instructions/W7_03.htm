Work Instructions for Service Desk Analysts

7.3.1

If the information in the Solution field was entered by a specialist, ensure that it is meaningful (i.e. that it explains how the solution was implemented). If this is not the case, contact the specialist directly and work with him/her to improve it. Also correct any spelling or grammatical mistakes if there are any in this field.

7.3.2

Inform the user of the manner in which the support request has been completed and ask the user to verify this solution. The user may be informed by telephone to verify the solution.
