Work Instructions for Specialists

5.7.1

Resolve the support request. Request support from external suppliers if necessary.

Note:

If, at any point during the resolution work, it becomes clear that the support request cannot be resolved before the date and time specified in the Due Date field, estimate a realistic time at which the request will be resolved and enter this information by adding an action line Incident updated.

Note:

Set the Status field to "Waiting for…" if the work cannot continue for a while (e.g. because you are waiting for a spare part or for information from a supplier). Add an action Incident update and specify what you are waiting for.

Note:

If you have completed your work on the support request, but further action is still required to get it resolved, specify what you have done by adding an action Incident updated to the support request. Enter the number of minutes you spent working on the support request in the Time spent (min.) field. If the additional work needs to be done by a specific staff within your group, use the "Assign To" button on the toolbar to select this person in the Staff field. On the other hand, if the additional work needs to be done by another group, use the "Assign To" button on the toolbar to select this group in the Queue field. Ensure that the Status field is set back to "Assigned".
