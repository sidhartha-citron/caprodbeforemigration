Work Instructions for Service Desk Analysts

1.12.1

Select the appropriate impact level in the Impact field. Only the following options will be available:

Low

-

Impacts 1 user

Medium

-

Impacts 1 user or degraded for several

High

-

Impacts several Users

1.12.2

Select the appropriate urgency level in the Urgency field. Only the following options will be available:

Low

-

Service degraded for 1 user

Medium

-

Service for 1 user or degraded for several

High

-

Service down for several Users

Note:

The service management application automatically determines the appropriate priority for requests for incident resolution using the support request's impact level and the urgency level. Having determined the appropriate priority for the support request, the service management application subsequently uses it, to calculate the due date and time for the support request.
