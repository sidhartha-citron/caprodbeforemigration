Work Instructions for Service Desk Analysts

1.6.1

If the user has provided additional information that is relevant to the resolution of the support request, add action incident information to the support request. Contact the specialist to whom the support request is assigned and provide this person with an update.

1.6.2

Provide the user with an update on the support request's status.

Note:

If the support request has not been updated recently, contact the specialist to whom it is assigned to obtain a status update so that you can pass this information on to the user.

1.6.3

Summarize your actions and the information that you provided to the user by adding an action incident updated to the support request.

1.6.4

Enter the number of minutes you spent on the update of the support request in the Time spent (min.) field.
