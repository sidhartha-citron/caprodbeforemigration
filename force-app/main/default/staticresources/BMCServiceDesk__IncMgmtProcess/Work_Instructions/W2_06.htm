Work Instructions for Group Coordinators

2.6.1

Continue with 2.7.1 if the support request was not assigned to the appropriate queue or if it does not contain all information that should have been provided in order to be able to resolve it. Otherwise go to 2.9.1.

Note:

If you are the group coordinator of the service desk, ensure that the support request contains all information that needs to be collected before the work can start on its completion. Once this is done, assign the support request to the service desk manager if its Category field is set to "Request for Support Improvement". If the support request is not a request for support improvement, determine whether or not the support request can be resolved by the service desk. If this is the case, assign the support request to a service desk analyst who is currently available for resolving support requests. If the support request cannot be completed by the service desk, assign it to the most appropriate queue.
