プロセスモデルヘルプ - ボタン


手順にアクセスすることができるプロセスフロー以外にも、プロセスモデルのプロセスをクリックすることで、その他たくさんの情報を参照することができます。スクリーンの左側にあるボタンをクリックして、この情報にアクセスすることができます。どのプロセスでも、同じボタンが使用できます。ボタンと、これらをクリックした際に表示される情報の説明は次の通りです。

ボタン

説明

プロセスフローのグラフィック表示にアクセスするには、このボタンをクリックします。これは、プロセスにアクセスした際に表示されるデフォルトのページです。

プロセスを、グラフィック表示ではなく文書で取得するには、このボタンをクリックします。

プロセスのミッションを取得するには、このボタンをクリックします。

プロセスの適用範囲を取得するには、このボタンをクリックします。

プロセスに関わる別の役割の責任をレビューするには、このボタンをクリックします。

プロセスをサポートするために使用されているサービス管理アプリケーション内で使用できるフォームのリストを取得するには、このボタンをクリックします。

プロセスのドキュメンテーションの、Microsoft Word 版をダウンロードするには、このボタンをクリックします。

用語や略語とその意味のリストを取得するにはこのボタンをクリックしてください。この用語集の内容は、プロセスに特有なものではありません。

プロセスモデルに戻るには、このボタンをクリックします。

プロセスモデルの使い方の概要を取得するには、このボタンをクリックします。


プロセスから手順にアクセスすれば、手順の各ステップの作業指示まで調べることが可能です。ここでも、スクリーンの左側にあるボタンをクリックして、詳細情報にアクセスすることができます。ボタンと、これらをクリックした際に表示される情報の説明は次の通りです。

ボタン

説明

手順フローのグラフィック表示にアクセスするには、このボタンをクリックします。これは、手順にアクセスした際に表示されるデフォルトのページです。

手順を、グラフィック表示ではなく文書で取得するには、このボタンをクリックします。

その手順が一部であるプロセスに関連した、別の役割の責任をレビューするには、このボタンをクリックします。

その手順が一部であるプロセスをサポートするために使用されているサービス管理アプリケーションで使用可能なフォームのリストを取得するには、このボタンをクリックします。

用語や略語とその意味のリストを取得するにはこのボタンをクリックしてください。用語集はプロセスまたは手順ごとではありません。

プロセスに戻るには、このボタンをクリックします。

プロセスに戻らずにプロセスモデルに戻るには、このボタンをクリックします。

プロセスモデルの使い方の概要を取得するには、このボタンをクリックします。
