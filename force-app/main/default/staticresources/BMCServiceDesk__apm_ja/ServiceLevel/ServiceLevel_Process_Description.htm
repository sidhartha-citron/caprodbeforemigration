プロセスの説明


サービスレベル管理プロセスは、次の手順で構成されています。

  • サービスカタログ保守は、変更コーディネータが新しく登録予定のカタログアイテムを要求するときに、サービスレベル管理者によって使用されます。

  • サービスの有効化は、顧客がカタログアイテムを選択して、カタログアイテムに指定されているレベルでサービスの申し込みを決定した後は、サービスレベル管理者に従います。

  • 顧客情報保守は、サービスレベル管理者がサービスプロバイダ組織の新しい顧客を登録するときに使用します。　 また、サービスレベル管理者が登録されている顧客の連絡先情報を更新するときに、この手順を使用します。

  • サービスの終了は、サービス、カタログアイテム、SLAが終了したときに、サービスレベル管理者が使用します。

  • SLAレビューとリクエスト処理は、それぞれの評価期間終了時に、サービスレベル管理者がSLAをレビューするときに使用します。また、サービスレベルマネージャは、顧客担当者がリクエストの送信依頼のために連絡してきたとき、この手順を使用します。

これら手順の詳細を確認するには、このプロセスのボタンをクリックしてグラフィック表示に戻り、該当する手順のボックスをクリックします。手順が、右のペインで説明と共にグラフィック表示されます。
