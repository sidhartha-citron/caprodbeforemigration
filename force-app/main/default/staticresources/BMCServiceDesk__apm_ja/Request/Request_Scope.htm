適用範囲


リクエスト管理プロセスの適用範囲は、次のサポートリクエストカテゴリに限定されます。

カテゴリ

説明

インシデント

ユーザーからの、正しく機能していないサービスを復元するリクエスト

メモ:

パスワードを忘れたユーザーからのパスワードリセットリクエストおよびデータを紛失したユーザーからのバックアップ復元リクエスト

変更リクエスト

編集、追加、一部またはすべてのサービス機能、アクセス、コンポーネントの移動

メモ:

データコピー（たとえば、異なる場所でのバックアップ作成および復元）リクエストおよびバッチジョブ実行リクエストはこのカテゴリに属します。

サービスリクエスト

次のシナリオで、事前定義済みワークフローの使用で解決できる、ユーザーからのリクエスト

  • このワークフロー用にテンプレートが存在する
  • リスクとインパクト分析が不要
  • 顧客以外の承認が不要

情報リクエスト

ユーザーからの、サービス関連の質問に対する回答リクエスト

称賛

ユーザーからの、お褒めや励ましの言葉

苦情

ユーザーからの、サービスプロバイダ組織のユーザーサポート改善についてのリクエスト
