サービスデスクアナリストへの作業指示

1.9.1

関連するメジャーインシデントを［ブロードキャスト］フィールドで選択します。

［エージェントツール］>［Incident Matching（インシデントの一致）］オプションから、既存のインシデントをリンクできます。

1.9.2

残りの必須フィールドの記入を完了して、リクエストを保存します。

メモ:

リクエストがメジャーインシデントにリンクされている場合は、直接リクエストを変更してはいけません。メジャーインシデントの更新が、すべての子リクエストにコピーされます。

ブロードキャストの選択およびインシデントの保存後、インシデントの次のフィールドに入力されます。
