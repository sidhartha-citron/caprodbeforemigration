作業指示について

手順

グループコーディネータへの作業指示

2.7.1

リクエストが適切なグループに割り当てられていない、またはリクエストの解決に必要な情報がそろっていない場合は、続けて2.8.1の作業を行います。その他の場合は、2.10.1に進みます。

メモ:

自分がサービスデスクのグループコーディネータである場合は、リクエストについて、作業の開始までに集める必要があるすべての情報がそろっていることを確認します。その後、［カテゴリ］フィールドが［苦情］または［称賛］に設定されている場合は、サービスデスクマネージャにそのリクエストを割り当てます。リクエストが苦情または称賛でない場合は、サービスデスクがリクエストを解決できるか判断します。サービスデスクで解決できる場合は、現在リクエスト解決のために作業可能なサービスデスクアナリストに、リクエストを割り当てます。サービスデスクで解決できない場合は、最も適切なグループにリクエストを割り当てます。
