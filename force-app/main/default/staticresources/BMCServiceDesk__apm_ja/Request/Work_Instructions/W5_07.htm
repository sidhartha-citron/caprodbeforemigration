手順

スタッフへの作業指示

5.7.1

リクエストを解決します。必要に応じて、外部サプライヤからのサポートをリクエストします。

メモ:

リクエストが［期限日］フィールドに指定されている日時までに解決できない場合は、リクエストががいつまでに解決可能か、現実的な時期を見積もり、［メモ］アクションを追加してその情報を記入します。

メモ:

作業を続行できない場合（たとえば、顧客からの予備部品または情報を待っている場合）、［ステータス］フィールドを［待機中］または［Waiting for customer（顧客待ち）］に設定します。［メモ］アクションを追加して、待機中の内容を指定します。サプライヤを含めてリクエストを登録した場合、このサプライヤと、［メモ］アクションのサプライヤから取得したリクエスト番号を入力します。

メモ:

リクエストの作業を終了しても、リクエストを解決するためにそれ以上のアクションが必要な場合、［メモ］アクションを追加して行った作業を指定します。［メモ］アクションの［所要時間(HH:MM)］フィールドで、リクエストの作業を行った時間を入力します。自分のグループ内の特定のスタッフメンバーが追加作業を行う必要がある場合、［オーナー］フィールドの横にあるルックアップアイコンを使用して、［オーナー］フィールドでスタッフメンバーを選択します。別のグループが追加作業を行う必要がある場合、［オーナー］ フィールドの横にあるルックアップアイコンを使用して、［オーナー］フィールドでグループを選択します。［ステータス］フィールドが［割り当て済み］であることを確認します。
