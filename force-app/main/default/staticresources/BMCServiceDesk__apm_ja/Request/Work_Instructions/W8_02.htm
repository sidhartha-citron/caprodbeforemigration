作業指示について

手順

サービスデスクアナリストへの作業指示

8.2.1

リクエストの［解決方法］フィールドに、どのようにリクエストを解決したか記入します。

8.2.2

未選択または誤ったCIが選択されている場合、CIフィールドで適切な構成アイテム（CI）を選択します。

メモ:

CIが、CMDBには未登録だが構成管理プロセスの適用範囲である場合は、そのCIをNORECORDコードで選択します。

コードがNORECORDのCIは、デフォルトでは表示されません。

メモ:

特定のCIへのリクエストではない、または構成管理プロセスの適用範囲にないCIの場合は、そのCIをN/Aコードで選択します。

8.2.3

［メモ］アクションの［所要時間(HH:MM)］フィールドに、リクエストの登録および解決にかかった時間を入力します。

8.2.4

適切な完了カテゴリを、［完了カテゴリ］フィールドで選択します。リクエストの［ステータス］フィールドを［終了済み］に設定します。

8.2.5

［オーナー］フィールドで自分が選択されていることを確認します。

8.2.6

最終レビューを実行し、リクエストが正確に記入されていることを確認します。

8.2.7

未完了問題の回避策を適用してリクエストを解決する場合、その問題にインシデントをリンクします。
