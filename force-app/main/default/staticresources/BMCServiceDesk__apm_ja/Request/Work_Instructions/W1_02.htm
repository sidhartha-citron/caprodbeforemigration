作業指示について

手順

サービスデスクアナリストへの作業指示

1.2.1

ユーザーが電話で、またはサービスデスクに来訪してきた場合、どのようなサポートが必要か質問します。

1.2.2

入手できた情報（電話、電子メール、その他、入手経路にかかわりなく）を利用して、過去に報告済みのリクエストについて問い合わせているのか、新しいリクエストを報告しているのかを判断します。

1.2.3

ユーザーが新しいリクエストを報告している場合は、関連するサービスを特定します。また、ユーザーのリクエスト内容を特定し、リクエストの解決に役立つその他の情報についてユーザーに質問します。

メモ:

リクエストを解決する詳細情報が必要で、現時点でユーザーと連絡を取れる状態にない（たとえばユーザーと通話中でない、またはユーザーがサービスデスクにいない）場合は、電話または電子メールでユーザーに連絡を取り、必要な追加情報の提供を依頼します。必要な情報がそろうまでは、リクエストを登録しないでください。
