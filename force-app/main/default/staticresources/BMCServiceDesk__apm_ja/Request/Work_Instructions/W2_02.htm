作業指示について

手順

サービスデスクアナリストへの作業指示

2.2.1

［メモ］アクションの［所要時間(HH:MM)］フィールドに、リクエストの登録にかかった時間を入力します。

2.2.2

［オーナー］フィールドの横にあるルックアップアイコンを使用して、サービスデスクマネージャを選択します。

2.2.3

ユーザーと通話中の場合は、ユーザーの苦情または称賛をサービスデスクマネージャに割り当て済みであることを説明して、インシデント番号を伝えます。この番号は、［インシデント番号］フィールドに指定されています。インシデント番号を伝えることで、ユーザーは苦情または称賛が登録されたことを確認できます。

メモ:

サポートリクエスト保存後の電子メールでもユーザーに通知されます。
