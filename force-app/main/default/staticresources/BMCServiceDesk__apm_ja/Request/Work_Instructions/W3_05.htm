作業指示について

手順

サービスデスクマネージャへの作業指示

3.5.1

必要に応じて、サービスデスクアナリスト、スタッフ、サービスレベルマネージャの手助けを得て、対象のユーザーがサービスプロバイダ組織の満足したユーザーになるようにします。

メモ:

苦情が［期限日］ フィールドに指定されている日時までに解決できないことが明らかな場合は、その苦情がいつまでに解決可能か、現実的な時期を見積もり、［メモ］アクションを追加してその情報を記入します。

メモ:

リクエストの［ステータス］フィールドを［待機中］ に、またはユーザー、スタッフからの情報待ちのため一時的に作業が継続できない場合は［Waiting for customer（顧客待ち）］に設定します。［メモ］アクションを追加して、待機中の内容を指定します。
