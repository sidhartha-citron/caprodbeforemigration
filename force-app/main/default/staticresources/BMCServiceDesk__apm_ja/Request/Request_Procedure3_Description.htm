手順3、苦情と称賛の処理


苦情または称賛が登録された後、サービスデスクマネージャはそれらの詳細を確認します。

苦情の場合、サービスデスクマネージャは、その苦情にかかわったサービスデスクアナリスト、スタッフ、サービスレベルマネージャ、顧客担当者、ユーザーと話し合い、苦情の原因を判断します。判断した後、サービスデスクマネージャは、サービスプロバイダ組織から提供されているサポートで、ユーザーの信頼を取り戻す方法について決定します。

称賛の場合、サービスデスクマネジャーは、従業員や部署に称賛を受けたことを通知します。

サービスデスクマネージャは、ユーザーの怒りを和らげるために必要なアクションをとるか、称賛を転送してから、リクエストを完了します。
