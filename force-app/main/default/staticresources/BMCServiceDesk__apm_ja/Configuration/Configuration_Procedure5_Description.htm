手順5、サポート契約の管理


作業依頼により、サポート契約の新規登録または既存更新が要求されている場合、構成管理者はサポート契約のサプライヤが既にサービス管理アプリケーションに登録されているか確認します。サポート契約のサプライヤ組織が既に登録されている場合、構成管理者は、登録されている契約の詳細が最新であるかを確認します。サポート契約のサプライヤ組織が登録されていない場合、または契約の詳細が最新でない場合、構成管理者は手順3、サプライヤ情報の保守に従い、サプライヤ情報が登録または更新されることを確認します。

サポート契約のサプライヤの連絡先詳細が登録済みであり、最新であることを確認した後、構成管理者は、［契約］フォームのフィールドの利用ガイドラインに従って、サポート契約を登録または更新します。

CMDB更新を完了した後、構成管理者は、行われた変更の概要により作業依頼を更新します。
