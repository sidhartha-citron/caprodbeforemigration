役割と責任


次の表に、構成管理プロセスに関する唯一の役割と、その責任を示します。プロファイルをレビューするには、役割をクリックします。

役割

責任

構成管理者

  • 購入申請を登録して、注文済みのCIを購入します。

  • 納品後は、構成管理者のグループの担当になったCIに、ラベルを貼付するようにします。

  • 構成管理者のグループが担当するCIの情報を、最新の状態に保ちます。

  • 構成管理者のグループが担当するCIを、提供またはサポートした社外組織の情報を保守します。

  • 構成管理者のグループが担当するCIのサポート契約を、登録及び更新します。

  • 構成管理者のグループが担当するCIのリンクを保守します。このリンクとは、CIと他のCI、CIのサプライヤ、CIのサポート契約およびCIの一部であるサービスインスタンスとのリンクです。
