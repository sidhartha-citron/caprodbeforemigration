サービスレベルアグリーメント（SLA）


次の表に、SLAフォームのフィールドと各フィールドの利用ガイドラインを示します。

フィールド

利用

アグリーメントタイトル

このフィールドには、サービスレベルアグリーメント（SLA）の名前を入力します。たとえば、UKPLANT向け人事管理P21（SAP）です。

説明

このフィールドには、SLAの簡単な説明を入力します。

一般情報

アグリーメントタイプ

このフィールドには、次のオプションから、アグリーメントタイプを選択します。

  • サービスレベルアグリーメント（SLA）

  • OLA

  • UC(請負契約)

サービス

このフィールドには、SLAを作成するサービスを選択します。

アカウント

このフィールドでは、SLAに含まれる組織を選択します。次のコストオプションを使用できます。

  • アグリーメントはすべてのアカウントに適用されます。

  • アグリーメントは選択したアカウントに適用されます。

［アグリーメントは選択したアカウントに適用されます。］を選択した場合、該当するアカントを選択します。

発効日

SLAの発効日を選択するには、このフィールドを使用します。

失効日

このフィールドでは、SLAが失効しないように更新する必要がある日を選択します。

予定されたレビュー

SLAを確認する必要がある日を選択するには、このフィールドを使用します。

サポート時間

このフィールドでは、SLAに該当するサポート時間を選択します。ここで選択されたサービス時間は、応答時間や解決時間の計算に使用されます。

アグリーメントオーナー

このフィールドでは、サービスレベルマネージャの役割を行うユーザーを選択します。

サービス目標

サービス目標のタイトル

このフィールドには、サービス目標のタイトルを入力します。

説明

このフィールドには、サービスの目標の簡単な説明を入力します。

サービス目標

適用先

このフィールドでは、目標を適用するモジュールを選択します。次のコストオプションを使用できます。

  • インシデント

  • 変更リクエスト

目標タイプ

このフィールドには、次のオプションから、目標タイプを選択します。

  • 解決時間

  • 応答時間

サポート時間

このフィールドでは、サービスの目標に該当するサポート時間を選択します。ここで選択されたサービス時間は、応答時間や解決時間の計算に使用されます。

目標

このフィールドには、日数、時間、分でサービスを完了する目標時間を入力します。または、［営業日］フィールドで、サービスを完了する目標時間を営業日で設定します。

認定条件

条件

このフィールドでは、条件を設定するフィールドと値を選択します。

計測

計測

このフィールドでは、SLAクロックを開始、一時停止、停止させるステータス値を選択します。

マイルストーンとアクション

マイルストーン

このフィールドでは、自動電子メール通知をいつ送信するかを選択します。マイルストーンを次のオプションから選択します。

  • 経過時間の比率

  • 残り時間の比率

  • サービス目標はキャンセルされました

  • サービス目標は達成されました

  • サービス目標は達成されませんでした

  • サービス目標の警告が出ています

  • 経過時間

  • 残り時間

アクション

このフィールドでは、マイルストーン達成時に送信する電子メールテンプレートを選択します。次のコストオプションを使用できます。

  • Incident Percentage of Time Elapsed（インシデントの経過時間の比率）

  • Incident Percentage of Time Remaining（インシデントの残り時間の比率）

  • Incident Service Target is cancelled（インシデントのサービス目標はキャンセルされました）

  • Incident Service Target is Met（インシデントのサービス目標は達成されました）

  • Incident Service Target is Missed（インシデントのサービス目標は達成されませんでした）

  • Incident Service Target Warning（インシデントのサービス目標の警告が出ています）

  • Incident Time Elapsed（インシデントの経過時間）

  • Incident Time remaining（インシデントの残り時間）

  • Service Request Percentage of time remaining（サービスリクエストの残り時間の比率）

  • Service Request Service Target is cancelled（サービスリクエストのサービス目標はキャンセルされました）

  • Incident Service Target is cancelled（インシデントのサービス目標はキャンセルされました）

  • Service Request Service target is met（サービスリクエストのサービス目標は達成されました）

  • Service Request Service Target is Missed（サービスリクエストのサービス目標は達成されませんでした）

  • Service Request Service Target Warning（サービスリクエストのサービス目標の警告が出ています）

  • Service Request Time Elapsed（サービスリクエストの経過時間）

  • Service Request Time remaining（サービスリクエストの残り時間）

変更先

このフィールドでは、電子メールメッセージを受信する必要がある電子メールアドレスを指定します。

CC

このフィールドでは、電子メールメッセージのCCリストに含める、電子メールアドレスを指定します。

関連リストまたは補足情報

履歴

このタブでは、アグリーメントに登録済みの履歴アクションをリストで示します。

インシデント

このセクションでは、SLAに関連するリクエストをリストで示します。このタブでは、リクエストをサービスにリンクします。

変更リクエスト

このセクションでは、SLAに関連する変更リクエストをリストで示します。このタブでは、変更リクエストをサービスにリンクします。

問題

このセクションでは、SLAに関連する問題をリストで示します。このタブでは、問題をSLAに添付します。

ドキュメンテーション

このセクションでは、SLAに関連するドキュメントをリストで示します。このタブには、SLAの完了のために追加情報を提供するファイルを添付します。

監査

このタブでは、SLA情報に対する変更をリストで示します。
