Instrucciones de trabajo para coordinadores de problemas

3.8.1

Determine el especialista apropiado para implementar la solución estructural propuesta. Puede ser el mismo especialista que analizó el problema. Use la opción Cambio junto al campo Propietario para seleccionar este especialista.

3.8.2

En el campo Duración (HH:MM)de la acción de Notas, introduzca el tiempo que empleó en la revisión y asignación del problema.

3.8.3

Establezca el campo Estatus en Asignado.
