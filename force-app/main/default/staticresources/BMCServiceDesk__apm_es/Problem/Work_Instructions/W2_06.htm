Instrucciones de trabajo para especialistas

2.6.1

Ahora que conoce la causa origen, encuentre modos de eliminarla o proporcione una solución permanente.

Nota:

Si el elemento de configuración (CI) que ha causado el problema está aún soportado por un proveedor externo (es decir, si el CI aún está cubierto por la garantía o un contrato de soporte), puede enviar al proveedor una petición de solución del problema.

Para averiguar si el CI aún está soportado por un proveedor externo, abra dicho CI y haga clic en la ficha Especificaciones para ver el campo Fecha de caducidad de la garantía.

2.6.2

Agregue una acción de Notas para describir cada solución estructural propuesta.

2.6.3

Determine cuál de las soluciones estructurales es la más práctica (basándose en consideraciones técnicas, financieras y de disponibilidad). Agregue una acción de Notas para explicar qué solución estructural encuentra más práctica.

2.6.4

Agregue una acción de Notas para describir en detalle el modo en el que debe implementarse la solución estructural preferida. Si se debe eliminar una solución provisional anterior, especifique el modo de hacerlo en el mismo campo.

Nota:

Agregue una acción de Notas con los motivos si no se encontró ninguna solución estructural.
