Instrucciones de trabajo para coordinadores de problemas

4.5.1

Agregue una acción de Notas para especificar por qué se solicita un nuevo análisis de problema.

4.5.2

Determine qué especialista es el más indicado (en cuanto a habilidades, disponibilidad y derechos de acceso) para realizar un análisis mejor. Puede ser el mismo que realizó el análisis anterior del problema. Use la opción Cambio junto al campo Propietario para seleccionar un especialista.

Nota:

Si el problema está vinculado a un CI que forma parte de una instancia de servicio que admite el servicio para el que actúa como coordinador de problemas, póngase en contacto con el coordinador de problemas del servicio complementario. Trate con él el problema y si el otro coordinador de problemas está de acuerdo con su razonamiento, asigne el problema al otro coordinador de problemas cambiando el nombre en el campo Propietario. En el campo Información, resuma por qué le pasó la responsabilidad del problema al otro coordinador de problemas.

4.5.3

En la acción de Notas, en el campo Duración (HH:MM), introduzca el tiempo que dedicó a identificar y registrar el problema.

4.5.4

Establezca el campo Estatus en Asignado
