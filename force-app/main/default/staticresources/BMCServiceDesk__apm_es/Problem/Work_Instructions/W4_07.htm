Instrucciones de trabajo para coordinadores de problemas

4.7.1

Realice comprobaciones periódicas para ver si hay nueva tecnología disponible o si se puede utilizar un nuevo enfoque para proporcionar un arreglo estructural para el problema.

Comience la búsqueda, por ejemplo, comunicándose con el proveedor del elemento de configuración (CI) en el que reside la causa origen del problema para averiguar si se ha podido obtener una solución estructural.
