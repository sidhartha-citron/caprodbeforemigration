Roles y responsabilidades


La siguiente tabla enumera los distintos roles involucrados en el proceso de gestión de problemas junto con sus respectivas responsabilidades. Haga clic en un rol para revisar el perfil.

Rol

Responsabilidad

Coordinador de problemas

  • Revisa las peticiones relacionadas con los servicios para los que actúa como coordinador de problemas a fin de identificar problemas.

  • Se asegura de que los problemas progresen en el proceso de gestión de problemas de forma adecuada y priorizada.

  • Se asegura de que la información introducida sobre los problemas sea precisa y esté completa.

  • Revisa periódicamente los problemas para los que no se ha podido encontrar una solución estructural práctica.

  • Verifica las soluciones estructurales y cierra problemas.

Especialista

  • Sugiere soluciones provisionales a los problemas.

  • Establece las causas origen de los problemas identificados.

  • Propone soluciones estructurales para los problemas.

  • Implementa soluciones estructurales para problemas si la gestión de cambios no es necesaria.

  • Actualiza problemas con información relevante y cambios de estatus.
