Paso del procedimiento

Instrucciones de trabajo para coordinadores de despliegue

1.3.1

Devuelva la petición de cambio al proceso de gestión de cambios.

1.3.2

Si la petición de cambio se recibe como una petición de soporte, agregue una acción de Notas que indique que el cambio puede implementarse sin el proceso de gestión de despliegues. En el campo Duración (HH:MM), en la acción de Notas, introduzca el tiempo que empleó para revisar la petición.

Haga clic en el vínculo Cambio junto al campo Propietario de la petición para seleccionar el coordinador de cambios que le asignó la petición de cambio. Establezca el campo Estatus de una petición de soporte como Asignado.

Nota:

No establezca el campo Estatus como Rechazado.

1.3.3

Si la petición de cambio se recibe en forma de un problema, agregue una acción de Notas que indique que el cambio puede implementarse sin el proceso de gestión de despliegues. En el campo Duración (HH:MM), en la acción de Notas, introduzca el tiempo que empleó para revisar el problema.

Haga clic en el vínculo Cambio junto al campo Propietario de la petición para seleccionar el coordinador de cambios que le asignó la petición de cambio. Establezca el campo Estatus del problema como Cambio solicitado.

Nota:

No establezca el campo Estatus como Rechazado.
