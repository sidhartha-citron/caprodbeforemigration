Paso del procedimiento

Instrucciones de trabajo para coordinadores de despliegue

3.4.1

Pida aprobación para el caso de negocio para garantizar que haya un presupuesto disponible para la implementación del nuevo despliegue.

3.4.2

Establezca el campo Estatus de la tarea de justificación comercial como Esperando… y agregue una acción de Notas que indique que está esperando la aprobación del caso de negocio.

3.4.3

En el campo Duración (HH:MM), en la acción de Notas, introduzca el tiempo que empleó para preparar el caso de negocio para su aprobación.
