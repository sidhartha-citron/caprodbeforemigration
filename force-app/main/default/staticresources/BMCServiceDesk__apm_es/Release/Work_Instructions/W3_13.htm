Paso del procedimiento

Instrucciones de trabajo para coordinadores de despliegue

3.13.1

Abra el cambio al que se vincularon las peticiones de cambio para el despliegue. Desde este cambio, abra la tarea para la preparación de documentos de requisitos para los coordinadores de cambios y la tarea para la reunión de postimplementación. La plantilla de cambios generó estas tareas cuando se registró el cambio.

3.13.2

Establezca el campo Estatus de estas tareas como Cancelado. En el campo Resolución de ambas tareas, especifique que se ha rechazado el caso de negocio para el despliegue.

3.13.3

En el campo Categoría de cierre del cambio al que están vinculadas estas tareas, seleccione Rejected by Approvers (Rechazado por aprobadores).

3.13.4

Establezca el campo Estatus del cambio como Terminado.

3.13.5

Agregue una acción de Notas que indique que el despliegue no pudo implementarse porque se rechazó su caso de negocio. Incluya también la razón por la que se rechazó.

3.13.6

Seleccione Caso de negocio rechazado en el campo Categoría de cierre del despliegue y establezca su campo Estatus como Cerrado.
