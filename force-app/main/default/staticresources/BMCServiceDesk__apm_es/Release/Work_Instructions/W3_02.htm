Paso del procedimiento

Instrucciones de trabajo para coordinadores de despliegue

3.2.1

Abra el cambio al que están vinculadas las peticiones de cambio para el despliegue. Desde la sección Tareas vinculadas de este cambio, abra la tarea para justificación comercial generada por la plantilla de cambios cuando se registró el cambio.

3.2.2

Establezca el campo Estatus de la tarea como Cancelado y, en el campo Resolución, especifique que el despliegue se puede implementar mediante los presupuestos ya disponibles para el propietario del servicio.
