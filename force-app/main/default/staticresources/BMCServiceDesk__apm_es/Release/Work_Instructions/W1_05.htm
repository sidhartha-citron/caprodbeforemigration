Paso del procedimiento

Instrucciones de trabajo para coordinadores de despliegue

1.5.1

Estime el esfuerzo de empleados internos y contratistas a largo plazo que se necesitarían para que se implemente la petición de cambio.

1.5.2

Estime el coste de los elementos de configuración, servicios y soporte que deben adquirirse a proveedores para poder implementar el cambio.

Nota:

Consulte con coordinadores de cambio y especialistas para estimar correctamente el cambio.

1.5.3

Si la petición de cambio se recibe como una petición, agregue una acción de Notas para introducir las aproximaciones e indicar que la gestión de despliegues ha aceptado la responsabilidad de la coordinación. Introduzca también las suposiciones en las que se basan las aproximaciones. Establezca el campo Estatus de la petición como Cambio pendiente.

1.5.4

Si la petición de cambio se recibe como un problema, agregue una acción de Notas para introducir las aproximaciones e indicar que la gestión de despliegues ha aceptado la responsabilidad de la coordinación. Introduzca también las suposiciones en las que se basan las aproximaciones. Establezca el campo Estatus del problema como Cambio pendiente.
