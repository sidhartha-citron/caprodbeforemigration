Instrucciones de trabajo para coordinadores de despliegue

1.2.1

Si no se necesita gestión de despliegues para una petición de cambios, continúe con el paso 1.3.1. De lo contrario, vaya al paso 1.4.1.

Nota:

Las peticiones de cambio que requieren la implementación de un cambio no estándar se deben procesar con gestión de despliegues. Esto excluye peticiones de cambios enviadas para impedir o corregir un problema, siempre que su implementación:

  • la pueda coordinar un solo coordinador de cambios

 

  • no cambie la funcionalidad de un servicio

 

  • no requiera financiación adicional.
