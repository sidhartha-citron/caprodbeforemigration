Part name (Nombre de pieza)

Use este campo para introducir el nombre de la pieza.

Brand (Marca)

Use este campo para introducir el número asignado a la pieza por el fabricante.

Subcategoría

Parameters (Y/N) (Parámetros (S/N))

Track Asset/Tag # (N.º de activo/etiqueta de seguimiento)

Marque este campo para hacer un seguimiento del número de activo o de etiqueta de los CI registrados en función de la pieza del catálogo de CI.

Track Serial # (N.º de serie de seguimiento)

Marque este campo para hacer un seguimiento del número de serie de los CI registrados en función de la pieza del catálogo de CI.

Track Version # (N.º de versión de seguimiento)

Marque este campo para hacer un seguimiento del número de versión de los CI registrados en función de la pieza del catálogo de CI.

Cantidad

Total

Este campo muestra automáticamente la cantidad de CI únicos de la pieza del catálogo de CI que están registrados en la CMDB.
