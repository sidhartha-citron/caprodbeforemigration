Despliegue


En la siguiente tabla encontrará los campos del formulario Despliegue y las pautas de uso de cada campo.

Campo

Uso

N.º de versión

Este campo contiene un número único de despliegue. La aplicación genera este número automáticamente.

Detalles de despliegue

Plantilla

Use este campo para seleccionar la plantilla usada para el despliegue.

Título de despliegue

Use este campo para introducir el título de despliegue.

Descripción

Use este campo para introducir una breve descripción que resuma el motivo principal para implementar el despliegue. Puede incluir el motivo de la creación del despliegue,como Problem Prevention or Fix (Prevención o corrección de problema), New Business Requirement (Nuevo requisito comercial) o Nuevo requisito legal.

Categoría

Use este campo para seleccionar la categoría del despliegue.

Tipo de despliegue

Use este campo para seleccionar en la siguiente lista de opciones el tipo de despliegue:

  • Principal

  • Secundario

  • Emergencia

Subtipo

Use este campo para seleccionar el subtipo en la siguiente lista de opciones:

  • Atrasos

  • Delta

  • Completo

  • Paquete

Tipo de aplicación

Use este campo para seleccionar en la siguiente lista de opciones el tipo de aplicación:

  • En fase

  • No en fase

Prueba finalizada

Use este campo para establecer la fecha y la hora en que se finalizó la prueba.

Estatus y prioridad

Impacto

Use este campo para seleccionar el impacto en la siguiente lista de opciones:

  • Bajo: servicio degradado para 1 usuario

  • Medio: fuera de servicio para 1 usuario o degradado para varios usuarios

  • Alto: fuera de servicio para varios usuarios

Urgencia

Use este campo para seleccionar el grado de urgencia entre las siguientes opciones:

  • Baja: responder utilizando procedimientos de funcionamiento estándar y según el tiempo lo permita.

  • Media: responder utilizando procedimientos de funcionamiento estándar según estructuras habituales de gestión y supervisión.

  • Alta: respuesta inmediata obligatoria

Prioridad

La aplicación selecciona la prioridad una vez seleccionados los niveles de impacto y de urgencia.

Estatus

Use este campo para seleccionar en la siguiente lista de opciones el estatus adecuado para el despliegue:

Abierto

El estatus inicial del despliegue.

Registrado

El despliegue ha sido registrado por su coordinador.

To Be Approved (Para aprobar)

Se está preparando o aprobando un caso de negocio para el despliegue.

Aprobado

Se ha aprobado el caso de negocio del despliegue.

En curso

Se han enviado los requisitos de la implementación del despliegue a los coordinadores de cambios.

Plan

El despliegue está en fase de planificación.

Aplicación

El despliegue está en fase de aplicación.

Terminado

Las labores de implementación del despliegue han finalizado por el motivo especificado en el campo Categoría de cierre.

Cerrado

Se ha revisado la implementación del despliegue durante la reunión de postimplementación.

Categoría de cierre

Use este campo para seleccionar la categoría de cierre adecuada para el despliegue una vez que se haya terminado. Encontrará las siguientes opciones:

  • Caso de negocio rechazado

  • Rechazado

  • Revertido

  • Implantado parcialmente

  • Implantado

Inactivo

Seleccione este campo si el registro de despliegue ya no es válido, pero debe retenerse para respetar la directiva de retención de registros. Por ejemplo, si un cliente ha abandonado la cuenta, todos los registros de despliegues del cliente dejarían de ser válidos.

Detalles de fecha y hora de despliegue

Fecha de vencimiento

Use este campo para establecer la fecha y la hora en que deben completarse los cambios vinculados al despliegue, excepción hecha del cambio al que están vinculadas las peticiones de cambio del despliegue.

Iniciar implementación

Use este campo para establecer la fecha y la hora en que debe comenzar la implementación del despliegue.

Fin de implementación

Use este campo para establecer la fecha y la hora en que debe completarse la implementación del despliegue.

Iniciar aplicación

Use este campo para establecer la fecha y la hora en que debe comenzar la aplicación del despliegue.

Fin de aplicación

Use este campo para establecer la fecha y la hora en que debe completarse la aplicación del despliegue.

Fecha de finalización

Use este campo para establecer la fecha y la hora en que debe completar el despliegue.

Inicio real de implementación

Use este campo para establecer la fecha y la hora en que comenzó realmente la implementación del despliegue.

Finalización real de implementación

Use este campo para establecer la fecha y la hora en que finalizó realmente la implementación del despliegue.

Inicio real de aplicación

Use este campo para establecer la fecha y la hora en que comenzó realmente la aplicación del despliegue.

Finalización real de aplicación

Use este campo para establecer la fecha y la hora en que se completó realmente la aplicación del despliegue.

Detalles empresariales y financieros

Presupuesto disponible

Use este campo para especificar si hay un presupuesto disponible para el despliegue.

Justificación comercial

Use este campo para seleccionar la justificación comercial adecuada para el despliegue. Encontrará las siguientes opciones:

  • Estrategia corporativa

  • Estrategia de unidad de negocios

  • Mantenimiento

  • Defecto

  • Actualización

  • Mejora

  • Compromiso del cliente

  • Sarbanes-Oxley

  • Otro

Comentarios

Use este campo para introducir los comentarios comerciales y financieros.

Evaluación de riesgos

Use este campo para seleccionar en la siguiente lista de opciones el nivel de riesgo adecuado del despliegue:

  • Bajo

  • Medio

  • Alto

Contratos requeridos verificados

Use este campo para indicar si los contratos requeridos han sido verificados.

Licencias requeridas verificadas

Use este campo para especificar si las licencias requeridas han sido verificadas.

CI requeridos disponibles

Use este campo para especificar si los elementos de configuración (CI) requeridos están disponibles.

Detalles de asignación de personal

Propietario

Este campo muestra la cola o el personal al que se asigna el despliegue.

Coordinador de despliegue

Use este campo para seleccionarse a usted mismo como coordinador de despliegue responsable del despliegue.

Listas relacionadas

Historial de acciones

En esta sección encontrará el historial de acciones registradas para el despliegue.

Tareas vinculadas

En esta sección encontrará las tareas relacionadas con el despliegue. Úsela para vincular tareas al despliegue.

Problemas vinculados

En esta sección encontrará los problemas relacionados con el despliegue. Úsela para vincular problemas al despliegue.

Incidencias vinculadas

En esta sección encontrará las incidencias relacionadas con el despliegue. Úsela para vincular peticiones al despliegue.

Historial de aprobaciones

En esta sección encontrará los registros de aprobaciones creados para el despliegue.

Peticiones de cambio vinculadas

En esta sección encontrará las peticiones de cambio relacionadas con el despliegue. Úsela para vincular peticiones de cambio al despliegue.

Servicios y CI vinculados

En esta sección encontrará los servicios y CI relacionados con el despliegue. Úsela para vincular los CI y servicios que se verán afectados por la ejecución del despliegue.

Notas y adjuntos

En esta sección encontrará los documentos relacionados con el despliegue. Úsela para adjuntar archivos que contengan información adicional sobre la finalización del despliegue.

Historial de despliegue

En esta sección encontrará las modificaciones que haya sufrido la información del despliegue.
