Artículo de conocimientos


En la siguiente tabla encontrará los campos del formulario Artículo de conocimientos y las pautas de uso de cada campo.

Campo

Uso

Tipo de registro del nuevo registro

Use este campo para seleccionar el tipo de registro en la siguiente lista de opciones:

  • Preguntas más frecuentes: artículos que contienen respuestas a las preguntas más frecuentes.

  • Cómo: artículos que describen los pasos que hay que seguir para llevar a cabo una tarea.

  • Error conocido: artículos que contienen información sobre errores conocidos y soluciones provisionales, en su caso.

  • Solución de problema : artículos que describen soluciones de problemas.

Información de artículo

Propietario

Este campo se establece en el nombre de la persona que creó el artículo de conocimientos.

Categoría

Use este campo para seleccionar la categoría del artículo de conocimientos.

Estatus

Use este campo para seleccionar el estatus en la siguiente lista de opciones:

  • Borrador

  • En revisión

  • Publicado

  • Retirado

Mostrar en Autoservicio

Use este campo para especificar si el artículo de conocimientos estará disponible en Autoservicio.

Detalles de artículo

Título

Use este campo para escribir un título breve para el artículo de conocimientos.

Palabras clave

Use este campo para introducir las palabras clave de este artículo. Al realizar una búsqueda, la cadena de búsqueda se compara con las palabras clave para mostrar los artículos de conocimientos que coinciden.

Tipo de registro: Pregunta frecuente y Cómo

Pregunta

Use este campo para introducir la pregunta del artículo Pregunta frecuente o Cómo.

Respuesta

Use este campo para introducir la respuesta del artículo Pregunta frecuente o Cómo.

Tipo de registro: Error conocido

Error

Use este campo para introducir la descripción del error conocido.

Causa origen

Use este campo para introducir la causa origen del error conocido.

Solución provisional

Use este campo para introducir la solución provisional del error conocido, si la hubiera.

Tipo de registro: Solución de problema

Problema

Use este campo para introducir una descripción del problema.

Solución

Use este campo para introducir la solución del problema.

Información adicional

Campo

Uso

Fecha de publicación

Este campo se establece en la fecha y hora en que se ha publicado el artículo.

Fecha de revisión

Este campo se establece en la fecha y hora en que se debe revisar el artículo.

Fecha de retiración

Este campo se establece en la fecha y hora en que el artículo se marcará como retirado.

Viewed in Self Service # (Visto en Autoservicio n.º)

En este campo aparece la cantidad de veces que se accede al artículo en Autoservicio.

Listas relacionadas

Notas y adjuntos

En esta sección encontrará los documentos relacionados con el artículo de conocimientos. Use esta sección para adjuntar archivos o notas que puedan contener información adicional relacionada con el artículo de conocimientos.

Valoración de conocimientos

En esta sección encontrará la valoración del artículo de conocimientos.

Incidencias vinculadas

En esta sección encontrará las incidencias relacionadas con el artículo de conocimientos. Úsela para vincular incidencias al artículo de conocimientos.

Historial de aprobaciones

En esta sección encontrará los registros de aprobaciones del artículo de conocimientos.
