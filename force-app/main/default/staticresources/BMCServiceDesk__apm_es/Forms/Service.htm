Listas relacionadas o información adicional

Historial

Este campo incluye las notas del historial creadas para el servicio.

Clientes

En esta ficha se enumeran los clientes relacionados con el servicio. Use esta ficha para seleccionar el cliente que usará el servicio o lo tiene reservado. Deje este campo vacío si ningún cliente específico usa o tiene reservado el servicio.

Cuentas

En esta ficha se enumeran las cuentas relacionadas con el servicio. Úsela para seleccionar las organizaciones que usarán el servicio o lo tienen reservado. Deje este campo vacío si ninguna organización específica usa o tiene reservado el servicio.

Incidencias

En esta sección encontrará las incidencias relacionadas con el servicio. Úsela para vincular incidencias al servicio.

Peticiones de cambio

En esta sección encontrará las peticiones de cambio relacionadas con el servicio. Úsela para vincular peticiones de cambio al servicio.

Problemas

En esta sección encontrará los problemas relacionados con el servicio. Úsela para vincular problemas al servicio.

Despliegues

En esta sección encontrará los despliegues relacionados con el servicio. Úsela para vincular despliegues al servicio.

Documentación

En esta sección encontrará los documentos relacionados con el servicio. Úsela para adjuntar archivos que proporcionen información adicional sobre el servicio.

Auditoría

En esta ficha se enumeran las modificaciones que haya sufrido la información del servicio.
