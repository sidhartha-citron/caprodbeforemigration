Acuerdo de nivel de servicio


En la siguiente tabla encontrará los campos del formulario de Acuerdo de nivel de servicio y las pautas de uso de cada campo.

Campo

Uso

Título del acuerdo

Use este campo para introducir el nombre del Acuerdo de nivel de servicio (SLA). Por ejemplo, Gestión de recursos humanos P21 (SAP) para UKPLANT.

Descripción

Use este campo para introducir la descripción del SLA.

Información general

Tipo de acuerdo

Use este campo para seleccionar en la siguiente lista de opciones el tipo de acuerdo:

  • Acuerdo de nivel de servicio

  • Acuerdo de nivel operacional

  • Contrato de mantenimiento

Servicio

Use este campo para seleccionar el servicio al que corresponde el SLA.

Cuenta

Use este campo para seleccionar las organizaciones cubiertas por el SLA. Encontrará las siguientes opciones:

  • El acuerdo se aplica a todas las cuentas

  • El acuerdo se aplica a cuentas seleccionadas

Si elige El acuerdo se aplica a cuentas seleccionadas, seleccione las cuentas adecuadas.

Vigente desde

Use este campo para especificar la fecha de entrada en vigor del SLA.

Vigente hasta

Use este campo para especificar antes de qué fecha se debe renovar el SLA para que no caduque.

Revisión programada

Use este campo para especificar la fecha en que será necesario revisar el SLA.

Horas de soporte

Use este campo para seleccionar las horas de soporte adecuadas para el SLA. Las horas de servicio que seleccione aquí servirán para calcular el tiempo de respuesta y el tiempo de resolución.

Propietario del acuerdo

Use este campo para seleccionar el usuario que actúa como gestor de nivel de servicio.

Objetivos de servicio

Título de Objetivo de servicio

Use este campo para introducir el título de objetivo de servicio.

Descripción

Use este campo para introducir la descripción del objetivo de servicio.

Objetivo de servicio

Se aplica a

Use este campo para seleccionar el módulo al que se aplica el objetivo. Encontrará las siguientes opciones:

  • Incidencia

  • Petición de cambio

Tipo de objetivo

Use este campo para seleccionar el tipo de objetivo entre la siguiente lista de opciones:

  • Tiempo de resolución

  • Tiempo de respuesta

Horas de soporte

Use este campo para seleccionar las horas de soporte adecuadas para el objetivo de servicio. Las horas de servicio que seleccione aquí servirán para calcular el tiempo de respuesta y el tiempo de resolución.

Objetivo

Use este campo para introducir el tiempo deseado para completar el servicio en días, horas y minutos. O bien, use el campo Días laborables para ajustar el tiempo deseado para completar el servicio en días laborables.

Calificación

Condiciones

Use este campo para establecer los campos y valores en función de los que desee establecer el criterio.

Medición

Medición

Use este campo para establecer los valores de estatus ante los que el reloj del SLA debe iniciarse, pausarse y detenerse.

Hitos y acciones

Hito

Use este campo para establecer cuándo se debe enviar una notificación automática por correo electrónico. Seleccione un hito en la siguiente lista de opciones:

  • Porcentaje de tiempo transcurrido

  • Porcentaje de tiempo restante

  • El objetivo de servicio se ha cancelado

  • El objetivo de servicio se ha alcanzado

  • No se ha alcanzado el objetivo de servicio

  • Hay una advertencia de objetivo de servicio

  • Tiempo transcurrido

  • Tiempo restante

Acción

Use este campo para seleccionar la plantilla de correo electrónico que desea enviar cuando se cumpla el hito. Encontrará las siguientes opciones:

  • Porcentaje de tiempo transcurrido de incidencia

  • Porcentaje de tiempo restante de incidencia

  • El objetivo de servicio de incidencia está cancelado

  • El objetivo de servicio de incidencia se ha alcanzado

  • No se ha alcanzado el objetivo de servicio de incidencia

  • Advertencia de objetivo de servicio de incidencia

  • Tiempo transcurrido de incidencia

  • Tiempo restante de incidencia

  • Porcentaje de tiempo restante de petición de servicio

  • El objetivo de servicio de petición de servicio está cancelado

  • El objetivo de servicio de incidencia está cancelado

  • El objetivo de servicio de petición de servicio se ha alcanzado

  • No se ha alcanzado el objetivo de servicio de petición de servicio

  • Advertencia de objetivo de servicio de petición de servicio

  • Tiempo transcurrido de petición de servicio

  • Tiempo restante de petición de servicio

A

Use este campo para especificar a qué direcciones de correo electrónico se debe enviar el mensaje.

CC

Use este campo para especificar qué direcciones de correo electrónico deben aparecer en la línea CC del mensaje.

Listas relacionadas o información adicional

Historial

En esta ficha encontrará el historial de acciones registradas para el acuerdo.

Incidencias

En esta sección encontrará las peticiones relacionadas con el SLA. Use esta ficha para vincular peticiones al servicio.

Peticiones de cambio

En esta sección encontrará las peticiones de cambio relacionadas con el SLA. Úsela para vincular peticiones de cambio al servicio.

Problemas

En esta sección encontrará los problemas relacionados con el SLA. Úsela para adjuntar problemas al SLA.

Documentación

En esta sección encontrará los documentos relacionados con el SLA. Use esta ficha para adjuntar cualquier archivo con información adicional sobre la finalización del SLA.

Auditoría

En esta ficha se enumeran las modificaciones que haya sufrido la información del SLA.
