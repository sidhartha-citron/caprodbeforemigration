Procedimiento 4, Actualización de CI


Si la tarea de gestión de cambios o de gestión de peticiones no requiere la actualización de la información de los elementos de configuración (CI) existentes, el administrador de configuración completará la tarea. Sin embargo, si el administrador de configuración se encarga de una tarea que requiere la actualización de los atributos y las relaciones de CI , este comprobará la configuración de la infraestructura para confirmar que las modificaciones solicitadas realmente son necesarias para que la CMDB esté actualizada.

Si hay que actualizar la CMDB, el administrador de configuración realizará la actualización de los atributos y las relaciones de CI necesarios.
