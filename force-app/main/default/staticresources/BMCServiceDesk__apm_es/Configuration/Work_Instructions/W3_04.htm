Instrucciones de trabajo para administradores de configuración

3.4.1

Si se ha registrado o actualizado la información de un proveedor de contrato de soporte, vaya a gestión de configuración en el paso 5.4.1. Si se ha registrado o actualizado la información de un proveedor de un elemento de configuración (CI), vaya a gestión de configuración en el paso 2.8.1.
