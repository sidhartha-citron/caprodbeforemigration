Instrucciones de trabajo para administradores de configuración

1.4.1

Después de enviar la petición de compra, ayude con la preparación de una petición de propuesta y la posterior revisión de las propuestas que han enviado los proveedores.

1.4.2

Una vez que el departamento de compras haya procesado el requerimiento de compra, compruebe el progreso para averiguar si el pedido del elemento de configuración (CI) solicitado ha sido enviado al proveedor. Si el pedido no se ha enviado, determine si el motivo de no haberlo hecho es que se ha rechazado el requerimiento de compra o si el departamento de compras todavía lo está procesando.
