Procedimiento 3, Mantenimiento de la información de proveedor


Los administradores de configuración son responsables del registro y la actualización de la información de las organizaciones proveedoras de elementos de configuración (CI) y soporte a la organización proveedora de servicios.

Un administrador de configuración se encarga de las tareas de mantenimiento de la información de proveedor, según proceda, antes de registrar nuevos CI, antes de registrar o actualizar contratos de soporte y siempre que se reciba información de contacto actualizada de un proveedor existente.

Si una organización proveedora no está registrada, el administrador de configuración se encargará de hacerlo. Si dicha organización ya existe en la aplicación de gestión de servicios, el administrador de configuración actualiza los detalles de contacto.
