Procedimiento 4, Implementación de cambios de infraestructura


Las primeras tareas de implementación se asignan cuando se aprueba el cambio. Si se trata de un cambio de aplicación, la implementación continúa en el procedimiento 5, Implementación de cambios de aplicación.

Si se trata de un cambio de infraestructura, los especialistas preparan la implementación para garantizar la disponibilidad operativa. Esta preparación puede incluir la petición de hardware, la configuración de un entorno de prueba y la realización de pruebas.

Los especialistas implementan el cambio de conformidad con el plan de cambio cuando se completa la preparación.
