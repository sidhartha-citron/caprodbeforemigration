Instrucciones de trabajo para coordinadores de cambios

4.1.1

Cuando el gestor de cambios haya establecido el campo Estatus del cambio como Aprobado, asigne las tareas de implementación. Cambie el campo Estatus de cada tarea de Registrado a Asignado.

Nota:

Asigne solo las tareas para las que no se debe esperar que se completen otras tareas de implementación.

Nota:

Si el orden de ejecución de tareas se ha activado dentro de las plantillas Cambio y Tarea, la siguiente tarea solo se mostrará una vez que se haya terminado la tarea anterior.

4.1.2

Establezca el campo Estatus de la petición de cambio como En curso.
