Procedimiento 4, Finalización del servicio


Si la tarea para gestión de nivel de servicio pide que se finalice un elemento de catálogo, el administrador de nivel de servicio se asegura de que los clientes ya no puedan acceder desde el catálogo de servicios basado en la Web de la organización proveedora de servicios.

Si la tarea solicita que se terminen uno o más SLA, el administrador de nivel de servicio actualiza el estatus de los SLA para indicar que se han finalizado. Además, el administrador de nivel de servicio comprobará el estatus de los otros SLA vinculados a los servicios cuyos SLA ha finalizado el administrador. Si todos los SLA vinculados a estos servicios están finalizados, el administrador de nivel de servicio también finalizará estos servicios actualizando su estatus.

Cuando no se necesitan más acciones de parte del administrador de nivel de servicio para actualizar la información de gestión de nivel de servicio, el administrador actualiza la tarea para informar al coordinador de cambios del éxito o fracaso (por ejemplo, no se aprobó el elemento de catálogo pedido o no se firmó el SLA solicitado).
