Instrucciones de trabajo para administradores de nivel de servicio

1.7.1

Establezca el campo Estatus de la tarea como En curso.

1.7.2

Agregue una acción de Notas para especificar por qué se rechazó el elemento de catálogo y qué debe cambiarse para que se apruebe.

1.7.3

Abra la copia digital del elemento de catálogo rechazado.

1.7.4

Ajuste el elemento de catálogo rechazado para garantizar que se obtengan todas la firmas de aprobación.
