Instrucciones de trabajo para administradores de nivel de servicio

2.5.1

Envíe las dos copias impresas del nuevo SLA al cliente que haya seleccionado el elemento de catálogo creado para el nuevo SLA. Pida al representante del cliente que firme ambas copias y se las vuelva a remitir.

2.5.2

Establezca el campo Estatus de la tarea como Esperando… y agregue una acción de Notas para especificar que está esperando a que el representante del cliente firme el SLA.

2.5.3

Cuando reciba las dos copias firmadas del SLA nuevo, déselas al propietario del servicio. Pídale al propietario del servicio que firme ambas copias y se las devuelva.

2.5.4

Agregue una acción de notas para especificar que está esperando a que el propietario del servicio firme el SLA. Mantenga el campo Estatus como Esperando…

2.5.5

Establezca el campo Estatus de nuevo como En curso cuando le hayan devuelto las dos copias del SLA con las firmas del representante del cliente y el propietario del servicio.
