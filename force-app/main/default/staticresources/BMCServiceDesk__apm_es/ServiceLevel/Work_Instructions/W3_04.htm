Paso del procedimiento

Instrucciones de trabajo para administradores de nivel de servicio

3.4.1

En el formulario de cliente, actualice los detalles de contacto después de recibir la información actualizada de un cliente individual existente.

3.4.2

Para actualizar los detalles de contacto de una organización de cliente existente, use el formulario Cuenta. Use el formulario de cliente para actualizar los detalles de contacto de un representante del cliente, y el formulario de cuenta para registrar o actualizar la información de contacto de otras personas a las que presta asistencia la organización proveedora de servicios.
