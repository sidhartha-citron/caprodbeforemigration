Perfil de administrador de nivel de servicio

Un administrador de nivel de servicio debe poseer las siguientes características:

  • Conocimiento básico de todos los servicios proporcionados por la organización proveedora de servicios.

  • Perfeccionista

  • Nociones básicas de los procesos de gestión de servicios

  • Amplios conocimientos sobre los procesos de gestión de nivel de servicio

  • Capacidad para usar la aplicación de gestión de servicios para desempeñar el rol de administrador de nivel de servicio

Recomendación:

Especialista del grupo de gestión de servicios con dedicación plena
