Perfil de analista de servicio de asistencia

Un analista de servicio de asistencia debe poseer las siguientes características:

  • Conocimiento medio sobre todos los servicios prestados por la organización proveedora de servicios para la que el servicio de asistencia constituye el punto de contacto único

  • Capacidad para transformar el lenguaje utilizado por los usuarios en información técnica para especialistas y viceversa

  • Excelentes habilidades de comunicación escrita y por teléfono

  • Actitud orientada al servicio

  • Nociones básicas de los procesos de gestión de servicios

  • Amplio conocimiento sobre los procedimientos de gestión de peticiones

  • Capacidad para usar la aplicación de gestión de servicios para desempeñar el rol de analista de servicio de asistencia

Recomendación:

Empleados con dedicación plena, no especialistas de otros grupos que lleven a cabo actividades en el servicio de asistencia de forma temporal
