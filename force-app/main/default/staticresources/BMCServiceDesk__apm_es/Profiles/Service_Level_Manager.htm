Perfil de gestor de nivel de servicio

Un gestor de nivel de servicio debe poseer las siguientes características:

  • Capacidad para establecer relaciones con los representantes de los clientes

  • Capacidad para comprender en cierta medida las implicaciones de las peticiones para cambiar el nivel del servicio o la funcionalidad que proporciona un servicio

  • Capacidad para trasladar los requisitos de negocio proporcionados por los representantes de clientes en SLR para los propietarios de servicio

  • Capacidad para negociar acuerdos de nivel de servicio

  • Capacidad para producir informes sobre el nivel actual de servicio prestado

  • Excelentes habilidades de comunicación escrita y por teléfono

  • Actitud orientada al servicio

  • Nociones básicas de los procesos de gestión de servicios

  • Amplios conocimientos sobre los procesos de gestión de nivel de servicio

  • Capacidad para usar la aplicación de gestión de servicios para desempeñar el rol de gestor de nivel de servicio

Recomendación:

Empleado con experiencia con clientes y como especialista
