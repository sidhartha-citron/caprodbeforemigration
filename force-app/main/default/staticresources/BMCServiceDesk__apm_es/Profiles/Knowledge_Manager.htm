Perfil del gestor de conocimientos

Un gestor de conocimientos debe tener las siguientes características:

  • Saber qué artículos deben formar parte de la base de conocimientos.

  • Nociones básicas de los procesos de gestión de servicios

  • Conocimientos pormenorizados de las instrucciones de trabajo de los gestores de conocimientos

  • Capacidad para utilizar la aplicación de gestión de servicios a fin de desempeñar las funciones de gestor de conocimientos.

Recomendación:

Un analista del servicio de asistencia sénior, un coordinador de problemas, un coordinador de cambios
