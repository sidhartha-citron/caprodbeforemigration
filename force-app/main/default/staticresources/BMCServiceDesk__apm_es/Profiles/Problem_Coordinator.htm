Perfil del coordinador de problemas

Un coordinador de problemas debe tener las siguientes características:

  • Capacidad para mejorar continuamente los indicadores del rendimiento clave del proceso de gestión de problemas.

  • Capacidad para tratar prioridades que entran en conflicto dentro del grupo entre el análisis de problemas y otras tareas

  • Conocimientos pormenorizados acerca de todas las instancias de servicio utilizadas para ofrecer los servicios de los que el coordinador de problemas es responsable

  • Afán incansable por la búsqueda de mejoras estructurales en lugar de correcciones rápidas

  • Nociones básicas de los procesos de gestión de servicios

  • Conocimientos pormenorizados de los procedimientos de gestión de problemas

  • Capacidad para utilizar la aplicación de gestión de servicios a fin de desempeñar las funciones de coordinador de problemas

Recomendación:

un especialista sénior por cada servicio o por cada grupo

Idealmente, este cargo lo ostentan personas que también realizan funciones de gestor de disponibilidad, gestor de capacidad y organizador de continuidad.
