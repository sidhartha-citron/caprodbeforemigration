Perfil del coordinador de cambios

Un coordinador de cambios debe tener las siguientes características:

  • Experiencia en la coordinación de varios cambios o proyectos simultáneamente

  • Nociones básicas de los procesos de gestión de servicios

  • Conocimientos pormenorizados de las instrucciones de trabajo de gestión de cambios.

  • Capacidad para utilizar la aplicación de gestión de servicios a fin de desempeñar las funciones de coordinador de cambios.

Recomendación:

un especialista sénior por cada servicio o por cada grupo
