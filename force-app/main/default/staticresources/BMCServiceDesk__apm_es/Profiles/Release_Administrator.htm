Perfil de administrador de despliegues

Un administrador de despliegues debe poseer las siguientes características:

  • Capacidad para utilizar las herramientas necesarias para transferir los despliegues a los entornos de prueba y de producción

  • Nociones básicas de los procesos de gestión de servicios

  • Amplios conocimientos sobre las instrucciones de trabajo de gestión de cambios para los administradores de despliegues

  • Capacidad para usar la aplicación de gestión de servicios para desempeñar el rol de especialista

Recomendación:

Un especialista con experiencia en desarrollo
