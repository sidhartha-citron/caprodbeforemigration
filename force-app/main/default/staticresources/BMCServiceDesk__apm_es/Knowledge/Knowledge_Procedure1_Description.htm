Procedimiento 1, Creación de conocimiento


Si se crea un nuevo artículo de conocimiento desde el proceso de gestión de peticiones, el especialista creará un nuevo artículo con el tipo de registro Preguntas frecuentes o Instrucciones. Una vez introducida toda la información necesaria, el especialista guardará el artículo y lo enviará para su aprobación.

Si se crea un nuevo artículo de conocimiento desde el proceso de gestión de problemas, el especialista creará un nuevo artículo con el tipo de registro Error o Solución de problema. Una vez introducida toda la información necesaria, el especialista guardará el artículo y lo enviará para su aprobación.

Si se crea un nuevo artículo de conocimiento desde el proceso de gestión de cambios, el especialista creará un nuevo artículo. Una vez introducida toda la información necesaria, el especialista guardará el artículo y lo enviará para su aprobación.
