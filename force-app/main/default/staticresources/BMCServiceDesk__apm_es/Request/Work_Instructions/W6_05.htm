Instrucciones de trabajo para analistas del servicio de asistencia / especialistas

6.5.1

En cada tarea, determine quién es el miembro del personal más adecuado del grupo (en cuanto a habilidades, disponibilidad y derechos de acceso) para ejecutarla. Use el icono de búsqueda junto al campo Propietario para seleccionar el miembro del personal o el grupo correcto en el campo Propietario de la tarea.

Si puede completar una tarea, use el icono de búsqueda junto al campo Propietario para seleccionarse a usted mismo en el campo Propietario de la tarea.

Si otro grupo debe completar una tarea, use el icono de búsqueda junto al campo Propietario para seleccionar el grupo en el campo Propietario.

Si una tarea de implementación debe ser completada por una organización externa, analice la tarea con su persona de contacto en la organización. Trabaje en conjunto con esta persona para determinar un objetivo de implementación realista. Use el icono de búsqueda junto al campo Propietario para seleccionarse a usted mismo en el campo Propietario de la tarea. Agregue una acción de Notas para especificar cuál es la organización externa a la que se le pidió que realice el trabajo. Cuando el trabajo comience, comuníqueselo al proveedor e introduzca el número de referencia con el que el proveedor registró la tarea en la acción de Notas.

Nota:

Asigne solo las tareas para las que no se debe esperar a que se completen otras tareas primero.
