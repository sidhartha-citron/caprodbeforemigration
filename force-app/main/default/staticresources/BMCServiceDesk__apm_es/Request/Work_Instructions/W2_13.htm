Instrucciones de trabajo para coordinadores de grupo

2.13.1

En el campo Duración (HH:MM), en la acción de Notas, introduzca el tiempo que empleó para revisar la petición.

2.13.2

Use el icono de búsqueda junto al campo Propietario para seleccionar al especialista más adecuado (en cuanto a habilidades, disponibilidad y derechos de acceso).

2.13.3

Si la petición debe procesarse lo antes posible (por ejemplo, en caso de un ataque de virus), comuníquese por teléfono o localizador con el especialista asignado, para asegurarse de que tenga conocimiento de la urgencia. Comuníquese con el especialista si es importante que este comience a trabajar en la petición incluso si no se espera que el especialista esté revisando nuevas asignaciones en ese momento (por ejemplo, después del horario de oficina).
