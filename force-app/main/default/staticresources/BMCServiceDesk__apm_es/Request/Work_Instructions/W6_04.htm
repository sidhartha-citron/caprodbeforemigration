Instrucciones de trabajo para analistas del servicio de asistencia / especialistas

6.4.1

Analice las tareas sin asignar y determine si puede completarlas todas.

Nota:

Una tarea no está asignada cuando el campo Estatus de la misma está establecido como Registrado.

6.4.2

Vaya al paso 6.12.1 si puede completar todas las tareas sin asignar. De lo contrario, continúe con el 6.5.1.
