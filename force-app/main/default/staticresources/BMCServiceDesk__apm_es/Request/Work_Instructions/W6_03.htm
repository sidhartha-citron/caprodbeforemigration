Paso del procedimiento

Instrucciones de trabajo para analistas del servicio de asistencia / especialistas

6.3.1

Agregue las tareas de implementación y aprobación requeridas para garantizar que se cumplan todos los requisitos de la petición. Asegúrese de seleccionar la categoría correcta.

Nota:

Cuando haya que asignar a la misma persona varias tareas que se deben completar en una secuencia ininterrumpida, puede crear una única tarea que las abarque todas.

6.3.2

Vincule cada tarea de implementación nueva a los CI que cambiarán (si los hubiera) cuando se ejecute la tarea. Haga clic en Vínculo nuevo en la lista relacionada Servicios y CI vinculados de tales tareas.

6.3.3

Asegúrese de que la fecha y hora especificadas en el campo Fecha de vencimiento de las nuevas tareas sean factibles. Actualice el campo Fecha de vencimiento de las tareas que se planifican con anterioridad y deben completarse cuando se hayan ejecutado las tareas nuevas.

6.3.4

Asegúrese de que el campo Estatus de las nuevas tareas esté establecido como Registrado.
