Paso del procedimiento

Instrucciones de trabajo para analistas del servicio de asistencia

1.1.1

Abra una nueva petición seleccionando la ficha Consola de incidencias y, a continuación, Nuevo.

1.1.2

Seleccione el ID de cliente correcto (seleccione Customer Portal User (Usuario del portal de clientes) para sus clientes y Usuario si el solicitante es un miembro del personal de soporte).

En el campo ID de cliente , seleccione el usuario. Para ello, seleccione la búsqueda de ID de cliente e introduzca el nombre o apellidos en el campo Buscar. Si los caracteres introducidos en el campo Buscar no son exclusivos, la aplicación de gestión de servicios mostrará una lista de usuarios cuyo nombre o apellidos comienzan con los caracteres introducidos. Seleccione el usuario de la lista.

Nota:

Si el usuario no está registrado, seleccione el registro de usuario NORECORD (No registrado) y escriba el nombre y la información de contacto del usuario en el campo Descripción. El registro podría no estar disponible de forma predeterminada.
