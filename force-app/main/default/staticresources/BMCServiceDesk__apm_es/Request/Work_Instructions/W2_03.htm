Instrucciones de trabajo para coordinadores de grupo

2.3.1

Continúe con el paso 6.1.1 si Resolución de peticiones puede procesar la petición. De lo contrario, continúe con el 2.4.1.

Nota:

Si la petición es una incidencia, un especialista determina en el paso 5.2.1 cómo resolverla. Para una incidencia, vaya al paso 2.4.1.

Nota:

Resolución de peticiones puede procesar una petición cuando la petición puede implementarse siguiendo el flujo de trabajo definido en una plantilla de petición aprobada por el propietario de servicio y usada para crear la petición.
Si la implementación de una petición requiere análisis de riesgos e impacto u otras aprobaciones distintas a las financieras, debe ser procesada por gestión de cambios.
