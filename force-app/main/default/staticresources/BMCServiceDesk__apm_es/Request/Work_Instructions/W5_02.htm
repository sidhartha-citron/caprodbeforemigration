Instrucciones de trabajo para coordinadores de grupo

5.2.1

Continúe con el paso 6.1.1 si Resolución de peticiones puede procesar una petición. De lo contrario, continúe con el 5.3.1.

Nota:

Resolución de peticiones puede procesar una petición cuando la petición puede implementarse siguiendo el flujo de trabajo definido en una plantilla de petición aprobada por el propietario de servicio y usada para crear la petición.

Gestión de cambios debe procesar la petición si la implementación de la petición requiere un análisis de riesgos e impacto o aprobaciones que no sean financieras.
