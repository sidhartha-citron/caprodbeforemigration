Instrucciones de trabajo para analistas del servicio de asistencia

2.4.1

Si el analista del servicio de asistencia puede resolver la petición (con la ayuda de otros analistas del servicio de asistencia, si fuera necesario), vaya al paso 8.1.1. De lo contrario, continúe con el 2.5.1.

Nota:

Si se usa una plantilla para registrar la petición, las instrucciones en el campo Descripción explican si el servicio de asistencia puede resolver la petición de soporte o no.

Nota:

Haga clic en Herramientas de agente > Búsqueda de conocimientos para comprobar si hay artículos de conocimientos disponibles que puedan proporcionar una solución. Seleccione Copy Article (Copiar artículo) o Copy Article Text (Copiar texto de artículo) para asegurarse de que la información del artículo se copie a la petición.
Seleccione la clasificación de artículo adecuada para el artículo de conocimientos y proporcione una valoración adicional. Seleccione Enviar.

Nota:

Puede que el servicio de asistencia no pueda resolver peticiones de soporte debido a restricciones de acceso, limitaciones de tiempo o conocimientos insuficientes.
