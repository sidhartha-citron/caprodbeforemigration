Paso del procedimiento

Instrucciones de trabajo para analistas del servicio de asistencia

8.2.1

Describa cómo se resuelve la petición en el campo Resolución.

8.2.2

Seleccione el elemento de configuración (CI) adecuado en el campo CI si no se ha hecho aún, o si no se ha seleccionado el CI correcto.

Nota:

Si el CI no está registrado en la CMDB, pero está dentro del alcance del proceso de gestión de configuración, seleccione el CI con el código NORECORD (No registrado).

El CI con el código NORECORD podría no estar disponible de forma predeterminada.

Nota:

Si la petición no se trata de un CI específico o si el CI está fuera del alcance del proceso de gestión de configuración, seleccione el CI con el código N/A (No disponible).

8.2.3

En el campo Duración (HH:MM), en la acción de Notas, introduzca el tiempo que empleó para registrar y resolver la petición.

8.2.4

Seleccione la categoría de cierre correspondiente en el campo Categoría de cierre. Establezca el campo Estatus de la petición como Terminado.

8.2.5

Asegúrese de estar seleccionado en el campo Propietario.

8.2.6

Realice una revisión final para asegurarse de que se haya rellenado correctamente la petición.

8.2.7

Si se aplica la solución provisional de un problema abierto para resolver la petición, vincule la incidencia al problema.
