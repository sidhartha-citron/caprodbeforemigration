Instrucciones de trabajo para analistas del servicio de asistencia

8.7.1

En el campo Duración (HH:MM) de la acción de Notas, introduzca el tiempo que empleó para verificar la solución. Cambie el valor del campo Estatus de Terminado a ClosedMail (Correo cerrado).

Nota:

La gestión de servicios puede establecer automáticamente el campo Estatus como Cerrado si, pasados 28 días, el usuario no ha respondido a la notificación automática por correo electrónico para indicar que la solución no es satisfactoria.

8.7.2

Si se pueden enviar peticiones similares, cree un artículo de conocimientos para la solución provista.
