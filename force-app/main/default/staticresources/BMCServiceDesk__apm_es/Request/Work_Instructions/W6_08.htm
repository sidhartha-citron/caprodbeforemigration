Instrucciones de trabajo para coordinadores de grupo

6.8.1

Continúe con el paso 6.9.1 si la tarea no está asignada al grupo adecuado o si no contiene toda la información que se necesita para resolverla. De lo contrario, vaya al paso 6.11.1.

Nota:

Si usted es el coordinador de grupo del servicio de asistencia, asegúrese de que la tarea contenga toda la información que se debe recopilar antes de poder comenzar a trabajar en la tarea. A continuación, asigne la tarea a un analista del servicio de asistencia actualmente disponible para resolver tareas. Si el servicio de asistencia no puede completar la tarea, asígnela al grupo que corresponda.
