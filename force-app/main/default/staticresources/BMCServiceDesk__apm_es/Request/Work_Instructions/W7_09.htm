Paso del procedimiento

Instrucciones de trabajo para propietarios de servicio

7.9.1

Pida a los especialistas más adecuados que implementen el cambio siguiendo los procedimientos de gestión de cambios de emergencia.

7.9.2

Tómese el tiempo suficiente para asegurarse de que los especialistas comprendan cómo se debe implementar el cambio para evitar riesgos de errores y su impacto en los clientes. Especifique qué listas de comprobación del plan de continuidad se deben usar y si se puede usar un plan de continuidad para minimizar el riesgo de error.

7.9.3

Envíe un mensaje de correo electrónico o de texto (por teléfono móvil o localizador) a los especialistas para confirmar que ha aprobado el uso de procedimientos de gestión de cambios de emergencia para la resolución de esta incidencia.
