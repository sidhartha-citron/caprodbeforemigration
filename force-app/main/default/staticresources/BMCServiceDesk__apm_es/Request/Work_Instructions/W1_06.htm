Instrucciones de trabajo para analistas del servicio de asistencia

1.6.1

Si el usuario ha proporcionado información adicional relevante para la resolución de la petición, agregue una acción de Notas para introducir dicha información y agregue a la petición los archivos relacionados. Comuníquese con el especialista asignado y proporcione una actualización.

1.6.2

Proporcione información actualizada al usuario sobre el estatus de la petición.

Nota:

Si la petición no se ha actualizado recientemente, comuníquese con el especialista asignado para obtener una actualización sobre el estatus para que así pueda proporcionar esta información al usuario.

1.6.3

Agregue una acción de Notas para resumir sus acciones y la información que le proporcionó al usuario.

1.6.4

En el campo Duración (HH:MM), en la acción de Notas, introduzca el tiempo que empleó para actualizar la petición.
