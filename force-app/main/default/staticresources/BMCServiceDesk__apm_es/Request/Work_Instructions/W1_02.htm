Paso del procedimiento

Instrucciones de trabajo para analistas del servicio de asistencia

1.2.1

Si el usuario llama por teléfono o acude al servicio de asistencia en persona, pregúntele cómo puede proporcionarle asistencia.

1.2.2

Use la información proporcionada (independientemente de si ha obtenido la información por teléfono, por correo electrónico o de otro modo) para determinar si el usuario se ha puesto en contacto con el servicio de asistencia a causa de una petición presentada anteriormente o si desea presentar una nueva petición.

1.2.3

Si el usuario desea presentar una nueva petición, determine el servicio relacionado. Determine también qué es lo que solicita el usuario y pregúntele sobre cualquier información adicional que pudiera ser de ayuda para resolver la petición.

Nota:

Si necesita más información para resolver la petición y no está en contacto con el usuario actualmente (por ejemplo, si no están hablando por teléfono o el usuario no se encuentra en el servicio de asistencia), comuníquese con el usuario por teléfono o por correo electrónico y solicite la información adicional necesaria. No registre una petición hasta que se le haya proporcionado esta información.
