Instrucciones de trabajo para analistas del servicio de asistencia

4.1.1

Si se registra una incidencia con el campo Impacto establecido como Alto o Grave si la petición es relativa a una indisponibilidad del servicio que afecta a varios usuarios, vaya al paso 4.3.1. De lo contrario, continúe con el 4.2.1.
