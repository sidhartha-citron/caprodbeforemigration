Instrucciones de trabajo para analistas del servicio de asistencia

1.5.1

Si el usuario no está satisfecho con la solución proporcionada para una de las peticiones que ha registrado anteriormente, vaya al paso 1.7.1. Si el usuario se pone en contacto con el servicio de asistencia para proporcionar información adicional sobre una petición que el usuario envió anteriormente o para solicitar una actualización sobre el estatus de una de las peticiones ya realizadas, continúe con el paso 1.6.1.
