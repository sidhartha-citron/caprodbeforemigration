Instrucciones de trabajo para analistas del servicio de asistencia

1.16.1

Use el icono de búsqueda junto al campo Propietario para seleccionar al propietario del servicio fallido.

Nota:

Haga clic en Herramientas de agente > Explorador de CI para ver el servicio y las relaciones. Seleccione Nombre de clase para ver los detalles de este servicio. La CMDB se abre con el servicio seleccionado. Seleccione las especificaciones del servicio para identificar al propietario del servicio fallido.

1.16.2

Comuníquese con el propietario del servicio e infórmele de la situación. Asegúrese de informar al propietario de que la incidencia ya se le ha escalado.

Nota:

Comuníquese con el gestor de guardia si el propietario de servicio no está disponible.

1.16.3

Agregue una acción de Notas para especificar que ha escalado la incidencia al propietario del servicio o al gestor de guardia.

1.16.4

En el campo Duración (HH:MM), en la acción de Notas, introduzca el tiempo que empleó para revisar la petición.
