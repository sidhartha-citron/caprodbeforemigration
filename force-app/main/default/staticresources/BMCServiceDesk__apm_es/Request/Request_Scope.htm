Alcance


El alcance del proceso de gestión de peticiones está limitado a las siguientes categorías de petición de soporte.

Categoría

Descripción

Incidencia

Petición de un usuario para restaurar un servicio que no está funcionando correctamente

Nota:

Peticiones para restablecer la contraseña de usuarios que la han olvidado y peticiones para restaurar copias de seguridad de los usuarios que han perdido sus archivos.

Petición de cambio

Petición de un usuario para modificar, agregar, reubicar o eliminar parte o toda la funcionalidad del servicio, el acceso, o los componentes

Nota:

Pertenecen a esta categoría las peticiones para copiar datos (por ejemplo, para realizar copias de seguridad o restaurar copias de seguridad en una ubicación diferente) y peticiones para ejecutar trabajos por lotes.

Petición de servicio

Petición de un usuario que se puede resolver utilizando un flujo de trabajo predefinido en los siguientes escenarios:

  • Existe una plantilla para este flujo de trabajo
  • No es necesario un análisis de riesgos e impacto
  • No se necesita aprobación otra que la del cliente

Petición de información

Petición de un usuario para la respuesta ante una duda relacionada con el servicio

Cumplido

Elogio o ánimo por parte del usuario

Queja

Petición de un usuario para mejorar la forma en la que la organización proveedora de servicios ofrece soporte al usuario.
