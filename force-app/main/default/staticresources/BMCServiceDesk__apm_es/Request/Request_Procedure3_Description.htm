Procedimiento 3: Tratamiento de quejas y cumplidos


Cuando se registra una queja o un cumplido, el gestor del servicio de asistencia revisa los detalles.

Si se trata de una queja, el gestor del servicio de asistencia hablará con los analistas del servicio de asistencia, los especialistas, el gestor de nivel de servicio, el representante del cliente y el usuario involucrados para determinar la causa de la queja. Tras determinar la causa, el gestor del servicio de asistencia decide qué método usar para restaurar la confianza del usuario en el soporte proporcionado por la organización proveedora de servicios.

Si se trata de un cumplido, el gestor del servicio de asistencia informa al empleado o departamento que lo recibió.

Una vez que el gestor del servicio de asistencia ha tomado las medidas necesarias para tranquilizar al usuario o remitir el cumplido, el gestor del servicio de asistencia completa la petición.
