Beneficiaries


The following table lists the roles that rely on the Change Management process and their respective requirements for the Change Management process.

Role

Requirement

CAB Member

Information regarding changes for which the approval of a CAB member is required

Change Coordinator

Information regarding the progress of tasks

Change Manager

Information regarding the risk and impact analysis to ensure that it was performed thoroughly

Information regarding the change implementation plans to ensure that they do not conflict with other changes or events

Configuration Administrator

Information regarding tasks for the registration or update of CI information

Customer Representative

Information regarding changes that affect services

Group Coordinator

Information regarding tasks that have been assigned to a group, and the changes to which the tasks belong

Problem Coordinator

Information regarding the progress of changes that were requested by problem coordinators

Service Desk Analyst

Information regarding the progress of changes for which a status update has been requested

Service Level Manager

Information regarding the status of changes for which the requests were registered by a service level manager after receiving the request from a customer representative

Information regarding the change implementation performance for change requests that were submitted using web request forms

Service Owner

Information regarding changes that affect services

Specialist

Information regarding tasks that have been assigned to a specialist, and the change to which the tasks belong

User

Information regarding the status of the user's requests for change

Information regarding changes that affect service
