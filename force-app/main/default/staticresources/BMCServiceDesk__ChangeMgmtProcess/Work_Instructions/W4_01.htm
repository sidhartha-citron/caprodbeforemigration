Work Instructions for Change Coordinators

4.1.1

After the change manager sets the Status field of the change to Approved, assign the implementation tasks. Change each task’s Status field from Registered to Assigned.

Note:

Assign only the tasks that do not have to wait for other implementation tasks to be completed.

Note:

After a task is completed, the next task in order is automatically set to the status Assigned, provided
that all previous tasks of each current task have reached the status Completed or Cancelled.

4.1.2

Set the Status field of the change to In Progress.
