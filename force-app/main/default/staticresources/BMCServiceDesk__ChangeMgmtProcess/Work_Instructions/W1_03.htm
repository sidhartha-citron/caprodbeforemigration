Work Instructions for Change Coordinators

1.3.1

If   the   change   request   was   received   from  Incident Management as a support request, create a change  request  and select  the  Incident  tab. Click the Actions button, select Link Incident, and then link the   existing   incident   to  the change request. Add  a Notes  action  to  the  support  request saying that the  "Responsibility  for  coordination  has  been accepted by Change  Management." In   the  Duration (HH:MM) field, enter  the  time that you spent reviewing the   support   request.

1.3.2

If   the   change   request   was   received  from Problem Management as a problem, select the Changes  tab. Click  the Actions button, select Link Change Request, and then link the new change to the problem. Add a Notes action to the problem saying that  the  "Responsibility   for  coordination  has been accepted by Change Management." In the Duration (HH:MM) field, enter the time that you spent reviewing the problem.
