Work Instructions for Specialists

4.4.1

When the Status field of your task changes from Registered to Assigned, open it and read the instructions in the Information field.

4.4.2

If you are not ready to start working on the task, set the Status field of the task to Accepted.

Note:

If the task should be assigned to another specialist, ask the change coordinator to change its assignment. To do so, set the Status field of the task to Rejected and specify in the Information update field why you rejected the task.

4.4.3

When you are ready to start the work, set the Status field of the task to In Progress and complete the implementation.

Note:

If you cannot continue the work on the task (for example, because you are waiting for equipment to be delivered), set its Status field to Waiting for, and specify in the Information update field what you are waiting for and when you expect to complete the task.

4.4.4

When you have completed the work, set the
Status field of the task to Completed.

If you could not complete the task, set the Status of the task to Failed and specify in the Result field why you could not complete the task successfully.

4.4.5

In the Time spent (min.) field, enter the number of minutes that you spent completing the task.
