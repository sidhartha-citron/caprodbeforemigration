Work Instructions for Change Coordinators

1.9.1

If the change request requires a prevention or fix of a problem, go to 1.12.1 provided the change request fulfills the following conditions:

The change implementation can be coordinated by a single change coordinator

The functionality of the service will not become different after the implementation.

No additional funding is required to fulfill the implementation.

For any other kind of change request, go to 1.10.1.
