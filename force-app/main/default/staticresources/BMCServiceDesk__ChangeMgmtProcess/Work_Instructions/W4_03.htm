Work Instructions for Specialists

4.3.1

The tasks for completing a change can include providing training for operators, testing new equipment in a lab environment, and so on. These tasks ensure operational readiness.

When the Status field of your task changes from Registered to Assigned, open the task and read the instructions in the Notes action details.

4.3.2

If you are not ready to start working on the task, set the Status field to Accepted.

Note:

If the task should be assigned to another specialist, ask the change coordinator to change its assignment. To do so, set the Status field of the task to Rejected and add a Notes action saying why you rejected the task.

4.3.3

When you are ready to start the work, set the Status field of the task to In Progress and make the final preparations before the change is implemented.

Note:

If you cannot continue your work on the task (for example, because you are waiting for equipment to be delivered), set its Status field to Waiting for, and add a Notes action specifying what you are waiting for and when you expect to complete the task.

4.3.4

As soon as you have completed the work, set the Status field of the task to Completed.

If you could not complete the task, set the Status field of the task to Failed and specify in the Resolution field why you could not complete the task successfully.

4.3.5

In the Duration (HH:MM) field, enter the time that you spent completing the task.
