Work Instructions for Change Coordinators

1.8.1

Return the change request to the process from which it was passed to Change Management.

1.8.2

If the change request was received from Incident Management as a support request, in the Information update field, specify that the change can be implemented within the Incident Management process. In the Time spent (min.) field, enter the number of minutes that you spent reviewing the support request. Use the Assign To option on the toolbar to select your group coordinator in the Member field of the support request. Ensure that the Status field of the support request is set to Assigned.

Note:

Do not set the Status field of the support request to Rejected, because that causes the service management application to automatically assign it to the service desk.

1.8.3

If the change request was received from Problem Management as a problem, in the Information update field, specify that the change can be implemented within the Problem Management process. In the Time spent (min.) field, enter the number of minutes that you spent reviewing of the problem. Set the Status field of the problem to Rejected, which causes the problem to be returned to its problem coordinator.
