Work Instructions for Change Coordinators

8.7.1

Set the Status field of the task for informing the service owner of the implemented emergency change to In Progress.

8.7.2

Inform the service owner (and the on-duty manager if they approved the emergency change implementation) that the emergency change was successfully implemented. Send an email summarizing the implemented change.

8.7.3

From the History tab of the task record, add an action record of the type Email Received and paste the content of this email in the Note field.

8.7.4

Set the Status field of the task to Completed.

8.7.5

Add a Notes action record and enter the time that you spent completing the task in the Duration (HH:MM) field.
