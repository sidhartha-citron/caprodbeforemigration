Arbeitsanweisungen für Change-Koordinatoren

1.3.1

Falls der Change Request vom Request Management in Form einer Anforderung erhalten wurde, verknüpfen Sie den bestehenden Change mit der Anforderung im Abschnitt Verknüpfte Change Requests, indem Sie auf die Schaltfläche Change Request mit Incident verknüpfen klicken. Fügen Sie in der Anforderung eine Hinweisaktion mit dem Text „Responsibility for coordination has been accepted by Change Management“ (Verantwortung für Koordination wurde von Change Management angenommen) hinzu.

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion die für das Prüfen der Anforderung aufgewendete Zeit ein.

Setzen Sie das Feld Status der Anforderung auf Change ausstehend.

1.3.2

Falls der Change Request vom Problem Management in Form eines Problems erhalten wurde, verknüpfen Sie den bestehenden Change mit dem Problem im Abschnitt Verknüpfte Change Requests, indem Sie auf die Schaltfläche Change Requests mit Problem verknüpfen. Fügen Sie dem Problem eine Hinweisaktion mit dem Text „Responsibility for coordination has been accepted by Change Management“ (Verantwortung für Koordination wurde von Change Management angenommen) hinzu.

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für das Prüfen des Problems ein.

Setzen Sie das Feld Status des Problems auf Change ausstehend.
