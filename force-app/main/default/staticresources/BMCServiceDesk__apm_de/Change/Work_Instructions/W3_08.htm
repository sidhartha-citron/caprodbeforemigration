Arbeitsanweisungen für CAB-Mitarbeiter

Hinweis:

Alle im Genehmigungs-Workflow definierten Genehmiger erhalten eine E-Mail.

3.8.1

Öffnen Sie den Genehmigungsdatensatz und prüfen Sie die Informationen zur Änderung im Abschnitt „Change Request“.

Öffnen Sie die Change-Anforderungen, die Anlayse für Risiko und Auswirkung und den Implementierungsplan.

3.8.2

Um den Change Request zu genehmigen, klicken Sie im Bereich Aktionsverlauf auf Genehmigen/Ablehnen in der Spalte Aktion, fügen Sie zusätzliche Informationen im Feld Kommentare ein und klicken Sie auf Genehmigen.

Um den Change Request abzulehnen, wählen Sie die Option Genehmigen/Ablehnen in der Spalte Aktion, geben Sie den Grund für die Ablehnung im Feld Kommentare ein und klicken Sie auf Ablehnen.

Hinweis:

Sie können den Change Request auch genehmigen bzw. ablehnen, indem Sie auf den Link in der E-Mail-Benachrichtigung klicken.
