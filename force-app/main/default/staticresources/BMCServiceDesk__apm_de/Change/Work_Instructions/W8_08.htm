Arbeitsanweisungen für Change-Koordinatoren

8.8.1

Nachdem das Feld Status für alle mit dem Notfall-Change verbundenen Aufgaben auf Storniert oder Fertiggestellt gesetzt wurde, setzen Sie das FeldSchließungskategorie der Anforderung auf Implementiert.
