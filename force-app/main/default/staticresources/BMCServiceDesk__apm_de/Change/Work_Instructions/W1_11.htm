Arbeitsanweisungen für Change-Koordinatoren

1.11.1

Überprüfen Sie die Informationen, die Sie vom Release Management erhalten, um die Anforderungen des Change Request nachzuvollziehen.

Hinweis:

Ein Change Request des Release Management ist ein Dokument, das die Change-Anforderungen festlegt. Alternativ ist der Change Request eine Anforderung oder ein Problem, wenn kein Release Management für eine Anforderung oder ein Problem erforderlich ist, das vorher an das Release Management übermittelt wurde.
