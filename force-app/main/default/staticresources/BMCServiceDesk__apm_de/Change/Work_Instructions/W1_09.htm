Arbeitsanweisungen für Change-Koordinatoren

1.9.1

Falls im Change Request die Vorbeugung oder Behebung eines Problems angefordert wird, gehen Sie zu Punkt 1.12.1, sofern:

  • Die Change-Implementierung von einem Change-Koordinator implementiert werden kann.

  • Die Implementierung keine Änderung der Funktionen des Service bewirkt.

  • keine zusätzlichen finanziellen Mittel für die Implementierung erforderlich sind.

Für alle anderen Change Requests gehen Sie zu Punkt 1.10.1.
