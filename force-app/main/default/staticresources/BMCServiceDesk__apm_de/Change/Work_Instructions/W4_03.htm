Arbeitsanweisungen für Spezialisten

4.3.1

Die für den Abschluss eines Change erforderlichen Aufgaben können Schulungen für Operatoren, Tests neuer Geräte in einer Laborumgebung usw. umfassen. Mit diesen Aufgaben wird die betriebliche Eignung gewährleistet.

Wenn das Feld Status der Aufgabe in „Zugewiesen“ geändert wird, öffnen Sie sie und lesen Sie die Anweisungen im Feld Beschreibung und in den Hinweisaktionen.

4.3.2

Wenn Sie noch nicht mit der Aufgabe beginnen möchten, setzen Sie das Feld Status auf Angenommen.

Hinweis:

Falls die Aufgabe einem anderen Spezialisten zugewiesen werden soll, bitten Sie den Change-Koordinator, die Zuweisung zu ändern. Dafür legen Sie das Feld Status der Aufgabe auf Abgelehnt fest und fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, warum sie abgelehnt wurde.

4.3.3

Wenn Sie mit der Arbeit beginnen können, legen Sie das Feld Status der Aufgabe auf In Bearbeitungs fest und nehmen Sie die letzten Vorbereitungen vor der Implementierung des Change vor.

Hinweis:

Wenn Sie die Arbeit an der Aufgabe nicht fortsetzen können (z. B. weil Sie auf zu liefernde Ausrüstung warten), legen Sie das Feld Status auf Waiting for... (Warten auf…) fest und fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, worauf Sie warten und wann Sie die Aufgabe wahrscheinlich abschließen werden.

4.3.4

Wenn Sie die Arbeit abgeschlossen haben, legen Sie das Feld Status der Aufgabe auf Fertiggestellt fest.

Wenn Sie die Aufgabe nicht abschließen konnten, legen Sie das Feld Status der Aufgabe auf Fehlgeschlagen fest und geben Sie im Feld Lösung an, warum Sie die Aufgabe nicht abschließen konnten.

4.3.5

Geben Sie im Feld Dauer (HH:MM) in einer Hinweisaktion die für den Abschluss der Aufgabe aufgewendete Zeit ein.
