Arbeitsanweisungen für Spezialisten

5.8.1

Wenn das Feld Status der Aufgabe zum Abschließen der letzten Vorbereitungsschritte in Zugewiesen geändert wurde, öffnen Sie die Aufgabe und lesen Sie die Anweisungen im Feld Beschreibung und in den Hinweisaktionen.

Hinweis:

Die Aufgaben zum Abschließen der letzten Vorbereitungen umfassen alle erforderlichen Schritte für die Sicherstellung der betrieblichen Eignung. Dazu gehören etwa folgende Aufgaben:

  • Erstellen einer Master-Version des neuen Release , die der Konfigurationsadministrator in der DML speichern kann.

  • Hinzufügen von Speicherkapazität für die Produktions- und Continuity-Umgebungen.

  • Bereitstellen von Schulungen für Benutzer

5.8.2

Wenn Sie noch nicht mit der Aufgabe beginnen möchten, setzen Sie das Feld Status der Aufgabe auf Angenommen fest.

Hinweis:

Falls die Aufgabe einem anderen Spezialisten zugewiesen werden soll, bitten Sie den Change-Koordinator, die Zuweisung zu ändern. Dafür legen Sie das Feld Status der Aufgabe auf Abgelehnt fest und fügen eine Hinweisaktion hinzu, in der Sie angeben, warum sie abgelehnt wurde.

5.8.3

Wenn Sie mit der Arbeit beginnen können, legen Sie das Feld Status auf In Bearbeitungs fest und nehmen Sie die letzten Vorbereitungen zum Übertragen des neuen Release in die Produktion vor.

Hinweis:

Wenn Sie nicht mit der Arbeit an der Aufgabe fortfahren können (z. B. weil Sie auf zu liefernde Ausrüstung warten), legen Sie das Feld Status auf Warten auf… fest und fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, worauf Sie warten und wann Sie die Aufgabe wahrscheinlich abschließen werden.

5.8.4

Wenn Sie die Arbeit abgeschlossen haben, legen Sie das Feld Status der Aufgabe auf Fertiggestellt fest.

Wenn Sie die Aufgabe nicht abschließen können, legen Sie das Feld Status der Aufgabe auf Fehlgeschlagen fest und geben Sie im Feld Lösung an, warum Sie die Aufgabe nicht erfolgreich abschließen konnten.

5.8.5

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion die für den Abschluss der Aufgabe benötigte Zeit ein.
