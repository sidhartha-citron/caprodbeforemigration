Arbeitsanweisungen für Change-Koordinatoren

1.13.1

Klicken Sie auf die Registerkarte Change Requests und öffnen Sie einen neuen Change Request, indem Sie auf Neu klicken.

1.13.2

Vergewissern Sie sich, dass Sie im Feld Initiator-ID des Change Request ausgewählt sind.

1.13.3

Wählen Sie im Feld Vorlage die Change-Vorlage für die Implementierung des angeforderten Change aus. Die Verwendung einer Vorlage gewährleistet, dass die Felder des Change mit Werten aus der ausgewählten Change-Vorlage ergänzt werden. Gleichzeitig wird dafür gesorgt, dass die erforderlichen Aufgaben erstellt und mit dem Change Request verknüpft werden, wenn dieser gespeichert wird.

1.13.4

Fügen Sie im Feld Beschreibung eine Kurzbeschreibung des Change hinzu.

1.13.5

Wählen Sie im Feld Change-Typ den geeigneten Typ aus.

1.13.6

Wählen Sie im Feld Change-Kategorie die geeignete Kategorie aus.

1.13.7

Speichern Sie den Change Request, um die dazugehörigen Aufgaben zu generieren und die Verknüpfung mit einer Anforderung, einem Problem oder einer Anlage zu ermöglichen.

1.13.8

Fügen Sie eine Hinweisaktion hinzu, um die Bedingungen für den Change zusammenzufassen.

1.13.9

Wählen Sie im Abschnitt Verknüpfte Services und CIs den Service aus, der von der Implementierung des Change betroffen wäre.

1.13.10

Falls der Change Request in Form einer Anforderung vom Request Management eingegangen ist, öffnen Sie die Anforderung und verknüpfen Sie den Change im Abschnitt Verknüpfte Change Requests der Anforderung. Fügen Sie der Anforderung in derHinweisaktion den folgenden Hinweis hinzu: Responsibility for coordination has been accepted by Change Management (Verantwortung für die Koordination wurde vom Change Management akzeptiert).

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für das Prüfen der Anforderung ein.

Setzen Sie das Feld Status der Anforderung auf Change ausstehend.

1.13.11

Falls der Change Request in Form eines Problems vom Problem Management eingegangen ist, öffnen Sie das Problem und verknüpfen Sie den Change im Abschnitt Verknüpfte Change Requests des Problems. Fügen Sie dem Problem in der Hinweisaktion hinzu den folgenden Hinweis hinzu: Responsibility for coordination has been accepted by Change Management (Verantwortung für die Koordination wurde vom Change Management akzeptiert).

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für das Prüfen des Problems ein.

Setzen Sie das Feld Status des Problems auf Change ausstehend.
