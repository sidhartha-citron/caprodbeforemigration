Arbeitsanweisungen für Change-Koordinatoren

4.1.1

Weisen Sie, nachdem der Change Manager das Feld Status für den Change Request auf Genehmigt gesetzt hat, die Implementierungsaufgaben zu. Ändern Sie das Feld Status der Aufgabe von Registriert auf Zugewiesen.

Hinweis:

Weisen Sie nur Aufgaben zu, bei denen nicht auf den Abschluss anderer Implementierungsaufgaben gewartet werden muss.

Hinweis:

Wenn die Ausführungsreihenfolge in den Change- und Aufgabenvorlagen aktiviert ist, wird die nächste Aufgabe erst nach Abschluss der vorangehenden Aufgabe angezeigt.

4.1.2

Setzen Sie das Feld Status des Change Request auf In Bearbeitung.
