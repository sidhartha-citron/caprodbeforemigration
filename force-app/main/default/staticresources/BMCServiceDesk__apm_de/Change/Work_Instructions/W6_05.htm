Arbeitsanweisungen für Spezialisten

6.5.1

Setzen Sie das Feld Status der Aufgabe zum Durchführen des Produktionstests auf Fehlgeschlagen.

6.5.2

Informieren Sie den Change-Koordinator (telefonisch oder persönlich) über die Situation. Geben Sie an, ob die Change-Implementierung zurückgesetzt wurde. Erklären Sie dem Change-Koordinator, ob Sie der Ansicht sind, dass die Change-Implementierung durchführbar ist oder ob weitere Maßnahmen erforderlich sind.
