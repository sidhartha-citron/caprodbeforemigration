Arbeitsanweisungen für Spezialisten

7.7.1

Implementieren Sie den Change wie mit dem Servicebesitzer geplant (bzw. dem diensthabenden Manager, falls der Servicebesitzer nicht verfügbar ist). Fordern Sie bei Bedarf Hilfe von anderen Spezialisten an.

7.7.2

Fügen Sie eine Hinweisaktion in der Anforderung hinzu, um zusammenzufassen, wie der Change implementiert wurde.

7.7.3

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für die Implementierung der Lösung ein.
