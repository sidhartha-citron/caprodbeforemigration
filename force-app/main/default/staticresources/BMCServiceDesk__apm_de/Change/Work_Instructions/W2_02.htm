Arbeitsanweisungen für Spezialisten

2.2.1

Öffnen Sie die Risiko- und Auswirkungsaufgabe und lesen Sie die Anweisungen im Feld Beschreibung und in den Hinweisaktionen.

2.2.2

Wenn Sie noch nicht mit der Aufgabe beginnen möchten, setzen Sie das Feld Status der Aufgabe auf „Angenommen“.

Hinweis:

Falls die Aufgabe einem anderen Spezialisten zugewiesen werden soll, bitten Sie den Change-Koordinator, die Zuweisung zu ändern. Legen Sie zu diesem Zweck das Feld Status der Aufgabe auf „Abgelehnt“ fest und fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, warum sie abgelehnt wurde.

2.2.3

Wenn Sie mit der Risiko- und Auswirkungsaufgabe beginnen möchten, setzen Sie ihr Feld Status auf „In Bearbeitung“ und beginnen Sie damit, die Antwort auf ihre Frage herauszufinden.

Hinweis:

Wenn Sie nicht mit der Arbeit an der Aufgabe fortfahren können (z. B. weil Sie auf Informationen eines Lieferanten warten), legen Sie das Feld Status auf „Warten auf...“ fest und fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, worauf Sie warten und wann die Antwort voraussichtlich vorliegen wird.

2.2.4

Wenn Sie die Antwort auf die Risiko- und Auswirkungsanalyse kennen, geben Sie sie in das Feld Lösung ein und legen Sie das Feld Status der Aufgabe auf „Fertiggestellt“ fest.

Hinweis:

Wenn Sie die Antwort auf die Frage der Risiko- und Auswirkungsanalyse nicht finden konnten, legen Sie das Feld Status der Aufgabe auf „Fehlgeschlagen“ fest und geben Sie im Feld Lösung an, warum die Frage nicht beantwortet werden konnte.

2.2.5

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion die für das Beantworten der Risiko- und Auswirkungsaufgabe aufgewendete Zeit ein.
