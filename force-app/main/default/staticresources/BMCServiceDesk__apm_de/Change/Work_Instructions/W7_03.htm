Arbeitsanweisungen für Release-Administratoren

7.3.1

Übertragen Sie den neuen Release in die Testumgebung.

7.3.2

Teilen Sie dem Spezialisten mit, dass die Übertragung durchgeführt wurde.

7.3.3

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion die Zeit ein, die Sie für die Übertragung des Release an die Testumgebung benötigt haben.
