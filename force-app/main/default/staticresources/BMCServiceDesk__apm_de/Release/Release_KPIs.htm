Key Performance Indicators


In der Tabelle unten werden die Key Performance Indicators (KPIs) aufgeführt, die ausgewählt wurden, um den Erfolg des Release Management-Prozesses zu verfolgen.

Erfolgreiche Releases

Die Anzahl abgeschlossener Releases, bei denen das Feld Completion Code (Abschlusscode) auf „Implementiert“ steht, geteilt durch die Gesamtzahl abgeschlossener Releases.

Die Anzahl von Releases, deren Feld Status noch nicht auf „Closed“ (Abgeschlossen) steht.
