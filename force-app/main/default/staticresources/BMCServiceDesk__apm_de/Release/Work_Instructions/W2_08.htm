Arbeitsanweisungen für Release-Koordinatoren

2.8.1

Zeigen Sie die verknüpften Anforderungen und Probleme in den Bereichen Verknüpfte Incidents und Verknüpfte Probleme des Change Request an, der als Platzhalter für die Change Requests des Service dient, die vom CAB geprüft werden sollen.

2.8.2

Informieren Sie alle Anforderer abgelehnter Change Requests, dass das CAB des Service ihren Change Request abgelehnt hat.

2.8.3

Falls es sich um einen in Form einer Anforderung eingegangenen Change Request handelt, geben Sie im Feld Lösung der Anforderung an, dass der Change Request vom Change Advisory Board des Service abgelehnt wurde. Legen Sie im Feld Closure Category (Abschlusskategorie) der Anforderung den Wert Unable - Not Able to Solve or in Conflict with Standard or Policy (Nicht möglich – keine Lösung möglich oder Konflikt mit Norm oder Richtlinie) fest. Setzen Sie das Feld Status der Anforderung auf Fertiggestellt. Der Service Desk ist dafür verantwortlich, den über das FeldKunden-ID mit der Anforderung verknüpften Kunden zu informieren.

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für die Fertigstellung der Anforderung ein.

2.8.4

Falls es sich um einen in Form eines Problems eingegangenen Change Request handelt, geben Sie im Bereich Aktionsverlauf an, dass der Change Request vom Change Advisory Board des Service abgelehnt wurde. Setzen Sie das Feld Status des Problems auf Abgelehnt. Das Problem wird wieder dem Problem-Koordinator zugewiesen.

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für die Fertigstellung des Problems ein.

2.8.5

Entfernen Sie die Verknüpfung zwischen jedem abgelehnten Change Request und dem als Platzhalter verwendeten Change. Um die Verknüpfung mit einer Anforderung aufzuheben, klicken Sie im Bereich Verknüpfte Incidents des Platzhalter-Change auf die Schaltfläche Del (Entf) in der Spalte Aktion. Um die Verknüpfung mit einem Problem aufzuheben, klicken Sie im Bereich Verknüpfte Probleme des Platzhalter-Change auf die Schaltfläche Del (Entf) in der Spalte Aktion.
