Arbeitsanweisungen für Release-Koordinatoren

3.2.1

Öffnen Sie den Change, mit dem die Change Requests für den Release verknüpft sind. Öffnen Sie in diesem Change im Bereich Verknüpfte Aufgaben die Geschäftsbegründungsaufgabe, die von der Change-Vorlage bei der Registrierung des Change generiert wurde.

3.2.2

Setzen Sie das Feld Status der Aufgabe auf Storniert und geben Sie im Feld Lösung an, dass der Release mit den dem Servicebesitzer bereits zur Verfügung stehenden Budgets implementiert werden kann.
