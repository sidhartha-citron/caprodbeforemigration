Arbeitsanweisungen für Release-Koordinatoren

4.12.1

Wählen Sie die Option Implementiert im Feld Closure Category (Schließungskategorie) für den Change, mit dem die Aufgabe für die Implementierungsnachbesprechung verknüpft ist.

Hinweis:

Das Feld Closure Category (Schließungskategorie) wird sichtbar, wenn alle mit dem Change verknüpften Aufgaben den endgültigen Status erreicht haben (d. h. Fehlgeschlagen, Storniert oder Fertiggestellt).

4.12.2

Vergewissern Sie sich, dass das Feld Status für den Change Request auf Fertiggestellt gesetzt ist.

4.12.3

Fügen Sie im Release eine Hinweisaktion hinzu, um anzugeben, dass die Implementierungsnachbesprechung abgeschlossen ist, und fügen Sie den Link zu den Notizen der Implementierungsnachbesprechung ein.

4.12.4

Setzen Sie das Feld Status des Release auf Geschlossen.
