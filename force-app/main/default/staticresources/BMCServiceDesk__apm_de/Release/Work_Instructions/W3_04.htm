Arbeitsanweisungen für Release-Koordinatoren

3.4.1

Fordern Sie die Genehmigung für den Geschäftsfall an, um sicherzustellen, dass ein Budget für die Implementierung des neuen Release erstellt wird.

3.4.2

Setzen Sie das Feld Status der Geschäftsbegründungsaufgabe auf Waiting for… (Warten auf) und fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, dass Sie auf die Genehmigung des Geschäftsfalls warten.

3.4.3

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion die Zeit ein, die Sie zum Vorbereiten des Geschäftsfalls auf die Genehmigung aufgewendet haben.
