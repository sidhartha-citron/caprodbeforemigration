Arbeitsanweisungen für Release-Koordinatoren

3.6.1

Fügen Sie den genehmigten Geschäftsfall der Geschäftsbegründungsaufgabe bei und geben Sie im Feld Lösung an, dass Sie den genehmigten Geschäftsfall beigefügt haben.

3.6.2

Öffnen Sie den Change, mit dem die Change Requests für den Release verknüpft sind. Öffnen Sie jede mit diesem Change verknüpfte Anforderung und fügen Sie eine Hinweisaktion hinzu, um den Betrag anzugeben, der dem Kunden für die Implementierung des angeforderten Change in Rechnung gestellt wird (gemäß genehmigtem Geschäftsfall).

3.6.3

Setzen Sie das Feld Status der Geschäftsbegründungsaufgabe auf Fertiggestellt.

3.6.4

Geben Sie im Feld Dauer (HH:MM) der Hinweisaktion die Zeit an, die Sie zum Aktualisieren der Aufgabe und der Change Requests für den Release aufgewendet haben.

3.6.5

Setzen Sie das Feld Status für den Release auf Genehmigt.
