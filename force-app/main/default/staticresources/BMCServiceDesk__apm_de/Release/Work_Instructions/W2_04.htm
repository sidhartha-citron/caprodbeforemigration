Arbeitsanweisungen für CAB-Mitarbeiter

2.4.1

Weisen Sie gegebenenfalls den Release-Koordinator auf die Change Requests hin, die abgelehnt werden müssen.

Hinweis:

Ein Change Request ist vom CAB abgelehnt, wenn mindestens zwei Drittel der CAB-Mitglieder sich einig sind, dass der Change Request abgelehnt werden muss.
