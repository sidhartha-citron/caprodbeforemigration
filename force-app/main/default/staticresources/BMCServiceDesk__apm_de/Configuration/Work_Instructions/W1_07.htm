Arbeitsanweisungen für Konfigurationsadministratoren

1.7.1

Registrieren Sie die bereits bekannten Attribute des bestellten Konfigurationselements (CIs) in Übereinstimmung mit den Feldverwendungsrichtlinien des Konfigurationselement-Formulars. Setzen Sie das Feld Status des bestellten CIs auf Bestellt. Verknüpfen Sie die bestellten CIs noch nicht mit anderen CIs, Services oder Benutzern.

Der bestellte Wert im Feld Status ist möglicherweise nicht standardmäßig verfügbar. Sie müssen das Kontrollkästchen Aufgabe für den Status Bestellt aktivieren.

1.7.2

Fügen Sie eine Hinweisaktion in der Aufgabe hinzu und geben Sie an, dass die Bestellung für die neuen CIs versandt wurde und Sie auf die Lieferung der CIs warten.

1.7.3

Setzen Sie das Feld Status der Aufgabe auf Waiting for… (Warten auf...).

1.7.4

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für den Einkauf und die Registrierung der CIs ein.
