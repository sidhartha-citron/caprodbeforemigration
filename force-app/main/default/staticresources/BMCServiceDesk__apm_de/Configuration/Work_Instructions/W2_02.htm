Arbeitsanweisungen für Konfigurationsadministratoren

2.2.1

Packen Sie die Lieferung aus und stellen Sie sicher, dass sie die richtige Hardware und Software sowie die richtigen Software-Lizenzzertifikate enthält, dass keine Artikel fehlen und dass die Konfigurationselemente (CIs) einwandfrei funktionieren.
