Arbeitsanweisungen für Konfigurationsadministratoren

2.4.1

Informieren Sie den Lieferanten über das Problem mit der Lieferung. Fragen Sie den Lieferanten, welche Maßnahmen ergriffen werden müssen, um das Problem zu lösen.

2.4.2

Folgen Sie den Anweisungen des Lieferanten und führen Sie die erforderlichen Schritte aus. Verpacken Sie beispielsweise die Konfigurationselemente (CIs) wieder und senden Sie sie an den Lieferanten zurück oder warten Sie auf die Lieferung der fehlenden CIs.
