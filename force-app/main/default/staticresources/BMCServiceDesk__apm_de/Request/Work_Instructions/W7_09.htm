Arbeitsanweisungen für Servicebesitzer

7.9.1

Bitten Sie die geeigneten Spezialisten, den Change unter Berücksichtigung der Managementverfahren für Notfall-Changes zu implementieren.

7.9.2

Investieren Sie ausreichend Zeit, um sicherzustellen, dass die Spezialisten verstehen, wie der Change implementiert werden muss. Auf diese Weise vermeiden Sie mögliche Fehlschläge und Auswirkungen auf die Kunden. Geben Sie an, welche Prüflisten für Continuity-Pläne verwendet werden müssen und ob ein Continuity-Plan verwendet werden kann, um das Risiko eines Fehlschlags zu minimieren.

7.9.3

Senden Sie eine E-Mail oder Textnachricht (mittels Mobiltelefon oder Pager) an die Spezialisten, um zu bestätigen, dass Sie die Verwendung der Managementverfahren für Notfall-Changes zur Lösung dieses Incidents genehmigt haben.
