Arbeitsanweisungen für Spezialisten

5.3.1

Stellen Sie fest, wie die Anforderung effizient zu lösen ist. Falls Sie keine Lösung finden können (z. B. aufgrund von Zugriffsbeschränkungen) oder die Anforderung ein CI betrifft, das von einer anderen Gruppe unterstützt wird, stellen Sie fest, welcher Gruppenkoordinator die Anforderung neu zuweisen soll.

Hinweis:

Wählen Sie Agenten-Tools > Knowledge-Suche, um zu prüfen, ob Knowledge-Artikel vorhanden sind, die möglicherweise eine Lösung enthalten. Wählen Sie den Link Artikel kopieren oder Artikeltext kopieren , um sicherzustellen, dass die Informationen aus dem Artikel in die Anforderung kopiert werden.
Wählen Sie die zutreffende Artikelbeurteilung aus und geben Sie zusätzliches Feedback.
