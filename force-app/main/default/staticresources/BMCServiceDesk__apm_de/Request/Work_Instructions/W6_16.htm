Arbeitsanweisungen für Service Desk Analysts/Spezialisten

6.16.1

Öffnen Sie die Anforderung und fassen Sie im Feld Lösung zusammen, wie die Anforderung implementiert wird. .

6.16.2

Legen Sie das Feld Abschlusskategorie auf Resolved (Gelöst) und das Feld Status der Anforderung auf Fertiggestellt fest.

6.16.3

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für die Lösung der Anforderung ein.

Hinweis:

Geben Sie lediglich die Zeit ein, die Sie zur Registrierung und Koordinierung der Anforderung aufgewendet haben. Die Zeit zur Lösung der einzelnen Aufgaben ist in den einzelnen Aufgaben aufgeführt.

6.16.4

Führen Sie eine letzte Prüfung der Anforderung durch, um sich zu vergewissern, dass die Informationen vollständig und korrekt sind.
