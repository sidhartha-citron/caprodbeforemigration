Arbeitsanweisungen für Service Desk Analysts

1.9.1

Wählen Sie den verbundenen schwerwiegenden Incident im Feld Nachricht aus.

Sie können einen vorhandenen Incident über die Option Agenten-Tools > Incident-Abgleich verknüpfen.

1.9.2

Füllen Sie die restlichen erforderlichen Felder aus und speichern Sie die Anforderung.

Hinweis:

Wenn eine Anforderung mit einem schwerwiegenden Incident verknüpft ist, sollte sie nicht direkt geändert werden. Änderungen am schwerwiegenden Incident werden direkt in sämtliche verbundenen Anforderungen kopiert.

Nachdem die Nachricht ausgewählt und der Incident gespeichert wurde, werden die folgenden Felder im Incident ausgefüllt:

  • Beschreibung

  • CI

  • Auswirkung

  • Priorität

  • Datum der Fertigstellung

  • Fälligkeitsdatum

  • Status

  • Lösung
