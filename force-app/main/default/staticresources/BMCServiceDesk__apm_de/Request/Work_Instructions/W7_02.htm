Arbeitsanweisungen für Gruppenkoordinatoren

7.2.1

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für das Prüfen der Anforderung ein.

7.2.2

Verwenden Sie das Suchsymbol neben dem Feld Besitzer, um den Spezialisten auszuwählen, der als Teamleiter für schwerwiegende Incidents im Feld Besitzer angegeben ist.
