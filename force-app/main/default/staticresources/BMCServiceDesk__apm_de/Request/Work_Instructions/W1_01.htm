Arbeitsanweisungen für Service Desk Analysts

1.1.1

Öffnen Sie eine neue Anforderung, indem Sie auf der Registerkarte Incident-Konsole auf Neu klicken.

1.1.2

Wählen Sie den korrekten Kunden ID-Typ aus („Kundenportal-Anwender“ für Ihre Kunden und „Benutzer“, falls der Anforderer ein Support-Mitglied ist).

Wählen Sie den Benutzer im Feld Kunden-ID aus. Wählen Sie dazu die Kunden-ID-Suche aus und geben Sie den Nach- oder Vornamen im Feld Suche ein. Wenn die im Feld Suche eingegebenen Zeichen nicht eindeutig sind, wird von der Service Management-Anwendung eine Liste der Benutzer angezeigt, deren Vor- oder Nachname mit den eingegebenen Zeichen beginnt. Wählen Sie den Benutzer aus dieser Liste aus.

Hinweis:

Falls der Benutzer nicht registriert ist, wählen Sie den Benutzer-Record NORECORD aus und geben Sie den Namen sowie die Kontaktdetails für den Benutzer im Feld Beschreibung ein. Dieser Record steht möglicherweise nicht standardmäßig zur Verfügung.
