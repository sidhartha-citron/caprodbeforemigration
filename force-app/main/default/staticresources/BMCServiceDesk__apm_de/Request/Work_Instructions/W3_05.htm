Arbeitsanweisungen für Service Desk Manager

3.5.1

Sorgen Sie dafür, dass der Benutzer mit den Leistungen des Serviceanbieters zufrieden ist, indem Sie auf die Hilfe der Service Desk Analysts, Spezialisten und Service Level Managers zurückgreifen.

Hinweis:

Falls klar zu erkennen ist, dass die Beschwerde nicht vor dem im Feld Fälligkeitsdatum festgelegten Zeitpunkt gelöst werden kann, legen Sie eine realistische Frist für die Lösung der Beschwerde fest und fügen Sie ein Hinweisfeld hinzu, in dem Sie diese Information eintragen.

Hinweis:

Setzen Sie das Feld Status der Anforderung auf Waiting for (Warten auf) oder Waiting for customer (Warten auf Kunden), falls die Arbeit für eine gewisse Zeit nicht fortgesetzt werden kann, da Sie noch auf Informationen des Benutzers bzw. eines Spezialisten warten. Fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, worauf Sie warten.
