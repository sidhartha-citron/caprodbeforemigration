Arbeitsanweisungen für Gruppenkoordinatoren

6.9.1

Kontaktieren Sie den Service Desk Analyst oder Spezialisten, der die Aufgabe Ihrer Gruppe zugewiesen hat mit der Erklärung, weshalb die Aufgabe der falschen Gruppe zugewiesen wurde oder Informationen fehlen.

6.9.2

Setzen Sie das Feld Status auf Abgelehnt. Fügen Sie eine Hinweisaktion mit dem Grund für die Ablehnung der Aufgabe hinzu.

6.9.3

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für das Prüfen und Ablehnen der Aufgabe ein.

6.9.4

Verwenden Sie das Suchsymbol neben dem Feld Besitzer, um den Service Desk Analyst oder Spezialisten, den Sie unter Schritt 6.9.1 kontaktiert haben, bzw. die korrekte Gruppe im Feld Besitzer der Aufgabe auszuwählen.
