Arbeitsanweisungen für Spezialisten

6.12.1

Öffnen Sie die Aufgabe und lesen Sie die Anweisungen im Feld Beschreibung.

6.12.2

Setzen Sie das Feld Status der Aufgabe auf Angenommen, wenn Sie noch nicht mit der Aufgabe beginnen möchten.

Hinweis:

Falls die Aufgabe einem anderen Spezialisten zugewiesen werden muss, bitten Sie den Gruppenkoordinator um die Änderung der Zuweisung. Setzen Sie das Feld Status der Aufgabe auf Abgelehnt und fügen Sie eine Hinweisaktion hinzu, um den Grund für die Ablehnung anzugeben.

6.12.3

Wenn Sie mit der angeforderten Arbeit beginnen können, legen Sie das Feld Status der Aufgabe auf In Bearbeitung fest.

Hinweis:

Wenn Sie die Arbeit an der Aufgabe nicht fortsetzen können, legen Sie das Feld Status auf Warten auf... fest und fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, worauf Sie warten und wann die Aufgabe abgeschlossen werden kann.

6.12.4

Wenn Sie die angeforderte Arbeit abgeschlossen haben, legen Sie das Feld Status der Aufgabe auf Fertiggestellt fest.

Wenn die angeforderte Arbeit nicht erfolgreich abgeschlossen werden kann, legen Sie das Feld Status der Aufgabe auf Fehlgeschlagen fest und geben Sie den Grund für das Scheitern im Feld Lösung an .

6.12.5

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion die für den Abschluss der Aufgabe benötigte Zeit ein.
