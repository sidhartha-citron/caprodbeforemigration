Arbeitsanweisungen für Spezialisten

5.1.1

Setzen Sie das Feld Status der Anforderung auf Angenommen, wenn Sie die Anforderung noch nicht bearbeiten möchten.

5.1.2

Setzen Sie das Feld Status auf In Bearbeitung, wenn Sie die Anforderung bearbeiten möchten.

5.1.3

Lesen Sie die Informationen in der Anforderung durch, um nachzuvollziehen, was angefordert wurde.
