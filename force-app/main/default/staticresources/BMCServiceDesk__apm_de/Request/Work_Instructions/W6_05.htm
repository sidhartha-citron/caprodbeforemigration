Arbeitsanweisungen für Service Desk Analysts/Spezialisten

6.5.1

Wählen Sie für die Aufgabe die geeigneten Mitarbeiter aus Ihrer Gruppe (in puncto Kenntnisse, Verfügbarkeit und Zugriffsrechte) aus. Verwenden Sie das Suchsymbol neben dem Feld Besitzer, um den Mitarbeiter oder die richtige Gruppe im Feld Besitzer der Aufgabe auszuwählen.

Falls Sie eine Aufgabe ausführen können, wählen Sie mithilfe des Suchsymbols neben dem Feld Besitzer sich selbst im Feld Besitzer der Aufgabe aus.

Falls eine andere Gruppe eine Aufgabe ausführen muss, wählen Sie mithilfe des Suchsymbols neben dem Feld Besitzer die Gruppe im Feld Besitzer aus.

Falls eine Implementierungsaufgabe durch eine externe Organisation ausgeführt werden muss, besprechen Sie die Aufgabe mit Ihrem Ansprechpartner im Unternehmen. Legen Sie gemeinsam mit dieser Person ein realistisches Implementierungsziel fest. Wählen Sie mithilfe des Suchsymbols neben dem Feld Besitzer sich selbst im Feld Besitzer der Aufgabe aus. Fügen Sie eine Hinweisaktion hinzu, um das externe Unternehmen anzugeben, dem die Arbeit übertragen wurde. Sobald mit der Arbeit begonnen wurde, informieren Sie den Lieferanten und geben Sie die Referenznummer, unter der der Lieferant die Zuweisung erfasst hat, in der Hinweisaktion ein.

Hinweis:

Weisen Sie nur Aufgaben zu, bei denen nicht auf den Abschluss anderer Aufgaben gewartet werden muss.
