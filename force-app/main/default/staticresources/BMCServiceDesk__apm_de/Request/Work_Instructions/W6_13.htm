Arbeitsanweisungen für Service Desk Analysts/Spezialisten

6.13.1

Wählen Sie sich selbst im Feld Besitzer jeder Aufgabe aus.

6.13.2

Öffnen Sie die erste nicht abgeschlossene Aufgabe und lesen Sie die Anweisungen in den Hinweisaktionen durch.

6.13.3

Setzen Sie das Feld Status der Aufgabe auf Angenommen, wenn Sie noch nicht mit der Aufgabe beginnen möchten.

6.13.4

Wenn Sie mit der angeforderten Arbeit beginnen können, legen Sie das Feld Status der Aufgabe auf In Bearbeitung fest.

Hinweis:

Wenn Sie die Arbeit an einer Aufgabe nicht fortsetzen können, legen Sie das Feld Status auf Warten auf... fest und fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, worauf Sie warten und wann die Aufgabe abgeschlossen werden kann.

6.13.5

Wenn Sie die angeforderte Arbeit abgeschlossen haben, legen Sie das Feld Status der Aufgabe auf Fertiggestellt fest.

Wenn die angeforderte Arbeit nicht erfolgreich abgeschlossen werden konnte, legen Sie das Feld Status der Aufgaben auf Fehlgeschlagen fest und geben Sie den Grund für das Scheitern im Feld Lösung an.

6.13.6

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion die für den Abschluss der Aufgabe benötigte Zeit ein.

6.13.7

Wenn alle Aufgaben abgeschlossen sind, fahren Sie mit Punkt 6.13.1 fort. Ansonsten gehen Sie zu Punkt 6.2.1.
