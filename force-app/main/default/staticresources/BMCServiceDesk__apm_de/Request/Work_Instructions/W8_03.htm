Arbeitsanweisungen für Service Desk Analysts

8.3.1

Wenn die Informationen im Feld Lösung von einem Spezialisten eingegeben wurden, prüfen Sie ihre Relevanz und Verständlichkeit (z. B. ob die Implementierung der Lösung durch die Angaben verdeutlicht wird). Kontaktieren Sie andernfalls den Spezialisten, um die Informationen gemeinsam mit ihm zu optimieren. Korrigieren Sie jegliche Rechtschreib- oder Grammatikfehler in dem Feld.

8.3.2

Teilen Sie dem Benutzer mit, wie die Anforderung abgeschlossen wurde, und bitten Sie ihn, die angegebene Lösung zu überprüfen. Der Benutzer kann zur Prüfung der Lösung telefonisch kontaktiert werden. Wahlweise kann der Benutzer auch informiert werden, indem das Feld Status auf Geschlossene Mail gesetzt wird, nachdem Sie im Feld Dauer (HH:MM) in der Hinweisaktion die für die abschließende Prüfung der Support-Anforderung aufgewendete Zeit (in Minuten) eingegeben haben.

Hinweis:

Durch Speichern der Anforderung mit dem Status Geschlossene Mail kann automatisch eine E-Mail an den Benutzer gesendet werden. Darin wird dem Benutzer mitgeteilt, dass die Anforderung abgeschlossen wird, und er wird darum gebeten, die Lösung zu prüfen. Außerdem wird der Benutzer in der E-Mail dazu aufgefordert, den Service Desk zu kontaktieren, falls die Lösung nicht zufriedenstellend ist.
