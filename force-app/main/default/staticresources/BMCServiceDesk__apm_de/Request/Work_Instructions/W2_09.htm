Arbeitsanweisungen für Service Desk Analysts

2.9.1

Prüfen Sie die Informationen in der Anforderung, um nachzuvollziehen, was angefordert wurde, und finden Sie heraus, warum die Anforderung von einem Gruppenkoordinator abgelehnt wurde.

2.9.2

Falls der Gruppenkoordinator die Anforderung ablehnt, da die Informationen unvollständig sind, beschaffen Sie die fehlenden Informationen und fügen Sie sie der Anforderung hinzu. (Wählen Sie z. B. im Feld CI das Konfigurationselement, für das die Anforderung registriert wurde). Fügen Sie eine Hinweisaktion mit einer Zusammenfassung der von Ihnen unternommenen Schritte hinzu.
