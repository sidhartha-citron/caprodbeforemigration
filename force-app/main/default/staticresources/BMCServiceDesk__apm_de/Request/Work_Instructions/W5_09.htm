Arbeitsanweisungen für Spezialisten

5.9.1

Fügen Sie eine Hinweisaktion hinzu, um anzugeben, weshalb die Anforderung einem anderen Spezialisten zugewiesen werden muss. Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für die Bearbeitung der Anforderung ein.

5.9.2

Falls die Anforderung einem anderen Spezialisten innerhalb Ihrer Gruppe zugewiesen werden muss, verwenden Sie das Suchsymbol neben dem Feld Besitzer, um Ihren derzeitigen Gruppenkoordinator im Feld Besitzer auszuwählen.

5.9.3

Falls ein anderer Spezialist einer anderen Gruppe die Anforderung lösen muss, verwenden Sie das Suchsymbol neben dem Feld Besitzer, um diese Gruppe im Feld Besitzer auszuwählen. Wählen Sie keinen bestimmten Mitarbeiter innerhalb der Gruppe im Feld Besitzer der Anforderung aus. Der Gruppenkoordinator muss den geeigneten Mitarbeiter (in puncto Kenntnisse, Verfügbarkeit und Zugriffsrechte) für die Bearbeitung der Anforderung bestimmen.

5.9.4

Setzen Sie das Feld Status auf Zugewiesen. Setzen Sie das Feld Status nicht auf Abgelehnt.
