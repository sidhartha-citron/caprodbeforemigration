Arbeitsanweisungen für Spezialisten

5.5.1

Kontaktieren Sie den Servicebesitzer und informieren Sie ihn über die Situation. Machen Sie deutlich, dass der Incident an den Besitzer eskaliert wurde und gemäß Verfahren 7: Incident-Eskalationsverarbeitung bearbeitet werden muss.

Hinweis:

Kontaktieren Sie den diensthabenden Manager, falls der Servicebesitzer nicht erreichbar ist.

5.5.2

Fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, dass Sie den Incident an den Servicebesitzer (oder diensthabenden Manager) eskaliert haben, da die von Ihnen vorgeschlagene Lösung nur mittels Change Management implementiert werden kann.
