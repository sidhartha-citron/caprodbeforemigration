Arbeitsanweisungen für Service Desk Analysts

2.2.1

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion die zum Registrieren der Anforderung aufgewendete Zeit ein.

2.2.2

Wählen Sie über das Suchsymbol neben dem Feld Besitzer den Service Desk-Manager.

2.2.3

Wenn Sie mit dem Benutzer am Telefon sprechen, erklären Sie, dass Sie seine Beschwerde bzw. sein Lob dem Service Desk-Manager zugewiesen haben, und teilen Sie dem Benutzer die Incident-Nummer mit. Diese Nummer finden Sie im Feld Incident-Nummer. Durch die Angabe der Incident-Nummer wird dem Benutzer versichert, dass die Beschwerde bzw. das Lob registriert wurde.

Hinweis:

Der Benutzer kann auch nach Speichern der Anforderung per E-Mail informiert werden.
