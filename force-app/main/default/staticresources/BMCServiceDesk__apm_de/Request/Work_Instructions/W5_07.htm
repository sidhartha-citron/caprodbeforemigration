Verfahrensschritt

Arbeitsanweisungen für Spezialisten

5.7.1

Lösen Sie die Anforderung. Fordern Sie bei Bedarf Support durch externe Anbieter an.

Hinweis:

Falls die Anforderung nicht vor dem im Feld Fälligkeitsdatum festgelegten Zeitpunkt gelöst werden kann, setzen Sie eine realistische Frist für die Lösung der Anforderung und fügen Sie eine Hinweisaktion hinzu, um diese Informationen in die Anforderung einzugeben.

Hinweis:

Setzen Sie das Feld Status auf Waiting for (Warten auf) oder Waiting for customer (Warten auf Kunden), falls die Arbeit nicht fortgesetzt werden kann (da Sie beispielsweise auf ein Ersatzteil oder Informationen von einem Kunden warten). Fügen Sie eine Hinweisaktion hinzu und geben Sie an, worauf Sie warten. Wenn Sie eine Anforderung bei einem Lieferanten registriert haben, geben Sie diesen Lieferanten sowie die Anforderungsnummer ein, die der Lieferant in der Hinweisaktion angegeben hat.

Hinweis:

Wenn Sie die Bearbeitung der Anforderung abgeschlossen haben, jedoch weitere Maßnahmen zum Lösen der Anforderung erforderlich sind, fügen Sie eine Hinweisaktion hinzu und geben Sie an, welche Schritte Sie unternommen haben. Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für die Bearbeitung der Anforderung ein. Falls ein bestimmter Mitarbeiter Ihrer Gruppe zusätzliche Aufgaben ausführen muss, wählen Sie mithilfe des Suchsymbols neben dem Feld Besitzer den Mitarbeiter im Feld Besitzer aus. Falls eine andere Gruppe die zusätzlichen Aufgaben ausführen muss, wählen Sie mithilfe des Suchsymbols neben dem Feld Besitzer die Gruppe im Feld Besitzer aus. Vergewissern Sie sich, dass das Feld Status auf Zugewiesen gesetzt ist.
