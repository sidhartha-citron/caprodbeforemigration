Arbeitsanweisungen für Service Desk Analysts

8.2.1

Beschreiben Sie im Feld Lösung, wie die Anforderung gelöst wird. .

8.2.2

Wählen Sie das betreffende Konfigurationselement (CI) im Feld CI aus, falls dies noch nicht geschehen ist oder das falsche CI ausgewählt wurde.

Hinweis:

Falls das CI nicht in der CMDB registriert ist, jedoch unter den Konfigurationsmanagement-Prozess fällt, wählen Sie das CI mit dem Code NORECORD.

Das CI mit dem Code NORECORD steht möglicherweise nicht standardmäßig zur Verfügung.

Hinweis:

Falls die Anforderung kein bestimmtes CI betrifft oder das CI nicht unter den Konfigurationsmanagement-Prozess fällt, wählen Sie das CI mit dem Code N/A.

8.2.3

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion die Zeit ein, die Sie für die Registrierung und Lösung der Anforderung benötigt haben.

8.2.4

Wählen Sie die entsprechende Schließungskategorie im Feld Schließungskategorie aus. Setzen Sie das Feld Status der Anforderung auf Fertiggestellt.

8.2.5

Vergewissern Sie sich, dass Sie im Feld Besitzer der Aufgabe ausgewählt sind.

8.2.6

Führen Sie eine letzte Prüfung durch, um sich zu vergewissern, dass die Anforderung korrekt ausgefüllt ist.

8.2.7

Falls die Umgehungslösung eines offenen Problems für die Lösung der Anforderung verwendet wird, verknüpfen Sie den Incident mit dem Problem.
