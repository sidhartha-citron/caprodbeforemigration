Arbeitsanweisungen für Service Level Administratoren

1.7.1

Setzen Sie das Feld Status der Aufgabe auf In Bearbeitung.

1.7.2

Fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, weshalb der Katalogartikel abgelehnt wurde und was geändert werden muss, damit der Artikel genehmigt werden kann.

1.7.3

Öffnen Sie die digitale Kopie des abgelehnten Katalogartikels.

1.7.4

Passen Sie den abgelehnten Katalogartikel gemäß den Anforderungen an, damit er genehmigt werden kann.
