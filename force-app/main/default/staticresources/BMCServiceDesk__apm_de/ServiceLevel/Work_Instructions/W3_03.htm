Arbeitsanweisungen für Service Level Administratoren

3.3.1

Um die Kontaktdetails eines einzelnen Kunden zu aktualisieren, verwenden Sie das Kundenformular.

3.3.2

Zum Aktualisieren der Informationen über ein Kundenunternehmen verwenden Sie das Formular „Konto“. Um die Kontaktdetails eines Kundenvertreters zu aktualisieren, verwenden Sie das Kundenformular. Verwenden Sie das Kundenformular auch zum Registrieren oder Aktualisieren der Kontaktinformationen anderer Personen, die Support von dem Serviceanbieter erhalten.
