Arbeitsanweisungen für Service Level Administratoren

3.4.1

Nachdem Sie aktualisierte Kontaktinformationen eines bestehenden einzelnen Kunden erhalten haben, aktualisieren Sie die Kontaktdaten im Kundenformular.

3.4.2

Verwenden Sie zum Aktualisieren der Kontaktdaten eines bestehenden Kundenunternehmens das Formular „Konto“. Um die Kontaktdetails eines Kundenvertreters zu aktualisieren, verwenden Sie das Kundenformular. Verwenden Sie das Formular „Konto“ auch zum Registrieren oder Aktualisieren der Kontaktinformationen anderer Personen, die Support von dem Serviceanbieter erhalten.
