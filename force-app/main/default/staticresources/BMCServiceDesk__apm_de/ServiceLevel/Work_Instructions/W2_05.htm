Arbeitsanweisungen für Service Level Administratoren

2.5.1

Senden Sie die beiden Ausdrucke des neuen SLA an den Kunden, der den für das neue SLA erstellten Katalogartikel ausgewählt hat. Bitten Sie den Kundenvertreter, beide Kopien unterzeichnet an Sie zurückzusenden.

2.5.2

Setzen Sie das Feld Status der Aufgabe auf Waiting for…( Warten auf) und fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, dass Sie auf die Unterzeichnung des SLA durch den Kundenvertreter warten.

2.5.3

Wenn Sie die beiden unterzeichneten Kopien des neuen SLA erhalten haben, leiten Sie sie an den Servicebesitzer weiter. Bitten Sie den Servicebesitzer, beide Kopien unterzeichnet an Sie zurückzusenden.

2.5.4

Fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, dass Sie auf die Unterzeichnung des SLA durch den Servicebesitzer warten. Behalten Sie im Feld Status den Wert Waiting for… (Warten auf) bei.

2.5.5

Setzen Sie das Feld Status auf In Bearbeitung zurück, nachdem beide Kopien des SLA mit den Unterschriften des Kundenvertreters und des Servicebesitzers an Sie zurückgesendet wurden.
