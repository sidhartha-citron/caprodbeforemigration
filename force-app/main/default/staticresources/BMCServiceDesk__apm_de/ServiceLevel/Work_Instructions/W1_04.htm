Arbeitsanweisungen für Service Level Administratoren

1.4.1

Sie können von den folgenden Personen eine Genehmigung für den neuen Katalogartikel anfordern:

  • dem Servicebesitzer, der für die Erfüllung der im Katalogartikel angegebenen SLTs verantwortlich ist

  • den Service Level-Managern der Kundenunternehmen, für die der Katalogartikel erstellt wurde

  • dem Finanzmanager des Unternehmens, das den Service bereitstellt

Hinweis:

Stellen Sie sicher, dass Sie eine gedruckte Fassung des Katalogartikels mit den Genehmigungsunterschriften erhalten.

Hinweis:

Wird der Katalogartikel nicht für einen bestimmten Kunden erstellt, wird ein bestimmter Kundentyp als Zielgruppe festgelegt. In diesem Fall sollten die Service Level-Manager den neuen Katalogartikel dennoch genehmigen.

1.4.2

Legen Sie im Feld Status der Aufgabe den Wert Waiting for… (Warten auf) fest und geben Sie im Bereich Action History (Aktionsverlauf) an, dass Sie auf die Genehmigung des Katalogartikels warten.

1.4.3

Geben Sie in den Hinweisaktionen im Feld Dauer (HH:MM) die zum Erstellen des Katalogartikels aufgewendete Zeit ein.
