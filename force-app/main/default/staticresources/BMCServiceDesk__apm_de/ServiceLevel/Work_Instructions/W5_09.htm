Arbeitsanweisungen für Service Level Manager

5.9.1

Halten Sie die Aufzeichnungen zur SLA-Prüfungssitzung in einem geeigneten Format in endgültiger Form fest. Wenn der Kundenvertreter Anforderungen einsendet, nehmen Sie deren Nummern mit auf.

5.9.2

Senden Sie eine E-Mail mit den Aufzeichnungen zur SLA-Prüfungssitzung an die folgenden Personen:

  • Kundenvertreter

  • Servicebesitzer, die Services für das Kundenunternehmen bereitstellen

  • Service Desk Manager
