Arbeitsanweisungen für Service Level Administratoren

4.5.1

Geben Sie im Feld Lösung der Aufgabe an, welche Service Level Management-Informationen Sie aktualisiert haben. Falls die Informationen nicht wie angefordert hinzugefügt oder geändert werden konnten, geben Sie den Grund dafür an (z. B. da der neue Katalogartikel oder das neue SLA nicht unterzeichnet wurde).

4.5.2

Setzen Sie das Feld Status in der Aufgabe auf Fertiggestellt, wenn die Service Level-Management-Informationen wie angefordert aktualisiert wurden. Legen Sie im Feld Status den Wert Fehlgeschlagen fest, wenn Sie die Informationen nicht aktualisieren konnten.

4.5.3

Geben Sie im Feld Dauer (HH:MM) in einer Hinweisaktion die für den Abschluss der Aufgabe benötigte Zeit ein.
