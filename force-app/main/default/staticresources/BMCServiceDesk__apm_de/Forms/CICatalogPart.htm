Unterkategorie

Verwenden Sie dieses Feld, um die passende Unterkategorie für das CI-Katalogteil aus der folgenden Liste von Optionen auszuwählen. Beachten Sie, dass Sie über dieses Feld nur Unterkategorien auswählen können, die für die ausgewählte Kategorie relevant sind.

Parameter (J/N)

Asset-/Kennzeichnungs-Nr. nachverfolgen

Aktivieren Sie das Kontrollkästchen in diesem Feld, um die Asset-/Kennzeichnungsnummer der Konfigurationselemente nachzuverfolgen, welche auf Grundlage des CI-Katalogteils registriert wurden.

Serien-Nr. nachverfolgen

Aktivieren Sie das Kontrollkästchen in diesem Feld, um die Seriennummer der Konfigurationselemente nachzuverfolgen, welche auf Grundlage des CI-Katalogteils registriert wurden.

Versions-Nr. nachverfolgen

Aktivieren Sie das Kontrollkästchen in diesem Feld, um die Versionsnummer der Konfigurationselemente nachzuverfolgen, welche auf Grundlage des CI-Katalogteils registriert wurden.

Gesamtzahl
