Feld

Verwendung

Attribute > Allgemein

Instanzname

Geben Sie in diesem Feld den eindeutigen Namen für den Service ein.

Beschreibung

Geben Sie in diesem Feld eine Beschreibung des Service ein.

Kurzbeschreibung

Geben Sie in diesem Feld eine Kurzbeschreibung des Service ein. Diese Beschreibung wird im CI-Explorer angezeigt.

Geben Sie in diesem Feld zusätzliche Informationen zu dem Service ein.

Zuletzt geändert von

Dieses Feld ist automatisch auf die Klassen-ID des Elements eingestellt, das zur Registrierung des Service verwendet wurde.

Verknüpfte Listen oder zusätzliche Informationen

Historie

In diesem Feld werden die für den Service erstellten Verlaufshinweise angegeben.

Kunden

Auf dieser Registerkarte werden die mit dem Service verknüpften Kunden aufgeführt. Wählen Sie auf dieser Registerkarte den Kunden, der den Service verwenden wird oder für den dieser reserviert ist. Wird der Service nicht von einem bestimmten Kunden verwendet oder ist er für keinen bestimmten Kunden reserviert, wählen Sie keinen Kunden.

Konten

Auf dieser Registerkarte werden die mit dem Service verknüpften Konten aufgeführt. Wählen Sie auf dieser Registerkarte die Unternehmen, die den Service verwenden werden oder für die dieser reserviert ist. Wird der Service nicht von einem bestimmten Unternehmen verwendet bzw. ist er für kein bestimmtes Unternehmen reserviert, wählen Sie kein Konto.

Incidents

Auf dieser Registerkarte werden die mit dem Service verknüpften Incidents aufgeführt. Verwenden Sie diese Registerkarte zum Verknüpfen von Incidents mit dem Service.

Change Requests

In diesem Bereich werden die mit dem Service verknüpften Change Requests aufgeführt. Verwenden Sie diese Registerkarte zum Verknüpfen von Change Requests mit dem Service.

Probleme

In diesem Bereich werden die mit dem Service verknüpften Probleme aufgeführt. Verwenden Sie diese Registerkarte zum Verknüpfen von Problemen mit dem Service.

Releases

In diesem Bereich werden die mit dem Service verknüpften Releases aufgeführt. Verwenden Sie diese Registerkarte zum Verknüpfen von Releases mit dem Service.

Dokumentation

In diesem Bereich werden die mit dem Service verknüpften Dokumente aufgeführt. Verwenden Sie diesen Bereich zum Anfügen von Dateien, die zusätzliche Informationen über den Service enthalten.

Prüfung

Auf dieser Registerkarte werden die an den Informationen zum Service vorgenommenen Änderungen aufgelistet.
