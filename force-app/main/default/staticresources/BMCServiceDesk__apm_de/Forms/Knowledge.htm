Knowledge-Artikel


In der untenstehenden Tabelle werden die Felder des Knowledge-Artikel-Formulars aufgeführt und Verwendungsrichtlinien für jedes Feld bereitgestellt.

Feld

Verwendung

Record-Typ des neuen Record

Verwenden Sie dieses Feld, um den Record-Typ aus der folgenden Liste auszuwählen:

  • FAQ-Artikel mit Antworten auf häufig gestellte Fragen;

  • Anleitungsartikel, welche Schritte zur Ausführung einer Aufgabe enthalten;

  • „Bekannte Fehler“-Artikel mit Informationen zu bekannten Fehlern und möglichen Umgehungslösungen;

  • Problemlösungsartikel, welche Lösungen zu einem bestimmten Problem enthalten.

Artikelinformationen

Besitzer

In diesem Feld wird der Name der Person angezeigt, die den Knowledge-Artikel erstellt hat.

Kategorie

Wählen Sie in diesem Feld die Kategorie des Knowledge-Artikels aus.

Status

Verwenden Sie dieses Feld, um den Status aus folgender Liste auszuwählen:

  • Entwurf

  • Wird überprüft

  • Veröffentlicht

  • Stillgelegt

In Selbstbedienung anzeigen

Verwenden Sie dieses Feld, um anzugeben, ob der Knowledge-Artikel in Selbstbedienung angezeigt werden kann.

Artikeldetails

Titel

Geben Sie in diesem Feld einen kurzen Titel für den Knowledge-Artikel ein.

Schlüsselwörter

Geben Sie in diesem Feld die Schlüsselwörter für den Knowledge-Artikel ein. Bei Suchvorgängen wird der Suchbegriff mit Schlüsselwörtern verglichen, um passende Knowledge-Artikel anzuzeigen.

Record-Typ: Häufig gestellte Frage (FAQ) und Anleitung

Frage

Geben Sie in diesem Feld die Frage für den FAQ- oder Anleitungsartikel ein.

Antwort

Geben Sie in diesem Feld die Antwort für den FAQ- oder Anleitungsartikel ein.

Record-Typ: Bekannter Fehler

Fehler

Verwenden Sie dieses Feld, um die Fehlerbeschreibung für den bekannten Fehler einzugeben.

Ursache

Geben Sie in diesem Feld die Ursache für den bekannten Fehler ein.

Umgehungslösung

Geben Sie in diesem Feld die Umgehungslösung (sofern verfügbar) für den bekannten Fehler ein.

Record-Typ: Problemlösung

Problem

Geben Sie in diesem Feld die Beschreibung des Problems ein.

Lösung

Geben Sie in diesem Feld die Lösung für das Problem ein.

Weitere Informationen

Feld

Verwendung

Veröffentlichungsdatum

In diesem Feld werden automatisch Datum und Uhrzeit der Veröffentlichung des Knowledge-Artikels eingegeben.

Überprüfungsdatum

In diesem Feld werden automatisch Datum und Uhrzeit für die nächste Überprüfung des Artikels eingegeben.

Stilllegungsdatum

In diesem Feld werden automatisch Datum und Uhrzeit für die Stilllegung des Artikels eingegeben.

Ansichten in Selbstbedienung

In diesem Feld wird angezeigt, wie oft der Artikel in Selbstbedienung angezeigt wurde.

Verknüpfte Listen

Notizen und Anhänge

In diesem Bereich werden die mit dem Knowledge-Artikel in Verbindung stehenden Dokumente aufgelistet. Verwenden Sie diesen Bereich zum Anhängen von Dateien oder Notizen, die zusätzliche Informationen zum Knowledge-Artikel enthalten.

Knowledge-Rückmeldung

Dieser Bereich enthält eine Liste mit Rückmeldungen für den Knowledge-Artikel.

Verknüpfte Incidents

In diesem Bereich werden die mit dem Knowledge-Artikel in Verbindung stehenden Incidents aufgelistet. Verwenden Sie diesen Bereich zum Anhängen von Incidents an den Knowledge-Artikel.

Genehmigungsverlauf

In diesem Bereich sind die Genehmigungs-Records für den Knowledge-Artikel aufgeführt.
