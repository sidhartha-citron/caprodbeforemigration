Arbeitsanweisungen für Problem-Koordinatoren

4.6.1

Setzen Sie das Feld Status auf Dead-End, um anzugeben, dass derzeit keine weitere Aktion erforderlich ist.

Hinweis:

Wenn das Feld Status bereits auf Dead-End gesetzt wurde, weil Sie eben eine der regelmäßigen Prüfungen auf neue Möglichkeiten durchgeführt haben, geben Sie dies in der Hinweisaktion an.

4.6.2

Wählen Sie im Feld Closure Category (Abschlusskategorie) die geeignete Kategorie aus.

4.6.3

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für das Prüfen des Problems ein.

4.6.4

Prüfen Sie den Knowledge-Artikel und aktualisieren Sie ihn mit den neuesten Informationen, falls erforderlich.
