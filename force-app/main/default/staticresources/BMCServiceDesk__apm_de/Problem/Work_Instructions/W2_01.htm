Arbeitsanweisungen für Spezialisten

2.1.1

Wenn Sie noch nicht am Problem arbeiten können, setzen Sie das Feld Status auf Angenommen.

2.1.2

Sobald Sie das Problem analysieren können, setzen Sie das Feld Status auf In Bearbeitung.

2.1.3

Überprüfen Sie die Informationen zum Problem, um dessen Ursache sowie die bisherigen und erwarteten Auswirkungen auf den verknüpften Service zu verstehen.

Hinweis:

Wenn das Problem einem anderen Spezialisten zugewiesen werden soll, bitten Sie den Problem-Koordinator, seine Zuweisung durch Ändern des Felds Status in Abgelehnt zu ändern und fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, warum Sie das Problem abgelehnt haben.
