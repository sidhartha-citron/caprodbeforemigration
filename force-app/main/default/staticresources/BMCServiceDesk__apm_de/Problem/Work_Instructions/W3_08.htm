Arbeitsanweisungen für Problem-Koordinatoren

3.8.1

Bestimmen Sie den Spezialisten, der sich am besten zum Implementieren der vorgeschlagenen strukturellen Lösung eignet. Dabei kann es sich um denselben Spezialisten handeln, der das Problem analysiert hat . Verwenden Sie die Option Ändern neben dem Feld Besitzer, um diesen Spezialisten auszuwählen.

3.8.2

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion die für das Prüfen und Zuweisen des Problems benötigte Zeit ein.

3.8.3

Setzen Sie das Feld Status auf Zugewiesen.
