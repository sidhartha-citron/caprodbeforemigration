Arbeitsanweisungen für Spezialisten

2.9.1

Wenn Sie noch nicht mit der Arbeit beginnen können, setzen Sie das Feld Status auf Angenommen.

2.9.2

Sobald Sie mit der Arbeit beginnen können, setzen Sie das Feld Status auf In Bearbeitung.

2.9.3

Lesen Sie die Angaben in den Hinweisaktionen des Problems, um zu erfahren, wie die strukturelle Lösung zu implementieren ist.

Hinweis:

Wenn das Problem einem anderen Spezialisten zugewiesen werden soll, bitten Sie den Problem-Koordinator, seine Zuweisung durch Ändern des Felds Status in Abgelehnt zu ändern.Fügen Sie eine Hinweisaktion hinzu, in der Sie angeben, warum Sie das Problem abgelehnt haben.
