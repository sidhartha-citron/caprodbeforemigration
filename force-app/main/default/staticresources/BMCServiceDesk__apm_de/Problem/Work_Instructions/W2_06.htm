Arbeitsanweisungen für Spezialisten

2.6.1

Nachdem Sie die Ursache ermittelt haben, entfernen Sie sie oder suchen Sie nach einer dauerhaften Umgehungslösung.

Hinweis:

Wenn das Konfigurationselement (CI), das das Problem verursacht hat, noch von einem externen Lieferanten unterstützt wird (d. h. das CI wird noch durch eine Garantie oder einen Support-Vertrag abgedeckt), können Sie eine Lösungsanforderung für das Problem an den Lieferanten senden.

Um zu bestimmen, ob das CI noch von einem externen Lieferanten unterstützt wird, öffnen Sie das CI und klicken Sie auf die Registerkarte Spezifikationen, damit das Feld Warranty-Ablaufdatum angezeigt wird.

2.6.2

Fügen Sie eine Hinweisaktion hinzu, um alle strukturellen Lösungen zu beschreiben.

2.6.3

Stellen Sie fest, welche der Vorschläge die praktischste strukturelle Lösung darstellt (unter Berücksichtigung technischer und finanzieller Gesichtspunkte bzw. der Verfügbarkeit). Fügen Sie eine Hinweisaktion hinzu, um anzugeben, welche strukturelle Lösung Sie für die praktischste halten. Geben Sie Ihre Gründe für diese Entscheidung an.

2.6.4

Fügen Sie eine Hinweisaktion hinzu, um konkret zu beschreiben, wie die bevorzugte strukturelle Lösung implementiert werden soll. Wenn eine frühere Umgehungslösung entfernt werden muss, geben Sie im selben Feld die Vorgehensweise hierfür an.

Hinweis:

Fügen Sie eine Hinweisaktion mit Gründen hinzu, falls keine praktische strukturelle Lösung gefunden wurde.
