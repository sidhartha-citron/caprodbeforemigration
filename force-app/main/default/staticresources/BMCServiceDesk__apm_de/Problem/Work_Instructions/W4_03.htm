Arbeitsanweisungen für Problem-Koordinatoren

4.3.1

Fassen Sie im Feld Lösung zusammen, wie die strukturelle Lösung implementiert wurde.

4.3.2

Geben Sie im Feld Dauer (HH:MM) in der Hinweisaktion den Zeitaufwand für das Verifizieren der strukturellen Lösung ein.

4.3.3

Setzen Sie das Feld Status auf Gelöst.

4.3.4

Wählen Sie im Feld Closure Category (Abschlusskategorie) die geeignete Kategorie aus.

Hinweis:

Wenn die Lösung im Feld Lösung des Problems angegeben ist, kann der Service Desk per E-Mail von der Lösung in Kenntnis gesetzt werden.

4.3.5

Falls die Ursache entfernt wurde und der Knowledge-Artikel für dieses Problem nicht länger benötigt wird, entfernen Sie den Artikel aus der Knowledge Base.

4.3.6

Falls erforderlich, erstellen Sie einen Knowledge-Artikel mit der Lösung aus der Knowledge Base.
