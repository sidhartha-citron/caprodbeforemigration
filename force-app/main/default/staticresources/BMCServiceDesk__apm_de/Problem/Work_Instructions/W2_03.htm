Arbeitsanweisungen für Spezialisten

2.3.1

Suchen Sie nach einer temporären Umgehungslösung, die zum Lösen von Incidents verwendet werden kann, die das Problem verursacht hat. Sehen Sie sich das Feld Lösung der Anforderungen an, die mit dem Problem verknüpft sind.

2.3.2

Wenn Sie eine Umgehungslösung anbieten können, beschreiben Sie sie (Schritt für Schritt) in einem neuen Knowledge-Artikel und fügen Sie im Feld Umgehungslösung des Problems einen Link zum Artikel ein.

Hinweis:

Die Service Desk Analysten und/oder Spezialisten können die Umgehungslösung zum Lösen von von diesem Problem verursachten Incidents verwenden, bis Sie eine strukturelle Lösung anbieten können.

Hinweis:

Wenn im Feld Umgehungslösung bereits eine Umgehungslösung angegeben ist, prüfen Sie sie und verbessern Sie sie wenn möglich.

2.3.3

Falls es nicht möglich ist, eine praktische Umgehungslösung anzubieten, geben Sie im Feld Umgehungslösung einen Grund hierfür an.
