Prozessmodell-Hilfe – Symbole


Die Prozesse und Verfahren des Alignability®-Prozessmodells werden mithilfe einer vereinfachten Konvention mit nur drei Symbolen dargestellt: dem Ereigniskreis, dem Aktionsfeld und der Entscheidungsraute. Die Bedeutung jedes dieser drei Symbole wird unten erklärt.

Symbol

Beschreibung

Das Aktionsfeld symbolisiert eine Aktion, die ausgeführt werden muss.
Ein Pfeil zeigt immer auf ein Aktionsfeld.
Ein Pfeil zeigt von einem Aktionsfeld weg, wenn nach Abschluss der Aktion weitere Maßnahmen erforderlich sind.

Die Entscheidungsraute steht für eine Entscheidung, die getroffen werden muss.
Ein Pfeil zeigt immer auf eine Entscheidungsraute.
Zwei Pfeile zeigen immer von einer Entscheidungsraute weg.

Der Ereigniskreis steht für ein Ereignis oder eine Situation, das bzw. die eingetreten ist.
Auf einen Ereigniskreis zeigt nie ein Pfeil.
Ein Pfeil zeigt immer von einem Ereigniskreis weg.
