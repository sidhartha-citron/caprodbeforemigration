Prozessmodell-Hilfe – Struktur


Die vierschichtige Struktur des Alignability®-Prozessmodells stellt sicher, dass die Informationen, nach denen Sie suchen, höchstens drei Mausklicks vom Modell selbst entfernt sind. Eine grafische Übersicht der Struktur wird unten angezeigt.

Struktur


Um vom Modell aus auf einen Prozess zuzugreifen, klicken Sie einfach auf das Feld, das den Prozess repräsentiert.

Nachdem der Prozess auf Ihrem Bildschirm dargestellt wurde, können Sie auf eines seiner Verfahren zugreifen, indem Sie auf eines der Felder klicken, die den Prozess darstellen.

Um auf die Arbeitsanweisungen für einen Schritt innerhalb eines Verfahrens zuzugreifen, klicken Sie einfach auf das Aktionsfeld oder die Ereignisraute, die für diesen Schritt steht.
