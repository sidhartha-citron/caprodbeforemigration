Verfahren 2, Knowledge-Genehmigung


Der Knowledge Manager erhält eine Benachrichtigung, wenn ein Artikel zur Genehmigung übermittelt wird. Der Knowledge Manager prüft den Artikel und nimmt gegebenenfalls Änderungen vor. Wenn ein Artikel zur Verfügung gestellt werden soll, wird er vom Knowledge Manager veröffentlicht. Andernfalls zieht der Knowledge Manager den Artikel zurück.
