Arbeitsanweisungen für Operatoren

1.1.1

Klicken Sie auf die Registerkarte Knowledge-Artikel und anschließend auf Neu. Wählen Sie den korrekten Record-Typ aus. Wählen Sie die entsprechende Kategorie aus und geben Sie den Titel ein. Geben Sie die Einzelheiten zum Knowledge-Artikel in den verfügbaren Feldern ein. Wählen Sie Zur Genehmigung übermitteln und übermitteln Sie den Artikel zur Genehmigung.

Hinweis:

Falls der Artikel über den Request Management-Prozess erstellt wird, wählen Sie den Record-Typ FAQ oder Anleitung aus. Falls der Artikel über den Problem Management-Prozess erstellt wird, wählen Sie den Record-Typ Bekannter Fehler oder Problemlösung aus. Falls der Artikel über den Change Management-Prozess erstellt wird, können Sie einen beliebigen Record-Typ auswählen.
