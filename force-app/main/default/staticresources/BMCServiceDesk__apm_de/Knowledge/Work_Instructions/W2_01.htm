Arbeitsanweisungen für Operations Manager

2.1.1

Wenn Sie eine Benachrichtigung erhalten, die die Genehmigung für eine vorgeschlagene Lösung anfordert, öffnen Sie sie und prüfen Sie die Lösungsinformationen auf Vollständigkeit und Richtigkeit.

2.1.2

Falls Sie die Lösung nicht verstehen oder nicht nachvollziehen können, warum diese zur Verfügung gestellt werden muss, wenden Sie sich an den Spezialisten, der die Lösung vorgeschlagen hat.
