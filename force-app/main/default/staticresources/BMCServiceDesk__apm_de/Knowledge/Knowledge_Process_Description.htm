Prozessbeschreibung


Der Knowledge Management-Prozess besteht aus folgenden Verfahren:

  • Knowledge-Erstellung: Wird von Spezialisten bei der Erstellung neuer Knowledge-Artikel verwendet.

  • Knowledge-Genehmigung: Wird von Knowledge Managern bei der Genehmigung von Knowledge-Artikeln verwendet.

  • Knowledge-Prüfung: Wird von Knowledge Managern bei der Überprüfung vorhandener Artikel verwendet.

Weitere Informationen zu diesen Verfahren erhalten Sie, wenn Sie auf Prozess klicken, um zur grafischen Darstellung zurückzukehren, und danach auf die Felder, welche die Verfahren verkörpern. Bei der grafischen Darstellung eines Verfahrens wird deren Beschreibung im rechten Bereich angezeigt.
