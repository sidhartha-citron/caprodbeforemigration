Top service requests are templates that fill in fields that are the same for a common issue. The system administrator creates these top service requests and makes these requests available for your use. Top service requests speed up data entry, improve data accuracy, and ensure consistent reporting.

To use a top service request

  1. In the Self Service Home page, in the Top Service Requests section, browse through the top service requests available.
  2. Click the top service request that suits your requirements.

    3. The values of the fields in the top service request are filled in the respective fields of the new service request.

  3. In the remaining fields of the new service request, make the required updates.

    4. For more information, see Creating a service request.

  4. Click Submit.
