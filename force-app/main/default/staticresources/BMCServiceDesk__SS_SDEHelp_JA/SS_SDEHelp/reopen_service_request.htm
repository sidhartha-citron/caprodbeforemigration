新しいサービスリクエストの作成の代わりにサービスリクエストの再開が必要になるのは、元のサービスリクエストで報告された問題が依然として解決されない場合です。またサービスリクエストに対して提示された解決策で、サービスリクエストが解決されず、さらに対応が必要になっている場合は、サービスリクエストを再度開く必要があります。

注意: サービスリクエストの再開ができるのは、Self Serviceクライアントがこのオプションを使用できるようにシステム管理者によって設定されている場合だけです。



サービスリクエストを再開するには

  1. 終了するサービスリクエストを検索するには、［検索］フィールドに検索文字列を入力して［検索］をクリックします。

    2. ［サービスリクエスト］セクションには、検索条件に一致するレコードが表示されます。詳細は、送信済みサービスリクエストの表示を参照してください。

  2. (オプション) 検索結果をクリアするには、［元に戻す］をクリックします。

    3. 検索結果がクリアされ、［サービスリクエスト］セクションのレコードのリストが更新されます。

  3. 再開させるサービスリクエストをダブルクリックします。
  4. ［サービスリクエストの再開］をクリックします。
  5. 確認ダイアログボックスで、［OK］をクリックします。
