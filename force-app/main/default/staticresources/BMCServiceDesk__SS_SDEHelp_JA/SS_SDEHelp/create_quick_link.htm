クイックリンクを作成するには

  1. サイドバーの右上のセクションにある［クイックリンク］ポートレットのアイコンをクリックします。
  2. ［クイックリンク］ウィンドウの［新規］ボタンをクリックします。

    4. この名前のクイックリンクが、サイドバーの［クイックリンク］ポートレットに表示されます。

  4. ［URL］フィールドに、クイックリンクのURLを入力します。

    5. このクイックリンクをクリックしたときに、このURLが新しいブラウザウィンドウで表示されます。

  5. ［送信］をクリックします。

    6. クイックリンクが［クイックリンク］ポートレットの［自分のリンク］セクションに表示されます。
