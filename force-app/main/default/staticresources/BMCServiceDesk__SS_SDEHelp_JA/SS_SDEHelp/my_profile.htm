自分の住所や電話番号に変更があった場合は、プロファイルのこれらの情報を更新することができます。更新された詳細はSelf Serviceによって作成したすべてのサービスリクエストにクライアント情報が記録される時に使用されます。



プロファイルの詳細を編集するには

  1. Self Serviceホームページの右上のセクションにある［自分のプロファイル］をクリックします。
  2. ［自分のプロファイル］ページで、必要に応じてプロファイルの詳細を更新します。
  3. 自分のプロファイルが［Salesforce］、［Salesforceプラットフォーム］、または［Chatterのみ］ライセンスを使用して作成されており、http://login.salesforce.com URLを使用してセルフサービスにログオンしている場合は、［設定］>［私の個人情報］>［個人情報］の順に移動し、［編集］をクリックします。

    4. 自分のプロファイルが［Salesforce］、［Salesforceプラットフォーム］、または［Chatterのみ］ライセンスを使用して作成されている場合は、［自分のプロファイル］リンクは表示されません。

    ［ブロードキャストティッカーの速度］リストを選択して、ブロードキャストティッカーの速度を調整します。［ブロードキャストティッカーの速度］フィールドが表示されない場合は、システム管理者に問い合わせてください。

  4. ［送信］をクリックします。
