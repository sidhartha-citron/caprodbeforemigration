In the Catalog console, you can create new folders and subfolders and add the QuickViews to these folders and subfolders. This allows you to organize your QuickViews in separate folders and quickly access the QuickViews that you use most frequently.

After you create the required folders, you can drag existing QuickViews to the folders to quickly access them. The drag functionality is available only for QuickViews. You cannot drag a folder to make it a subfolder of an existing folder. To make all QuickViews in a folder as a subfolder of an existing folder, you must create the required subfolder and drag the QuickViews to the subfolder.

Note: You cannot drag QuickViews from the All Available QuickViews folder to the My QuickViews folder or its subfolders. In addition, you cannot drag any QuickViews from the My QuickViews folder or its subfolders to the All Available QuickViews folder.

The following sections describe how to create folders and manage the QuickViews in the folders:

    • Creating new folders
    • Creating subfolders
    • Deleting folders
    • Adding QuickViews to folders
    • Deleting QuickViews

Creating new folders

You can create a new folder in the QuickViews folder to group the QuickViews that you use frequently or to quickly access related QuickViews. For example, you can create a folder called My Incidents and add a QuickView that displays the incidents that you must work on, or a QuickView of all your incidents sorted by priority, and so on.

To create a new folder

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Create Dashboards and Reports tile, and from the menu select Dashboards (Remedyforce).
  3. From the list of Catalog options, expand QuickViews.
  4. Click Add Folder.
  5. In the Add Folder window, in the Folder Name field, type the name of the folder.
  6. Click OK.

Creating subfolders

You can create subfolders for an existing folder to organize your QuickViews at a detailed level. For example, you can create a My Work folder. In this folder, you can create these subfolders: My Incidents, My Tasks, and My Broadcasts. In these subfolders, you can add the QuickViews that display the incidents, tasks, broadcasts, change requests, and problems assigned to you. In the My Incidents subfolders, you can add QuickViews that display all incidents assigned to you, all high priority incidents, and all incidents whose due date is in the coming week. Similarly, you can add the required task and broadcast QuickViews in the My Tasks and My Broadcasts subfolders.

To create a subfolder

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Create Dashboards and Reports tile, and from the menu select Dashboards (Remedyforce).
  3. From the list of Catalog options, expand QuickViews.
  4. Select the folder where you want to create a subfolder.
  5. Click Add Folder.
  6. In the Add Folder window, in the Folder Name field, type the name of the subfolder.
  7. Click OK.

Deleting folders

If you have created QuickViews that are no longer being used and all these QuickViews reside in one folder, you can delete all the QuickViews by deleting the folder. For example, if you have a My Broadcasts folder that contains all QuickViews for broadcasts assigned to you and you no longer work on broadcasts, you can delete the My Broadcasts folder, and all its QuickViews are deleted.

To delete a folder

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Create Dashboards and Reports tile, and from the menu select Dashboards (Remedyforce).
  3. From the list of Catalog options, expand QuickViews.
  4. Select the folder you want to delete.
  5. Click Delete Folder.
  6. In the confirmation dialog box, click Yes.

    7. Note: When you delete a folder, all QuickViews in the folder are deleted.

Adding QuickViews to folders

You can add QuickViews to a folder by creating a new QuickView in the folder or copying an existing QuickView into the folder.

To add a QuickView to a folder

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Create Dashboards and Reports tile, and from the menu select Dashboards (Remedyforce).
  3. From the list of Catalog options, expand QuickViews.
  4. To add an existing QuickView to the folder, perform the following actions:
    1. Navigate to the folder that contains the QuickView you want to add.
    2. Select the required QuickView.
    3. Click Copy Query.
    4. Select the folder where you want to paste the QuickView.
    5. Click Paste Query.
    6. In the Add Query window, in the QuickView Name field, type the name of the QuickView.
    7. Click OK.
  5. To create a new QuickView in the folder, perform the following actions:
    1. Navigate to the folder where you want to create the QuickView.
    2. Click New Query.
    3. In the QuickViews wizard, create the QuickView.

      4. For more information, see Creating QuickViews.

Deleting QuickViews

When you no longer need to use a QuickView, you can choose to delete the QuickView from the folder where it resides or you can permanently delete the QuickView. When you delete the QuickView from the folder only, a copy of the QuickView is retained in the All Available QuickViews folder.

Note: You cannot delete out-of-the-box QuickViews.

To delete a QuickView

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Create Dashboards and Reports tile, and from the menu select Dashboards (Remedyforce).
  3. From the list of Catalog options, expand QuickViews.
  4. Select the QuickView that you want to delete.
  5. Click Delete Query from System.
  6. In the confirmation dialog box, click Yes.
