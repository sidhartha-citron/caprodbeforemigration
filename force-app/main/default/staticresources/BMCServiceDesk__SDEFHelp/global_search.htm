You can use Global Search to search for information that might help resolve the incidents that are assigned to you. You can search for information in other incidents, tasks, broadcasts, configuration items, change requests, problems, knowledge articles, documents uploaded to the Documents tab of your Salesforce.com organization, and Google.

First few words of text that you save in the Description field are used as the default text to be searched. To search for text other than the text available in the Description field, delete the text from the text box, type the required text, and click .

To refine search, use operators such as AND, OR, AND NOT, *, and?. For example, to search records with “Networking” or “Networks” but not “Coverage,” enter Network* AND NOT Coverage. Records that contain any word that begins with “Network” but do not contain the word “Coverage” are displayed. For more information, see https://na12.salesforce.com/help/doc/user_ed.jsp?section=help&target=search_unified_refine.htm&loc=help&hash=topic-title.

To use Global Search on the incident form

  1. Click the Remedyforce Workspaces tab.
  2. If you are using the Visualforce page layout UI, perform the following actions:
    1. In the BMC Remedyforce home page, click Workspaces.
    2. From the list of Workspaces, expand Incident Management.
  3. In the Incident Management list, click Global Search.
  4. In the Global Search window, enter the required keywords that you want to search in the text box.

    5. If you have entered more than one word as keywords and no match is found for the keywords, then BMC Remedyforce searches for the individual keywords.

  5. To limit your search, click .
  6. Select the data source that must be searched:
    • Search Support Content—Searches for matching values in incident, task, broadcast, change request, problem, configuration records, and knowledge articles of BMC Remedyforce and Salesforce.
    • Search Support Documents—Searches for matching values in documents uploaded to the Documents tab of your Salesforce.com organization.
    • Search the Web—Searches for matching values in the Internet by using the Google search engine.

      • Note: Global Search searches in all internal fields and fields that are not visible on the form of an object. When the search results display data that is not present in a form, this data might be stored in an internal field or a field that is not visible on the form.

  7. In the Maximum Results field, type the maximum matching results to be displayed in each tab.

    8. Note: This limit is not applicable to the search results in the Web tab.

  8. In the Search field, type the required keywords.

    9. Global Search allows you to search values in string fields only. For example, fields of type Text, Text Area, Long Text Area, Phone, and Email.

    Searching the fields of type Number, Date/Time, and Formula is not supported by the platform.

    Note: * (asterisk) and? (question mark) are valid wildcard characters. If you use ? as a wildcard character, the search matches only one character from the middle or end of your search term.
    For example, there are two KB articles - How to upgrade iphone? and How to backup iphones?. In this case, if you search for iphone?, the search only returns How to backup iphones? knowledge article by taking the s at the end of iphones as a wildcard character. For more information about wildcards, see https://help.salesforce.com/apex/HTViewHelpDoc?id=search_unified_wildcards.htm&language=en_US.

  9. Click .

    10. The search results are displayed in tabs. A tab opens for each data source and the data fields in the tab display the matching values. Contact your system administrator to configure the data sources and data fields that appear in the search results.

    If the search criteria match the file name of a document uploaded to the Documents tab, the document appears in the Documents tab. When you click the document in the Documents tab, you can download the document or open it. When you click the Web Search tab, the matching search results from the Google search engine are displayed in a new browser tab or window. When you click a search result in the Salesforce Knowledge tab, it opens in a new tab in the same browser window.

    Note: If no records are found in Salesforce knowledge, no article is displayed in the Salesforce tab.

  10. To view a knowledge article, on the Knowledge Articles tab, click the required article.
  11. In the Global Search window, select the rating for the article.

    12. This field is required if the system administrator has configured it on the Global Search Settings page.

    The following options are provided:

    • Helped
    • Not Helped

      • If you select Not Helped, only the Return to search results option is enabled.

      If you select a custom rating defined by your system administrator, all the options are enabled. To add a custom rating, contact your system administrator

  12. In the Feedback/Comments field, enter your feedback on the article.
  13. To use the knowledge article text or the link to the knowledge article in the Resolution field on the incident form, select the appropriate option, Copy article text or Copy article link.

    14. The system administrator configures the default option on the Global Search Settings page. If you select the Copy article text option, the text of the Answer, Solution, or Workaround field, depending on the article type of the knowledge article, is copied to the Resolution field. If you select Copy article link, the link to the knowledge article is copied to the Resolution field.

  14. Click Submit.
