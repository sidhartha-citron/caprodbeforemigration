You can configure which Incident templates and service request definitions are available to Self Service clients as top requests. Self Service clients can use these top requests to quickly request help when they face an issue or require a service. Top requests speed up data entry, improve data accuracy, and ensure consistent reporting.

To configure the availability of top requests

  1. In the BMC Remedyforce home page, click Configuration.
  2. From the list of Configuration options, expand Self Service.
  3. In the Self Service list, click Self Service Settings.

    4. Note: In the BMC Remedyforce tabbed view, Self Service Settings is available in the Remedyforce Administration tab.

  4. Click the Top Requests tab.

    5. The Selected Top Requests list displays all top requests that are available to Self Service clients.

  5. In the Type of Top Requests list, select the type of top requests that you want to make available to Self Service clients.
  6. In the Available Top Requests list, select the top requests that you want to make available to Self Service clients.
  7. To move the selected top requests from the Available Top Requests list to the Selected Top Requests list, click the right arrow.
  8. (Optional) To prevent Self Service clients from using top requests, select the top requests in the Selected Top Requests list and click the left arrow.

    9. If you select only Incident templates in the Selected Top Requests list and click the left arrow, the selected Incident templates move to the Available Top Requests list. The value of the Type of Top Requests list changes to Incident Templates and the Available Top Requests list displays only Incident templates. If you select only service request definitions in the Selected Top Requests list and click the left arrow, the selected service request definitions move to the Available Top Requests list. The value of the Type of Top Requests list changes to Request Definitions and the Available Top Requests list displays only service request definitions.

    However, if you select Incident templates and service request definitions in the Selected Top Requests list and click the left arrow, the selected Incident templates and service request definitions move to the Available Top Requests list. The value of the Type of Top Requests list remains the same and the Available Top Requests list displays the type of top requests that were displayed before you clicked the left arrow. For example, the value of the Type of Top Requests list is Request Definitions. If you select Incident templates and service request definitions in the Selected Top Requests list and click the left arrow, the selected service request definitions appear in the Available Top Requests list and the value of the Type of Top Requests list remains as Request Definitions.

    Tip: Double-clicking acts as a toggle to move top requests back and forth between the Available Top Requests and Selected Top Requests lists.

  9. Define the display order in which the top requests appear to Self Service clients by selecting the required top request and clicking the up and down arrows to define the top request's position in the display order.

    10. You can also click the top or bottom arrow to move a top request to the top or bottom of the list.

  10. Click Save.
  11. In the confirmation dialog box, click OK.
