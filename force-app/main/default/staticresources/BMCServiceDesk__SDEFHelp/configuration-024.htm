During the package installation, you cannot assign the ServiceDesk Client level of access to Chatter Only (also known as Chatter Plus) profiles. You must configure the ServiceDesk Client permissions for the Chatter Only (also known as Chatter Plus) profiles after you complete the installation.

To configure permissions for profiles with the Chatter Only license

  1. Navigate to Setup > Manage Users > Profiles.
  2. In the Action column next to the profile with the Chatter Only license, click Edit.
  3. In the Custom App Settings section, select the BMC Remedyforce check box.
  4. In the Standard Tab Settings section, select Default On for the Chatter tab.
  5. In the Custom Tab Settings section, select Default On for the Remedyforce Self Service tab.
  6. For the remaining tabs in the Custom Tab Settings section, select Tab Hidden.
  7. To grant the required permissions to the profile with the Chatter Only license for custom objects, perform the following actions:
    1. Ensure that the Read check box is selected for the following custom objects:

        • Agreements
        • Configuration Item Histories
        • Process Controls
        • Agreement Histories
        • Custom Actions
        • Request Definitions
        • Base Elements
        • Entitlements
        • SelfService Images
        • Broadcasts
        • FAQs
        • SelfService Themes
        • Broadcast Account Link
        • FAQ Categories
        • Service Targets
        • Business Services
        • Fulfillment Inputs
        • Service Target Criteria
        • Categories
        • Fulfillment Mappings
        • SLA Milestones
        • Change Assessments
        • Impacts
        • SLA Milestone Actions
        • Change Histories
        • Incident Histories
        • SLA Relations
        • Change Requests
        • Knowledge Articles
        • Status
        • CI Client Link
        • SYSPopupObject
        • Urgencies

    2. Ensure that the Read and Create check boxes are selected for the following custom objects:
      • CustomAttachment
      • Quick Links
      • Tasks
    3. Ensure that the Read, Create, and Edit check boxes are selected for the following custom objects:
      • Change Request Service Targets
      • Incidents
      • Incident Service Targets
      • Knowledge Feedback
      • Knowledge Incident Links
      • Request Details
      • Request Detail Inputs
      • SLA Milestone Change Transactions
      • SLA Milestone Incident Transactions
    4. Ensure that the Read, Create, Edit, and Delete check boxes are selected for the Knowledge Articles Viewed custom object.
    5. Click Save.
  8. To assign the correct layouts to the profile with the Chatter Only license, perform the following actions:
    1. Navigate to Setup > Manage Users > Profiles.
    2. In the Profile Name column, click the name of the profile with the Chatter Only license.
    3. For the Home Page Layout in the Standard Object Layouts section, click the View Assignment link.
    4. Click Edit Assignment.
    5. Select the required profiles with the Chatter Only license.
    6. From the Page Layout To Use list, select Remedyforce Client Home.
    7. Click Save.
    8. Navigate to Setup > Manage Users > Profiles.
    9. In the Profile Name column, click the name of the profile with the Chatter Only license.
    10. For the Change Request object in the Custom Object Layouts section, click the View Assignment link.
    11. Click Edit Assignment.
    12. Select the required profiles with the Chatter Only license.
    13. From the Page Layout To Use list, select Remedyforce Change for Clients Version 1.0.

      14. To view the details of the change request that is sent for approval to users of this profile, you must assign the Remedyforce Change for Clients Version 1.0 page layout. After viewing the details of the change request, the user can approve or reject the change request.

    14. Click Save.
    15. For the Incident object in the Custom Object Layouts section, click the View Assignment link.
    16. Click Edit Assignment.
    17. Select the required profiles with the Chatter Only license.
    18. From the Page Layout To Use list, select Remedyforce Incident for Clients Version 1.0.

      19. With this layout, users of this profile can log on by using the https://login.salesforce.com URL. Users can access the Home and Self Service tabs. If chatter is enabled in the Salesforce organization, they can access the Chatter tab too. Users can also follow an incident by clicking the Follow link for the incident record from the Home tab. If the user is assigned as an Approver in an approval process, the user can approve change requests and incidents from the Home tab. In the Recent Items component of the Sidebar, users can view new incidents. However, these users cannot edit these incidents.

    19. Click Save.
  9. To configure the field-level security for the profile with the Chatter Only license, perform the following actions:
    1. Navigate to Setup > Manage Users > Profiles.
    2. In the Profile Name column, click the name of the profile with the Chatter Only license.
    3. In the Field-Level Security section for the Change Request object, click the View link.
    4. Click Edit.
    5. Ensure that the Visible check box is selected for the following fields:

        • Impact~

        • Closed Date

        • Feedback

        • First Name
        • Urgency
        • Impact
        • Urgency~

    6. Click Save.
    7. Click Back to Profile.
    8. Repeat step c through step g for the following objects:
      • Change Assessment
      • Change History
      • Incident
      • Incident History

    4. Click Save.
