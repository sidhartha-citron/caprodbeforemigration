Portlet titles of the portlets available in the Sidebar, such as Action Items, Broadcasts, Search, World Time, Chatter Groups, and Top Requests portlets are mapped to custom labels. These labels are localized on the custom label details page. For more information about localizing custom labels, see Localizing custom labels.

Note: Only the Broadcasts portlet title is mapped to plural object label.

UI labels that are used in the Action Items portlet are mapped to object plural labels. For more information about localizing object labels, see Localizing object labels.
