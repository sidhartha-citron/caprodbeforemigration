An urgency record identifies the speed to resolve an incident, task, broadcast, change request, or problem and determine how the record escalates through the service desk.

The following are the default urgency values:

    • HIGH
    • MEDIUM
    • LOW

The system administrator can create more urgency records as part of the configuration tasks to set up BMC Remedyforce for use in the organization.

To create an urgency record

  1. In the BMC Remedyforce home page, click Configuration.
  2. From the list of Configuration options, expand Configuration.
  3. In the Configuration list, click Urgency.
  4. In the Urgencies List view, click New.
  5. In the Urgency field, type a unique identifier that you want to assign to the current urgency record.
  6. Click Save.

    7. Note: When you open a new form to create a record and you save the record, you cannot navigate to the next or previous record by using the Next Record and Previous Record buttons in the upper right section of the form. These buttons are disabled in a new form.

  7. (Optional) To designate the current record as inactive, select the Inactive check box.

    8. Note: Records marked as inactive are not included in any search performed on the database.
