You can create field updates and associate the fields updates with workflows and approvals.

To create email alerts

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Manage Workflows and Other Processes tile, and from the menu select Field Updates.
  3. To create a new field update, click New Field Update.

    4. For more information about creating field updates, see https://help.salesforce.com/htviewhelpdoc?err=1&id=workflow_defining_field_updates.htm&siteLang=en_US.
