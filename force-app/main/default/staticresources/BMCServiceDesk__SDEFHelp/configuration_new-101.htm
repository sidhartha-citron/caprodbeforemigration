When you are viewing the tasks for an incident, change request, or problem record in the Supporting Information section, you can configure BMC Remedyforce to display inactive tasks that are added for the record. By default, BMC Remedyforce does not display inactive tasks in the Supporting Information section.

To display inactive tasks in the Supporting Information section

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Application Settings tile, and from the menu select Custom Settings.
  3. For Remedyforce Settings, in the Action column, click Manage.
  4. For the Show inactive Task in Supporting Info list, in the Action column, click Edit.
  5. In the Value field, type TRUE.
  6. Click Save.
