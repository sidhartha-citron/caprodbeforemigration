An impact record creates the values that are used to specify the impact that an incident, task, broadcast, change request, or problem has on an account.

The following are the default impact values:

    • HIGH
    • MEDIUM
    • LOW

The system administrator can create more impact records as part of the configuration tasks to set up BMC Remedyforce for use in the organization.

To create an impact record

  1. In the BMC Remedyforce home page, click Configuration.
  2. From the list of Configuration options, expand Configuration.
  3. In the Configuration list, click Impact.
  4. In the Impacts List view, click New.
  5. In the Impact ID field, type a unique alphanumeric identifier that you want to assign to the current impact record.
  6. Click Save.

    7. Note: When you open a new form to create a record and you save the record, you cannot navigate to the next or previous record by using the Next Record and Previous Record buttons in the upper right section of the form. These buttons are disabled in a new form.

  7. (Optional) To designate the current record as inactive, select the Inactive check box.

    8. Note: Records marked as inactive are not included in any search performed on the database.
