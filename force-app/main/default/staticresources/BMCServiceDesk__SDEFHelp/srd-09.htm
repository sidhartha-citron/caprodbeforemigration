When you select Radio Button as the Response Type option for an Input field on the Fulfillment tab of a Request Definition, you need to add the radio button options from which the users can select an option.

To add radio button options to the service request

  1. In the Response Type list, select Radio Button.
  2. When you select Radio Button as the field type, the Radio Button Value section appears.
  3. In the Displayed Value field of the Radio Button Value section, type the displayed value of the radio button option.
  4. In the Stored Value field of the Radio Button Value section, type the stored value of the radio button option.
  5. Click Add.

    6. The radio button option, displayed value, and stored value are added to the list of radio buttons.

  6. If you want the radio button option to be selected by default, select the check box in the Default Value column.
  7. (Optional) To delete a radio button option, select it and click the Delete icon.
