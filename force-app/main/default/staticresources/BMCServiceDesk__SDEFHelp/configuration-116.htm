To make the translations available, you can import the translation files in the appropriate organization.

To import translation files

  1. Navigate to Setup > Translation Workbench > Import.
  2. In the Step 3. Select and upload your file(s) section, click Browse and select the .stf file that you exported, translated, and saved.
  3. Click Import.

    4. You receive an email to confirm the status of the import.

    Note: If there is a failure, check for invalid keys in the appropriate .stf file on the Documents page. You can also check for the “If No, it is a regular class” sentence. If the sentence is in a new line, move the sentence to the previous sentence. Save the file and import it again.
