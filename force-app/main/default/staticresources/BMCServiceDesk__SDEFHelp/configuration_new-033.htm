When you create a configuration item, you can use custom attributes to record information about the configuration item that is not available in the out-of-the-box attributes.

You can create the required custom fields for the custom object.

To create custom fields for a custom object

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click Manage Objects tile, and from the menu, select Create and Edit Objects.
  3. From the Object list, select All Custom Objects.
  4. Click the required <custom object name>.
  5. In the Custom Fields & Relationships section, click New.
  6. Select the data type of the custom field.

    7. Note: Custom fields of the Lookup Relationship data type do not appear in the Custom tab of the Instance Editor form.

  7. Click Next.
  8. In the Field Label field, type the label of the custom field.
  9. In the attribute fields, enter the attributes of the custom field.

    10. For more information, see https://na11.salesforce.com/help/doc/user_ed.jsp?section=help&target=custom_field_attributes.htm&loc=help&hash=topic-title.

  10. Click Next.
  11. Select the profiles to which you want to grant edit access to this custom field.
  12. Click Next.
  13. Select the page layouts that display the custom field as an editable field.

    14. By default, the custom field is selected for page layout of the custom object.

  14. Click Save.
