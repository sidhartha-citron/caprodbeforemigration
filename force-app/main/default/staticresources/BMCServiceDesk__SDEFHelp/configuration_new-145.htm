You can view and manage the data storage usage for your Salesforce organization. You can also view the top users based on storage usage.

To view storage usage

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the License and Storage Usage tile, and from the menu select Storage Usage.
  3. To view the storage used by files, click the Overall Top Files by Storage Utilization link that is provided at the end of the page.

For more information, see https://help.salesforce.com/htviewhelpdoc?err=1&id=admin_monitorresources.htm&siteLang=en_US.
