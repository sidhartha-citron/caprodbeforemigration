After saving the release, you can add user-defined actions and system-defined actions to a release in the Action History section.

To add an action history record to a release

  1. Navigate to the record to which you want to add an action.

    2. For more information, see Navigating to a release record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click the Actions menu and select New Action.

      • Alternatively, click Record Details and in the Action History section, click New Action.

    • If you are using the Releases tab (Salesforce page layout), click New Action in the Action History section.
  3. Select the appropriate action that you want to add to the Action History section.

    4. If you select the Notes option, a new browser window is displayed in which you can enter additional notes. The note is displayed in the Action History section.

    Note: To get the Note field on your page layout, edit your layout by clicking the Edit Layout link on the upper-right corner of the page. Click Visualforce Pages and drag-and-drop Add Notes to the required section.

    BMC recommends that you create a separate section on your layout with a meaningful name and add the Add Notes option to that section; otherwise, a text box without a label is displayed on the layout.

Viewing actions related to releases

You can view the actions that are related to a release by using the View Action History option. The View Action History option allows you to create a new action, filter actions based on type, and print the actions.

To view actions related to a release

  1. Navigate to the release record for which you want to view actions.

    2. For more information, see Navigating to a release record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click the Actions menu, and select View Action History.

      • Alternatively, click Record Details and in the Action History section, click View Action History.

    • If you are using the Releases tab (Salesforce page layout), in the Action History section, click View Action History.

      • The Release History window displays the actions in the view-only mode. You can not edit or delete the actions that are displayed in the Release History window.

  3. (Optional) To filter the records based on the action type, select the required action type from the list.
  4. (Optional) To print the actions related to a release, click Print.
