Custom labels can be localized on the custom labels details page.

Note: Custom labels are not part of the Translation Workbench and the translation files that can be exported from Salesforce. You must localize custom labels for each organization individually.

To localize custom labels

  1. Navigate to Setup > Create > Custom Labels.
  2. To open the label, select the custom label name.
  3. Click New Local Translations / Overrides.
  4. In the Language field of the Translation Information section, select the language that you are translating into.
  5. Enter the translation text.

    6. This text overrides the value specified in the Value field of the label when the default language of a user is the translation language.

  6. Click Save.

For more information, see https://na7.salesforce.com/help/doc/user_ed.jsp?section=help&target=cl_translate_edit.htm&loc=help&hash=topic-title.
