This section provides steps to view additional information about a record in the Visualforce page layouts UI.

The following sections are provided:

    • Viewing user and system actions in the Visualforce page layouts UI
    • Viewing linked incidents
    • About the list view

Viewing user and system actions in the Visualforce page layouts UI

You can view the user and system-defined actions performed on a broadcast record by using the View menu in the History tab of the Supporting Information section.

To view the user and system actions in the Visualforce page layouts user interface

  1. Navigate to the record for which you want to view the user and system-defined actions.

    2. For more information, see Navigating to a broadcast record.

  2. In the Supporting Information section, click the History tab.
  3. Click the View menu.
  4. To view the user-defined actions performed on a broadcast, select User Entries.
  5. To view the system-defined actions performed on a broadcast, select System Entries.
  6. To view all the actions (both user-defined and system-defined) performed on a broadcast, select All Entries.

Viewing linked incidents

You can view the incidents linked to a broadcast by using the View menu in the Incidents tab of the Supporting Information section.

To view linked incidents in the Visualforce page layout user interface

  1. Navigate to the broadcast for which you want to view the linked incidents.

    2. For more information, see Navigating to a broadcast record.

  2. In the Supporting Information section, click the Incidents tab.
  3. In the Incidents tab, click the View menu.
  4. To view all the linked incidents, select All.
  5. To view the linked incidents in the Open stage, select Open.
  6. To view the linked incidents in the Acknowledged stage, select Acknowledged.
  7. To view the linked incidents in the In Process stage, select In Process.
  8. To view the linked incidents in the Closed stage, select Closed.

About the list view

The following table provides description about the options available in the list view in the Visualforce page layout user interface:

Option
Description
New
To create a new broadcast
Copy
To copy the selected broadcast
Delete
To delete the selected broadcast
Views > Assigned to myself
To view the broadcasts assigned to the logged on user
Views > Open Items
To view the broadcasts that are in Open state
Views > Closed Items
To view the broadcasts that are in Closed state
Views > Active Items
To view all active broadcasts
Views > Inactive Items
To view all inactive broadcasts
Refresh every __ minutes
To set the time in which the list view must be refreshed
Group by
To show broadcasts with same value in a field, such as Category.

To search for a broadcast, enter keywords and click .
