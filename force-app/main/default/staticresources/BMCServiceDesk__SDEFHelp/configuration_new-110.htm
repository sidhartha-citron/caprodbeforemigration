The following list of documents is available to be displayed in the left side bar component:

    • Release Notes
    • Upgrade Guide
    • User's Guide for Analysts
    • User's Guide for Self Service clients
    • Installation and Configuration Guide
    • Administrator's Guide

The documents are not available by default. To show these documents to the user, you must assign the correct home page layout to the required user profile.

To add documentation to the left side component in home page

  1. Navigate to Setup > Customize > Home > Home Page layouts.
  2. Click Page Layout Assignment.
  3. Click Edit Assignment.
  4. For the required profile, select Remedyforce Home from the list.
  5. Click Save.
