BMC Remedyforce is available in the following languages:

    • Japanese
    • French
    • Spanish
    • German

To use a localized version of BMC Remedyforce, you must enable the following:

    • Translation Workbench
    • Language

Translation Workbench

If you are using localized BMC Remedyforce in an organization, you must first enable the Translation Workbench in Salesforce.

To enable the Translation Workbench

  1. Navigate to Setup > Translation Workbench > Translation Settings.
  2. On the Translation Workbench page, click Enable.

    3. If the Translation Workbench is already enabled on your organization, the following message is displayed at the end of the page, “The Translation Workbench is currently enabled for your organization. To disable it, click the Disable button.”

    Note: By default, the Manage Translation permission is enabled in the System Administrator profile and can only be edited in a custom profile. If you want only certain administrators to enable, disable, or manage the Translation Workbench, you must create a new profile. You can clone the new profile from the System Administrator profile, and then disable the Manage Translation permission.

    For more information, see https://na7.salesforce.com/help/doc/user_ed.jsp?section=help&target=customize_wbench.htm&loc=help&hash=topic-title.

Language

To use the localized version of BMC Remedyforce, you must activate the language in Salesforce. For example, if you want to use BMC Remedyforce in French, you must enable the French language. Language activation is required to ensure that the fields of data type picklist, such as Change Type on the Change Request form, are localized correctly.

To enable a language

  1. Navigate to Setup > Translation Workbench > Translation Settings.
  2. In the Action column, click Edit beside the language that you want to enable.
  3. In the Language section, select Active.
  4. Click Save.
