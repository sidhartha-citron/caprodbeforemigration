If an approval process has been set up to submit a change request for approval, submit the change request for approval.

To submit a change request for approval

  1. Navigate to the change request that you want to submit for approval.

    2. For more information, see Navigating to a change request record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Actions menu, select Submit for Approval.

      • Alternatively, you can click Record Details and in the Approval History section, click Submit for Approval.

      Note: If your system administrator has configured a manual approval process, after you select the approver, you must close the Submit for Approval window, manually.

    • If you are using the Change Requests tab (Salesforce page layout), in the Approval History section, click Submit for Approval.
  3. In the confirmation dialog box, click OK.
  4. If you are using the Visualforce page layout UI, perform the following actions:
    1. In the Supporting Information section, click the Change Approvals tab.
    2. From the menu, select Submit For Approval.
    3. In the Comments text box, type comments that are included in the change approval.

      4. The comments appear when you move your mouse over the balloon icon in the Comment column.

    4. Click OK.
    5. (Optional) To recall a change request from approval, from the menu, select Recall Approval Request.

      6. Only you or the system administrator can recall a change request from approval.

      The overall status of approvals or rejections for this change request is displayed in the Overall Status field in the upper-right section of the Supporting Information section.
