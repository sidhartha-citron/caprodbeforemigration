You can create incidents and link those incidents to the client record.

To create and link incidents to a client

  1. In the BMC Remedyforce home page, click Configuration.
  2. From the list of Configuration options, expand Configuration.
  3. Click Clients.
  4. From the clients list view, select the client record for which you want to create and link incidents.
  5. In the Supporting Information section, click the Incidents tab.
  6. From the Actions menu, select Add.

    7. The incident form is opened.

  7. Enter the incident details.

    8. A link to the incident that is created is established in the Incidents tab in the Supporting Information section of the Client.

  8. (Optional) To edit an incident record linked to a client, select the incident record, and from the Actions menu, select Edit.
