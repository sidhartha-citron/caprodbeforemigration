While importing large number of records, if you get OutofMemoryError or java heap size error, increase the heap size in the Spoon.bat file.

To resolve the OutofMemoryError error

  1. Go to the \pdi-ce-4.x.0-stable\data-integration or \pdi-ce-5.0.1.A-stable\data-integration folder.
  2. Right-click the Spoon.bat file, and select Edit.
  3. In the If "%PENTAHO_DI_JAVA_OPTIONS%"=="" set PENTAHO_DI_JAVA_OPTIONS="-Xmx512m" "-XX:MaxPermSize=256m" line, replace 512 with a higher value.
  4. Click Save.
  5. Relaunch the Spoon.bat file and rerun the job file.
