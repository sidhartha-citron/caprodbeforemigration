If you are a staff member who is a member of the Change Manager profile, you can create, edit, and delete change requests.

If you are a staff member who is not a member of the Change Manager profile, you can only view change requests, and you cannot create, edit, or delete change requests.

To erase the updates you have done to the record and revert the record to the earlier saved state, click Refresh.

Tracking changes

When you edit a change request, the modifications are tracked in the Change Request History section. When you modify a field in a change request, the following message appears in the Change Request History section:

Changed Record Update Count from <x> to <y>.

In this message, x and y are numbers.

Contact your system administrator to mark a field to be tracked.

If your system administrator has not enabled field tracking for the Record Update Count field, the following message does not appear in the Change Request History section: 

Changed Record Update Count from <x> to <y>.

Editing multiple change requests

  1. Click the Remedyforce Console tab.
  2. From the View menu, select Change Requests.
  3. Select the check boxes for the records that you want to edit, and click Update.

    4. You can update only 200 records at the same time.

  4. From the Field list, select the field that you want to update for the selected records.

    5. Only the fields that are shown on the list view are available in the Field list. Fields of type Auto Number and Formula are not available in the Field list. If you do not have permission to edit any of the fields in the list view, the field is not available in the Field list.

  5. Enter or select a value for the selected field.
  6. Click Apply.

    7. You can update one field of multiple records at the same time.

  7. To update another field of multiple records, repeat step 4 to step 6.
  8. Click OK.
