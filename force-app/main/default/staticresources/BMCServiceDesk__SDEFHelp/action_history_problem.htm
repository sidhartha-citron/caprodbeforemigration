After saving a problem, you can add user-defined actions and system-defined actions to a problem in the Action History section (in the History tab of the Supporting Information section in the Visualforce page layout UI.

To add an action history record to a problem

  1. Navigate to the problem to which you want to add an action.

    2. For more information, see Navigating to a problem record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click Record Details and in the Action History section, click New Action.
    • If you are using the Problems tab, (Salesforce page layout), click New Action in the Action History section.
    • If you are using the Visualforce page layout UI, click the History tab in the Supporting Information section and from the , select Add.
  3. Select the appropriate action that you want to add to the Action History section.

    4. If you select the Notes option, a new browser window is displayed where you can enter additional notes. This note is displayed in the Action History section.

    In the Visualforce page layout UI, this note is displayed in the History tab. Your system administrator can create a workflow to send an email message to the client when a particular action is added to the History tab.

    Note: You cannot save a Client Note action after adding the action. The Client Note actions are for the exclusive use of Self Service clients. This restriction is added to allow Self Service clients to edit the Client Note actions.

Viewing actions related to problems

You can view the actions related to a problem by using the View Action History option. The View Action History option allows you to create a new action, filter actions based on type, and print the actions.

Note: The Problem History window displays the actions in the view-only mode. You can not edit or delete the actions that are displayed in the Problem History window.

To view actions related to a problem

  1. Navigate to the problem for which you want view actions.

    2. For more information, see Navigating to a problem record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click the Actions menu, and select View Action History.

      • Alternatively, click Record Details and in the Action History section, click View Action History.

    • If you are using the Problems tab (Salesforce page layout), in the Action History section, click View Action History.

      • The Problem History window displays the actions in the view-only mode. You can not edit or delete the actions that are displayed in the Problem History window.

  3. (Optional) To filter the records based on the action type, select the required action type from the list.
  4. (Optional) To print the actions related to a problem, click Print.
