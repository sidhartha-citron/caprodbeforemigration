You can view survey reports in the Dashboards tab. You can use reports to see the response to specific questions and incidents or service requests to which the response is related.

To view survey reports

  1. Click the Dashboards tab.
  2. Click the BMC Remedyforce folder.
  3. Click the BMC Remedyforce Survey dashboard.

    4. A graph is shown for responses on each key metric.

  4. To see more details of a key metric, click the required graph.

    5. Below the graph, a report is shown. You can click the incident # to view the details of the relevant incident or service request.

If the survey contains custom questions, you must create a custom report. For more information about creating reports, see https://help.salesforce.com/HTViewHelpDoc?id=reports_builder_create.htm&language=en_US.
