The following table provides description about the options available in the list view in the Visualforce page layout UI:

Option
Description
New
To create a new problem
Copy
To copy the selected problem
Delete
To delete the selected problem
Views > Assigned to myself
To view the problems assigned to the logged on user
Views > Open Items
To view the problems that are in Open state
Views > Closed Items
To view the problems that are in Closed state
Views > Active Items
To view all active problems
Views > Inactive Items
To view all inactive problems
Refresh every __ minutes
To set the time in which the list view must be refreshed
Group by
To show problems with same value in a field, such as Category.
Search
To search for a problem, enter keywords and click .
