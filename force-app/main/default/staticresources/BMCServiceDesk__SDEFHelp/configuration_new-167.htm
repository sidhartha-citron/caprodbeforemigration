You can view the incidents, tasks, and broadcasts records by using the Remedyforce mobile application on Salesforce1 platform.

To add or remove fields in the record list view

  1. Navigate to Setup > Create > Objects.
  2. In the Label column, click the required object, such as Incident.
  3. In the Field Sets section, in the Action column for the Salesforce1 - List View field set, click Edit.
  4. From the object palette, drag the preferred fields to the In the Field Set list.

    5. The form displays all fields that are added to the In the Field Set list. The order in which fields appear in the custom form depends on the order in which the fields are arranged in the object palette. You can rearrange the fields in the Field Set list by dragging and dropping.

  5. (Optional) To remove the selected fields, click the x button that appears on the field name.
  6. Click Save.
  7. Repeat step 2 to step 6 for the Task and Broadcast object.
