By default, the following information is displayed on the Client ID pop-up window:

    • Title
    • Email
    • Phone
    • Manager
    • Extension

In the Salesforce page layout, to add more information on the Client ID pop-up window, perform the steps mentioned in this section.

In the console layout, to show more information about a client in the Client ID pop-up window, add the required fields in the Data Source and Data Fields columns on the Global Search page. For more information, see Configuring Global Search settings.

To add more information on the Client ID pop-up window

  1. Navigate to Setup > Customize > Users > Page Layouts.
  2. In the Action column beside Remedyforce User page layout, click Edit.
  3. On the right corner of the page, click Mini Page Layout.
  4. From the Available list, select the fields that you want to display on the Client ID pop-up window and click the right arrow to move the selected fields to the Selected list.
  5. Click Save.
