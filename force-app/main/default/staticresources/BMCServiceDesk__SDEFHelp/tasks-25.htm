You can view the user-defined and system-defined actions performed on a task record in the History tab in the Supporting Information section. To view these actions, you can use the View menu. The following options are provided in the View menu:

View menu options
Description
User Entries
Displays all user-defined actions that have been performed on the record
System Entries
Displays all system-defined actions that have been performed on the record
All Entries
Displays all actions (both user-defined and system-defined) that have been performed on the record
