You can add attachments to a broadcast before and after saving a record.

Note: On the Remedyforce Console tab, to add attachments to a broadcast before saving, from the Actions menu, select Attach/View Files.

To add an attachment to a broadcast

  1. Navigate to the broadcast for which you want to add an attachment.

    2. For more information, see Navigating to a broadcast record.

  2. Perform one of the following actions:
    • On the Remedyforce Console tab, from the Actions menu, select Attach/View Files.
    • If you are using the Broadcasts tab (Salesforce page layout), in the Action History section, click Attach File.
  3. In the Attach/View Files window, to select the file that you want to attach to the incident, click Choose File.

    4. Note: The size of one attachment cannot exceed 10 MB.

  4. Select the file that you want to add and click Open.
  5. Click Close.

    6. The selected file is attached to the record. In the Attach/View Files window, you can view or delete the existing attachments.
