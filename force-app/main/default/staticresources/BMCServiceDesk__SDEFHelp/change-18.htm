You can include additional information about the change request as notes in the Action History section.

To add a note to a change request

  1. Navigate to the change request to which you want to add a note.

    2. For more information, see Navigating to a change request record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click the Actions menu, and select Add Note.

      • Alternatively, click Record Details and in the Action History section, click New Note.

    • If you are using the Change Requests tab (Salesforce page layout), in the Action History section, click New Note.
    • If you are using the Visualforce page layout UI, add the updates in the Note field.
  3. Enter the note details.
  4. (Optional) In the Duration field, enter the time that you have spent on the action for which you are adding a note.
  5. Click Add.

    6. If the Note field is provided on the change request form, you can add the note in the Note field also. If the Note field is not available on the change request details page, contact your system administrator to customize the problem layout to display the Add Notes Visualforce page. Request the system administrator to give a logical name, such as Note, to the section where the Add Notes Visualforce page is added. You can also record the time that you spent working on the change request in the Duration field.
