You can manage licenses for BMC Remedyforce on the Package Details BMC Remedyforce page. You can add or remove licensed users or remove multiple licensed users.

To manage Remedyforce licenses

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the License and Storage Usage tile, and from the menu select Remedyforce License Usage.
  3. To add licensed users, click Add Users.
  4. To remove a licensed user, click Remove next to the name of the user.
  5. To remove multiple licensed users, click Remove Multiple Users.

    6. For more information, see https://help.salesforce.com/htviewhelpdoc?id=distribution_managing_licenses.htm&siteLang=en_US.
