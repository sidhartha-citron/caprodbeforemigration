The UI labels that are mapped to field labels can be localized using the Translation Workbench.

To localize field labels

  1. Navigate to Setup > Translation Workbench > Translate.
  2. Select the language that you are localizing to.
  3. Select a Setup Component.

    4. Select Custom Field to localize field labels.

  4. In the Object list, select the form that you want to localize from.

    5. For example, to localize the UI labels on the Incident form, select Incident.

  5. If required, select the appropriate aspect of the form to be localized from the Aspect list.

    6. To localize the Tooltips that are mapped to field labels, select the Help Text option.

  6. To enter new values, double-click the Field Label Translation column.

    7. You can press Tab to advance to the next editable field or SHIFT+Tab to go to the previous editable field.

    Note: The Out of Date column indicates that the item has been updated and the term might need to be localized. When editing a button or link label, you see the Button or Link Name column, which sees the component when using the web services API.

  7. Click Save.

For more information, see https://na7.salesforce.com/help/doc/user_ed.jsp?section=help&target=adding_and_editing_translated_languages.htm&loc=help&hash=topic-title.
