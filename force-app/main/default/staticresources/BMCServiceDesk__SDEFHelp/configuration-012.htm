If you have installed BMC Remedyforce for the first time, by default, the following options are displayed in the User License field on the Client form:

    • Customer Portal Manager Custom
    • Salesforce Platform
    • Salesforce

To enable more licenses in this field, perform the actions mentioned in this section.

To add other licenses to create clients

  1. Navigate to Setup > Develop > Custom Settings.
  2. In the Action column of Remedyforce Portal Licenses, click Manage.
  3. Click New.
  4. In the Name field, enter a number.

    5. Ensure that this number is not used in the list.

  5. In the Value field, enter the license that you want to add to the User License field.
  6. Click Save.
