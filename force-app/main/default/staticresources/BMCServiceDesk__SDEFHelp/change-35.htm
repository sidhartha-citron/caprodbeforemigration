You might need to reopen a change request instead of creating a new one if the initiator is not satisfied with change that you have implemented or the review of the change request has brought more change activities that must be performed to complete the change request.

To reopen a change request

  1. Navigate to the change request that you want to reopen.

    2. For more information, see Navigating to a change request record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Actions menu, select Reopen.
    • If you are using the Change Requests tab (Salesforce page layout), click Reopen.
    • If you are using the Visualforce page layout UI, from the menu, select Reopen Incident.
