As a system administrator, you must create the BMC Remedyforce folder before exporting a report as the exported reports are saved in this folder.

To create the BMC Remedyforce folder

  1. Click All tabs (+).
  2. Click Documents.

    3. Ensure that you are clicking the Documents tab that has a page icon.

  3. Click Create New Folder.
  4. In the Document Folder Label field, enter BMC Remedyforce.
  5. In the Public Folder Access list, select Read/Write.
  6. Click Save.
