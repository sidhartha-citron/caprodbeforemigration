BMC Remedyforce provides a variety of predefined reports, QuickViews, and charts to give you quick access to information. The following table lists the tabs that you can access to view this information.

Tab
Additional information
Reports
On the Reports tab, staff members and the system administrator can view, edit, and export the predefined reports that BMC Remedyforce provides.
Based on the requirements, staff members and the system administrator can also create new reports. However, only the system administrator can create additional report types.
  • Creating reports
  • Creating report types
  • Accessing predefined reports
Dashboards
On the Dashboards tab, the system administrator can access the BMC Remedyforce Dashboard, which displays the following data in the form of charts and views:
  • Number of open records per client (Bar chart)
  • Number of incidents by category (Pie chart)
  • Tasks by Owner (Table view)
  • Incidents by Priority (Dial view)
To enable users with the Salesforce or Salesforce Platform license to access the Dashboards tab, the system administrator must grant them the Manage Dashboards profile-level permission.
Additionally, to make the BMC Remedyforce Dashboard available to staff members, the system administrator must perform the following actions:
  • Configure the Running User property of the dashboard to be a user with a Salesforce Platform license
  • Select a user who is high in the role hierarchy because the data visibility of the Running User is applied to the dashboard
  • Viewing the BMC Remedyforce dashboard
  • Configuring the BMC Remedyforce dashboard
Remedyforce Dashboard
On the Remedyforce Dashboard tab, staff members and the system administrator can view a key set of data in one place and monitor metrics regularly. Staff members can use the dashboard to view all their work immediately when they log on to BMC Remedyforce.
The system administrator uses the default dashboard to set up the QuickViews that appear on the dashboard of staff members when they log on.
  • Accessing the Remedyforce Dashboard tab
  • Configuring QuickViews on the Remedyforce Dashboard tab
