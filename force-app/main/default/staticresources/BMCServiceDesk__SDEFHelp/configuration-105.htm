Most of the supporting information UI labels are mapped to custom labels and are prefixed with “Supporting_”. For example, the Auditing UI label is mapped to Supporting_Auditing.

Tab names that are used under the Supporting Information section are mapped to custom labels. The column names used in the tables provided under each of these tabs are mapped to field labels, or these column names are standard fields. For example, the History tab under the Supporting Information section on the Incidents page is mapped to custom label. However, the Date & Time column name under the History tab is a standard field. By default, the standard fields are localized in the language that is assigned to the logged-on user.

To localize the Supporting Information UI labels that are mapped to field labels, see Localizing field labels. To localize the Supporting Information UI labels that are mapped to custom labels, see Localizing custom labels.
