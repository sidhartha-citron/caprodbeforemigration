You can print a task record on a printer or as a PDF output when required. If you want to add a specific field to the PDF output, contact your system administrator.

To print a task record

  1. Navigate to the task that you want to print.

    2. For more information, see Navigating to a task record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Actions menu, select Print.

      • To print a task in the PDF format, from the Actions menu, select Print to PDF.

    • If you are using the Task Console tab, from the Actions menu, select Print.

      • To print a task in the PDF format, from the Actions menu, select Print to PDF.

    • If you are using the Tasks tab (Salesforce page layout), on the right corner of the page, click Printable View and on the top-right corner of the page, click Print This Page.
    • If you are using the Visualforce page layout UI, from the menu, select Print.

      • To print a task in the PDF format, from the menu, select Print to PDF.

      The page layout of the Task object is opened. To change the page layout in which you want to print the task, contact your system administrator.
