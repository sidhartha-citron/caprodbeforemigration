The BMC Remedyforce dashboard displays the following data in the form of charts and views:

    • Number of open records per client (Bar chart)
    • Number of incidents by category (Pie chart)
    • Tasks by Owner (table view)
    • Incidents by Priority (Dial view)

To view the BMC Remedyforce dashboard

  1. In the upper right section of the BMC Remedyforce home page, click the Setup link.
  2. In the confirmation dialog box, click Yes.
  3. Click the Home tab.
  4. Click the Dashboards tab.

    5. Note: If the Dashboards tab is not visible, click the All Tabs icon, and click Dashboards.

  5. From the View Dashboard list, select BMC Remedyforce Dashboard.

    6. The BMC Remedyforce Dashboard displays with updated data.

  6. (Optional) To update the charts and views in the dashboard on a periodic basis, click Refresh.
