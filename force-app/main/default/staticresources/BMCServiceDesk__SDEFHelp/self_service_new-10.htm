You can edit custom themes based on your requirements. If you edit a custom theme while users are logged on to Self Service, the Self Service users must refresh their browsers to view the changes.

To edit a custom theme

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Configure Self Service tile, and from the menu, select Branding.
  3. From the User Interface Theme list, click Edit next to the custom theme that you want to modify.
  4. In the new browser tab that opens, modify the theme name or the colors of the required UI elements.

    5. For information about modifying the colors of the UI elements, see step 5.

  5. Click and then click OK.

For information about changing the images that are assigned to a custom theme, see Configuring images for a theme.
