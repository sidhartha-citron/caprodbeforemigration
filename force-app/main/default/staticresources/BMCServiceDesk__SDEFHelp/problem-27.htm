To close a problem

  1. Navigate to the problem that you want to close.

    2. For more information, see Navigating to a problem record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Actions menu, select Close.
    • If you are using the Problems tab (Salesforce page layout), click Close.
    • If you are using the Visualforce page layout UI, from the menu, select Close.

      • The Close Problem form appears with the Closed Date field filled with the current date and time. The Category field is filled with the category that is entered in the problem form.

      Note: This procedure describes the out-of-the-box fields only. If your system administrator has added or removed fields, complete all the fields on the form.

  3. From the Staff ID Closed By list, select the staff member who has worked on the problem.

    4. By default, this field displays the Staff ID of the currently logged-on staff member. If you are closing the problem on behalf of another staff member, select the staff member who has worked on the problem.

  4. From the Status list, select a close status.

    5. By default, this field displays the default close status that is configured by the system administrator. If required, select a different close status from the Status list. All status records with the state of Close are displayed in the Status list.

  5. From the Closure Category list, select the category that suggests the reason for closing the record.
  6. If the original category was incorrect or no longer applies, select the required category from the Category field.
  7. In the Resolution field, type the resolution of the problem.

    8. By default, this field displays the value of the Resolution field in the problem form. You can modify this field to include the confirmed resolution of the problem or any new steps taken to resolve the problem.

  8. Select the Update Standard Description check box to add the contents of the problem form Description field as a new standard description for the category of this problem.

    9. You might need to add a new standard description to describe a frequently occurring issue that belongs to the category of the current problem.

  9. Select the Update Standard Resolution check box to add the contents of the problem form Resolution field as a new standard resolution for the standard description of the issue.

    10. You might need to add this resolution for the standard resolution of this issue because it provides the information about resolving the issue or it might be an alternative method of resolving the issue.

  10. To close all the tasks linked with this problem, select the Close Tasks check box.

    11. When you have completed the problem, you might want to close all linked tasks because these tasks are no longer required.

    Note: If linked records, such as linked tasks and incidents, are assigned to a staff member who is higher in the role hierarchy than the staff member who is closing the record, these linked records might not be closed if the Private sharing model is applied on the linked objects. By default, BMC Remedyforce has configured the Private sharing model for the Incidents and Tasks objects. However, administrators can configure the Private sharing model for Change, Broadcast, and Problems objects too.

  11. To close all incidents linked with this problem, select the Close Incidents check box.

    12. If you have incidents that are linked to a problem, the problem resolution is appended to those incidents when you close the problem. If you select the check box, the following message is appended to the Resolution field of the linked incidents:

    [The incident is closed because the linked Problem <problem number> has been closed. The problem resolution is: <problem resolution>]

    If you do not select the check box, the following message is appended to the Resolution field of the linked incidents:

    [The linked problem <problem number> has been closed. The problem resolution is: <problem resolution>]

  12. Click Save.
