When customers call the service desk to provide updates about their incidents, the staff member can type these updates in the Action History section.

To add a note to an incident or a service request

  1. Navigate to the incident or service request to which you want to add a note.

    2. For more information, see Navigating to an incident or a service request record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click the Actions menu, and select Add Note.

      • Alternatively, click Record Details and in the Action History section, click New Note.

    • If you are using the Incident Console tab, click the Actions menu, and select Add Note.

      • Alternatively, click Record Details and in the Action History section, click New Note.

    • If you are using the Incidents tab (Salesforce page layout), in the Action History section, click New Note.

      • You can add a note in the Note field also. To add the Note field on the incident details page, contact your system administrator to customize the incident layout to display the Add Notes Visualforce page. Request the system administrator to give a logical name, such as Note, to the section where the Add Notes Visualforce page is added.

    • If you are using the Visualforce page layout UI, add the updates in the Note field.
  3. Enter the note details.
  4. (Optional) In the Duration field, enter the time that you have spent on the action for which you are adding a note.
  5. Click Add.
