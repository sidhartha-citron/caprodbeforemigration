When an issue or outage is reported, you create a broadcast for it. However, if the issue or outage is resolved without any action from you, you can delete the broadcast that you created earlier.

For example, if all of computers on your network have Internet connectivity issues, you create a broadcast for this issue. However, if your Internet service provider fixes the Internet connectivity issue, this broadcast is no longer valid and can be deleted.

To delete a broadcast

  1. Navigate to the record that you want to delete.

    2. For more information, see Navigating to a broadcast record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, select the check box for the required broadcast, and click Delete.

      • To delete multiple records, select the check boxes for all of the records that you want to delete.

    • If you are using the Broadcasts tab (Salesforce page layout), click Delete.
    • If you are using the Visualforce page layout user interface, click .
  3. In the confirmation dialog box, click Yes.

    4. The broadcast record is deleted and the broadcast form is closed.
