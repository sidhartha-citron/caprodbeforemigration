Incident Management enables you to enter, maintain, and resolve client issues. BMC Remedyforce enables you to build a knowledge base as you create and resolve incidents. As you work on an issue, BMC Remedyforce maintains a complete audit trail and records the date and time each action occurred, the staff member who performed the work, and the time spent on working on the issue. This information is available to help solve the next issue or to generate a report for management.

The following topics are provided:

  • Registering and assigning incidents
  • Working with incidents and service requests
  • Closing and reopening incidents or service requests
  • Viewing the Alignability Process Model for Incident Management
  • Linking records of other objects in console layout UI and Salesforce page layout

    • Note: BMC Remedyforce supports keyboard shortcuts for performing a few common actions on the Remedyforce Console and Incident Console tabs. You must use the same shortcuts for all languages and browsers that are supported in BMC Remedyforce. For more information about the supported keyboard shortcuts, see Keyboard shortcuts.

  • Linking records of other objects in the Visualforce page layout UI
