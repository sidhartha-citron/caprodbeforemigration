You can add attachments to an incident or a service request record before and after saving a record.

Note: On the Remedyforce Console and Incident Console tabs, to add attachments to an incident or a service request before saving, from the Actions menu, select Attach/View Files.

If you are using the Incidents tab (Salesforce page layout) and you cannot see the documents that you attached to a record from the Visualforce page layout UI of the same record, contact your system administrator.

To add an attachment to an incident or a service request

  1. Navigate to the incident or service request to which you want to add an attachment.

    2. For more information, see Navigating to an incident or a service request record.

  2. Perform one of the following actions:
    • On the Remedyforce Console tab, from the Actions menu, select Attach/View Files.
    • On the Incident Console tab New form, from the Actions menu, select Attach/View Files.
    • If you are using the Incidents tab (Salesforce page layout), in the Action History section, click Attach File.
  3. In the Attach/View Files window, to select the file that you want to attach to the incident, click Choose File.

    4. Note: The size of one attachment cannot exceed 10 MB.

  4. Select the file that you want to add and click Open.
  5. Click Close.

    6. The selected file is attached to the record. In the Attach/View Files window, you can view or delete the existing attachments.

  6. If you are using the Visualforce page layout UI, perform the following actions:
    1. In the Supporting Information section, click the Documentation tab.
    2. From the menu, select Add.
    3. To attach a file, click Choose File and select the required file.
    4. To attach a URL, select URL, and provide the required URL.
    5. Click OK.
    6. (Optional) To open an attached document or URL, select the document or URL and from the menu, select Open.
    7. (Optional) To delete an attached document or URL, select the document or URL and from the menu, select Delete.
