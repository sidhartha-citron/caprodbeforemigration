By default, a broadcast is assigned to the user who creates it. After saving the broadcast, you can assign the record to another user or queue.

In the Assignment Details section, based on the setting configured by the system administrator, either the Owner field or Queue and Staff fields are displayed.

To change the broadcast owner

  1. Navigate to the broadcast that you want to change the owner.

    2. For more information, see Navigating to a broadcast record.

    Note: On the Remedyforce Console tab, you can change the owner for multiple records. In the list view, select the required records and click Change Owner.

  2. Perform one of the following actions:
    • In the Assignment Details section on the Remedyforce Console tab, click .
    • In the Staff Assignment Details section, on the Broadcasts tab (Salesforce page layout), click Change beside the name of the user to whom the broadcast is currently assigned.
  3. From the Owner list, select the appropriate option:
    • To assign the broadcast to a user, select User.
    • To assign the broadcast to a queue, select Queue.

      • The Customer Portal User option is not required for broadcasts.

  4. To select the user or queue, click and select the required option.
  5. Click Save.

    6. Note: To use the staff and queue assignment feature, you must add the Edit Layout > Visualforce Pages > StaffAssignmentPage to the Staff Assignment Details section. BMC recommends that you remove the Owner field from the Staff Assignment Details section. You must adjust the height of the StaffAssignmentPage component to avoid displaying white space on the form.

    If the StaffAssignmentPage Visualforce page is displayed on your layout, the Staff and Queue fields are displayed.

  6. In the Staff field, select the required staff member.

    7. All staff members are displayed.

  7. To assign the record to a queue, in the Queue field, select the required queue.
  8. To assign the record to a staff member who belongs to a queue, first click the row of the queue record and from the list of staff members belonging to the queue, select the required staff member.

    9. When you select a queue, the staff members who belong to the queue are displayed in the Staff field.

  9. If you are using the Visualforce page layout user interface, perform the following actions:
    1. Click the Assigned To menu.

      2. The following table provides the options provided in the Assign To menu:

Assigning a broadcast to a suggested expert

System administrators can designate a user to be an expert for a category. On the Broadcasts tab (Salesforce page layout), on the broadcast details page, you can assign the broadcast to an expert in the selected category.

The Assign to Suggested Expert button is not provided by default. Contact your system administrator to edit the broadcast layout to display the button on the page.

To assign a broadcast to a suggested expert

  1. Navigate to the broadcast that you want to assign to a suggested expert.

    2. For more information, see Navigating to a broadcast record.

  2. Click Assign to Suggested Expert.
  3. Select an expert.

    4. By default, suggested staff experts are displayed.

  4. To view the suggested queue, select Queue from the Views menu, and click Apply.

    5. Note: To use the staff and queue assignment feature, contact your system administrator to customize the broadcast layout and add the StaffAssignmentPage Visualforce page. Also, the administrator must enable the Enable Queue and User Assignment check box on the Record Assignment page.

    If a staff member is not available or is out of the office, the status is displayed in the Out of Office column in the Search for a user window. Contact your system administrator to configure the Out of Office column in the Search for a user window.

    To set your out-of-office status, navigate to Setup > My Personal Information > Personal Information. In the BMC Remedyforce section, click Edit, and select the Out of Office check box.

    Note: If you have enabled the improved Setup user interface in your Salesforce organization, navigate to My Settings > Personal > Personal Information. For more information about the improved Setup user interface, see http://help.salesforce.com/apex/HTViewHelpDoc?id=admin_setup_improved.htm&language=en_US.

    If the StaffAssignmentPage Visualforce page is displayed on your layout, the Staff and Queue fields are displayed.

  5. In the Staff field, select the required staff member.

    6. All staff members are displayed.

  6. To assign a record to a queue, in the Queue field, select the required queue.
  7. To assign a record to a staff member of a queue, first select the row of the required queue and then select the queue member.

    8. When you select a queue member, the selected staff member is displayed in the Staff field.
