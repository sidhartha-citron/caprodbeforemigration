You can assign translators of the language in which you are localizing BMC Remedyforce and assign correct permissions to these translators.

To assign translators

  1. Navigate to Setup > Translation Workbench > Translation Settings.
  2. In the Action column, click Edit beside the language for which you want to assign translators.
  3. In the Identify Translators for this Language section, select the translators from the Available list, and click Add.

    4. If you do not see the member that you want to add, enter keywords in the search box and click Find.

    Important: Ensure that all translators have the View Setup and Configuration permissions so that they can begin localizing. Users can only translate in languages that they are assigned to.

  4. Click Save.

For more information, see https://na7.salesforce.com/help/doc/user_ed.jsp?section=help&target=adding_and_editing_translated_languages.htm&loc=help&hash=topic-title.
