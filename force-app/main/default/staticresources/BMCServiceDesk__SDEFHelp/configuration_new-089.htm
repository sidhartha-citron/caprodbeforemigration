By default, the Configuration Item~ column is displayed in the Linked Services and CIs or Linked CIs sections. However, if the column is not displayed, system administrators can change the page layout to display the column.

To display the Configuration Item~ column in the Linked CIs section

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Manage Objects tile, and from the menu, select Create and Edit Objects.
  3. On the Create and Edit Objects page, from the Object list, select All Other Objects.
  4. In the Label column, select the required object.
  5. In the Page Layouts section, ensure that Remedyforce Incident Version 3.0 is assigned.
  6. If the Remedyforce Incident Version 3.0 layout is not present, click Page Layout Assignment.
  7. Click Edit Assignment.
  8. From the Page Layout To Use list, select Remedyforce Incident Version 3.0.
  9. From the Profiles list, select the required profiles.
  10. Click Save.
