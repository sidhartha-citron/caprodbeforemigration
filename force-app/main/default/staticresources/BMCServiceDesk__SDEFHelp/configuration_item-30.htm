Click the Performance Metrics icon, in the upper-right section of the Instance Editor or the Relationship Editor forms, to display the performance metrics for these forms. The following performance metrics are displayed in a ToolTip:

    • Server Time—The time taken by the Force.com server to process a request.
    • Network Time—The time taken to complete the data transfer to the browser over the network. The accuracy of this time depends on the how closely your computer's time is synchronized with correct time of your time zone. If your computer's time is incorrect, contact your system administrator to correct it.
    • Page Render—The time taken by the browser to load all the data after receiving a response from the Salesforce server.

      • Note: The unit of measurement is milliseconds.

If you observe that it is taking a long time to open the Instance Editor or the Relationship Editor form, you can use the performance metrics to identify problem area. For example, if the Network Time metric is high and the Page Load metric is low, the problem is with your Internet connection, which is slow and is taking a long time to receive a request sent by the Force.com server.
