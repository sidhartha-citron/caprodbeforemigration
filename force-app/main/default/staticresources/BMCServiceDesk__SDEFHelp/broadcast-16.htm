You can add information about tasks in the Action History section.

To add a note to a broadcast

  1. Navigate to the broadcast to which you want to add a note.

    2. For more information, see Navigating to a broadcast record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click the Actions menu, and select Add Note.

      • Alternatively, click Record Details and in the Action History section, click New Note.

    • If you are using the Broadcasts tab (Salesforce page layout), in the Action History section, click New Note.
  3. If you are using the Visualforce page layout UI, add the updates in the Note field.
  4. Enter the note details.
  5. (Optional) In the Duration field, enter the time that you have spent on the action for which you are adding a note.
  6. Click Add.
