Performance Metrics for the Change Request form

Click the Performance Metrics icon, in the upper-right section of the form, to display the performance metrics for this form. The following performance metrics are displayed in a tooltip:

    • Server Time—The time taken by the Force.com server to process a request.
    • Network Time—The time taken to complete the data transfer to the browser over the network. The accuracy of this time depends on the how closely your computer's time is synchronized with correct time of your time zone. If your computer's time is incorrect, contact your system administrator to correct it.
    • Page Render—The time taken by the browser to load all data after receiving a response from the Force.com server.

      • Note: The unit of measurement is milliseconds.

If you observe that it is taking a long time to open the Change Request form, you can use the performance metrics to identify problem area. For example, if the Network Time metric is high and the Page Load metric is low, the problem is with your Internet connection, which is slow and is taking a long time to receive a request sent by the Force.com server.

Note: You can also view performance metrics for lookup windows, such as the Category and Client lookup windows. However, the Performance Metrics icon is disabled in the list views, and it does not display performance metrics for the list views.
