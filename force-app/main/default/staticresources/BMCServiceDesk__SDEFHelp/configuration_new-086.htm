Out-of-the-box page layouts are provided that are shown when you are printing a record. You can customize these page layouts as per your requirements.

To customize page layouts for printing

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Manage Objects tile, and from the menu, select Create and Edit Objects.
  3. On the Create and Edit Objects page, from the Object list, select All Other Objects.
  4. In the Label column, click the required object.
  5. To customize a page layout, in the Page Layouts section, click Edit for the required page layout in the Action column.

    6. For the console layout, customize fields in the Remedyforce Incident Console version 1.0 page layout.

  6. To create a new page layout, click New.
  7. To assign the page layout that you have created to the required profile, click Page Layout Assignment.
