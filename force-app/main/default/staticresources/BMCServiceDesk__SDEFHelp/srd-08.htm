When you select Picklist as the Response Type option for an Input field on the Fulfillment tab of a Request Definition, you need to add the picklist options from which the users can select an option.

To add a picklist response type

  1. In the Response Type list, select Picklist.
  2. When you select Picklist as the field type, the Picklist Values section appears.
  3. In the Displayed Value field of the Picklist Values section, type the displayed value of the picklist value.
  4. In the Stored Value field of the Picklist Values section, type the stored value of the picklist value.
  5. Click Add.
  6. The displayed value and stored value are added to the list of picklist values.
  7. If you want the picklist value to be selected by default, select the check box in the Default Value column.
  8. (Optional) To delete a picklist value, select it and click the Delete icon.
