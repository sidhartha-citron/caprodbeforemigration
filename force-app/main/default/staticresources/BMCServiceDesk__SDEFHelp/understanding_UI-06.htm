The console layout is available for the Incident, Task, Problem, Change Request, Release, and Broadcast objects in BMC Remedyforce.

The console layout provides the following features:

    • The lookup fields display all of the records and provide pagination and search options
    • The broadcast ticker is available and enables users to follow or unfollow broadcasts on Chatter
    • The category tree is available when users are creating a record
    • Users can set the owner while creating a record
    • The total number of records is displayed for all modules in the list views. If there are more than 10,000 records in a module, the total number of records is displayed as 10,000+.
    • System administrators can configure columns of the list view
    • System administrators can configure fields and related lists that suit the business requirements
    • System administrators can add buttons to forms to enable custom actions

For new installations of BMC Remedyforce 20.14.01, only the Remedyforce Console tab is available out-of-the-box. The Incident, Problem, Change Request, Task, Release, and Broadcast objects are available as modules within the integrated Remedyforce Console tab. In previous versions of BMC Remedyforce, your system administrator could configure the Incident Console and Task Console tabs.
