In BMC Remedyforce, the UI labels that can be localized are mapped to one of the following labels:

    • Field labels
    • Custom labels
    • Object labels

      • Note: If you want to use the <BLANK> field to manage the space between two fields on the console form, ensure that the field with the label <BLANK> is not translated into any other language.

The UI labels of a form that are mapped to field labels are represented with the same name in the Translation Workbench. However, UI labels that are mapped to custom labels are prefixed with intuitive keywords to help you identify these custom labels.

Object names (both singular and plural) are mapped to object labels. For example, the title of the Incidents form is mapped to plural object label. For more information about localizing object labels, see Localizing object labels.

By default, the standard fields on a form are localized in the language that is assigned to the logged-on user.
