Staff members can create and edit custom views on the Remedyforce Console, Incident Console, and Task Console tabs. To allow staff members to create a custom view, ensure that you configure a remote site on the Remedyforce Administration > Manage Objects tab. If you do not configure the remote site or do not reopen the Manage Objects tab after configuring the remote site, the staff members cannot create a new custom view. However, they can edit the view on the console. For more information about configuring the remote site, see Configuring objects.

Also, after the remote site is configured, ensure that you refresh the Remedyforce Console tab after 5 minutes to enable the staff members to create custom view.

Note: The fields that are shown on the custom list view is configurable by the Column Headers field set only. For more information, see Configuring fields shown in the list views on the Remedyforce Console tab.
