You must enable the integration between BMC Remedyforce and Dell KACE to allow staff members to work with Dell KACE console in BMC Remedyforce and view the asset summary of devices imported from Dell KACE.

For more information about how to import records from Dell KACE, see Importing data from Dell KACE by using Pentaho.

Ensure that the Dell KACE server is accessible over the Intranet.

To configure the integration between BMC Remedyforce and Dell KACE console

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Integrations tile, and from the menu select Dell KACE.
  3. If you are using the Visualforce page layout user interface, navigate to Configuration > Application Settings > Dell KACE.
  4. To enable the integration between BMC Remedyforce and Dell KACE, select the Enable Dell KACE Integration check box.
  5. In the Server URL field, type the valid URL of the Dell KACE server.

    6. For example, http://<Dell-KACE-Server-Name>.

    Note: If the URL is invalid, you receive an error message in the Configuration Items List view when you launch the Dell KACE console to view the details of the records that has been imported from Dell KACE.

  6. Click Save.
