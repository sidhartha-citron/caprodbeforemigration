By default, the SRD is available to all accounts and profiles.

The portal and nonportal users can view the SRD in Self Service based on specific conditions. For more information, see Visibility of SRD in Self Service for portal and nonportal users.

Before enabling request definitions for accounts or profiles, ensure that you have created the request definition and added the fields to get more information from clients. For more information, see Creating service request definitions and Getting more information about the request.

To enable request definitions for accounts or profiles

  1. Click the Entitlement tab of the request definition.
  2. To define the accounts that can use the SRD, perform one of the following actions:
    • To make the SRD available to users of all accounts, select All Accounts.
    • To make the SRD available to users of all selected accounts, select Selected Accounts.
  3. If you select Selected Accounts, in the Available Accounts list, select the required accounts.

    4. You can make the SRD available to a large number of accounts at one time. You can also search for a particular account by entering the search criteria in the Search field and clicking Search. To clear the search criteria in the Search field and refresh the list of accounts in the Available Accounts list, click Undo. You can use the * wildcard character to match one or more characters at the end of your search criteria, but not the beginning or the middle of your search criteria. You can navigate to the next or previous page of accounts by using Next Page and Previous Page next to the Undo.

  4. Click the right arrow to move the selected accounts from the Available Accounts list to the Selected Accounts list.
  5. To define the profiles that can use the SRD, perform one of the following actions:
    • To make the SRD available to users of all profiles, select All Profiles.
    • To make the SRD available to users of the selected profiles and the accounts you have already selected in the Selected Accounts list, select Selected Profiles.
  6. If you select Selected Profiles, in the Available Profiles list, select the required profiles.
  7. Click the right arrow to move the selected profiles from the Available Profiles list to the Selected Profiles list.
  8. Click .

To complete creating a request definition, perform the actions mentioned in Configuring the service request information to be shown to clients.
