You can assign out-of-the-box or custom themes to one or more accounts. When users who belong to an account log on to Self Service, the theme that is assigned to that account is applied to the Self Service UI.

To assign a theme to an account

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Configure Self Service tile, and from the menu, select Branding.
  3. From the User Interface Theme list, select the theme that you want to assign to one or more accounts.

    4. The box next to the User Interface Theme list displays the most prominent color in the selected theme. This color corresponds to the Broadcast Ticker Bar > Top Gradient Color element in the theme.

  4. Click the Assign Theme to Accounts link.
  5. In the Select Accounts for the Theme window, select the required accounts.

    6. You can search for a particular account by specifying the search criteria. You can use the * wildcard character to match one or more characters at the end of your search criteria, but not at the beginning or in the middle of your search criteria.

  6. Click the right arrow to move the selected accounts from the Available Accounts list to the Selected Accounts list.
  7. (Optional) To remove accounts that should not be assigned to the selected theme, select the accounts in the Selected Accounts list and click the left arrow to move the selected accounts from the Selected Accounts list to the Available Accounts list.
  8. To save the account assignment to the selected theme, click Apply.

    9. The selected theme becomes the default theme and is available to Self Service users after they log on. The Self Service logon page appears in the default theme for all Self Service users.

  9. In the confirmation dialog box, click OK.
  10. Close the Select Accounts for the Theme window and return to the Branding tab.
