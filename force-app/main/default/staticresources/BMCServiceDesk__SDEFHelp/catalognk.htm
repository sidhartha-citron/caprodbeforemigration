In the Catalog console, you can create new folders and subfolders and add the QuickViews to these folders and subfolders. This allows you to organize your QuickViews in separate folders and quickly access the QuickViews that you use most frequently.

After you create the required folders, you can drag existing QuickViews to the folders to quickly access them. The drag-and-drop functionality is available only for QuickViews. You cannot drag a folder to make it a subfolder of an existing folder. To make all QuickViews in a folder as a subfolder of an existing folder, you must create the required subfolder and drag the QuickViews to the subfolder.

Note: You cannot drag QuickViews from the All Available QuickViews folder to the My QuickViews folder or its subfolders. In addition, you cannot drag any QuickViews from the My QuickViews folder or its subfolders to the All Available QuickViews folder.

The following QuickView folders are available out-of-the-box:

    • My QuickViews—You can use the folder to store all the QuickViews and folders that you create.
    • All Available QuickViews—This folder contains the out-of-the-box QuickViews described in the following table. You cannot edit the out-of-the-box QuickViews. If you want to update an out-of-the-box QuickView, you can copy the required QuickView and then customize the QuickView according to your requirements.

If you have changed your language from English to another language, the My Change Request Service Targets with warnings, My Incident Service Targets with warnings, and My Service Requests out-of-the-box QuickViews do not display any records when you view these QuickViews.

You can edit your language by navigating to Setup > My Personal Information > Personal Information. For more information about editing personal information, see https://na11.salesforce.com/help/doc/user_ed.jsp?section=help&target=usersetup.htm&loc=help&hash=topic-title.

The My Change Request Service Targets with warnings and My Incident Service Targets with warnings QuickViews do not display records because the value of the Status data field in the query condition is Warning and remains in English. This value is not translated in the new language. The My Service Requests QuickView does not display records because the value of the Service Request data field in the query condition is Yes and remains in English. This value is not translated in the new language.

BMC recommends that you make a copy of these QuickViews and edit the values of the Status and the Service Request data fields in the Query Conditions section of the QuickViews wizard. Update the value of the Status data field from Warning to a value in the new language. Update the value of the Service Request data field from Yes to a value in the new language. For example, if you have changed the language to French, edit the My Change Request Service Targets with warnings and My Incident Service Targets with warnings QuickViews and update the Warning value of the Status data field to Avertissement.
