To make the BMC Remedyforce dashboard available to users of the ServiceDesk Staff profile, you must configure the Running User property of the dashboard to be a user with a Salesforce Platform license. Additionally, you must select a user who is high in the role hierarchy because the data visibility of the Running User is applied to the dashboard.

To configure the BMC Remedyforce dashboard

  1. Click the Dashboards tab.
  2. From the View Dashboard list, select BMC Remedyforce Dashboard.

    3. The BMC Remedyforce Dashboard appears with updated data.

  3. Click Edit.

    4. You can select the user for whom you want to view the dashboard from the View dashboard as list.

  4. Click the View dashboard as list.

    5. The dashboard running user details are displayed. By default, the Run as selected user option is selected.

  5. In the View Dashboard as text box, type the name of the user for whom you want to view the dashboard.

    6. If you select Run as logged in user, the dashboard for the logged-in user is displayed.

  6. Click Save.
