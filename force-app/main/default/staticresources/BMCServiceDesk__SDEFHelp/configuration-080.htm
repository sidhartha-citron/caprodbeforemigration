You can use only Visualforce email templates in the Email Message window.

To ensure that you can use the email template that you are creating, select User in the Recipient Type list and <namespace of organization>__<object name>__c in the Relate To Type list while creating the template.

Also, to allow users to use email templates in email conversations, enable the Email Template field in Remedyforce Administration > Application Settings.

To create an email template

  1. Navigate to Setup > Communication Templates > Email Templates.
  2. Click New Template.
  3. In the Step 1. Email Template: New Template page, select Visualforce, and click Next.
  4. In the Email Template Information section, from the Folder list, select the folder for your template.
  5. To enable users to use the template when sending email messages, select the Available For Use check box.
  6. In the Email Template Name field, enter a name for the template.

    7. The following table displays the text that you must always include in the subject of the email message to track the email conversation:

    Object
    Text
    Change Requests
    Ref:CR:<Change Request Number>
    Incidents
    Ref:IN:<Incident Number>
    Problems
    Ref:PR:<Problem Number>
    Tasks
    Ref:TA:<Task Number>

  7. In the Visualforce Template Attributes section, in the Email Subject field, enter the subject of the email template.
  8. From the Recipient Type list, select User.
  9. From the Relate To Type list, select <namespace of organization>__<object name>.
  10. Click Save.

Enter the email text after creating the email template. For information about using fields available on the object forms, see Customizing an email template.
