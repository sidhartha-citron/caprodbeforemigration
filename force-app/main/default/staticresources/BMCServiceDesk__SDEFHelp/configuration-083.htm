If you want your clients to provide feedback after their incident or service request is closed, you must configure the following workflows to send an email message with a link to a survey:

    • Notify the client when a service request is closed
    • Notify the client when an incident is closed

Clients can click the survey link in the email message and provide their feedback. BMC Remedyforce uses the following email templates to send the email messages to clients when their incident or service request is closed:

    • Email Template for submitting feedback when incident is closed
    • Email Template for submitting feedback when service request is closed

You must update these email templates to use the URL of the Force.com site that you have configured for your clients.

The following steps provide an overview of how to collect feedback through an email survey:

Step 1 Copy the URL of the Force.com site where you want to host the Feedback Survey Visualforce page and paste the URL in a Notepad file.

Step 2 Update the required email templates to use the URL of the Force.com site.

Step 3 Assign the updated email templates to the required workflows.

Step 4 Make the Feedback Survey Visualforce page accessible to the Force.com site.

To collect feedback through an email survey

  1. Navigate to Setup > Develop > Sites.
  2. If you have configured Self Service for BMC Remedyforce, copy the URL of the BMC Remedyforce Self Service site and paste it in a Notepad file.
  3. If you have not set up a Force.com site on your organization, perform the following actions:
    1. Navigate to Setup > Develop > Sites.
    2. In the Sites page, type the domain that you want to associate with the site.
    3. Click Check Availability.

      4. If the domain is available, the following message appears: 

      Success: The Force.com domain name "<domain>-force.com" is 
available.
    4. Select the I have read and accepted the Force.com Sites Terms of Use check box.
    5. Click Register My Force.com Domain.

      6. Note: You cannot modify the domain name after you complete the registration process.

    6. In the confirmation dialog box, click OK.
    7. In the Sites (<domain name>) section, click New.
    8. In the Site Label field, type the display label of this site.
    9. In the Site Name field, type the name of this site.
    10. In the Site Description field, type the description of this site.

      11. By default, your name is populated in the Site Contact field.

    11. Select the Active check box.
    12. In the Active Site Home Page and Inactive Site Home Page fields, ensure that SSSiteLogin and InMaintenance are entered.
    13. In the Site Template field, type SiteTemplate.
    14. Select the Enable Feeds check box.
    15. Click Save.
    16. Copy the URL of the site and paste it in a Notepad file.
  4. If you have a Force.com site and want to use the site to host the BMC Remedyforce Feedback Survey Visualforce page, perform the following actions:
    1. Navigate to Setup > Develop > Sites.
    2. Copy the URL of the site and paste it in a Notepad file.
  5. Navigate to Setup > Communication Templates > Email Templates.
  6. From the Folder list, select BMC Remedyforce Emails.
  7. In the Email Template Name column, click Email Template for submitting feedback when incident is closed.
  8. To view the Visualforce code for this template, click Edit Template in the Email Template section.

    9. Note: BMC recommends that you save a copy of the existing Visualforce code for this template before you make updates to the code.

  9.   10. Click Save.
  11. Navigate to Setup > Communication Templates > Email Templates.
  12. From the Folder list, select BMC Remedyforce Emails.
  13. In the Email Template Name column, click Email Template for submitting feedback when service request is closed.
  14. Repeat step 8 through step 10.
  15. To assign the Email Template for submitting feedback when incident is closed email template to the Notify the client when an incident is closed workflow rule, perform the following actions:
    1. Navigate to Setup > Create > Workflows and Approvals > Workflow Rules.
    2. In the Rule Name column, click the Notify the client when task is closed workflow rule.
    3. Click Activate.
    4. In the Workflow Actions section, click Edit.
    5. From the Add Workflow Action list, select Select Existing Action.
    6. From the Choose Action Type list, select the Email Alert type of action.
    7. In the Available Actions list, select Email Alert notify_client_when_incident_is_closed.
    8. To move the selected action from the Available Actions list to the Selected Actions list, click the right arrow.
    9. Click Save.
    10. In the Immediate Workflow Actions section, click notify_client_when_incident_is_closed.
    11. In the Email Alert Detail section, click Edit.
    12. To configure this workflow to use the Email Template for submitting feedback when incident is closed email template, click the Email Template lookup.

      13. This workflow currently uses the Incident Email Template 6 Version 2.0 email template.

    13. Select the Email Template for submitting feedback when incident is closed email template.
    14. Click Save.
  16. To assign the Email Template for submitting feedback when service request is closed email template to the Notify the client when a service request is closed workflow rule, perform the following actions:
    1. Navigate to Setup > Create > Workflows and Approvals > Workflow Rules.
    2. In the Rule Name column, click the Notify the client when service request is closed workflow rule.
    3. If the notify_client_when_service_request_is_closed email alert is present in the Immediate Workflow Actions section, proceed to step k.
    4. Click Activate.
    5. In the Workflow Actions section, click Edit.
    6. From the Add Workflow Action list, select Select Existing Action.
    7. From the Choose Action Type list, select the Email Alert type of action.
    8. In the Available Actions list, select Email Alert notify_client_when_service_request_is_closed.
    9. To move the selected action from the Available Actions list to the Selected Actions list, click the right arrow.
    10. Click Save.
    11. In the Immediate Workflow Actions section, click notify_client_when_service_request_is_closed.
    12. In the Email Alert Detail section, click Edit.
    13. To configure this workflow to use the Email Template for submitting feedback when service request is closed email template, click the Email Template lookup.

      14. This workflow currently uses the Service Request Closed email template.

    14. Select the Email Template for submitting feedback when service request is closed email template.
    15. Click Save.
  17. To make the Feedback Survey Visualforce page accessible to your Force.com site, perform the following actions:
    1. Navigate to Setup > Develop > Sites.
    2. In the Site Label column, click the label of the site.
    3. In the Site Visualforce Pages section, click Edit.
    4. In the Available Visualforce Pages list, select BMCServiceDesk.FeedbackSurveyPage.
    5. To move the selected Visualforce page from the Available Visualforce Pages list to the Enabled Visualforce Pages list, click the right arrow.
    6. Click Save.
