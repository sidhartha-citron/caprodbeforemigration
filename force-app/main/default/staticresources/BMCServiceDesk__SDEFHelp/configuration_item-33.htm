The following table list the CI Explorer toolbar buttons.

Table 13-3:  CI Explorer toolbar buttons  
Toolbar buttons
Description
Reload
Updates the CI Explorer with the latest information.
Setting
Opens the Setting window where you can configure the line and field settings of the CI Explorer.
For more information, see Configuring the Configuration Items Explorer.
Show/Hide Information Pane
Displays or hides the Information Pane where you can view incidents, tasks, problems, change requests, and releases that are linked to the selected CI in the CI Explorer. You can open a linked record by double-clicking the required record in the Information Pane.

Note:  In the Information Pane, you cannot view closed records that are linked to the selected CI.
Layout
Displays the CI image in the layout style you select from the Layout list.
Reposition Nodes
Rearranges the nodes in the CI Explorer for the current layout. You can continue clicking this toolbar button until the current layout suits your requirements.

Note:  This toolbar button is enabled for the Grid, Anneal, and Spring layouts.

For more information, see Configuring the Configuration Items Explorer.
Zoom bar
Resizes the size of the nodes in the CI Explorer for easier viewing. You can resize the nodes by dragging the slider between theicon and theicon on the zoom bar or by clicking the required section of the zoom bar.
