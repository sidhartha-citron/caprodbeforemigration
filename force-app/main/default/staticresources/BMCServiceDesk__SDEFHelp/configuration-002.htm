You can create staff members for your Salesforce organization on the Active Users page.

To add staff members

  1. Navigate to Setup > Manage Users > Users.
  2. On the Active Users page, click New.
  3. From the Role list, select a role for the staff member.
  4. From the Profile list, select the required profile.

    5. The profiles are available for BMC Remedyforce:

    • ServiceDesk Change Manager
    • ServiceDesk Staff
    • ServiceDesk System Administrator

      • Note: BMC recommends that you use the Client form to add client users.

      For more information, see https://help.salesforce.com/htviewhelpdoc?err=1&id=admin_users.htm&siteLang=en_US.

If you are creating a staff member, customize the default Salesforce.com User page layout to use the User Layout page layout, which is available when you install the managed package. After customizing the User page layout, edit existing user records or create new user records, and select the ServiceDesk Staff check box to designate the user as a BMC Remedyforce staff member. Also, assign the ServiceDesk Staff permission set to the users. For more information, see Assigning permission sets.

You can view the details of these staff members in the Clients form. However, you cannot edit the staff member records and the Save toolbar button is disabled in the Clients form. Additionally, the Account field appears as an optional field in the Clients form when you view these staff member records in the Clients form. These staff members can log on to BMC Remedyforce and Self Service.

Note: If you do not select the ServiceDesk Staff check box for these staff members, they do not appear in the Select from Staff window that appears when you use the Assign To menu to assign a record to staff members.
