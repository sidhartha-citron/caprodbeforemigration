BMC Remedyforce uses Java Web Start to launch the BMC Client Management console. You must install the latest version of the Oracle® Java Runtime Environment (JRE™) as listed in the following table. When you install the JRE, Java Web Start is also installed on your computer. For more information about Java Web Start, see http://docs.oracle.com/javase/7/docs/technotes/guides/javaws.

BMC Client Management version
Required JRE version
Version 11.7
Version 7 Update 21 or later
Version 11.6
Version 6 or later

Note:  BMC recommends that you install Version 7.

You can download JRE from http://www.java.com/getjava.
