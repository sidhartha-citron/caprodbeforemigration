In Visualforce page layouts UI, additional information about a record is available in the Supporting Information section.

The following sections are provided:

    • Viewing linked change requests
    • Viewing linked tasks
    • Viewing linked incidents
    • Performance Metrics for the problem form

Viewing user and system actions

You can view the user-defined and system-defined actions performed on a problem record in the History tab in the Supporting Information section. To view these actions, you can use the Views menu. The following options are provided in the Views menu:

View menu commands
Description
User Entries
Displays all user-defined actions that have been performed on a problem
System Entries
Displays all system-defined actions that have been performed on a problem
All Entries
Displays all actions (both user-defined and system-defined) that have been performed on a problem

Viewing linked change requests

You can filter the change requests linked to a problem by using the Views menu.

To view linked change requests

  1. Click the record whose linked change requests you want to view.
  2. In the Supporting Information section, click the Change Requests tab.
  3. Click .
  4. To view all the linked change requests, select All.
  5. To view the linked change requests that are in the Open stage, select Open.
  6. To view the linked change requests that are in the Acknowledged stage, select Acknowledged.
  7. To view the linked change requests that are in the In Process stage, select In Process.
  8. To view the linked change requests that are in the Closed stage, select Closed.

Performance Metrics for the problem form

Click the Performance Metrics icon, in the upper-right section of the form, to display the performance metrics for this form. The following performance metrics are displayed in a tooltip:

    • Server Time—The time taken by the Force.com server to process a request.
    • Network Time—The time taken to complete the data transfer to the browser over the network. The accuracy of this time depends on how closely your computer's time is synchronized with correct time of your time zone. If your computer's time is incorrect, contact your system administrator to correct it.
    • Page Render—The time taken by the browser to load all data after receiving a response from the Force.com server.

      • Note: The unit of measurement is milliseconds.

If you observe that it is taking a long time to open the problem form, you can use the performance metrics to identify problem area. For example, if the Network Time metric is high and the Page Load metric is low, the problem is with your Internet connection, which is slow and is taking a long time to receive a request sent by the Force.com server.

Note: You can also view performance metrics for lookup windows, such as the Category and Client lookup windows. However, the Performance Metrics icon is disabled in the list views, and it does not display performance metrics for the list views.
