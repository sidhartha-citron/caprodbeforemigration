To enable users of a profile to use a custom form, you must assign the custom version of the form to the required profile. You need to assign the custom version of a form to a profile only for the first time. However, whenever there are updates to the profiles to which the customized form must be assigned, you must follow this process.

To assign the custom version of a form to a profile

  1. In the BMC Remedyforce home page, click Configuration.
  2. From the list of Configuration options, expand Configuration.
  3. In the Configuration list, click Application Settings.
  4. In the Form Assignment tab, from the Form to Assign list, select the type of form to be assigned.
  5. In the Assign Standard Form list, select the profiles to which you want to assign the custom form.
  6. To move the selected profiles from the Assign Standard Form list to the Assign Custom Form list, click the right arrow.
  7. Click Save.

    8. Users of the selected profiles view the custom version when they open the type of form selected in the Form to Assign list. The custom form is available when users log on to BMC Remedyforce after the current time. The custom form is not displayed for currently logged-on users of the selected profiles.
