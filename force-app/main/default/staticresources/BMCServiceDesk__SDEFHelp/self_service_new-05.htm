BMC Remedyforce provides out-of-the-box themes that you can assign to one or more accounts. If required, you can apply an out-of-the-box theme, replace the default images with your company logo, and specify your company URL. To meet the branding requirements of your company or the different accounts that you support, you can also create custom themes based on existing themes.

For each theme, you can configure multiple UI elements such as the banner, tiles, broadcast ticker bar, and active foreground tab. When creating a custom theme, you can use the legend to understand the location of the different UI elements in Self Service.

You can create an unlimited number of custom themes for Self Service. However, the out-of-the-box or custom themes that you assign to an account are displayed only after users log on to Self Service. The default theme is assigned to the Self Service login page for all users. If you do not assign a theme to an account, or the users do not belong to an account, the default theme is also displayed to the users after they log on to Self Service.

The following table describes a few scenarios in which you might want to create a custom theme.

Goal
Recommended action
Use an out-of-the-box color scheme for two accounts, but assign different images and company URL for each account
Create a new theme based on the out-of-the-box theme, retain the color scheme, and change the images and company URL.
You can assign the out-of-the-box theme to one account and the custom theme to the other account.
Make a few changes to the colors in an existing theme, and assign it to all accounts
Create a new theme based on the existing theme, update the colors for specific UI elements, change or retain the default images, and specify the company URL.
You can assign this custom theme to all accounts.
Implement specific branding requirements for multiple accounts that you support
Create multiple custom themes and modify the color scheme, default images, and company URL for each.
You can assign each custom theme to the appropriate account.
