You can integrate BMC Remedyforce with BMC ProactiveNet Performance Management to create incidents in BMC Remedyforce using the events that are generated in BMC ProactiveNet Performance Management. This integration helps you to create, update, and close incidents based on the events generated in BMC ProactiveNet Performance Management. Data in the events is mapped to the fields in the Incident object.

To enable sending emails using events from BMC ProactiveNet Performance Management, you must configure email service in BMC Remedyforce.

For information about email limits of your Salesforce organization, see http://www.salesforce.com/us/developer/docs/apexcode/Content/apex_gov_limits.htm.

To integrate BMC Remedyforce with BMC ProactiveNet Performance Management, perform the following:

    • Customizing event rules in BMC ProactiveNet Performance Management
    • Creating email services
    • Configuring Application Settings

For a use case scenario, see Use case scenario.
