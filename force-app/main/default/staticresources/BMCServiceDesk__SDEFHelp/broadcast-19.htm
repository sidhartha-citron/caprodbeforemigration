You can search for information that might help you for the records that are assigned to you. You can search for information in other incidents, tasks, broadcasts, configuration items, change requests, contacts, problems, knowledge articles, documents uploaded to the Documents tab of your Salesforce.com organization, users, knowledge articles of RightAnswers, and Google.

The first few words of text that you save in the Description field are used as the default text to be searched. To search for text other than the text available in the Description field, delete the text from the text box, type the required text, and click .

To refine search, use operators such as AND, OR, AND NOT, *, and ?. For example, to search records with “Networking” or “Networks” but not “Coverage,” enter Network* AND NOT Coverage. Records that contain any word that begins with “Network” but do not contain the word “Coverage” are displayed. For more information, see https://na12.salesforce.com/help/doc/user_ed.jsp?section=help&target=search_unified_refine.htm&loc=help&hash=topic-title.

To search knowledge

  1. Navigate to the record for which you want to search knowledge.

    2. For more information, see Navigating to a broadcast record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Agent Tools menu, select Knowledge Search.
    • If you are using the Broadcasts tab (Salesforce page layout), click Knowledge Search.
  3. In the Global Search window, enter the text that you want to search for in the text box.

    4. For more information about the options available in the Global Search window, see Searching knowledge to resolve a record.
