After saving the task, you can add user-defined actions and system-defined actions to a task in the Action History section (in the History tab of the Supporting Information section in the Visualforce page layout UI).

To add an action history record to a task record

  1. Navigate to the task that you want to add action history.

    2. For more information, see Navigating to a task record.

  2. Perform one of the following actions:
    • If you are using Remedyforce Console tab, from the Actions menu, select New Action.

      • Alternatively, click Record Details and in the Action History section, click New Action.

    • If you are using the Task Console tab, from the Actions menu, select New Action.

      • Alternatively, click Record Details and in the Action History section, click New Action.

    • If you are using the Tasks tab (Salesforce page layout), click New Action in the Action History section.
    • If you are using the Visualforce page layout UI, click the History tab in the Supporting Information section and from the menu, select Add.
  3. Select the appropriate action that you want to add.

    4. If you select the Notes option, a new browser window is displayed where you can enter additional notes. This note is displayed in the Action History section in Salesforce page layout. If you are using Salesforce page layout and do not see the Note field on your layout, contact your system administrator to edit your page layout.

    In the Visualforce page layout UI, this note is displayed in the History tab. You can create a workflow to send an email message to the client when a particular action is added to the History tab.

  4. Add the notes, and click Save.

Viewing actions related to tasks

You can view the actions related to a task by using the View Action History option. The View Action History option allows you to create a new action, filter actions based on type, and print the actions.

Note: The Task History window displays the actions in the view-only mode. You can not edit or delete the actions that are displayed in the Task History window.

To view actions related to a task

  1. Navigate to the task for which you want to view actions.

    2. For more information, see Navigating to a task record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click the Actions menu, and select View Action History.

      • Alternatively, click Record Details and in the Action History section, click View Action History.

    • If you are using the Task Console tab, click the Actions menu, and select View Action History.

      • Alternatively, click Record Details and in the Action History section, click View Action History.

    • If you are using the Tasks tab (Salesforce page layout), in the Action History section, click View Action History.
  3. (Optional) To filter the records based on the action type, select the required action type from the list.
  4. (Optional) To print the actions related to a task, click Print.
