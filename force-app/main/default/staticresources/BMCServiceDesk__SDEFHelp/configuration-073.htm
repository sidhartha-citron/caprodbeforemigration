You can add QuickViews to a folder by creating a new QuickView in the folder or copying an existing QuickView into the folder.

To add a QuickView to a folder

  1. In the BMC Remedyforce home page, click Catalog.
  2. From the list of Catalog options, expand QuickViews.
  3. To add an existing QuickView to the folder, perform the following actions:
    1. Navigate to the folder that contains the QuickView you want to add.
    2. Select the required QuickView.
    3. Click Copy Query.
    4. Select the folder where you want to paste the QuickView.
    5. Click Paste Query.
    6. In the Add Query window, in the QuickView Name field, type the name of the QuickView.
    7. Click OK.
  4. To create a new QuickView in the folder, perform the following actions:
    1. Navigate to the folder where you want to create the QuickView.
    2. Click New Query.
    3. In the QuickViews wizard, create the QuickView.

      4. For more information, see Creating QuickViews.
