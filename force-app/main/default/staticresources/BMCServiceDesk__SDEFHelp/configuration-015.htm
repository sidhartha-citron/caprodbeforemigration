You can view the incidents that linked to a client.

To view incidents linked to a client

  1. In the BMC Remedyforce home page, click Configuration.
  2. From the list of Configuration options, expand Configuration.
  3. Click Clients.
  4. From the clients list view, select the client record for which you want to view linked incidents.
  5. In the Supporting Information section, click the Incidents tab.
  6. To view all the linked incidents, from the View menu, select All.
  7. To view the linked incidents that are in the Open stage, from the View menu, select Open.
  8. To view the linked incidents that are in the Acknowledged stage, from the View menu, select Acknowledged.
  9. To view the linked incidents that are in the In Process stage, from the View menu, select In Process.
  10. To view the linked incidents that are in the Closed stage, from the View menu, select Closed.
