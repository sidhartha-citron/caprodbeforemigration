Service Request Management supports the service desk function of the Information Technology Infrastructure Library® (ITIL®). The service desk is specifically defined within ITIL because the service desk is critical in IT user support. The goal of the service desk is to provide a single point of contact between the IT user and the IT organization.

The following topics are provided:

  • Overview of Service Request Management
  • Overview of the request definition creation process
  • Creating service request definitions
  • Configuring service requests settings
