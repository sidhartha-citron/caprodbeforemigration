On the Banner Configuration page, you can configure settings related to the banner displayed in Self Service.

To configure banner settings

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Configure Self Service tile, and from the menu select Banner.
  3. Select the Display application banner check box to display the banner of the BMC Remedyforce application.

    4. The application banner in Self Service consists of the banner that you select and My Profile, About, and Log Out links.

  4. Select the Display My Profile on application banner check box to display the My Profile link in the banner.

    5. The My Profile link allows the logged-on users to add or update their details.

  5. To enable Self Service users to control broadcast ticker speed, select the Allow clients to control Broadcast ticker speed check box

    6. If you select this check box, the Broadcast Ticker Speed field is displayed on the My Profile page in Self Service.

  6. In the Default ticker speed field, set the required speed of broadcast ticker speed in the BMC Remedyforce application.
  7. Click Save.
  8. In the confirmation dialog box, click OK.
