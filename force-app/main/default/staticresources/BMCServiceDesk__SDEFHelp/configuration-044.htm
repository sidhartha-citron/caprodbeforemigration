This section provides information about the field sets of the object forms that can be customized. The available fields and the fields that are displayed on the existing forms are also listed.

The following topics are provided:

    • Change Request field sets
    • Client field sets
    • Incident field sets
    • Problem field sets
    • Self Service: Service Request field sets
    • Task field sets

Change Request field sets

To customize the Change Request custom form, BMC Remedyforce provides out-of-the-box field sets with commonly used out-of-the-box fields. By default, some fields of these field sets are displayed on the Change Request custom form. You can add more fields to the custom form or remove existing fields from the custom form.

The Change Request object contains the following out-of-the-box field sets:

    • Main Panel: First Column
    • Main Panel: Second Column
    • Details Section
    • Additional Details

By default, the Last Queue field is not available on the Salesforce page layout and in any custom forms. You must customize the layout of the forms to add the Last Queue field. Also, you must enable read-only permission for the newly added field.

If you create custom fields for the Change Request object, these custom fields are added to the Available for the Field Set list of all field sets of the Change Request object.
