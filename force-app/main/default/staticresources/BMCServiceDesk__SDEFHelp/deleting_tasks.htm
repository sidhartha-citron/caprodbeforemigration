When a client reports an issue, you create an incident for the issue and a task for the steps that must be completed to resolve the incident. However, if the client calls back later saying that the issue has been resolved without any action from you, you can delete the incident and the task that you had created.

To delete a task

  1. Navigate to the task that you want to delete.

    2. For more information, see Navigating to a task record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, select the check box for the required task, and click Delete.
    • If you are using the Task Console tab, select the check box for the required task, and click Delete.
    • If you are using the Tasks tab (Salesforce page layout), click Delete.

      • To delete multiple records, select the check boxes for all the records that you want to delete.

    • If you are using the Visualforce page layout UI, click .
  3. In the confirmation dialog box, click Yes.

The task record is deleted, and the list of tasks is displayed.
