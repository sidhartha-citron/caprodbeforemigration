The Salesforce platform enables you to translate Self Service by using the custom labels.

Understanding UI labels mapping

UI labels in Self Service are mapped to custom labels. Appendix A, Custom labels for Self Service 2.0 provides the list of custom labels that must be translated to localize Self Service. For more information about localizing custom labels, refer to Localizing custom labels. However, as an exception, the UI labels on the following pages are mapped to standard fields:

    • My Profile
    • Register

By default, the standard fields are localized in the language that is assigned to the logged-on user.
