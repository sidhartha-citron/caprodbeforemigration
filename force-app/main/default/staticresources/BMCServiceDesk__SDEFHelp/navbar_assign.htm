The navigator appears in the left pane after you log on to BMC Remedyforce and allows you to access objects and their forms. BMC Remedyforce includes the following out-of-the-box navigators:

    • HELPDESK
    • SYSTEM ADMINISTRATOR
    • CHANGE MANAGEMENT

The system administrator can assign one of these navigators to a profile. Profiles assigned with the HELPDESK navigator can create and manage incidents, tasks, broadcasts, and configuration items; search for data; view out-of-the-box QuickViews and the Alignability Process Model for Incident Management, and create new QuickViews using the QuickViews wizard.

Profiles assigned with the CHANGE MANAGEMENT navigator can access all objects and forms available in the HELPDESK navigator. Additionally, these profiles can also create and manage change requests, change assessments, projected service outages, approvals, agreements, and service requests; and view the change schedule and the Alignability Process Model for Change Management.

Profiles assigned with the SYSTEM ADMINISTRATOR navigator can access all objects and forms available in the HELPDESK and CHANGE MANAGEMENT navigators. Additionally, these profiles can also configure the Application Settings and Global Search settings; set up the standard information in BMC Remedyforce; and create and manage folders and QuickViews in the Catalog console. Ensure that at least one profile has SYSTEM ADMINISTRATOR navigator assigned to it.

To assign a navigator

  1. In the BMC Remedyforce home page, click Configuration.
  2. From the list of Configuration options, expand Configuration.
  3. In the Configuration list, click Navigator Assignment.
  4. From the Navigator list, for each profile, select the required navigator.

    5. Note: Ensure that at least one profile has SYSTEM ADMINISTRATOR navigator assigned to it.

  5. Click Save.
