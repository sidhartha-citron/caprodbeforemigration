In the Categories tab, you can create new categories and filter the categories that are available to Self Service clients. Self Service clients can create incidents and service requests for the available categories only.

To define the Self Service categories

  1. In the BMC Remedyforce home page, click Configuration.
  2. From the list of Configuration options, expand Self Service.
  3. In the Self Service list, click Self Service Settings.

    4. Note: In the BMC Remedyforce tabbed view, Self Service Settings is available in the Remedyforce Administration tab.

  4. Click the Categories tab.

    5. The Categories list displays all categories that are available to Self Service clients.

  5. To create a new category, click New or to edit the category in the Category form, click an existing category.

    6. For more information, see Creating category records.

  6. Click Actions > Manage Categories.
  7. To filter the categories that are available to Self Service clients, in the Manage Categories form, perform the following actions:
    1. In the Available Categories list, select the categories that you want to make available to Self Service clients.
    2. To move the selected categories from the Available Categories list to the Selected Categories list, click the right arrow.
    3. (Optional) To move the selected categories from the Selected Categories list to the Available Categories list to prevent Self Service clients from using these categories, click the left arrow.

      4. Note: Double-clicking acts as a toggle to move categories back and forth between the Available Categories and Selected Categories lists.

    4. Define the display order in which the categories appear to Self Service clients by selecting the required category and clicking the up and down arrows to define the category's position in the display order.
    5. Click Save.
    6. In the confirmation dialog box, click OK.

      7. The categories that you have made available now appear in the Categories list.
