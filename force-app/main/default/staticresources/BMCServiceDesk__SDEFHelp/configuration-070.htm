You can create a new folder in the QuickViews folder to group the QuickViews that you use frequently or to quickly access related QuickViews. For example, you can create a folder called My Incidents and add a QuickView that displays the incidents that you must work on, or a QuickView of all your incidents sorted by priority, and so on.

To create a new folder

  1. In the BMC Remedyforce home page, click Catalog.
  2. From the list of Catalog options, expand QuickViews.
  3. Click Add Folder.
  4. In the Add Folder window, in the Folder Name field, type the name of the folder.
  5. Click OK.
