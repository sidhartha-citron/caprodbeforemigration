Linking of records of one object to the records of other objects in the Visualforce page layout UI is done in the Supporting Information section of a record. The options to link, unlink, add, or edit a record are provided in the menu in the respective tabs.

The following table provides you the list of objects that can be linked together and the options that are available in the menu for this purpose.

Object

Edit: To edit the selected incident.
 
 
Link To Incident: To link an existing incident.
 
 
Unlink Incident: To unlink the selected incident.

Unlink Problem: To unlink the selected problem.
