Lucy Diamond, a BMC Server Automation operator, and Bob Heart, a BMC Remedyforce administrator, want to create a change request in BMC Remedyforce for provisioning a new server.

Step 1 Bob creates an email listener in BMC Remedyforce and adds the ChangeRequestEmailListener out-of-the-box Apex class to the email service.

Step 2 In BMC Remedyforce, Bob creates a template named CreateBSAChange that contains Priority, Initiator ID, Change Category, Description, and Status fields for a change request form for creating a new change request from an email message. He follows steps 4 through 7 in Configuring BMC Server Automation to create change requests in BMC Remedyforce specifying the CreateBSAChange template in the cloned workflow.

Step 3 Lucy creates a job in the BMC Server Automation Console by performing the steps in Configuring jobs in BMC Server Automation Console.

Step 4 An email message is generated and sent to BMC Remedyforce and a change request record is created in BMC Remedyforce from the email message received.
