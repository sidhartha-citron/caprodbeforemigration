When you double-click a CI, you can view the related CIs. You can move from one CI to another to view relationships at all levels for all CIs in BMC Remedyforce.

Note: A red cross on a CI icon in the CI Explorer () signifies that the CI is impacted because it is linked to one or many Problems or Incidents. Incidents which are of service request type are not considered while showing the impacted CI.

To view configuration items and their relationships

  1. Click the Remedyforce CMDB tab.
  2. If you are using the Visualforce page layout user interface, perform the following actions:
    1. Go to BMC Remedyforce Home and click Workspaces.
    2. From the list of Workspaces, expand Configuration Management.
    3. In the Configuration Management list, click Configuration Items.
  3. In the left pane of the CMDB Manager, select the class whose CI you want to view in the CI Explorer.
  4. In the right pane of the CMDB Manager, select the CI that you want to view in the CI Explorer.

    5. To search for a CI, type initial characters of the name of the CI or the full name of the class name in the Search text box and click . For example, to search for Email Service CI, you can type Email or Email Service in the Search text box and click . You can also search using data in the Instance ID, Instance Name, Description, Class Name, Asset #, Token ID, and Serial # fields. If you select a class in the left pane, only the CIs of that class are searched. To search all classes, select the Base Element class in the left pane.

  5. Click Launch CI Explorer.

    6. The CI Explorer opens and displays the CI and its related CIs. Each CI represents a node in the CI Explorer. You can expand each node by double-clicking the CI to view its related CIs. By default, the CI Explorer displays the number of relationship levels that you have configured for the CI Explorer. For more information, see Configuring the Configuration Items Explorer.

  6. Double-click the CI whose relationships you want to view.

    7. The CI Explorer reappears and displays the CI and its related CIs. You can double-click the related CI to view the CIs that are related to it.

    Important: When you open the CI Explorer for the first time, you must download and install Microsoft Silverlight from the link provided in the CI Explorer.
