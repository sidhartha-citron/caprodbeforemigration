Using the Tab key to auto populate values in forms
Provided to you out-of-the-box.
Pressing the Tab key after entering first few characters of the required value in the lookup or list fields does not populate the complete value. However, when you click Save, the matching value is populated in the field. For example, if you type Net in the Category field and press the Tab key, the value in the Category field is Net. However, when you click Save, the value Network populates in the Category field. Similarly, field values of a template and related field values are populated after you select the template and click Save.
