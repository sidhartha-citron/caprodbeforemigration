By default, if you attach a document to an incident record in the Salesforce page layout user interface (UI) or new self service UI or incident console, the attachment is not visible when you access the incident record in the Visual force page layout UI.

To enable users to view attachments attached from the Salesforce page layout in the Visualforce page layout UI

  1. Navigate to Setup > Develop > Custom Settings.
  2. In the Action column beside Remedyforce Settings, click Manage.
  3. In the Action column beside EnableMixedModeAttachments, click Edit.
  4. In the value field, enter TRUE.
  5. Click Save.

The documents attached before configuring these settings are not be visible in Visualforce page layouts; however, the documents attached after configuring these settings are visible in both Salesforce and Visualforce page layouts.
