If you forgot to select an account while creating a staff record, perform the tasks mentioned in this section to assign an account to a staff member or system administrator user.

To assign an account to a staff member or system administrator

  1. Navigate to Setup > Create > Tabs.
  2. Click New.
  3. From the Objects list, select User Account Link.
  4. From the Tab Style list, select an appropriate style for the tab, and click Next.
  5. Select the appropriate option on the Step 2. Add to Profiles page, and click Next.
  6. On the Step 3. Add to Custom Apps page, select the appropriate options, and click Save.
  7. Click the User Accounts Link tab.
  8. Click New.
  9. From the Force.com Quick Access menu, select Edit Layout.
  10. Add the User and Account fields to the layout.
  11. From the User field, select the required user.
  12. From the Account field, select the required account, and click Save.

    13. You cannot link an account to a user whose license is Customer Portal Manager Custom, Customer Portal Manager, or Customer Portal Manager Standard. Also, you cannot link more than one account to a user.
