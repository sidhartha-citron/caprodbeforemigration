Revoke access to the BMC MyIT application
BMC Remedyforce
Clients can authorize themselves for the BMC Remedyforce MyIT OnDemand App. System administrator can revoke authorization (OAuth token) for individual clients or block the use of BMC MyIT for the clients.
To revoke authorization for a client:
  1. Navigate to Setup > Manage Apps > Connected Apps OAuth Usage.
  2. In the User Count column for BMC Remedyforce MyIT OnDemand App, click the user count.
  3. On the Connected App User’s Usage page, in the Action column for the client, click Revoke.
To block all clients from using BMC MyIT:
  1. Navigate to Setup > Manage Apps > Connected Apps OAuth Usage.
  2. On the Connected Apps OAuth Usage page, in the Action column, click Block for the BMC Remedyforce MyIT OnDemand application.
