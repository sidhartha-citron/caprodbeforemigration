If you have created QuickViews that are no longer being used and all these QuickViews reside in one folder, you can delete all the QuickViews by deleting the folder. For example, if you have a My Broadcasts folder that contains all QuickViews for broadcasts assigned to you and you no longer work on broadcasts, you can delete the My Broadcasts folder, and all its QuickViews are deleted.

To delete a folder

  1. In the BMC Remedyforce home page, click Catalog.
  2. From the list of Catalog options, expand QuickViews.
  3. Select the folder you want to delete.
  4. Click Delete Folder.
  5. In the confirmation dialog box, click Yes.

    6. Note: When you delete a folder, all QuickViews in the folder are deleted.
