If an approval process has been set up so that an incident must be submitted for an approval, you must submit the incident for approval.

To submit an incident for approval

  1. Navigate to the incident record that you want to submit for approval.

    2. For more information, see Navigating to an incident or a service request record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click Record Details and in the Approval History section, click Submit for Approval.
    • If you are using the Incident Console tab, click Record Details and in the Approval History section, click Submit for Approval.
    • If you are using the Incidents tab (Salesforce page layout), in the Approval History section, click Submit for Approval.
    • If you are using the Visualforce page layout UI, click the Approvals tab in the Supporting Information section, and select Submit For Approval from the menu.
  3. In the confirmation dialog box, click OK.

    4. The incident is submitted for approval and current approval status is displayed in the Approval History section (in the Status column in the Visualforce page layout UI). To recall the incident from approval, click Recall Approval Request.
