The Views menu allows you to filter the records that are displayed in the incidents list view. You can see records that are assigned to the logged-on user, open records, closed records, active records, inactive records, only service request records, and only incident records. You can also define the refresh rate of the incidents list view and display the duration when the incidents list view was last refreshed.

Views menu commands
Description
Assigned to myself
Displays all records that are assigned to the logged-on user
Open Items
Displays all records in the Open state
Closed Items
Displays all records in the Close state
Active Items
Displays all active records
Inactive Items
Displays all inactive records
Service Requests
Displays only service request records
Incident
Displays only incident records
Refresh every minutes
Type the refresh rate of the list view in this field.
The default refresh rate is 10 minutes and the minimum refresh rate is one minute. You can modify this value depending on your requirements. A faster refresh rate impacts your actions because the list view would refresh when you are performing an action. BMC recommends that you use a refresh rate that is higher than the minimum refresh rate.
The new refresh rate is applied when you click the Apply command and this value is stored in the browser's cookies. When you log on, the list view refreshes at the new refresh rate. If a staff member deletes the browser's cookies, the refresh rate of the list view is reset to 10 minutes. To disable the automatic refresh of the list view, enter zero as the refresh rate.

Note:  The Last Refreshed <Time> label displays the earlier time when the List view had refreshed.
