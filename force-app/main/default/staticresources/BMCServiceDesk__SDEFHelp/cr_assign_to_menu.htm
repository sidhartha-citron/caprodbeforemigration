By default, a change request is assigned to the user who creates it. After saving the change request, you can change the assignment to another user or queue.

Changing assignment of a change request owner on the Remedyforce Console tab

In the Assignment Details section, based on the setting configured by the system administrator, either the Owner field or Queue and Staff fields are displayed.

To change assignment of a change request on the Remedyforce Console tab

  1. Navigate to the record that you want to assign to another user or queue.

    2. For more information, see Navigating to a change request record.

    You can also select a record in the list view, and click Change Owner.

  2. If the Owner field is displayed in the Assignment Details section, perform the following actions:
    1. Click the Owner lookup .

      2. If staff members exist in your organization, by default, all the staff members are displayed in the Suggested Owners window. If a staff member does not exist, queues are displayed.

    2. To assign the record to a staff member, click the required staff member record.
    3. To assign the record to a queue, select the Queue option and click the required queue name.
    4. To assign the record to a staff member who belongs to a queue, first click the row of the required queue and from the list of staff members belonging to the queue, select the required staff member.
  3. If the Staff and Queue fields are displayed in the Assignment Details section, perform the following actions:
    1. To assign the record to a staff member, select the required staff from the Staff field.
    2. To assign the record to a queue, select the required queue from the Queue field.
    3. To assign the record to a staff member who belongs to a queue, first click the row of the required queue and from the list of staff members belonging to the queue, select the required staff member.

      4. When you select a queue in the Queue field, the staff members belonging to the queue are displayed in the Staff field lookup.

Changing assignment of a change request owner on the Change Requests tab

In the Staff Assignment Details section, based on the setting configured by the system administrator, either the Owner field or Queue and Staff fields are displayed.

To change assignment of a change request

  1. Navigate to the change request that you want to assign to another user or queue.

    2. For more information, see Navigating to a change request record.

  2. In the Staff Assignment Details section, click Change beside the user name to whom the change request is assigned.
  3. From the Owner list, select appropriate option.
    • To assign the change request to a user, select User.
    • To assign the change request to a queue, select Queue.

      • If the staff member is not available or out of office, the status can be displayed in the Out of Office column in the Search for a user window. Contact your system administrator to configure the Out of Office column in the Search for a user window.

      To set your out of office status, navigate to Setup > My Personal Information > Personal Information. In the BMC Remedyforce section, click Edit and select the Out of Office check box.

      Note: If you have enabled the improved Setup user interface in your Salesforce organization, navigate to My Settings > Personal > Personal Information. For more information about the improved Setup user interface, see http://help.salesforce.com/apex/HTViewHelpDoc?id=admin_setup_improved.htm&language=en_US.

  4. To select the user or queue, click and select the required option.
  5. Click Save.

    6. Note: To use the staff and queue assignment feature, contact your system administrator to edit the change request layout and display the StaffAssignmentPage Visualforce page.

    The Change Request form Assign To menu assigns open change requests to yourself, other staff members, and staff members of your profile.

    When you use the Assign To menu to assign the current record to staff members, some staff members might not be displayed in the Select from Staff window that appears.

Changing assignment of a change request owner from the Visualforce page layout UI

If you are using the Visualforce page layout UI, perform the steps mentioned in this section.

To change incident owner in the Visualforce page layout UI

  1. Navigate to the record that you want to assign to another user or queue.

    2. For more information, see Navigating to a change request record.

  2. Click the Assigned To menu.

    3. The following table describes the commands of the Assign To menu.

    Menu Commands
    Description
    Staff
    Opens the Select from Staff window that allows you to select a staff member that you can assign to a change request.     If the staff member is not available or out of office, the status is displayed in the Out of Office column in the Select From Staff window.
    To set your out of office status, navigate to Setup > My Personal Information > Personal Information. In the BMC Remedyforce section, click Edit and select the Out of Office check box.
