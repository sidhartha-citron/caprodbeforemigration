To enable users to follow broadcasts, you must ensure that feed tracking for the Broadcast object is enabled.

To enable users to follow broadcasts

  1. Navigate to Setup > Customize > Chatter > Feed Tracking.
  2. From the list of objects, select Broadcast.
  3. Select the Enable Feed Tracking check box.
  4. Click Save.

    5. Also, ensure that Chatter is enabled for your Salesforce organization and Remedyforce. For more information, see Configuring Chatter settings and Enabling or disabling Chatter.
