When you create a CI, you can use custom attributes to record information about the CI that is not available in the out-of-the-box attributes.

You can enter the custom attribute values for the CI if the required custom fields for the custom object are created and available in the Instance Editor form. For more information, contact your system administrator.

Note: If you do not have the Create, Edit, and Delete permissions on the class of a CI, you cannot create, edit, or delete the CI. The New and Save toolbar buttons are disabled if you do not have these permissions.

To enter custom attribute values for the configuration item

  1. Click the Remedyforce CMDB tab.
  2. If you are using the Visualforce page layout user interface, perform the following actions:
    1. Go to BMC Remedyforce Home and click Workspaces.
    2. From the list of Workspaces, expand Configuration Management.
    3. In the Configuration Management list, click Configuration Items.
  3. Create a CI of the custom object.
  4. In the custom fields that you have created for the custom object, enter the custom attribute values for the CI.

    5. For more information, see Creating a configuration item.
