David is a system administrator in a company. He is using BMC ProactiveNet Performance Management to monitor CPU usage of virtual machines (VMs). If the CPU usage exceeds 85%, an event is generated. David has integrated BMC ProactiveNet Performance Management with BMC Remedyforce to create an incident when the event is generated. The event is received as an email message by BMC Remedyforce. An incident is created by using the email message. This incident is assigned to a staff user and necessary steps are taken.

To create an incident in this scenario, David needs to perform the following tasks:

    • Create an email service and add the GenericEmailListener Apex class to the email service in BMC Remedyforce. This task is performed in Salesforce settings (Remedyforce Administration > Configure Email > Email Services). If you are using the Visualforce page layout user interface (UI), navigate to Setup > Develop > Email Services.
    • Create an event in BMC ProactiveNet Performance Management that generates an event when the CPU usage of VMs exceeds 85%.
    • While creating the event, David specifies the email address configured for BMC Remedyforce in the event. The alerts that are generated for the event in BMC ProactiveNet Performance Management are received at this email address and are used to create incidents.
    • Map the data in the alert to the fields in the Incident object. This task is performed on the Remedyforce Administration > Configure Email > Incident - Email Settings page. If you are using the Visualforce page layout UI, navigate to Configuration > Application Settings > Incident - Email Settings.

      • In this tab, you specify that an incident is created when BMC Remedyforce receives an email message with a specified subject. You can specify the subject in the Subject fields of event rules in BMC ProactiveNet Performance Management. Next, you map the email message content to the Incident fields. For example, you can choose to save the body of the email message as the Description of the incident.
