If a similar release is required, you can clone a release, modify the details, and reuse the release record.

To clone a release

  1. Navigate to the record that you want to clone.

    2. For more information, see Navigating to a release record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Actions menu, select Clone.
    • If you are using the Releases tab (Salesforce page layout), click Clone.

      • The New Release form is displayed, and the values of the fields in the release that you have cloned are displayed.

  3. Revise the values as necessary for the new record.

    4. For field descriptions, see Release fields.

  4. Click Save.
