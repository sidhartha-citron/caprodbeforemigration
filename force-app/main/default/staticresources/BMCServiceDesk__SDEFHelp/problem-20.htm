You can create knowledge articles from a problem. The knowledge articles that you create from a problem are displayed in the Linked Knowledge Articles section.

To create a knowledge article from a problem

  1. Navigate to the required problem from which you want to create a knowledge article.

    2. For more information, see Navigating to a problem record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Agent Tools menu, select Create Knowledge Article.
    • If you are using the Problems tab (Salesforce page layout), click Create Knowledge Article.
  3. In the Title field, enter a title for the knowledge article.

    4. The first 30 characters of the Description field of the problem appear in the Title field. You can choose to enter a different title. Maximum of 255 characters are allowed in this field.

  4. From the Type field, select the required type of the knowledge article, and click Create.

    5. The article is saved in the Draft status. If you select the Known Error check box in the problem form, the Known Error article type is selected in the Type field.

    To enable the article to be searched for and displayed in Self Service, edit the article and select the Display in Self Service check box. For more information, see Creating a knowledge article.

    Based on the type of knowledge article that you are creating, the data of some of the fields of the problem is saved in the appropriate fields of the knowledge article.   5. In the Knowledge Article window, click Edit.
  6. Enter the required details for the knowledge article.

    7. For more information, see Creating a knowledge article.

  7. Click Save.

    8. To view the Linked Problems related list, contact your system administrator to edit the layout of the knowledge article form.
