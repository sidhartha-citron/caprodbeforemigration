You can add clients to an account after creating the account.

To add clients to an account

  1. In the BMC Remedyforce home page, click Configuration.
  2. From the list of Configuration options, expand Configuration.
  3. In the Configuration list, click Accounts.
  4. Double-click the account to which you want to add clients.
  5. In the Supporting Information section, click the Clients tab.
  6. From the Actions menu, select Add.
  7. On the Client form, enter the required details.

    8. For more information, see Creating clients.

  8. (Optional) To edit client details of a client linked to an account, select the client record, and from the Actions menu, select Edit.
