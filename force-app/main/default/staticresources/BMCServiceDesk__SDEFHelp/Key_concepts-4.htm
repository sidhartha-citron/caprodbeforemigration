BMC Remedyforce is a 100% cloud based application that resides on a webserver that allows for high availability of your Information Technology Service Management environment. The application may be accessed from anywhere within an organization via the Internet using a standard web browser. BMC Remedyforce is hosted on the Force.com platform.

Data is hosted in the cloud as well. For this reason, there are limits set to how much data is allowed to be stored within the cloud. You must check your subscription for the limits.

Customizations to the interface are performed through easy-to-use web administration tools with no additional programming. Users can be authenticated through the Force.com platform to the domain, if using LDAP or directly to Salesforce.com instance. The user is then authenticated to BMC Remedyforce with the proper Role, automatically assigning the user correct preferences and permissions within the system.
