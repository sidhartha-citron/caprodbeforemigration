On the BMC Communities BMC Remedyforce page, a sample batch file is provided which you can use as a base to create a batch file for scheduling data import. For more information, see http://wiki.Pentaho.com/display/EAI/Kitchen+User+Documentation.

Each data source folder on the BMC Communities BMC Remedyforce page contains a Scheduling Files folder. This folder contains sample batch files that you can use as a base to create your batch files to schedule a job.

To schedule importing of data

  1. From the \pdi-ce-4.x.0-stable\data-integration or \pdi-ce-5.0.1.A-stable\data-integration folder, make a backup copy of the Kitchen.bat file.
  2. From the BMC Communities BMC Remedyforce page, navigate to the Scheduling Files folder available in the appropriate data source folder, such as LDAP, ADDM, SCCM, or Dell KACE.
  3. Copy the Kitchen.bat file and paste it in the \pdi-ce-4.x.0-stable\data-integration or \pdi-ce-5.0.1.A-stable\data-integration folder.
  4. To schedule a job, create a batch file by using the appropriate .bat sample file, such as SchedulingComputerSystem.bat, available in the Scheduling Files folder.
  5. In your batch file, perform the following steps:
    1. Enter the location of the Kitchen.bat file in KETTLE_HOME="<path>\pdi-ce-4.x.0-stable\data-integration" or KETTLE_HOME="<path>\pdi-ce-5.0.1.A-stable\data-integration".
    2. Enter the location of the Kitchen.bat file and the KJB file in <path>\pdi-ce-4.x.0-stable\data-integration\kitchen.bat /file:"<JobFile>\<Job File Name>.kjb"/level:Detailed or <path>\pdi-ce-5.0.1.A-stable\data-integration\kitchen.bat /file:"<JobFile>\<JobFileName>.kjb"/level:Detailed.

      3. In this case, <path> is the location where you have extracted the pdi-ce-4.x.0-stable.zip or the pdi-ce-5.0.1.A-stable.zip file, and <JobFile> refers to the location where you have extracted the job files. The level option specifies the level of logging for the job. Its values can be Minimal, Basic, Detailed, Debugging, or Row Level.

      The log files are saved in the same folder where KJB files are saved.

  6. Go to Start > Settings > Control Panel > Scheduled Tasks > Add Scheduled Task.
  7. In the scheduled tasks wizard, click Next and then click Browse to locate the batch file that you have created to schedule the job.
  8. Type the name of the task, select the frequency at which you want to execute the batch file, and click Next.
  9. Select start time, frequency, and start date and click Next.
  10. Enter your Windows credentials and click Next.
  11. Click the Open advanced properties for this task when I click Finish check box, and click Finish.

    12. The <name of the task> window is displayed.

  12. (Optional) In the <name of the task> window, click the Schedule tab.
  13. In the Schedule tab, click Advanced to set the advanced scheduling options for the job.
  14. Click OK.
