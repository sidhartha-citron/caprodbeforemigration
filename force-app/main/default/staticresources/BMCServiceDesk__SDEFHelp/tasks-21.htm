When you select a service, BMC Remedyforce applies all SLAs that match the service. You can view the service target transactions that occur when an SLA is applied to a task by clicking Service Targets.

To view the service targets related to a task

  1. Navigate to the task for which you want to view service targets.

    2. For more information, see Navigating to a task record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Agent Tools menu, select Service Targets.

      • Alternatively, click .

    • If you are using the Task Console tab, from the Agent Tools menu, select Service Targets.

      • Alternatively, click .

    • If you are using the Tasks tab (Salesforce page layout), click Service Targets.
