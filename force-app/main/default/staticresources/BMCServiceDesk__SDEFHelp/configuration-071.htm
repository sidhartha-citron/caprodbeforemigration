You can create subfolders for an existing folder to organize your QuickViews at a detailed level. For example, you can create a My Work folder. In this folder, you can create these subfolders: My Incidents, My Tasks, and My Broadcasts. In these subfolders, you can add the QuickViews that display the incidents, tasks, broadcasts, change requests, and problems assigned to you. In the My Incidents subfolders, you can add QuickViews that display all incidents assigned to you, all high priority incidents, and all incidents whose due date is in the coming week. Similarly, you can add the required task and broadcast QuickViews in the My Tasks and My Broadcasts subfolders.

To create a subfolder

  1. In the BMC Remedyforce home page, click Catalog.
  2. From the list of Catalog options, expand QuickViews.
  3. Select the folder where you want to create a subfolder.
  4. Click Add Folder.
  5. In the Add Folder window, in the Folder Name field, type the name of the subfolder.
  6. Click OK.
