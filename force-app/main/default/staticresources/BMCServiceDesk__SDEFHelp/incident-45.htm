You can link and unlink multiple incident and service request records to a broadcast. To select multiple records, press the Ctrl key and click the required records. You can select multiple consecutive records by pressing the Shift key and clicking the first and last records of the consecutive records. Clicking a selected record clears it.

To link broadcasts to an incident

  1. Navigate to Workspaces > Incidents Management > Incidents.
  2. In the incident list view, select the incident that you want to link or unlink to a broadcast.
  3. To link to a broadcast, from the menu, select Link to Broadcast.
  4. From the Select From Broadcasts window, select the required broadcast.
  5. To unlink broadcast, from the menu, select Unlink Broadcast.
