Page layouts version 3.0 are provided for the following objects:

    • Broadcast
    • Change Request
    • Incident
    • Problem
    • Release
    • Task

The version 3.0 page layouts contain all the features available for an object. If your users want to use all new features, you can assign the version 3.0 page layouts to these objects.

To assign version 3.0 page layouts to objects

  1. Navigate to Setup > Create > Objects.
  2. Click Broadcast.
  3. In the Page Layouts section, click Page layout Assignment.
  4. Click Edit Assignment.
  5. Select the required profiles.
  6. From the Page Layout To Use list, select Remedyforce Broadcast Version 3.0.
  7. Click Save.
  8. Repeat step 2 to step 7 for the following objects:
    • Change Request
    • Incident
    • Problem
    • Release
    • Task

      • Note: Ensure that you select the Version 3.0 page layout for these objects.
