Before importing data from BMC BladeLogic Client Automation into BMC Remedyforce, you must meet the following requirements:

    • Install JRE. For more information, see Installing JRE.
    • Download the Pentaho Data Integration tool. For more information, see Downloading the Pentaho Data Integration tool.
    • Register at the BMC Communities website. For more information, see Registering at the BMC Communities website.
    • Configure Windows authentication to enable communication between the Pentaho Data Integration tool and BMC BladeLogic Client Automation. For more information, see Connecting to Windows authenticated MS SQL server.
