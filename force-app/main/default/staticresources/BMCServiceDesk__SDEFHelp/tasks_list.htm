The following table provides description about the options available in the list view in the Visualforce page layout UI:

Option
Description
New
To create a new task
Copy
To copy the selected task
Delete
To delete the selected task
Views > Assigned to myself
To view the tasks assigned to the logged on user
Views > Open Items
To view tasks that are in Open state
Views > Closed Items
To view the tasks that are in Closed state
Views > Active Items
To view all active tasks
Views > Inactive Items
To view all inactive tasks
Refresh every __ minutes
To set the time in which the list view must be refreshed
Group by
To show tasks with same value in a field, such as Category
Search
To search for a task, enter keywords and click .
