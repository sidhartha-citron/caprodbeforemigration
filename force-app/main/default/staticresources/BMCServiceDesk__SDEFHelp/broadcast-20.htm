You can print a broadcast record on a printer or as a PDF output when required. If you want to add a specific field to the PDF output, contact your system administrator.

To print a broadcast record

  1. Navigate to the broadcast record that you want to print.

    2. For more information, see Navigating to a broadcast record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Actions menu, select Print.

      • To print a broadcast in PDF format, from the Actions menu, select Print to PDF.

      Note: On the Remedyforce Console tab, if you print a broadcast record, the Display Broadcast section is not printed.

    • If you are using the Broadcasts tab (Salesforce page layout), on the right corner of the page, click Printable View, and on the top-right corner of the page, click Print This Page.
