The Remedyforce Console tab enables users to manage all records (incidents, service requests, tasks, problems, change requests, releases, and broadcasts) from a single console. Each record opens in a different tab within the Remedyforce Console tab. Users can easily navigate between records without opening multiple windows. For more information about the benefits of using the Remedyforce Console tab, see Overview of the Remedyforce Console tab.

Note: For new installations of BMC Remedyforce, the Remedyforce Console tab is enabled out of the box. In this case, you do not have to perform the steps that are described in Enabling the Remedyforce Console tab.

However, if you are upgrading from a earlier version of BMC Remedyforce to version 20.14.01 or later, you must manually set up and enable the Remedyforce Console tab. For more information, see Enabling the Remedyforce Console tab.
