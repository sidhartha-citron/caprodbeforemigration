You might need to reopen a problem instead of creating a new one if the issue reported in the original problem still exists. Another reason for reopening the problem might be that the reported resolution did not resolve the problem, and further work is needed.

To reopen a problem

  1. Navigate to the problem that you want to reopen.

    2. For more information, see Navigating to a problem record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Actions menu, select Reopen.
    • If you are using the Problems tab (Salesforce page layout), click Reopen.
    • If you are using the Visualforce page layout UI, from the menu, select Reopen Problem.
