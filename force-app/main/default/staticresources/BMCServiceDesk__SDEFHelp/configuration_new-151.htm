You can configure the fields that are shown in the list views of an object on the Remedyforce Console tab.

Note: The changes that you configure for the Incident and Task objects are also applied on the Incident Console and Task Console tabs. These tabs are available only if you enabled them (by selecting the Enable and Use the Console check box) before upgrading to BMC Remedyforce 20.14.01 or later.

To configure the fields shown in the list view on the Remedyforce Console tab

  1. Navigate to Setup > Create > Objects.
  2. In the Label column, click the required object.
  3. In the Field Sets section, in the Action column for the Column Headers field set, click Edit.
  4. Select and drop the required fields from the palette in the In the Field Set list.
  5. Click Save.

Configuring the fields shown in the Impact Analysis window

The fields that are shown in the Impact Analysis window lists are configured in the Column Headers field set of the Base Element object. You can add or remove fields from the field set to show or hide fields in these lists.

The Impact Analysis window is shown when a user selects Impact Analysis from the Agent Tools menu in the Change Requests module.

Guidelines for using images in list views

Apply the following guidelines to use images in list views on the Remedyforce Console, Incident Console, and Task Console tabs:

    • 16 x 16 images are the best size.
    • Load images from secured (HTTPS sources) and trusted sources.
