You can delete custom themes that are no longer required. Before you delete a custom theme, make sure that the theme is not assigned to any account.

To delete a custom theme

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Configure Self Service tile, and from the menu, select Branding.
  3. From the User Interface Theme list, click Edit next to the custom theme that you want to delete.
  4. To delete the custom theme, click .

    5. Note: If you delete a custom theme that was configured as the default theme, the BMC Theme is reset as the default theme.
