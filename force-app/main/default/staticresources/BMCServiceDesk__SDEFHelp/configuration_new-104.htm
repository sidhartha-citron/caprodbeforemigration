If you are using Internet Explorer to access the BMC Remedyforce Salesforce page layout, you must configure Internet Explorer to address the following two actions:

    • Pop-up windows
    • Updated records

Pop-up windows

You need to configure settings in Internet Explorer to handle pop-ups correctly in BMC Remedyforce.

To configure pop-up windows

  1. In your Internet Explorer browser window, in the Tools menu, click Internet Options.
  2. In the Tabs section, click Settings.
  3. In the When a pop-up is encountered section in the Tabbed Browsing Settings window, select Let Internet Explorer decide how pop-ups should open.
  4. In the Open links from other programs in section, select The current tab or window.
  5. Click OK.

Updated records

After you perform an action in a pop-up window that updates the record, the updated record is displayed in a new tab. You must configure Internet Explorer to display the updated records in the same tab. For example, a record is refreshed after you link a change request to an incident. To see the updated incident record with the linked change request in the same tab, you must configure your Internet Explorer browser.

To configure Internet Explorer to see updated records in the same tab

  1. In your Internet Explorer browser window, in the Tools menu, click Internet Options.
  2. Click the Security tab.
  3. In the Security level for this zone section, click Custom Level.
  4. From the Miscellaneous list of options, select Enable for the Navigate windows and frames across different domains option.
  5. Click OK.
