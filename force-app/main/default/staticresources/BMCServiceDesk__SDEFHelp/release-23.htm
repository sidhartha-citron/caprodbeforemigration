When you have implemented the change requests linked to a release and all the linked tasks are also complete, you can close the release.

Before closing a record, ensure that all the linked tasks are closed. If your system administrator has selected the Releases check box in the Closing Records with Linked Open Tasks section on the Remedyforce Administration > Application Settings page, you cannot close a release with linked open tasks.

If you have set status and order numbers for the asks and change requests of a release in the release manifest, all the tasks and change requests must be closed before you can close the release record.

To close a release

  1. Navigate to the record that you want to close.

    2. For more information, see Navigating to a release record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Actions menu, select Close.

      • If you want to close linked change requests and tasks select appropriate check box.

    • If you are using the Releases tab (Salesforce page layout), click Close.

      • Note: This procedure describes the out-of-the-box fields only. If your system administrator has added or removed fields, complete all the fields on the form.

  3. From the Closed By list, select the user who is closing the release.

    4. By default, this field displays the user ID of the currently logged-on user. If you are closing the release on behalf of another user, select the user who has worked on the release.

  4. From the Status list, select a close status.

    5. By default, this field is filled with the default close status that is configured by the system administrator. If required, you can use this field to select a different close status from the Status list. All statuses with the state of Close are displayed in the Status list.

  5. From the Closure Category list, select the category that suggests the reason for closing the record.
  6. If the original category was incorrect or no longer applies, assign a new category to the release.
  7. In the Close Comments field, enter comments as needed.
  8. To close all change requests associated with this release, select the Close Change Requests check box.
  9. To close all tasks associated with this release, select the Close Tasks check box.
  10. Click Save.
