You can configure the type and color of connecting lines and the number of levels you want to expand the relationships in the CI Explorer.

Note: You can save any changes you make in the settings of the CI Explorer only if you belong to the System Administrator profile. For more information about profiles, see https://na11.salesforce.com/help/doc/user_ed.jsp?section=help&target=admin_userprofiles.htm&loc=help&hash=topic-title.

To configure the CI Explorer

  1. Click the Remedyforce CMDB tab.
  2. If you are using the Visualforce page layout user interface, perform the following actions:
    1. Go to BMC Remedyforce Home and click Workspaces.
    2. From the list of Workspaces, expand Configuration Management.
    3. In the Configuration Management list, click Configuration Items.
  3. In the left pane of the CMDB Manager, select the class whose CI you want to view in the CI Explorer.
  4. In the right pane of the CMDB Manager, select the CI that you want to view in the CI Explorer.

    5. To search for a CI, type initial characters of the name of the CI or the full name of the class name in the Search text box and click . For example, to search for Email Service CI, you can type Email or Email Service in the Search text box and click . You can also search using data in the Instance ID, Instance Name, Description, Class Name, Asset #, Token ID, and Serial # fields. If you select a class in the left pane, only the CIs of that class are searched. To search all classes, select the Base Element class in the left pane.

  5. To open the CI Explorer and display the CI and its related CIs, click Launch CI Explorer.
  6. Click Settings.
  7. In the Line Settings tab of the CI Explorer Settings window, perform any of the following actions:
    • From the Expand CI Relationships to levels list, select the number of levels you want to expand the relationships between CIs.
    • In the Relationship section, click the type of relationship that you want to configure.

      • For the selected relationship type, configure the following properties:

      • From the Line Direction list, select the direction of the relationship line.
      • From the Line Style list, select the style of the relationship line.
      • From the Line Thickness list, select the thickness of the relationship line.
      • From the Line Color palette, select the color of the relationship line.
  8. In the Field Settings tab of the CI Explorer Settings window, perform any of the following actions:
    1. From the Display Fields for list, select one of the following CI Explorer components:
      • ToolTip—The selected fields and their values appear as a ToolTip when you move your mouse over a node in the CI Explorer. You can select a maximum of five fields for the ToolTip.
      • Label—The selected fields display their values in the label of the node in the CI Explorer. You can select a maximum of two fields for the label of the node.
      • Information Pane—When you select a CI, its linked open incidents, tasks, change requests, or problems appear in the Information Pane. For each open record, its Class Name and Instance ID is displayed by default. The selected fields and their values appear at the top of the Information Pane in the CI Explorer. You can select a maximum of 10 fields for the Information Pane. All the field values that are displayed in the Configuration Item section are hyperlinked to the Instance Editor form of the Configuration Item.
    2. In the Available Fields list, select the fields you want to display.
    3. To move the selected fields from the Available Fields list to the Selected Fields list, click the right arrow.
    4. (Optional) To move the selected fields from the Selected Fields list to the Available Fields list, click the left arrow.
    5. Click Save.
  9. Click Close.
