All the changes on service targets are applied to new incidents, service requests, tasks, problems, or change requests only. If you modify a service target, no changes are made to the targets of the incidents, service requests, tasks, problems, or change requests for which the service target transactions are created and the clock has started. Therefore, BMC recommends that you do not modify a service target.

If you want to create a service target that is similar to a service target that you have already created, you can copy the service target and modify the required details. Also, if you do not need a service target in an agreement, you can delete it from the agreement.

To edit, copy, or delete a service target

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Manage Workflows and Other Processes tile, and from the menu select Agreements.

    3. Alternatively, you can also navigate to Remedyforce Workspaces > Service Level Management > Agreements.

  3. If you are using the Visualforce page layout user interface, perform the following actions:
    1. In the BMC Remedyforce home page, click Workspaces.
    2. From the list of Workspaces, expand Service Level Management.
    3. In the Service Level Management list, click Agreements.
  4. In the Service Level Management list view, use one of the following options to select the agreement in which you want to edit a service target:
    1. From the Group By list, select a field, navigate to the required group, and select the required agreement.
    2. Type a search string in the Search field, click Search, and select the required agreement from the search results.
    3. Scroll through the list of agreements, and select the required agreement.

      4. The agreement is opened in a new tab.

  5. Click the Service Target tab.

    6. A list of existing service targets is displayed.

  6. Perform one of the following actions:

    7. Goal
    Step
    Edit the selected service target
    Click the service target that you want to edit and click .

    Note:  If you edit a service target, you cannot change the object for which the service target is created. For example, if you edited a service target that was created for Incidents, you cannot change the object to Change Request in the Applied To field.

    The service target is opened in a new window.
    Create a service target that is similar to the selected service target
    Click the service target that you want to copy and click .

  7. Click Save.
