To meet the branding requirements of your company or the accounts that you support, you can use custom themes. You can create a custom theme only by copying an existing theme. However, the number of custom themes that you can create for Self Service users is unlimited.

To create a custom theme

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Configure Self Service tile, and from the menu select Branding.
  3. From the User Interface Theme list, select Create New Theme.
  4. In the new window that opens, perform the following actions:
    1. From the Copy From Theme list, select the existing theme based on which you want to create the custom theme.
    2. In Enter the Theme Name, specify a unique name for the custom theme.
    3. To create the custom theme, click Save Theme.



  5. In the new browser tab that opens, define the colors for the various elements in the Self Service UI. The default colors of the UI elements are the same as the colors in the existing theme from which you copied the custom theme.

    6. Note: The default Hex color code of the Bottom Gradient Color (Active Foreground Tab and Inactive Background Tabs) and Top Gradient Color (Tiles) UI elements is #FFFFFF (White) and cannot be changed.

    1. From the User Interface Elements list, select the element whose color you want to change.
    2. In the Element Color area, click the text box and select the appropriate color from the picklist.

      3. If you know the Hex color code for the selected UI element, you can also enter that code in the text box.

    3. Click Apply.

      4. You can view your changes in the Preview section. The first tile in the Preview section always displays the color that is applied to the Tile On Mouse Over UI element.

    4. To save your changes to the colors in the custom theme, click and then OK.

      5. Tip: When creating a custom theme, you can use the legend to understand the location of the different UI elements in Self Service. To view the location of the selected UI element in Self Service, move your mouse on the icon next to User Interface Elements. The numbers next to the element in the User Interface Elements list correspond to the numbers displayed in the legend.

      For information about changing the default images that are assigned to the custom theme, see Configuring images for a theme.

      For information about assigning the custom theme to one or more accounts, see Assigning a theme to an account.

      For information about setting the custom theme as the default theme, see Configuring the default theme for Self Service users.
