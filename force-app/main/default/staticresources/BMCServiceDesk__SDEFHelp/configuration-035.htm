You can filter the records that are displayed in a List view by using the Views menu, the Group By list, or searching for the specific record. The Views menu displays only records that are assigned to the logged-on user, open records, closed records, active records, and inactive records. The Group By list groups records based on the field selected in this list. If you know a keyword of the required record, you type the keyword and execute a search to display the required record only. In the List view, when you open the last record and click Next Record, the first record in the List view is displayed. If you open the first record and click Previous Record, the last record in the List view is displayed.

Note: Double-clicking the column header acts as a toggle to sort the column in ascending or descending order.

To filter records in a List view

  1. In the List view, perform the following actions to filter records by using the Views menu:
    1. Click the Views menu.
    2. Select the required commands from the Views menu. The following table describes the commands of the Views menu.

      3. Menu Commands
      Description
      Assigned to myself
      Displays all records that are assigned to the logged-on user
      Open Items
      Displays all records in the Open state
      Closed Items
      Displays all records in the Close state
      Active Items
      Displays all active records
      Inactive Items
      Displays all inactive records
      Refresh every minutes
      Type the refresh rate of the List view in this field.
      The default refresh rate is 10 minutes and the minimum refresh rate is one minute. You can modify this value depending on your requirements. A faster refresh rate impacts your actions because the List view would refresh when you are performing an action. BMC recommends that you use a refresh rate that is higher than the minimum refresh rate.
      The new refresh rate is applied when you click the Apply command and this value is stored in the browser's cookies. When you log on, the List view refreshes at the new refresh rate. If a staff member deletes the browser's cookies, the refresh rate of the List view is reset to 10 minutes. You can disable the automatic refresh of a List view by entering zero as the refresh rate.

      Note:  The Last Refreshed <Time> label displays the earlier time when the List view had refreshed.

    3. Click the Apply command.

      4. Depending on the commands selected in the Views menu, the List view refreshes and displays the matching records.

  2. From the Group By list, select a field.

    3. The List view refreshes and creates groups of matching records based on the field selected in the Group By list.

  3. To filter records by searching for the specific record, perform the following actions:
    1. In the Search field, type a search string.
    2. Click Search.

      3. The search results display records that match the search string.

    3. To clear the search results, click Undo.
