When a client reports an issue, you create an incident for the issue. However, if the client calls back later saying that the issue has been resolved without any action from you, you can delete the incident that you had created earlier.

To delete an incident or a service request

  1. Navigate to the record that you want to delete.

    2. Fore more information, see Navigating to an incident or a service request record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, select the check box for the required incident, and click Delete.

      • To delete multiple records, select the check boxes for all the records that you want to delete.

    • If you are using the Incident Console tab, select the check box for the required incident, and click Delete.

      • To delete multiple records, select the check boxes for all the records that you want to delete.

    • If you are using the Incidents tab (Salesforce page layout), click Delete.
    • If you are using the Visualforce page layout UI, click .
  3. In the confirmation dialog box, click Yes.

    4. The record is deleted and the incident form is closed.
