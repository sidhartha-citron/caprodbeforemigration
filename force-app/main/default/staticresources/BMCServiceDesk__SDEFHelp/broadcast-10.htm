Navigation to a record varies based on the UI that your system administrator has configured. For more information, see Determining the UI that you are using.

To navigate to a broadcast record

  1. Based on the UI and tab that your system administrator has configured, perform one of the following actions:
    • Click the Remedyforce Console tab and select View > Broadcasts.
    • Click the Broadcasts tab (Salesforce page layout).
  2. From the list of records, click the broadcast record that you want to use.
  3. If you are using the Visualforce page layout UI, perform the following actions:
    1. In the BMC Remedyforce home page, click Workspaces.
    2. From the list of Workspaces, expand Incident Management.
    3. From the Incident Management list, click Broadcasts.
    4. In the broadcasts list view, select the broadcast that you want to copy by using one of the following options:
      • Select a field from the Group By list, navigate to the required group, and select the required broadcast.
      • Type a search string in the Search field, click Search, and select the required broadcast from the search results.
      • Scroll through the list of broadcasts, and select the required broadcast.
    5. Double-click the selected broadcast record.
