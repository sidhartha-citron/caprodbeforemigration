In a new Change Request form, the Change Request Chatter Feed is disabled for editing. It is enabled for editing after you save the change request record. In the Change Request Chatter Feed, you can perform the following actions:

    • View updates that are made to the Change Request fields that are enabled for feed tracking
    • Comment on the updates that are made to the Change Request fields that are enabled for feed tracking
    • Create posts that you want to share with followers of the record
    • Follow and unfollow the change request record. You can follow a maximum of 500 records.

To use the Change Request Chatter Feed

  1. Navigate to the change request that you own or follow.

    2. For more information, see Navigating to a change request record.

    Note: By default, when you create or own a record, you automatically follow it. For more information, see Automatically following records.

  2. Click Change Request Chatter Feed.
  3. Perform the following actions to create a post that you want to share with followers of the change request record:
    1. Type the post that you want to add to the Chatter Feed.
    2. Click Share.

      3. The post appears in the Change Request Chatter Feed and in your Chatter Feed. You can view your Chatter Feed by navigating to Workspaces > Chatter > Chatter Feed.

  4. Perform the following actions to comment on the updates that are made to the Change Request fields:
    1. Click the comment link below the update that you want to comment.
    2. Type the comment.
    3. Click the Add Comment link.

      4. The comment appears below the update in the Change Request Chatter Feed and in your Chatter Feed.

  5. Perform one of the following actions to follow or unfollow the change request record:
    • In the Change Request Chatter Feed, click the Unfollow link to stop following the change request and stop seeing updates to the change request in your Chatter Feed.

      • The contents of the Change Request Chatter Feed are deleted.

    • In the Change Request Chatter Feed, click the Follow link to restart following the change request and viewing updates to the record in your Chatter Feed.

      • The updates that are made after clicking the Follow link are displayed in the Change Request Chatter Feed and in your Chatter Feed.

  6. Click Save.

Automatically following records

By default, you follow records that meet the following conditions:

    • You have created the record or you own the record.
    • The system administrator has enabled feed tracking for the object of the record.
    • Another user has assigned the record to you and the system administrator has enabled the Owner field on the object of the record for feed tracking.

You can stop following records by navigating to Setup > My Chatter Settings > My Feeds and selecting the Stop automatically following records check box.

Note: If you have enabled the improved Setup user interface in your Salesforce organization, navigate to My Settings > Chatter > My feeds. For more information about the improved Setup user interface, see http://help.salesforce.com/apex/HTViewHelpDoc?id=admin_setup_improved.htm&language=en_US.
