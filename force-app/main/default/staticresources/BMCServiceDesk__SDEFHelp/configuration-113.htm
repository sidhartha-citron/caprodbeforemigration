Page titles of the objects in BMC Remedyforce are mapped to plural object labels.

Note: Object labels are not part of the Translation Workbench and the translation files that can be exported from Salesforce. You must localize object labels for each organization individually.

To localize object labels

  1. Navigate to Setup > Customize> Tab Names and Labels > Rename Tabs and Labels.
  2. From the Select Language list, select the appropriate language to which you want to localize to.
  3. In the Custom Tabs section, in the Action column beside the tab name that you want to localize, click Edit.
  4. Enter the appropriate translation.
  5. Click Save.
