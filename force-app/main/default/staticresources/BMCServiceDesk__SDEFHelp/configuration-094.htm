By default, Salesforce allows you to track 20 fields on a form. BMC recommends that you always track the Record Update Count field. For all the fields of the object, this field helps you in tracking the user who makes the change and the time when the record is changed.

To enable field history tracking

  1. Navigate to Setup > Create > Objects.
  2. For the Incident, Task, Broadcast, Change Request, and Problem objects, perform the following actions:
    1. In the Label column, click the <object name>.
    2. In the Custom Fields & Relationships section, click Set History Tracking.
    3. In the Track old and new values section, select the required fields.
    4. Click Save.

      5. Note: BMC Remedyforce displays the auditing data in the Auditing tab of the Supporting Information section.
