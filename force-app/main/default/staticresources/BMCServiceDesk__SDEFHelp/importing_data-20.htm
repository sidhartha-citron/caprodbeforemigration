Different job files are available for different jobs. These job files are bundled with related files and provided to you in folders. The following table provides the names of the folders and the description of the job for which the job files in these folders are used.

Folder name
(CI Type)
Description
Job file
Computer System
Transfers information about the BMC_ComputerSystem CI type
TransferADDM(CS)InfotoCMDB.kjb
OS
Transfers information about the BMC_OperatingSystem CI type
TransferADDM(OS)InfotoCMDB.kjb
Processor
Transfers information about the BMC_Processor CI type
TransferADDM(CPU)InfotoCMDB.kjb
IPEndpoints
Transfers information about the BMC_IPEndPoint CI type (both IPv4 and IPv6)
TransferADDMIPEndpointInfotoCMDB.kjb
LANEndpoints
Transfers information about the BMC_LANEndpoint CI type
TransferADDMLANEndpointInfotoCMDB.kjb
Software Server
Transfers information about the BMC_SoftwareServer CI type

Note:  Depending on the organization size, running the Software Server job transfers large amounts of data to the Salesforce organization, which might lead to exceeding the organization storage limit.
TransferADDMSoftwareServerInfotoCMDB.kjb
Combined Jobs
There are two batch files in the folder. The TransferAdvancedADDMinfotoCMDB.bat batch file transfers information related to the BMC_ComputerSystem, BMC_OperatingSystem, BMC_Processor, BMC_IPEndPoint, and BMC_LANEndpoint CI types.
The TransferComputerSystemAndSoftwareADDMinfotoCMDB.bat batch file transfers information related to the BMC_ComputerSystem and BMC_SoftwareServer CI types.
TransferAdvancedADDMinfotoCMDB.bat and TransferComputerSystemAndSoftwareADDMinfotoCMDB.bat
