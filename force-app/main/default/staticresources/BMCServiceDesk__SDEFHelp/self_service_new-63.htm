BMC Remedyforce supports keyboard shortcuts for performing a few common actions in Self Service. Based on your requirement, you can deactivate these keyboard shortcuts. If you deactivate the setting, keyboard shortcuts are disabled in Self Service and on the Remedyforce Console, Incident Console, and Task Console tabs. To deactivate the keyboard shortcuts, you must create a custom setting Deactivate Keyboard Shortcuts and set the value as True. For more information, see Deactivating keyboard shortcuts.
