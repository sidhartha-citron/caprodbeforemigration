You can print a problem record on a printer or as a PDF output, when required. If you want to add a specific field to the PDF output, contact your system administrator.

To print a problem record

  1. Navigate to the problem record that you want to print.

    2. For more information, see Navigating to a problem record.

  2. If you are using the Remedyforce Console tab, perform the following actions:
    1. From the Actions menu, select Print.
    2. To print a record in the PDF format, from the Actions menu, select Print to PDF.
  3. If you are using the Problems tab (Salesforce page layout), perform the following actions:
    1. On the upper-right corner of the page, click Printable View.
    2. On the upper-right corner of the page, click Print This Page.
  4. If you are using the Visualforce page layout UI, perform the following actions:
    1. From the menu, select Print.

      2. The page layout of the Problem object is opened. To change the page layout in which you want to print the problem, contact your system administrator.

    2. To print a record in the PDF format, from the menu, select Print to PDF.
