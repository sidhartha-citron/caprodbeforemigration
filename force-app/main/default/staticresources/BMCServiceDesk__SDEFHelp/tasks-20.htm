You can view the configuration items (CIs) that you have linked to a task in the CI Explorer.

Note: There is no limit on the number of CIs that you can link to a record; however, at a time, you can view only 150 linked CIs in the CI Explorer.

To view CIs linked to a task in the CI Explorer

  1. Navigate to the task for which you want to see linked CIs in the CI Explorer.

    2. For more information, see Navigating to a task record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Agent Tools menu, select CI Explorer.
    • If you are using the Task Console tab, from the Agent Tools menu, select CI Explorer.
    • If you are using the Tasks tab (Salesforce page layout), click CI Explorer.

The linked CIs are displayed in the CI Explorer window.

Note: If you are using the Remedyforce Console tab, each time you click a record from the CI explorer page, the record opens in a new browser tab or window.
