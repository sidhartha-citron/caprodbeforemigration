This section provides instructions for configuring jobs in BMC Server Automation Console for creating an incident.

  1. In the BMC Server Automation Console, click Jobs.
  2. Select the required Job entry.
  3. Click the Default Notifications tab.
  4. Select the Send email to field in the Compliance Results Notifications section or Job Run Notification section, depending on the data that you want to send.
  5. In the Send email to field, type the email address of the newly created email listener that was configured in BMC Remedyforce.
