After you implement the back-out plan for restoring the configuration item to its initial state before the change request was implemented, you must delete the change request because it is no longer applicable.

To delete a change request

  1. Navigate to the change request that you want to delete.

    2. For more information, see Navigating to a change request record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, select the check box for the required change request, and click Delete.

      • To delete multiple records, select the check boxes for all the records that you want to delete.

    • If you are using the Change Requests tab (Salesforce page layout), click Delete.
    • If you are using the Visualforce page layout UI, click .
  3. In the confirmation dialog box, click Yes.

The change request record is deleted and the list of change requests is displayed.
