After saving an account, you can attach documents or URLs to it.

To attach documents or URLs to an account

  1. In the BMC Remedyforce home page, click Configuration.
  2. From the list of Configuration options, expand Configuration.
  3. In the Configuration list, click Accounts.
  4. Double-click the account to which you want to attach documents or URL.
  5. In the Supporting Information section, click the Documentation tab.
  6. To attach a document or URL, from the Actions menu, select Add.
  7. To open an attached document, select the record, and from the Actions menu, select Open.
  8. To delete an attached document or URL, select the record, and from the Actions menu, select Delete.
