You can use the Users page to add system administrators and staff members.

Based on your requirements, you can enable a few staff members to perform administrative tasks in BMC Remedyforce. You must assign the Remedyforce Administrator permission set and select the Remedyforce Administrator check box for these staff members. You must also ensure that the View Setup and Configuration check box is selected for the profile that you have assigned to the staff members.

For information about the additional access that is granted by the Remedyforce Administrator permission set and the limitations of this permission set, see BMC Remedyforce profiles and permission sets.

You can also grant staff members the permission to create or edit staff members and system administrators from the Client form by selecting the Manage ServiceDesk Staff Member check box.

Note: If you are creating staff members, customize the default Salesforce.com User page layout to use the Remedyforce User page layout, which is available when you install the BMC Remedyforce managed package.

To add or edit a user

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Manage Users tile, and from the menu select Add and Edit User.
  3. Perform one of the following actions:
    • To add a user, click New User.
    • To edit a user detail, select the check box for the required user record, and click Edit in the Action column.
  4. From the User License list, select the appropriate license.

    5. The following table provides the information about which license and profile you must select for a type of user that you are adding:

  5. From the Role list, select a role for the staff member.
  6. From the Profile list, select the required profile.

    7. Note: BMC recommends that you use the Client form to add client users.

  7. To designate the user as a BMC Remedyforce staff member, select the ServiceDesk Staff check box. Also, assign the ServiceDesk Staff permission set to the users. For more information, see Assigning permission sets.

    8. The ServiceDesk Staff check box is not available by default. Edit the layout of the form and add the check box to the layout. If you have customized the default Salesforce.com User page layout to use the Remedyforce User page layout, this check box is available in the BMC Remedyforce section.

    Note: If you do not select the ServiceDesk Staff check box for these staff members, they do not appear in the Select from Staff window.

  8. To permit the user to perform administrative tasks in BMC Remedyforce, select the the Remedyforce Administrator check box.

    9. The Remedyforce Administrator check box is not available by default. Edit the layout of the form and add the check box to the layout.

    Note: You must select the Remedyforce Administrator check box to enable the user to create, edit, and delete custom QuickViews, and copy out-of-the-box QuickViews.

    To configure the user as a Remedyforce administrator, you must also perform the following tasks:

    • Assign the Remedyforce Administrator permission set to the user. For more information, see Assigning permission sets.
    • In the Administrative Permissions section of the user’s profile, ensure that the View Setup and Configuration check box is selected.
  9. (Optional) To permit the user to create or edit staff member and system administrator records by using the Client form, select the Manage ServiceDesk Staff Member check box.

    10. The Manage ServiceDesk Staff Member check box is not available by default. Edit the layout of the form and add the check box to the layout. If you have customized the default Salesforce.com User page layout to use the Remedyforce User page layout, this check box is available in the BMC Remedyforce section.

    All users who can access the Clients form can view details of staff members. However, only those users for whom the Manage ServiceDesk Staff Member check box is selected can edit the information about staff members and system administrators on the Clients form. Additionally, the Account field appears as an optional field when you view these staff member records on the Clients form. These staff members can log on to BMC Remedyforce and Self Service.

  10. Click Save.

For more information, see https://help.salesforce.com/htviewhelpdoc?err=1&id=admin_users.htm&siteLang=en_US.

Note: BMC recommends that you assign out-of-the-box permission sets to the users you set up in your organization. For more information, see Assigning permission sets.
