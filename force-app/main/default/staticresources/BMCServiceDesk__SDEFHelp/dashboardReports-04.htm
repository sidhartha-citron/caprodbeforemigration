A report type defines the set of records and fields available to a report.

To create a report type

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Create Dashboards and Reports tile, and from the menu select Report Types.
  3. On the All Custom Report Types page, click New Custom Report Type.

    4. For more information about creating report types, click the Help for this page link.
