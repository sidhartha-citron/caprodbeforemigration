You can view configuration items (CIs) that you have selected for an incident in the CI Explorer. However, the Information Pane in the CI Explorer does not display closed records that are linked to the selected CI.

Note: There is no limit on the number of CIs that you can link to a record; however, at a time, you can view only 150 linked CIs in the CI Explorer.

To view CIs linked to an incident in the CI Explorer

  1. Navigate to the incident for which you want to view CIs in the CI Explorer.

    2. For more information, see Navigating to an incident or a service request record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Agent Tools menu, select CI Explorer.
    • If you are using the Incident Console tab, from the Agent Tools menu, select CI Explorer.
    • If you are using the Incidents tab (Salesforce page layout), click CI Explorer.

The selected CI is displayed in the CI Explorer window. A red cross on a CI icon in the CI Explorer window () signifies that the linked CI is affected because of the incident. This red cross is not shown for a CI linked to a service request.

Note: If you are using the Remedyforce Console tab, each time you click a record from the CI explorer page, the record opens in a new browser tab or window.
