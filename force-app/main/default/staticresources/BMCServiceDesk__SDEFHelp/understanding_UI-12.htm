System administrators can add or remove Visualforce page components on the Salesforce page layout. The following table describes the components that you can add to or remove from the page layout:

Visualforce page component
Applies to
Description
Broadcast Ticker
By default, this component is displayed on the layout of all objects.
Displays the broadcast message on the page layout.
Stage Indicator
By default, this component is displayed on the layout of all objects.
Displays the stage of a record.
Add Notes
This component is available for Incident, Task, Change Request, and Problem objects.
Allows you to add notes to a record. The system administrator must add this component for the users of the layout.
BMC recommends that you create a separate section on the layout with a meaningful name and add the Add Notes component to that section. Otherwise a text box without label is displayed on the layout.
Display Broadcasts
This component is available for the Broadcast object only.
Allows you to set options to display broadcasts to users.
Category Tree
This component is available for Incident, Task, Change Request, and Problem objects.
Displays all categories in a category tree.

Note:  This component must be added to the object page layout to display the Due Date Calculation window if the When the Priority changes, recalculate the Due Date based on the Current Date without user input check box is clear on the General Application Settings page.
StaffandQueueAssignment
NA
Enables your users to assign a record to a staff or queue. This component displays Staff and Queue fields on the layout.
Service Request Detail
This component is available for the Incident object only.
Allows you to enter details for the selected request definition. This components is displayed after an incident is saved.
