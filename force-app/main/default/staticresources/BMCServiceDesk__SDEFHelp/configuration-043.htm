BMC Remedyforce allows you to customize the BMC Remedyforce forms of the following objects:

    • Change Request
    • Client
    • Incident
    • Problem
    • Self Service: Service Request
    • Task

Each form is composed of one or many fields. The location of the field on the user interface depends upon the field set to which the field is added. A field set is a group of fields that is used to store related information. For example, a field set that provides details of a user can contain fields such as First Name, Last Name, and Designation. An administrator can add or remove fields in a field set. After you add a field to a field set, it is displayed in the UI.

You can customize forms by adding or removing fields from the field sets that are available for the object form. When you add fields from a field set, these fields appear on the custom form in the order in which these fields appear in the field sets. You can create custom fields for the object and add these custom fields to the form from the field sets. When you create a custom field, it is available on each of the field sets that are to be added on the custom form.

The following steps provide an overview of how to customize forms in BMC Remedyforce:

Step 1 In Application Settings, assign the custom version of the Incident, Change Request, Client, Problem, Self Service: Service Request, or Task forms to the required profiles.

Step 2 From the out-of-the-box field sets, add, remove, and reorder the fields that you want to add to the custom form.

Step 3 Create any more custom fields that you need and add those fields to the custom form.
