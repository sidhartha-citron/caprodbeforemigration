You can work with incidents, and view tasks and broadcasts, by using BMC Remedyforce on the Salesforce1 mobile platform. You must enable the appropriate settings.

To enable Remedyforce mobile application on the Salesforce1 platform

  1. To enable Salesforce1 for the appropriate user, perform the following actions:
    1. Click the Remedyforce Administration tab.
    2. On the Home page, click the Manage Users tile, and from the menu select Add and Edit User.
    3. In the Action column click Edit next to the required user.
    4. In the General Information section, select the Salesforce1 User check box.
    5. Click Save.
  2. To enable users to access the Remedyforce mobile application on the Salesforce1 platform, select the API Enabled check box for their profile.
    1. Navigate to Setup > Manage Users > Profiles.
    2. On the User Profiles page, click Edit next to the appropriate profile.
    3. In the Administrative Permissions section, select the API Enabled check box.
    4. Click Save.
  3. Navigate to Setup > Mobile Administration > Salesforce1 > Settings and select the Enable the Salesforce1 mobile browser app check box and click Save.
  4. To add Remedyforce Mobile application to the Salesforce1 platform, perform the following steps:
    1. Navigate to Setup > Mobile Administration > Mobile Navigation.
    2. On the Navigation Menu Item page, from the Available list, select Remedyforce Mobile and use the right arrow key to move it to the Selected list.

      3. Note: If you want to set Remedyforce mobile application as the landing application after you log on to Salesforce1, you must keep it at the top of the Selected list.

    3. Click Save.
