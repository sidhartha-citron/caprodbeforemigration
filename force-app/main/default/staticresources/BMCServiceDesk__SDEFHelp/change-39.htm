You can view tasks linked to a change request or release in the Tasks pane.

To view linked tasks

  1. Navigate to the change request for which you want to view linked tasks in Change Schedule.

    2. For more information, see Navigating to a change request record.

    To view the linked tasks of a release, navigate to the required release record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Agent Tools menu, select Change Schedule.
    • If you are using the Change Requests tab (Salesforce page layout), click Change Schedule.

      • Alternatively, you can navigate to Remedyforce Administration > Manage Workflows and Other Processes > Change Schedule or you can navigate to Remedyforce Workspaces > Change Management > Change Schedule.

    • If you are using the Visualforce page layout UI, from the Change Management list, click Change Schedule.
  3. Click the Tasks pane at the bottom of the page.
  4. From the Linked To list, select the change request or release whose tasks you want to view.

    5. The tasks linked to the selected change request or release are displayed. The tasks that are not yet created for the change request or release are displayed in the Unscheduled Tasks pane. The tasks for which no value is selected in the Scheduled Start Date or Scheduled End Date fields are also displayed in the Unscheduled Tasks pane.

  5. To view the details of a task, click the task.
