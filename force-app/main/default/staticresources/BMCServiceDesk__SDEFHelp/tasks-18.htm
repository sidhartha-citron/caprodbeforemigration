If an approval process has been set up to submit a task for approval, submit the task for approval.

To submit a task for approval

  1. Navigate to the task that you want to submit for approval.

    2. For more information, see Navigating to a task record..

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Actions menu, select Submit for Approval.

      • Alternatively, you can click Record Details and in the Approval History section, click Submit for Approval.

      Note: If your system administrator has configured a manual approval process, after you select the approver, you must close the Submit for Approval window, manually.

    • If you are using the Tasks tab (Salesforce page layout), in the Approval History section, click Submit for Approval.
  3. In the confirmation dialog box, click OK.
