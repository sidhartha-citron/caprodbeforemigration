When the users are created, you must assign a license to them for the BMC Remedyforce.

To assign a license to staff members and clients

  1. Navigate to Setup > Installed Packages.
  2. In the Package Name column, click BMC Remedyforce.
  3. Click Add Users.
  4. Select the credentials of the user that were sent in the email message.
  5. Click Add.
