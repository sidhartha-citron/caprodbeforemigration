Broadcasts allow you to create messages that can be communicated to multiple users at the same time. A broadcast is helpful in situations in which you are working on an issue. For example, if a network server is affected, you can create a broadcast that notifies users that the server is down or will be unavailable for some time. BMC Remedyforce enables you to create and display broadcasts to all profiles and Self Service users.

The following topics are provided:

  • Creating and assigning broadcasts
  • Working with broadcasts
  • Closing and reopening broadcasts
