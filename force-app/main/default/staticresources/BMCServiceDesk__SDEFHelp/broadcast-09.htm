You can create or edit the custom views on the Remedyforce Console tab. If the setting to create a custom view is not enabled on the Remedyforce Console tab, contact your system administrator.

To create or edit a custom view for broadcast records

  1. Click the Remedyforce Console tab and select View > Broadcasts.
  2. From the list view, perform one of the following actions:
    • To create a custom view, click Create New View.
    • To edit an existing view, click next to the required view.

      • For more information about creating or editing a custom view, see https://help.salesforce.com/htviewhelpdoc?id=customviews.htm&siteLang=en_US.

      You cannot configure the fields that are displayed in custom list views. Contact your system administrator to customize the Column Headers feildset to add the desired fields to the list view.

      Note: After you create a new view, you must refresh the Remedyforce Console tab to see the it in the list view.
