After saving a broadcast, you can add user-defined and system-defined actions to the broadcast in the Action History section (in the History tab of the Supporting Information section in the Visualforce page layout user interface (UI)).

To add an action history record

  1. Navigate to the broadcast to which you want to add a history record.

    2. For more information, see Navigating to a broadcast record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click the Actions menu, and select New Action.

      • Alternatively, click Record Details and in the Action History section, click New Action.

    • If you are using the Broadcasts tab, (Salesforce page layout), click New Action in the Action History section.
  3. If you are using the Visualforce page layout UI, perform the following actions:
    1. In the Supporting Information section, click the History tab.
    2. From the Actions menu, select Add.
  4. Select the action that you want to add.

    5. If you select the Notes option, a new browser window is displayed where you can enter additional notes. These notes are displayed in the Action History section in Salesforce page layout. If you are using the Salesforce page layout and do not see the Note field on your layout, contact your system administrator to edit the page layout.

Viewing actions related to broadcasts

You can view the actions related to a broadcast by using the View Action History option. This option enables you to create a new action, filter actions based on type, and print actions.

To view actions related to a broadcast

  1. Navigate to the broadcast for which you want to view actions.

    2. For more information, see Navigating to a broadcast record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click the Actions menu, and select View Action History.

      • Alternatively, click Record Details and in the Action History section, click View Action History.

    • If you are using the Broadcasts tab (Salesforce page layout), in the Action History section, click View Action History.

      • The Broadcast History window displays the actions in view-only mode. You cannot edit or delete the actions that are displayed in the Broadcast History window.

  3. (Optional) To filter the actions based on the type, select the required action type from the list.
  4. (Optional) To print the actions related to an incident or a service request record, click Print.
