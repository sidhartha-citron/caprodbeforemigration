Navigation to a record varies based on the UI that your system administrator has configured. To determine the UI that you are using, see Determining the UI that you are using.

To navigate to a task record

  1. Based on the UI and tab that your system administrator has configured, perform one of the following actions:
    • Click the Remedyforce Console tab and select View > Tasks.
    • Click the Task Console tab.
    • Click the Tasks tab (Salesforce page layout).
  2. From the list of records, click the task record that you want to use.
  3. If you are using the Visualforce page layout UI, perform the following actions:
    1. In the BMC Remedyforce home page, click Workspaces.
    2. From the list of Workspaces, expand Incident Management.
    3. In the Incident Management list, click Tasks.
    4. In the tasks list view, select the required task by using one of the following options:
      • Select a field from the Group By list, navigate to the required group, and select the required task.
      • Type a search string in the Search field, click Search, and select the required task from the search results.
      • Scroll through the list of tasks, and select the required task.
    5. Double-click the selected task record.
