After saving a change request, you can add user-defined actions and system-defined actions to the change request in the Action History section.

To add an action history record to a change request

  1. Navigate to the change request to which you want to add an action.

    2. For more information, see Navigating to a change request record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click the Actions menu, and select New Action.

      • Alternatively, click Record Details and in the Action History section, click New Action.

    • If you are using the Change Requests tab (Salesforce page layout), click New Action in the Action History section.
  3. In the Action History section, click New Action.
  4. Select the appropriate action that you want to add to the Action History section.

    5. If you select the Notes option, a new browser window is displayed where you can enter additional notes. This note is displayed in the Action History section.

  5. If you are using the Visualforce page layout UI, perform the following actions:
    1. In the Supporting Information section, click the History tab.
    2. From the menu, select Add.

      3. In the Visualforce page layout UI, this note is displayed in the History tab. You can create a workflow to send an email message to the client when a particular action is added to the History tab.

      Note: You cannot save a Client Note action after adding the action. The Client Note actions are for the exclusive use of Self Service clients. This restriction is added to allow Self Service clients to edit the Client Note actions.

  6. Add the required details, and click Save.

Viewing actions related to change requests

You can view the actions related to a change request by using the View Action History option. The View Action History option allows you to create a new action, filter actions based on type, and print the actions.

To view actions related to a change request

  1. Navigate to the change request for which you want to view actions.

    2. For more information, see Navigating to a change request record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click the Actions menu and select View Action History.

      • Alternatively, click Record Details and in the Action History section, click View Action History.

    • If you are using the Change Requests tab (Salesforce page layout), in the Action History section, click View Action History.

      • The Change Request History window displays the actions in the view-only mode. You can not edit or delete the actions that are displayed in the Change Request History window.

  3. (Optional) To filter the actions based on the type, select the required action type from the list.
  4. (Optional) To print the actions related to a change request, click Print.
