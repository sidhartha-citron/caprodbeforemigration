Coordinate Format
The format in which the geographical position of the element is specified. The World Geodetic System 1984 (WGS84) standard, used by virtually all geographic information systems, is preferred.
In this context
  • Latitude is a decimal value between-180.000 and +180.000 (positive values represent the number of degrees north of the equator; negative values represent the number of degrees south).
  • Longitude is a decimal value between -180.000 and +180.000 (positive values represent the number of degrees east of the prime meridian; negative values represent the number of degrees west).
  • Altitude is a decimal value that specifies height (in meters) above mean sea level (MSL); negative values indicate depth below MSL.
Other values can be used in the Coordinate Format field, but they might not work with partner products or other BMC products. Geographic positioning values may be converted to more human-readable formats (such as degrees, minutes, and seconds; North, West, South, and East; and so on) to show in user interfaces.
