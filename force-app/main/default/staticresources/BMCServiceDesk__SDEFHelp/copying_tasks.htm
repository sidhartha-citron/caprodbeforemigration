On the Remedyforce Console, Task Console, and Tasks tab (Salesforce page layout), if you have a task that needs to be performed for multiple categories, you can clone the task, modify the category, and reuse the task record.

The option to copy a record is provided in the Visualforce page layout UI only.

To clone or copy a task

  1. Navigate to the task that you want to clone.

    2. For more information, see Navigating to a task record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, from the Actions menu, select Clone.
    • If you are using the Task Console tab, from the Actions menu, select Clone.
    • If you are using the Tasks tab (Salesforce page layout), click Clone.
    • If you are using the Visualforce page layout UI, click .

      • The New Task form is displayed, and the values of the fields in the task that you have cloned are displayed.

  3. In the New Task form, make the required modifications.
  4. Click Save.
