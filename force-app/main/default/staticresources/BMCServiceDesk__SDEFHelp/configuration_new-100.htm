The Alignability tab enables users to view the Alignability Process Model. The model provides easy and fast access to detailed information about actions that the users need to perform in any given situation. By default, the Alignability tab is not visible for the ServiceDesk Staff and ServiceDesk Change Manager profiles. System administrators must make this tab visible for the required profiles.

Note: If you have enabled the enhanced profile user interface in your Salesforce organization, the steps to configure visibility of the Alignability tab for profiles are different. For information about using the Enhanced Profile User Interface, see http://help.salesforce.com/HTViewHelpDoc?id=users_profiles_using_enhanced_ui.htm&language=en_US.

To configure visibility of the Alignability tab for profiles

  1. Click the Remedyforce Administration tab.
  2. On the Home page, click the Manage Users tile, and from the menu select Profiles.
  3. In the Action column of the required profile, click Edit.
  4. In the Tab Settings section, select Default On for the Alignability tab.
  5. Click Save.
  6. (Optional) To enable a user to download the process documents for languages other than English, perform the following steps:
    1. In the Enabled Visualforce Page Access section, click Edit.
    2. In the Available Visualforce Pages list, select the BMCServiceDesk.APMProcDocLoader and move it to the Enabled Visualforce Pages list by using the right arrow.
    3. Click Save.
