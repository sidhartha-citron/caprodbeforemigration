BMC Remedyforce supports keyboard shortcuts for performing a few common actions on the Remedyforce Console, Incident Console, and Task Console tabs. You must use the same shortcuts for all languages and browsers that are supported in BMC Remedyforce. If you do not want to use keyboard shortcuts, contact your system administrator to deactivate them.

The following table describes the keyboard shortcuts that are supported on the different tabs.

Tab
Shortcut
Description
  • Remedyforce Console
  • Incident Console
  • Task Console

    • Note:  These shortcuts are also supported if you have configured the BMC Remedyforce modules in the Service Cloud console. However, the shortcuts work only if the cursor is in the form details area.
Ctrl+Alt+N
  • On the Remedyforce Console tab: Opens a new record in another tab based on the selected view.

    • For example, if Tasks is selected in the View list, the Ctrl+Alt+N shortcut opens a new task record.

  • On the Incident Console tab: Opens a new incident record.
  • On the Task Console tab: Opens a new task record.
Ctrl+Alt+S
Saves the active record
Ctrl+Alt+A
Opens the Actions menu for the active record.
Ctrl+Alt+T
Opens the Agent Tools menu for the active record.
Remedyforce Console
Ctrl+Alt+W
Closes the active tab on the Remedyforce Console tab.
Ctrl+Alt+0
Navigates to the first tab on the Remedyforce Console tab.
Ctrl+Alt+1 to Ctrl+Alt+6

Note:  These shortcuts are not supported if you are using BMC Remedyforce in the Safari browser on Macintosh operating system.
Open a list view from the View list on the Remedyforce Console tab. These shortcuts are dynamic and open the list views based on the configurations that your system administrator. The Ctrl+Alt+1 shortcut opens the list of records for the first module in the View list, the Ctrl+Alt+2 shortcut opens the second module in the View list, and so on.
For example:
If all modules are configured, then the shortcut Ctrl+Alt+6 displays the list of records for the last module from the View list.
If only the Tasks, Problems, and Releases modules are configured, then the shortcut Ctrl+Alt+1 displays the list of records for the Tasks module, Ctrl+Alt+2 displays the list of records for the Problems module, and Ctrl+Alt+3 displays the list of records for the Releases module.
