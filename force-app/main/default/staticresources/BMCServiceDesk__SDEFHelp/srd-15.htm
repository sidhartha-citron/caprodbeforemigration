The Request Definitions List view Actions menu allows you to make online and take offline multiple service request definition records. To select multiple records, press the Ctrl key and click the required records. You can select multiple consecutive records by pressing the Shift key and clicking the first and last records of the consecutive records. Clicking a selected record clears it.

The following table describes the commands of the Actions menu.

Table 17-3:  Actions menu commands  
Actions menu commands
Description
Make Online
Makes the selected service request definition records online
Take Offline
Takes the selected service request definition records offline
