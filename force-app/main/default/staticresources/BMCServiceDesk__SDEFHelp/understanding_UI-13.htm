The following table describes items that are common to all forms:

Common items
Description

Lookup icon. This icon is displayed for fields for which you can select a value from a list. When you click a lookup icon, a separate window is displayed with a list of active records. To search in the Lookup window, enter the first few characters of the value that you want to search for in the Search text box and click Go. In the list of matching items that is displayed, click the appropriate value to select it. You can sort records in ascending or descending order based on the columns in the lookup window. By default, 25 records are displayed on one page.

Red side bar. A red side bar indicates that the field is required to save a form.

Tooltip icon. When you move your mouse over this icon, the tooltip for the field is displayed.

Remedyforce Search
Left side bar search component. This portlet allows a logged-on staff member to search specific records based on given input search text within all objects of an organization. Copy link, Copy text, Return to Results options are unavailable on the Knowledge article detail page launched from the search results. Remedyforce Search settings depend on the settings that your system administrator has configured for global search.
BMC Remedyforce
Left side bar documentation component. This portlet provides links to Remedyforce documentation and BMC Communities.
