You can attach add attachments to a change request either before or after you save the record. you save the record.

On the Remedyforce Console tab, if you want to add attachments to a record before you save it, contact you system administrator to configure the Attach/View Files option in the Actions menu.

If you do not see the documents attached to a record from the Visualforce page layout UI of the same record, contact your system administrator.

To add an attachment to a change request before you save the record

  1. On the Remedyforce Console tab New form, from the Actions menu, select Attach/View Files.
  2. In the Attach/View Files window, click Choose File.

    3. Note: The size of one attachment cannot exceed 10 MB.

  3. Select the file you want to add and click Open.

    4. The selected file is attached to the record. In the Attach/View Files window, you can view or delete the existing attachments.

To add an attachment to a change request after you save the record

  1. Navigate to the change request to which you want to attach a file.

    2. For more information, see Navigating to a change request record.

  2. Perform one of the following actions:
    • If you are using the Remedyforce Console tab, click Record Details, and in the Notes & Attachments section, click Attach File.
    • If you are using the Change Requests tab (Salesforce page layout), in the Action History section, click Attach File.
  3. To select the file that you want to attach to the incident, click Choose File.

    4. Note: The size of one attachment cannot exceed 10 MB.

  4. Click Attach File.
  5. Click Done.
  6. If you are using the Visualforce page layout UI, perform the following actions:
    1. In the Supporting Information section, click the Documentation tab.
    2. From the menu, select Add.
    3. To attach a file, click Choose File and select the required file.
    4. To attach a URL, select URL, and provide the required URL.
    5. Click OK.
    6. (Optional) To open an attached document or URL, select the document or URL and from the menu, select Open.
    7. (Optional) To delete an attached document or URL, select the document or URL and from the menu, select Delete.
