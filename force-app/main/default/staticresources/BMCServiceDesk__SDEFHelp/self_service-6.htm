In the Branding tab, you can apply a User Interface Theme to Self Service and select the images that allow you to brand your Self Service. You can also override the images that appear in the Login page to customize the look and feel of the Login page for your Self Service clients. When you click Login Page: Company Logo or Banner: Application Title image, you will be redirected to the selected website.

You can change the following images of Self Service:

  1. Login Page: Application Title—This image appears in the upper left section of the Login page.
  2. Login Page: Company Logo—This image appears in the upper right section of the Login page.
  3. Login Page: Brand Background—This image appears in the middle section of the Login page, next to the fields that allow a Self Service client to enter their logon credentials.
  4. Banner: Application Title—This image appears in the upper left section of the Self Service home page. You can replace this image with your company's logo to customize Self Service for your clients.

    5. Note: If you make modifications while users are logged on to Self Service, Self Service clients must refresh their browsers to view the changes made to Self Service. They can refresh their browsers by clicking Refresh in the browser or pressing the F5 key.

    Tip: BMC recommends that you use images that are smaller in size to improve the performance of BMC Remedyforce.

To configure the Self Service branding

  1. In the BMC Remedyforce home page, click Configuration.
  2. From the list of Configuration options, expand Self Service.
  3. In the Self Service list, click Self Service Settings.

    4. Note: In the BMC Remedyforce tabbed view, Self Service Settings is available in the Remedyforce Administration tab.

  4. Click the Branding tab.
  5. From the User Interface Theme list, select the theme to be applied to Self Service.

    6. The selected User Interface Theme is applied to Self Service when you refresh your browser. The following out-of-the-box themes are available:

    • BMC Theme
    • Dark Yellow Theme
    • Deep Green Theme
    • Neutral Theme
    • Rich Red Theme
    • Steel Blue Theme

      • The Set as default check box appears next to the current theme. Select the check box if you want the selected theme to be set as default theme for the users. By default, this check box is not selected.

      If you select another theme from the User Interface Theme list, the Set as default field is clear.

  6. (Optional) To assign a theme to an account, perform the following actions:
    1. Click the Assign Theme to Accounts link.

      2. In the Select Accounts for the Theme window, select the required accounts. You can make the theme available to a large number of accounts at one time. You can also search for a particular account by entering the search criteria in the Search field and clicking Search. To clear the search criteria in the Search field and refresh the list of accounts in the Available Accounts list, click Undo. You can use the * wildcard character to match one or more characters at the end of your search criteria, but not the beginning or the middle of your search criteria. You can navigate to the next or previous page of accounts by using Next Page and Previous Page next to the Undo.

    2. Click the right arrow to move the selected accounts from the Available Accounts list to the Selected Accounts list.
    3. (Optional) To remove the accounts that should not be assigned to the selected theme, select the accounts in the Selected Accounts list and click the left arrow to move the selected accounts from the Selected Accounts list to the Available Accounts list.
    4. Click Apply to save the accounts assignment to the selected theme.

      5. The selected theme becomes the default theme and is available to Self Service clients after they log on. The Self Service log on page appears in the default theme for all Self Service clients. Managers in the role hierarchy cannot click Save because this button is disabled for these users. Only the system administrator can click Save and change the default theme. For more information, see Implementing configuration and data segregation.

    5. In the confirmation dialog box, click OK.
    6. Click Cancel to close the Select Accounts for the Theme window and return to the Branding tab.
  7. For each image, perform one of the following actions:
    • Select Use Default to use the out-of-the-box images for the Login page.
    • Select Replace, click Browse to browse to the location where the image you want to use is located, select the required image, and click Open.

      • The selected image replaces the out-of-the-box image in the Login page.

  8. In the Company URL field, type the URL address assigned to the company logo and the application title. Ensure that you include http:// or https:// at the beginning of the URL.
  9. In the URL Tip Text field, type the text that appear as a ToolTip when you move your mouse over the Login page: Company Logo and Banner: Application Title image.
  10. Click Save.

    11. The Login page and the home page of Self Service appears to clients with the images that you have selected.
