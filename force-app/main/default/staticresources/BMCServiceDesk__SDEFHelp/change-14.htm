Navigation to a record varies based on the UI that your system administrator has configured. To determine the UI that you are using, see Determining the UI that you are using.

To navigate to a record

  1. Based on the UI and tab that your system administrator has configured, perform one of the following actions:
    • Click the Remedyforce Console tab and select View > Change Requests.
    • Click the Change Requests tab (Salesforce page layout).
  2. From the list of records, click the incident record that you want to use.
  3. If you are using the Visualforce page layout UI, perform the following actions:
    1. In the BMC Remedyforce home page, click Workspaces.
    2. From the list of Workspaces, expand Change Management.
    3. In the Change Management list, click Change Requests.
    4. In the change requests list view, select the change request that you want to copy by using one of the following options:
      • Select a field from the Group By list, navigate to the required group, and select the required change request.
      • Type a search string in the Search field, click Search, and select the required change request from the search results.
      • Scroll through the list of change requests, and select the required change request.
