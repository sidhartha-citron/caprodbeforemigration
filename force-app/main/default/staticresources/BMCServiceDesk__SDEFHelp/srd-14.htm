The Views menu allows you to filter the records that are displayed in the Request Definitions List view. The Views menu allows you to display only active records and inactive records. The Views menu also allows you to define the refresh rate of the Request Definitions List view and displays when the Request Definitions List view was last refreshed.

The following table describes the commands of the Views menu.

Table 17-2:  Views menu commands  
Views menu commands
Description
Active Items
Displays all active records
Inactive Items
Displays all inactive records
Refresh every minutes
Type the refresh rate of the List view in this field.
The default refresh rate is 10 minutes and the minimum refresh rate is one minute. You can modify this value depending on your requirements. A faster refresh rate impacts your actions because the List view would refresh when you are performing an action. BMC recommends that you use a refresh rate that is higher than the minimum refresh rate.
The new refresh rate is applied when you click the Apply command and this value is stored in the browser's cookies. When you log on, the List view refreshes at the new refresh rate. If a staff member deletes the browser's cookies, the refresh rate of the List view is reset to 10 minutes. You can disable the automatic refresh of a List view by entering zero as the refresh rate.

Note:  The Last Refreshed <Time> label displays the earlier time when the List view had refreshed.
