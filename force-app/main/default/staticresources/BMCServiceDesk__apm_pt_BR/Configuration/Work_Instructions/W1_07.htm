Work Instructions for Configuration Administrators

1.7.1

Register the already known attributes of the ordered configuration item (CIs). Do this by following the field utilization guidelines for the Configuration Item form. Set the Status field of the ordered CIs to Ordered. Do not yet link the ordered CIs to any other CIs, services, or users.

The ordered value in the Status field might not be available by default. You must select the Task check box for the Ordered status.

1.7.2

Add a Note action in the task specifying that the purchase order for the new CIs has been sent and that you are waiting for the CIs to be delivered.

1.7.3

Set the Status field of the task to Waiting for….

1.7.4

In the Duration (HH:MM) field, in a Notes action, enter the time that you spent purchasing and registering the CIs.
