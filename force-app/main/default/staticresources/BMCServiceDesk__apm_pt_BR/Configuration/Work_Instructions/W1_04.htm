Work Instructions for Configuration Administrators

1.4.1

After submitting the purchase request, assist with the preparation of a request for proposal and the subsequent review of proposals that have been submitted by suppliers.

1.4.2

After the purchasing department has processed the purchase requisition, check the progress to find whether the purchase order for the requested configuration item (CI) has already been sent to the supplier. If the purchase order has not been sent, determine whether the reason for not sending the purchase order is that the purchase requisition is rejected or the purchasing department is still processing it.
