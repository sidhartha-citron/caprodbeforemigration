Work Instructions for Configuration Administrators

1.1.1

When the value in the Status field of your Configuration Management task changes from Registered to Assigned, open it and read the instructions in the Description field.

1.1.2

If you are not ready to work on the task, set the Status field of the task to Accepted.

1.1.3

When you are ready to work on the task, set the Status field to In Progress.
