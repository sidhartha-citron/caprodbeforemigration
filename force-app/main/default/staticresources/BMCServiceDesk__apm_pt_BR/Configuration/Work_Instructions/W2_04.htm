Work Instructions for Configuration Administrators

2.4.1

Inform the supplier of any issue with the delivery. Ask the supplier what needs to be done for resolving the issue.

2.4.2

Take the necessary actions based on the instructions from the supplier. For example, put the defect or incorrectly delivered configuration items (CIs) back in their box and return them to the supplier, or wait for the missing CIs to be delivered.
