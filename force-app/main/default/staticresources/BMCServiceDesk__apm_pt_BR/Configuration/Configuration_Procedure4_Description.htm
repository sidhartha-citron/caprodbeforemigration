Procedure 4, CI Update


If the task from Change Management or Request Management does not require updating of existing configuration item (CI) information, the configuration administrator completes the task. However, if the configuration administrator is dealing with a task that requests the update of CI attributes and relations, the configuration administrator checks the actual configuration of the infrastructure to confirm that the requested modifications are really required to bring the CMDB back up to date.

If CMDB needs to be updated, the configuration administrator performs the update of the necessary CI attributes and relations.
