Process description


The Configuration Management process consists of the following procedures:

  • CI Requisition is used by configuration administrators when new configuration items (CIs) need to be ordered.

  • CI Receipt is used by configuration administrators when new CIs have been delivered.

  • Supplier Information Maintenance is used by configuration administrators when they register new suppliers and when they update the contact details of previously registered suppliers.

  • CI Update is used by configuration administrators when they update the attributes and relations of CIs that were already registered in the CMDB.

For details about these procedures, click Process to return to the graphical representation of this process and click the boxes that represent the procedures. The graphical representation of this procedure appears with its description in the right-hand pane.
