Procedure 3, Supplier Information Maintenance


The configuration administrators are responsible for registering and updating the information of organizations that supply configuration items (CIs) and support to the service provider organization.

A configuration administrator performs the supplier information maintenance tasks as needed before registering new CIs, before registering or updating support contracts, and whenever updated contact information has been received from an existing supplier.

If a supplier organization is not already registered, the configuration administrator adds it. If the supplier organization already exists in the service management application, the configuration administrator updates its contact details.
