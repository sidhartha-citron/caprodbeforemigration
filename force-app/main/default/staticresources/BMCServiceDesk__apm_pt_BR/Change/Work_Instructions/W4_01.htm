Work Instructions for Change Coordinators

4.1.1

After the change manager sets the Status field of the change request to Approved, assign the implementation tasks. Change the Status field of each task from Registered to Assigned.

Note:

Assign only the tasks that do not have to wait for other implementation tasks to be completed.

Note:

If the task execution order is enabled in the Change and Task templates, the next task is visible once the previous task has been completed.

4.1.2

Set the Status field of the change request to In Progress.
