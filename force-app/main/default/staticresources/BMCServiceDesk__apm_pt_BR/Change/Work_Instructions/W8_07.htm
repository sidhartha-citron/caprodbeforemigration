Work Instructions for Change Coordinators

8.7.1

Set the Status field of the task for informing the service owner of the implemented emergency change to In Progress.

8.7.2

Inform the service owner (and the on-duty manager if the duty manager approved the emergency change implementation) that the emergency change is successfully implemented. Send an email message summarizing the implemented change.

8.7.3

Add a Notes action of type Email Received and paste the content of this email message into the note.

8.7.4

Set the Status field of the task to Completed.

8.7.5

In the Duration (HH:MM) field, in the Notes action, enter the time that you spent completing the task.
