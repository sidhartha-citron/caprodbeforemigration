Work Instructions for Change Coordinators

1.10.1

Assign the change request to the release coordinator of the service for which the change is requested.

1.10.2

If the change request was received from Request Management as a request, in the Duration (HH:MM) field of the Notes action, enter the time that you spent reviewing the request.

In the Owner field of the request, select the release coordinator of the service for which the change is requested. Set the Status field of the request to Assigned.

1.10.3

If the change request was received from Problem Management as a problem, in the Duration (HH:MM) field of the Notes action, enter the time that you spent reviewing the problem.

In the Owner field of the problem, select the release coordinator of the service for which the change is requested. Set the Status field of the problem to Change Requested.
