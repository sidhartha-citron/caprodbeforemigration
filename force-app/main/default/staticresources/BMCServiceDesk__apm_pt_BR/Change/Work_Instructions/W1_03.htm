Work Instructions for Change Coordinators

1.3.1

If the change request was received from Request Management as a request, link the existing change to the request in the Linked Change Requests section by clicking Select and Link Change Requests to Incident. Add a Notes action in the request saying that the Responsibility for coordination has been accepted by Change Management.

In the Duration (HH:MM) field, in the Notes action, enter the time that you spent reviewing the request.

Set the Status field of the request to Change Pending.

1.3.2

If the change request was received from Problem Management as a problem, link the existing change request to the problem in the Linked Change Requests section by clicking Select and Link Change Requests to Problem. Add a Notes action to the problem saying that the Responsibility for coordination has been accepted by Change Management.

In the Duration (HH:MM) field, in the Notes action, enter the time that you spent reviewing the problem.

Set the Status field of the problem to Change Pending.
