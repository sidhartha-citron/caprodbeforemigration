Work Instructions for Release Coordinators

1.6.1

If the change request is received as a request, in the Linked Change Requests section of the request, select the change request that acts as a placeholder for requests that are waiting to be reviewed during the next CAB meeting of the service.

In the Duration (HH:MM) field, in the Notes action, enter the time that you spent reviewing the request.

1.6.2

If the change request is received as a problem, in its Linked Change Requests section, click Select and Link Change Requests to Release and link the problem to the change request that acts as a placeholder for requests that are waiting to be reviewed during the next CAB meeting of the service.

In the Duration (HH:MM) field, in the Notes action, enter the time that you spent reviewing the problem.

Note:

Linking the change request to this change request ensures that it is reviewed in Procedure 2, Release Definition during the next CAB meeting of the service.
