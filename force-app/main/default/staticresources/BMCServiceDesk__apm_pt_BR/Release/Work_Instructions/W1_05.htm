Work Instructions for Release Coordinators

1.5.1

Estimate the effort of internal employees and long-term contractors that is required to get the requested change implemented.

1.5.2

Estimate the cost of configuration items, services, and support that need to be acquired from suppliers to get the requested change implemented.

Note:

Consult with change coordinators and specialists to estimate for the change correctly.

1.5.3

If the change request is received in the form of a request, add a Notes action to enter the estimates and to indicate that the responsibility for coordination has been accepted by Release Management. Also, enter the assumptions on which the estimates are based. Set the Status field of the request to Change Pending.

1.5.4

If the change request is received in the form of a problem, add a Notes action to enter the estimates and to indicate that the responsibility for coordination has been accepted by Release Management. Also, enter the assumptions on which the estimates are based. Set the Status field of the problem to Change Pending.
