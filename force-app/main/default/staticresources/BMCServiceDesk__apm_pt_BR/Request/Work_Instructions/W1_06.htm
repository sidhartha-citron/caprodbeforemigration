Work Instructions for Service Desk Analysts

1.6.1

If the user has provided additional information relevant to the resolution of the request, add a Notes action to enter this information and attach related files to the request. Contact the assigned specialist and provide an update.

1.6.2

Provide an update to the user on the request's status.

Note:

If the request is not recently updated, contact the assigned specialist to obtain a status update so that you can provide this information to the user.

1.6.3

Add a Notes action to summarize your actions and the information that you provided to the user.

1.6.4

In the Duration (HH:MM) field, in the Notes action, enter the time that you spent updating the request.
