Work Instructions for Service Desk Analysts

1.9.1

Select the related major incident in the Broadcast field.

You can link an existing incident from the Agent Tools > Incident Matching option.

1.9.2

Complete the remaining required fields and save the request.

Note:

When a request is linked to a major incident, it should not be directly modified. Updates to the major incident are copied to all child requests.

After selecting the broadcast and saving the incident, the following fields are populated in the incident:

  • Description

  • Configuration Item

  • Impact

  • Priority

  • Completion Date

  • Due Date

  • Status

  • Resolution
