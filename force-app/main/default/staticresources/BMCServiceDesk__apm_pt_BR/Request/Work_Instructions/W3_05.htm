Work Instructions for Service Desk Managers

3.5.1

Turn the user into a satisfied user of the service provider organization with help from the service desk analysts, specialists, and service level managers, if necessary.

Note:

If it is clear that the complaint cannot be resolved before the date and time specified in the Due Date field, estimate a realistic time by which the complaint will be resolved and add a Notes field to add this information.

Note:

Set the Status field of the request to Waiting for or Waiting for customer if the work cannot continue for a while because you are waiting for information from the user or a specialist. Add a Notes action to specify what you are waiting for.
