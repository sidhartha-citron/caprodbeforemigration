Work Instructions for Service Desk Analysts / Specialists

6.16.1

Open the request and summarize how the request is implemented in the Resolution field.

6.16.2

Set the Closure Category field to Resolved and set the Status field of the request to Completed.

6.16.3

In the Duration (HH:MM) field, in the Notes action, enter the time that you spent resolving the request.

Note:

Specify only the time spent on registering and coordinating the request. The time spent on the resolution of the individual tasks is specified in each task.

6.16.4

Perform a final review of the request to ensure that its information is accurate and complete.
