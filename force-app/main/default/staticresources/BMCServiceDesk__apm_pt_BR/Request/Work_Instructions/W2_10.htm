Work Instructions for Group Coordinators

2.10.1

If Change Management must handle the request, continue with 2.11.1. Otherwise, go to 2.13.1.

Note:

If the request is an incident, a specialist determines in 5.2.1 how to resolve the incident to decide if Change Management is required to resolve the incident. For incidents, go to 2.13.1.

Note:

Change Management must coordinate a request only when the implementation of the request has one of the following effects:

  • A service is unavailable or degraded during service hours

  • The functionality of a service changes

  • The CMDB requires an update
