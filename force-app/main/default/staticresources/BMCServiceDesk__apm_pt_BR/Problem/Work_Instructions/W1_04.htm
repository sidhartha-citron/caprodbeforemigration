Work Instructions for Problem Coordinators

1.4.1

If the request is significant, continue with 1.5.1. If the request is insignificant, go back to 1.2.1 to look at the details of the next request.

Note:

A request is considered significant if:

  • The Impact level is High or Major.

  • There have been multiple occurrences.

  • It is expected to recur.
