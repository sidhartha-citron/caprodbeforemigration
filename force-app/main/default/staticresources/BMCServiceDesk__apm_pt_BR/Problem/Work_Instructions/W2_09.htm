Work Instructions for Specialists

2.9.1

If you are not yet ready to start working on it, set the Status field to Accepted.

2.9.2

As soon as you are ready to start working on it, set the Status field to In Progress.

2.9.3

Examine the information in the Notes actions of the problem to find out how to implement the structural solution.

Note:

If the problem should be assigned to another specialist, ask the problem coordinator to change its assignment by setting its Status field to Rejected. Add a Notes action to specify why you rejected it.
