Work Instructions for Problem Coordinators

1.7.1

From the Remedyforce Console tab, click View > Problems. To open a new problem, click New.

1.7.2

Ensure that you are selected in the Owner field.

1.7.3

In the Problem Source field, select the appropriate source.

1.7.4

In the Category field, select the appropriate category.

1.7.5

In the Description field, enter a short description.

1.7.6

in the Impact and Urgency fields, select the appropriate impact and urgency. Click Save to save the problem.

1.7.7

Add a Notes action with a detailed description of the symptoms of the problem. In the Duration (HH:MM) field, in the Notes action, enter the time that you spent identifying and registering the problem.

1.7.8

To link the service affected by the problem, in the Linked Services and Configuration Items section, click Select and Link Services or Configuration Items to Problem.

1.7.9

To link the CI affected by the problem, in the Linked Services and Configuration Items section, click Select and Link Services or Configuration Items to Problem.

1.7.10

In the Linked Change Requests section, click Select and Link Change Requests to Problem. You can also link a previously reviewed request that was not significant, but had caused service disruption.
