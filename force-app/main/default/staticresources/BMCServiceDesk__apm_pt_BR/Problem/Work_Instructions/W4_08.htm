Work Instructions for Problem Coordinators

4.8.1

If a new knowledge article is submitted to you for approval, review and edit it to make the necessary modifications. Publish the article to make it available to your customer and/ or via the self service portal.

4.8.2

If an existing knowledge article needs to be reviewed, review the article and if it is not applicable anymore, remove it. Otherwise, read the attached feedback if applicable, review and edit it to make the necessary modifications.
