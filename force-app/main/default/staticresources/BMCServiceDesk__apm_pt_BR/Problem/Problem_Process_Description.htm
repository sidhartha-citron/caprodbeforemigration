Process description

The Problem Management process consists of four procedures.

  • Request Review is used by problem coordinators when they review requests to identify problems within the services for which they are responsible.

  • Root Cause Analysis is used by specialists when they analyze a problem.

  • Analysis Review is followed by problem coordinators when they review the results of a root cause analysis performed by a specialist.

  • Problem Closure is used by problem coordinators when they close out problems.
