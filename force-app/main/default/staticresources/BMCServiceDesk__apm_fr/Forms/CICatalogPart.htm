CI Catalog Part (Elément du catalogue des EC)


Le tableau ci-dessous répertorie les champs du formulaire Elément du catalogue des EC et propose des consignes de saisie pour chaque champ.

Page

Principal

Champ

Saisie

CI Information (Informations de l'EC)

Part name (Nom de l'élément)

Ce champ permet de saisir le nom de l'élément.

Brand (Marque)

Ce champ permet de saisir la marque de l'élément.

Manufacturer part # (N° de référence du fabricant)

Ce champ permet de saisir le numéro donné à l'élément par le fabricant.

Sous-catégorie

Ce champ permet de sélectionner la sous-catégorie appropriée d'élément de catalogue des EC parmi les options suivantes. Veuillez noter qu'il autorise uniquement la sélection des sous-catégories correspondant à la catégorie sélectionnée.

Les options suivantes sont disponibles pour la catégorie Adresse :

  • Adresse IP

  • Adresse MAC

Les options suivantes sont disponibles pour la catégorie Carte :

  • Carte LAN

  • Carte d'interface réseau

  • Carte à processeur

  • Carte vocale

  • WAN Card (Carte WAN)

  • Autre type de carte

Les options suivantes sont disponibles pour la catégorie Lecteur de disque :

  • CD-ROM Drive (Lecteur CD-ROM)

  • CD-ROM Writer (Graveur CD-ROM)

  • Lecteur DVD

  • DVD Writer (Graveur DVD)

  • Floppy Disk Drive (Lecteur de disquette)

  • Hard Disk Drive (Lecteur de disque dur)

  • Jaz Drive (Lecteur Jaz)

  • Zip Drive (Lecteur Zip)

  • Other Type of Disk Drive (Autre type de lecteur de disque)

Les options suivantes sont disponibles pour la catégorie Documentation :

  • Manuel d'installation

  • Process Document (Document de processus)

  • Manuel technique

  • Manuel utilisateur

  • Other Type of Document (Autre type de document)

Les options suivantes sont disponibles pour la catégorie Modem  :

  • Modem téléphonique analogique

  • Modem en bande de base

  • Modem DSL

  • Modem à fibre optique

  • Modem RNIS

Les options suivantes sont disponibles pour la catégorie Imprimante/multifonction :

  • Inkjet Printer (Imprimante à jet d'encre)

  • Label Printer (Imprimante d'étiquettes)

  • Laser Printer (Imprimante laser)

  • Matrix Printer (Imprimante matricielle)

  • Pen Plotter (Traceur)

  • Wax Printer (Imprimante à sublimation)

Les options suivantes sont disponibles pour la catégorie Routeur :

  • Routeur d'accès

  • Serveur d'accès distant

Les options suivantes sont disponibles pour la catégorie Logiciel :

  • Logiciel de sauvegarde/Récupération

  • Application de navigation/visionnage

  • Logiciel d'échange communication/données

  • Logiciel de compression

  • Application de gestion des contacts

  • Base de données

  • Application de planification des ressources/financière

  • Logiciel de (gestion du) réseau/système

  • Logiciel système d'exploitation

  • Logiciel de planification

  • Logiciel de sécurité

  • Logiciel/outil de développement logiciel

  • Suite logicielle

  • Application de productivité utilisateur

  • Serveur Web/moteur de recherche

  • Autre type de logiciel

Les options suivantes sont disponibles pour la catégorie Certificat de licence de logiciel :

  • Licence utilisateur coexistante

  • CPU-based License (Licence par processeur)

  • Licence utilisateur installée

  • Licence utilisateur nommée

  • Licence utilisateur temporaire

  • Licence utilisateur illimitée

Les options suivantes sont disponibles pour la catégorie Commutateur :

  • Commutateur de données

  • Commutateur téléphoniques (PBX)

  • Autre type de commutateur

Les options suivantes sont disponibles pour la catégorie Lecteur de bandes :

  • 1/4 inch (1/4 pouce)

  • 1/2 inch (1/2 pouce)

  • 4 mm (4 mm)

  • 8 mm (8 mm)

  • Autre type de lecteur de bandes

Les options suivantes sont disponibles pour la catégorie Terminal :

  • Text Terminal (Terminal texte)

  • X-terminal (Terminal X)

Les options suivantes sont disponibles pour la catégorie Poste de travail :

  • Ordinateur de bureau

  • Ordinateur portable

Parameters (Y/N) (Paramètres (O/N))

Track Asset/Tag # (Suivi du n° d'équipement/identifiant)

Sélectionnez cette option pour suivre le numéro de bien ou le numéro d'identification des EC enregistrés sous ce type d'élément du catalogue des EC.

Track Serial # (N° de série de suivi)

Sélectionnez cette option pour suivre le numéro de série des EC enregistrés sous ce type d'élément du catalogue des EC.

Track Version # (N° de version de suivi)

Sélectionnez cette option pour suivre le numéro de version des EC enregistrés sous ce type d'élément du catalogue des EC.

Quantité

Total

Ce champ affiche automatiquement le nombre d'EC uniques enregistrés sous ce type d'élément du catalogue des EC dans la CMDB.
