Etape de procédure

Instructions de travail pour les administrateurs de niveau de service

2.4.1

Saisissez les détails dans le modèle d'accord de niveau de service.

2.4.2

Imprimez deux exemplaires du modèle de SLA complété.

2.4.3

Imprimez deux exemplaires de l' élément de catalogue du nouveau SLA auquel le client s'est inscrit.

2.4.4

Joignez un exemplaire imprimé de l'élément de catalogue à chaque exemplaire imprimé du modèle de SLA complété afin de préparer deux exemplaires du nouveau SLA.
