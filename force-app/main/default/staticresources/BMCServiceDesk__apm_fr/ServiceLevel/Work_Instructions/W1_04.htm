Instructions de travail pour les administrateurs de niveau de service

1.4.1

Vous pouvez demander une approbation pour le nouvel élément de catalogue auprès des intervenants suivants :

  • le propriétaire du service responsable du respect des SLT de l'élément de catalogue ;

  • les gestionnaires de niveau de service des entreprises clientes pour lesquelles l'élément de catalogue a été préparé ;

  • le gestionnaire financier du prestataire de services.

Remarque :

obtenez les signatures sur une version papier du nouvel élément de catalogue.

Remarque :

si l'élément de catalogue n'a pas été préparé pour un client en particulier, il doit viser un certain type de client. Dans un tel cas, les gestionnaires de niveau de service doivent tout de même approuver le nouvel élément de catalogue.

1.4.2

Définissez le champ Statut de la tâche sur Waiting for... (En attente de...) et, dans la section Historique des actions, précisez que vous attendez l'approbation de l'élément de catalogue.

1.4.3

Dans le champ Durée (HH:MM) de l'action Remarques, saisissez le temps que vous avez consacré à la création de l'élément.
