Etape de procédure

Instructions de travail pour les administrateurs de niveau de service

3.2.1

Si un client n'est pas enregistré, utilisez le formulaire de client pour l'enregistrer.

3.2.2

Si une nouvelle entreprise cliente doit être enregistrée, utilisez le formulaire de compte. Utilisez le formulaire de client pour ajouter les informations relatives au représentant client de l'entreprise, ainsi que toute autre personne susceptible de bénéficier d'un service de support de la part du prestataire de service.
