Etape de procédure

Instructions de travail pour les administrateurs de configuration

5.7.1

Dans le champ Résultat de l'intervention, précisez les changements apportés dans la CMDB et/ou les informations du contrat de support.

Remarque :

si ni les informations de l' EC ni les informations du contrat de support ne doivent être mises à jour, précisez la raison pour laquelle la mise à jour demandée n'est pas nécessaire dans le champ Résultat de l'intervention.

5.7.2

Définissez le champ Statut de l'intervention sur Terminé.

5.7.3

Saisissez le temps (en minutes) que vous avez consacré à l'exécution de la tâche depuis la dernière mise à jour du champ Durée.
