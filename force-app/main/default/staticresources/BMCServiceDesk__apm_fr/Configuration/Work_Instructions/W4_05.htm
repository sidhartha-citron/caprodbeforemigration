Etape de procédure

Instructions de travail pour les administrateurs de configuration

4.5.1

Dans le champ Résolution de la tâche, précisez ce qui a été modifié dans les informations de la CMDB .

Remarque :

si les informations de l'EC ne nécessitent pas de mise à jour, précisez pourquoi la mise à jour n'est pas nécessaire dans le champ Résolution de la tâche.

4.5.2

Définissez le champ Statut de la tâche sur Terminé.

4.5.3

Dans le champ Durée (HH:MM) de l'action Remarques, saisissez le temps que vous avez consacré à l'exécution de la tâche.
