Etape de procédure

Instructions de travail pour les administrateurs de configuration

2.8.1

Enregistrez les informations sur le nouvel élément de configuration (EC). Si une demande d'achat est soumise pour le nouvel EC, celui-ci est déjà enregistré dans la CMDB avec le champ Statut défini sur Ordered (Commandé). Dans ce cas, mettez à jour le statut des EC et ajoutez les informations manquantes (par exemple, le numéro de série).

Si aucune demande d'achat formelle n'est soumise (par exemple, dans le cas d'un logiciel développé en interne), enregistrez le nouvel EC dans la CMDB.

Associez tout nouvel EC à d'autres EC, clients et comptes, selon le besoin. Pour cela, suivez les consignes de saisie relatives aux champs du formulaire d'élément de configuration.

2.8.2

Imprimez le code EC unique de chaque nouvel EC matériel sur une étiquette. Apposez chaque étiquette sur l'EC matériel correspondant de façon à ce qu'elle soit visible une fois l'EC installé.

2.8.3

Imprimez le code EC unique de chaque nouvel EC logiciel sur une étiquette. Apposez chaque étiquette sur la copie principale de l'EC logiciel correspondant de façon à ce qu'il soit visible une fois que la copie principale a été ajoutée dans la DML ; Ajoutez la copie principale de chaque nouvel EC logiciel dans la DML.

2.8.4

Imprimez le code EC unique de chaque nouveau certificat logiciel sur une étiquette. Apposez chaque étiquette sur le certificat correspondant de façon à ce qu'il soit facilement identifiable une fois qu'il a été ajouté dans la DML. Ajoutez chaque nouveau certificat de licence logicielle dans la DML.
