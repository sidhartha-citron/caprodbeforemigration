Etape de procédure

Instructions de travail pour les administrateurs de configuration

5.2.1

Si un nouveau contrat de support doit être enregistré, vérifiez si le fournisseur associé a déjà été enregistré dans l'application de gestion des services. Si tel est le cas, vérifiez si les coordonnées de ce fournisseur sont exactes et complètes.

5.2.2

Si un contrat de support doit être mis à jour, ouvrez-le et vérifiez si les coordonnées du fournisseur associé sont toujours exactes et complètes.
