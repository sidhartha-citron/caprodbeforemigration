Instructions de travail pour les administrateurs de configuration

1.7.1

Enregistrez les attributs connus des éléments de configuration (EC) commandés. Pour cela, suivez les consignes de saisie relatives aux champs du formulaire d' élément de configuration. Définissez le champ Statut des EC commandés sur Ordered (Commandé). N'associez pas immédiatement les EC commandés à un autre EC, service ou utilisateur.

La valeur commandée dans le champ Statut n'est peut-être pas disponible par défaut. Vous devez cocher la case Tâche pour le statut Ordered (Commandé).

1.7.2

Ajoutez une action Remarques dans la tâche pour indiquer que le bon de commande des nouveaux EC a été envoyé et que vous attendez leur livraison.

1.7.3

Définissez le champ Statut de la tâche sur Waiting for (En attente de).

1.7.4

Dans le champ Durée (HH:MM) d'une action Remarques, saisissez le temps que vous avez consacré à l'achat et l'enregistrement des EC.
