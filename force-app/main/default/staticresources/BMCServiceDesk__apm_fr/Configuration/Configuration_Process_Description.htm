Description du processus


Le processus de gestion des configurations est constitué des procédures suivantes :

  • Demande d'achat d'EC : procédure utilisée par les administrateurs de configuration lorsque de nouveaux éléments de configuration (EC) doivent être commandés.

  • Reçu d'EC : procédure utilisée lorsque de nouveaux EC ont été livrés.

  • Maintenance des informations de fournisseur : procédure utilisée par les administrateurs de configuration lorsqu'ils enregistrent de nouveaux fournisseurs et qu'ils mettent à jour les coordonnées de fournisseurs enregistrés antérieurement.

  • Mise à jour d'EC : procédure utilisée par les administrateurs de configuration lorsqu'ils mettent à jour les attributs et relations d'EC déjà enregistrés dans la CMDB.

Pour obtenir des informations sur ces procédures, cliquez sur Processus pour revenir à la représentation graphique de ce processus et cliquez sur les encadrés représentant les procédures. La représentation graphique de la procédure s'affiche avec sa description dans le volet de droite.
