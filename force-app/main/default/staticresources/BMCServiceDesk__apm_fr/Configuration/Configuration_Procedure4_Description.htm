Procédure 4 - Mise à jour d'EC


Si la tâche envoyée par la gestion des changements ou la gestion des demandes ne requiert pas la mise à jour d'informations d'éléments de configuration (EC) existantes, l'administrateur de configuration termine la tâche. Cependant, si l'administrateur de configuration traite une tâche demandant la mise à jour d'attributs et de relations d'EC, il doit vérifier la configuration effective de l'infrastructure pour s'assurer que les modifications demandées sont vraiment requises pour mettre à jour la CMDB .

Si la CMDB doit être mise à jour, l'administrateur de configuration effectuera la mise à jour des attributs et relations d'EC requis.
