Procédure 2 - Reçu d'EC


Si la tâche envoyée par la gestion des changements ou la gestion des demandes ne requiert pas l'ajout de nouvel EC dans la CMDB, l'administrateur de configuration passe directement à la procédure 4 - Mise à jour d'EC.

Si, en revanche, la tâche requiert l'enregistrement d'un ou plusieurs EC pour lesquels aucune demande d'achat formelle n'a été soumise (par exemple, dans le cas d'un logiciel développé en interne), ou si un ou plusieurs EC ont été commandés, l'administrateur de configuration vérifie d'abord les EC, après leur livraison. Il procède ainsi afin de s'assurer que le matériel, les logiciels et les certificats de licence logicielle adéquats ont bien été reçus, qu'aucun élément n'est manquant et que les EC ne sont pas endommagés. Si la livraison est incomplète, l'administrateur de configuration informe le fournisseur puis prévient le coordinateur de changement du retard en mettant à jour la tâche.

A l'inverse, si les EC ont été reçus dans un état satisfaisant, l'administrateur de configuration vérifie si les coordonnées du fournisseur sont à jour dans l'application de gestion des services. Si le fournisseur des EC n'est pas déjà enregistré ou que ses coordonnées ne sont plus à jour, l'administrateur de configuration s'assure que les informations de fournisseur sont enregistrées ou mises à jour lors de la procédure 3 - Maintenance des informations de fournisseur.

Après avoir mis à jour ou enregistré les informations de fournisseur, l'administrateur de configuration met à jour le statut des nouveaux EC et ajoute toute information manquante (par exemple, le numéro de série) si les EC ont déjà été enregistrés lors de la procédure 1 - Demande d'achat d'EC. Si, en revanche, aucune demande d'achat formelle n'est soumise pour les nouveaux EC, l'administrateur de configuration enregistre les nouveaux EC dans la CMDB. Il s'assure que tous les attributs des nouveaux EC ainsi que les liens entre ces derniers et les autres EC, services et utilisateurs sont correctement enregistrés.

Après avoir vérifié que les nouveaux EC sont correctement enregistrés dans la CMDB, il joint à chacun d'entre eux un intitulé mentionnant le code EC, le cas échéant.
