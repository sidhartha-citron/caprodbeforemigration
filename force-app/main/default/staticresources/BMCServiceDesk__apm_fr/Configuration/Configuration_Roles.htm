Rôles et responsabilités


Le tableau ci-dessous montre le seul rôle impliqué dans le processus de gestion des configurations, ainsi que ses responsabilités. Cliquez sur un rôle pour afficher son profil.

Rôle

Responsabilité

Administrateur de configuration

  • Soumettre des demandes d'achat d'EC commandés.

  • S'assurer que les EC, placés sous la responsabilité du groupe de l'administrateur de configuration, sont bien référencés suite à leur livraison.

  • Maintenir à jour les informations des EC dont le groupe de l'administrateur de configuration est responsable.

  • Gérer les informations d'organisations externes qui ont fourni ou assurent le support des EC dont le groupe de l'administrateur de configuration est responsable.

  • Enregistrer et mettre à jour les contrats de support pour les EC dont le groupe de l'administrateur de configuration est responsable.

  • Gérer les liens des EC dont le groupe de l'administrateur de configuration est responsable. Il s'agit des liens entre un EC et d'autres EC, son fournisseur, son contrat de support et les instances de service dont il fait partie.
