Aide du modèle de processus - Boutons


Outre les procédures accessibles à partir du flux de processus, bien d'autres d'informations peuvent être affichées après être entré dans un processus du modèle de processus. Pour cela, il suffit de cliquer sur les boutons à gauche de votre écran. Chaque processus dispose des mêmes boutons. Ces boutons sont répertoriés ci-dessous, accompagnés d'une description des informations qui s'affichent une fois que vous avez cliqué dessus.

Bouton

Description

Cliquez sur ce bouton pour accéder à une représentation graphique du flux de processus. La page par défaut d'accès à un processus s'affiche.

Cliquez sur ce bouton pour obtenir une description écrite du processus au lieu de sa représentation graphique.

Cliquez sur ce bouton pour accéder à la mission du processus.

Cliquez sur ce bouton pour accéder au champ d'application du processus.

Cliquez sur ce bouton pour afficher les responsabilités des différents rôles impliqués dans le processus.

Cliquez sur ce bouton pour obtenir la liste des formulaires disponibles dans l'application de gestion des services prenant en charge ce processus.

Cliquez sur ce bouton pour télécharger la version Microsoft Word de la documentation relative au processus.

Cliquez sur ce bouton pour obtenir la liste des termes et abréviations ainsi que leurs définitions. Le glossaire n'est pas spécifique au processus.

Cliquez sur ce bouton pour revenir au modèle de processus.

Cliquez sur ce bouton pour obtenir une présentation rapide de l'utilisation d'un modèle de processus.


Lorsque vous accédez à une procédure à partir d'un processus, vous pouvez explorer les instructions de travail pour chaque étape de la procédure. Vous pouvez également utiliser les boutons à gauche de votre écran pour obtenir davantage d'informations. Ces boutons sont répertoriés ci-dessous, accompagnés d'une description des informations qui s'affichent une fois que vous avez cliqué dessus.

Bouton

Description

Cliquez sur ce bouton pour accéder à une représentation graphique du flux de procédures. La page par défaut d'accès à une procédure s'affiche.

Cliquez sur ce bouton pour obtenir une description écrite de la procédure au lieu de sa représentation graphique.

Cliquez sur ce bouton pour afficher les responsabilités des différents rôles impliqués dans la procédure du processus.

Cliquez sur ce bouton pour obtenir la liste des formulaires disponibles dans l'application de gestion des services prenant en charge la procédure du processus.

Cliquez sur ce bouton pour obtenir la liste des termes et abréviations ainsi que leurs définitions. Le glossaire n'est pas spécifique au processus ni à la procédure.

Cliquez sur ce bouton pour revenir au processus.

Cliquez sur ce bouton pour revenir au modèle de processus sans passer par le processus.

Cliquez sur ce bouton pour obtenir une présentation rapide de l'utilisation d'un modèle de processus.
