Aide du modèle de processus - Symboles


Les processus et procédures du modèle de processus Alignability® ont été conçus selon une convention simplifiée reposant sur trois symboles : cercle (Evénement), carré (Action) et losange (Décision). La signification de ces trois symboles est expliquée ci-dessous.

Symbole

Description

Un carré (Action) représente une action à réaliser.
Il est toujours précédé d'une flèche.
Lorsqu'une autre action est requise ensuite, une autre flèche part du carré.

Un losange (Décision) représente une décision à prendre.
Il est toujours précédé d'une flèche.
Deux flèches partent toujours d'un losange.

Un cercle (Evénement) représente un événement ou une situation.
Il ne peut pas être précédé par une flèche.
Une flèche part toujours d'un cercle.
