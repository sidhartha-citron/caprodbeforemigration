Description du processus

Le processus de gestion des demandes est composé de huit procédures de traitement des demandes.

  • L'enregistrement de demande est utilisé par les analystes du centre de services lorsqu'ils enregistrent des demandes des utilisateurs.

  • L'affectation de demande est utilisée par les analystes du centre de services lorsqu'ils affectent des demandes à des groupes ou au gestionnaire du centre de services en cas de plaintes ou d'éloges. Les analystes du centre de services utilisent également cette procédure lorsqu'ils traitent des demandes refusées par les coordinateurs de groupe. Les coordinateurs de groupe utilisent cette procédure lorsque des demandes ont été affectées à leurs groupes. Les coordinateurs de groupe utilisent cette procédure pour affecter ces demandes à un coordinateur de changement ou à un spécialiste spécifique pour résolution.

  • La gestion des éloges et des plaintes est utilisée par le gestionnaire du centre de services pour traiter de manière satisfaisante des demandes de type plainte ou éloge.

  • Le suivi des demandes est utilisé par les analystes du centre de services lorsqu'ils suivent l'évolution des demandes.

  • La résolution de demande par un spécialiste est utilisée par les spécialistes pour traiter les demandes qui leur sont affectées.

  • La gestion de l'exécution des demandes est utilisée par les analystes du centre de services ou les spécialistes pour exécuter les demandes pouvant être traitées à l'aide d'un workflow prédéfini.

  • La gestion des escalades d'incident est utilisée par les propriétaires de service lorsqu'un incident est escaladé. Le propriétaire du service affecté utilise cette procédure pour déterminer comment résoudre efficacement l'incident.

  • La clôture de demande est utilisée par les analystes du centre de services lorsqu'ils résolvent et ferment les demandes terminées.

Pour obtenir des informations sur ces procédures, cliquez sur Processus pour revenir à la vue graphique de ce processus et cliquez sur les encadrés représentant les procédures. La représentation graphique de la procédure s'affiche avec sa description dans le volet de droite.
