Procédure 3 - Gestion des plaintes et des éloges


Une fois une plainte ou un éloge enregistré, le gestionnaire du centre de services en examine les détails.

Dans le cas d'une plainte, le gestionnaire du centre de services échange avec les analystes du centre de services, les spécialistes, le gestionnaire de niveau de service, le représentant client et l'utilisateur impliqués pour déterminer la cause de la plainte. Une fois la cause définie, le gestionnaire du centre de services décide de la méthode à appliquer pour rétablir la confiance de l'utilisateur dans le support fourni par le prestataire de services.

Dans le cas d'un éloge, le gestionnaire du centre de services informe l'employé ou le département ayant reçu l'éloge.

Une fois que le gestionnaire du centre de services a appliqué les mesures nécessaires pour rassurer l'utilisateur ou transmettre l'éloge, il passe la demande au statut Terminé.
