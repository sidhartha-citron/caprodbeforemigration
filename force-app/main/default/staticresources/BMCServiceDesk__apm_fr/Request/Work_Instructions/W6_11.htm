Etape de procédure

Instructions de travail pour les coordinateurs de groupe

6.11.1

Dans le champ Durée (HH:MM) de l'action Remarques, saisissez le temps que vous avez consacré à l'examen de la tâche.

6.11.2

Utilisez l'icône de recherche située à côté du champ Propriétaire pour sélectionner le spécialiste le plus approprié (en termes de compétences, de disponibilité et de droits d'accès) dans le champ Propriétaire.
