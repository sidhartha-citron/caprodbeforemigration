Etape de procédure

Instructions de travail pour les analystes du centre de services

1.12.1

Lisez les instructions dans le champ Description de la nouvelle demande pour déterminer les informations à recueillir auprès de l'utilisateur et définir comment résoudre la demande.

1.12.2

Recueillez les informations requises auprès de l'utilisateur et ajoutez-les à la demande conformément aux instructions du champ Description.

1.12.3

Enregistrez la demande.

Si la définition de demande est sélectionnée, à savoir si le type de demande est défini sur Demande de service, les interventions correspondantes sont générées et une fenêtre contextuelle s'affiche, dans laquelle vous devez répondre aux questions relatives à la demande de service.
