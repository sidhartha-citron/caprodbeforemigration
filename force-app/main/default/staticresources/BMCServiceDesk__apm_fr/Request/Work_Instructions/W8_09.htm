Etape de procédure

Instructions de travail pour les coordinateurs de problème

8.8.1

Si un nouvel article de connaissance vous est envoyé pour approbation, examinez-le, cliquez sur Modifier et effectuez les modifications nécessaires. Cliquez sur Publier pour le rendre accessible aux clients via Libre service, ainsi qu'au personnel de support.

8.8.2

Si un article de connaissance existant nécessite une révision, révisez-le. Si l'article n'est pas applicable, cliquez sur Retirer. Dans le cas contraire, lisez les commentaires, cliquez sur Modifier et apportez les modifications nécessaires. Une fois les modifications effectuées, enregistrez l'article.
