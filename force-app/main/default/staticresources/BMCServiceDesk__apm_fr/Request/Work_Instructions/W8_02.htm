Etape de procédure

Instructions de travail pour les analystes du centre de services

8.2.1

Décrivez comment la demande est résolue dans le champ Résolution .

8.2.2

Sélectionnez l'élément de configuration (EC) approprié dans le champ EC s'il n'est pas déjà renseigné ou si l'EC sélectionné n'est pas correct.

Remarque :

si l'EC n'est pas enregistré dans la CMDB, mais qu'il fait partie du processus de gestion des configurations, sélectionnez l'EC ayant le code NORECORD.

L'EC présentant le code NORECORD n'est peut-être pas disponible par défaut.

Remarque :

si la demande ne concerne pas un EC spécifique ou que l'EC ne fait pas partie du processus de gestion des configurations, sélectionnez l'EC ayant le code N/A.

8.2.3

Dans le champ Durée (HH:MM) de l'action Remarques, saisissez le temps que vous avez consacré à l'enregistrement et à la résolution de la demande.

8.2.4

Sélectionnez la catégorie de clôture appropriée dans le champ Catégorie de clôture. Définissez le champ Statut de la demande sur Terminé.

8.2.5

Assurez-vous d'être bien sélectionné dans le champ Propriétaire.

8.2.6

Vérifiez une dernière fois que la demande est remplie correctement.

8.2.7

Si la solution de contournement d'un problème ouvert est utilisée pour résoudre la demande, associez l'incident au problème.
