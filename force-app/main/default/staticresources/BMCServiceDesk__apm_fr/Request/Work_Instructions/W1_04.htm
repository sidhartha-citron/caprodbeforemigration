Instructions de travail pour les analystes du centre de services

1.4.1

Recherchez la demande pour laquelle l'utilisateur a contacté le centre de services.

Remarque :

pour obtenir une vue d'ensemble des demandes ouvertes pour l'utilisateur, sélectionnez l'option Outils d'agent > Incidents for Customer (Incidents pour le client).

1.4.2

Examinez les détails, l'évolution et le statut de la demande.
