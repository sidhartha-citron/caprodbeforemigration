Instructions de travail pour les analystes du centre de services

8.1.1

Résolvez la demande. Si nécessaire, demandez de l'aide à l'utilisateur et à vos collègues du centre de services.

Remarque :

si un modèle est utilisé pour enregistrer la demande, les informations contenues dans la demande peuvent préciser comment elle doit être résolue.

Remarque :

si la demande est due à un problème ouvert proposant une solution de contournement dans son champ Solution de contournement, suivez les instructions pour la mettre en œuvre.
