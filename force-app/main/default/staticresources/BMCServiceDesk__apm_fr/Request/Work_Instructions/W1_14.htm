Etape de procédure

Instructions de travail pour les analystes du centre de services

1.14.1

Si type de la demande est Incident, recherchez les demandes ouvertes pour le même service et utilisez les informations fournies par l'utilisateur pour déterminer à quel point le service est affecté. Sélectionnez le niveau d'impact approprié dans le champ Impact . Les options suivantes sont disponibles :

  • Faible

-

Service détérioré pour un utilisateur

  • Moyen

-

Service interrompu pour un utilisateur ou détérioré pour plusieurs utilisateurs

  • Elevé

-

Service indisponible pour plusieurs utilisateurs

  • Majeur

-

Service critique ou central indisponible pour plusieurs utilisateurs Ce niveau d'impact n'est peut-être pas disponible par défaut.

Sélectionnez le niveau d'urgence approprié dans le champ Urgence. Les options suivantes sont disponibles :

  • Faible

-

Responding using standard operating procedures and as time allows (Réponse dans le cadre des procédures d'exploitation standard et selon le temps disponible)

  • Moyen

-

Responding using standard procedures and operating within normal supervisory management structures (Réponse dans le cadre des procédures standard et des structures normales de supervision)

  • Elevé

-

Immediate response required (Réponse immédiate requise)


Remarque :

si le type de demande est Incident, ne sélectionnez pas le niveau d'impact None - No Degradation of Service (Aucun - Aucune détérioration de service).

Ce niveau d'impact n'est peut-être pas disponible par défaut.

Remarque :

si le type de demande n'est pas Incident, définissez le niveau d'impact sur None - No Degradation of Service (Aucun - Aucune détérioration de service).

Ce niveau d'impact n'est peut-être pas disponible par défaut.

Remarque :

l'application de gestion des services détermine automatiquement la priorité des demandes à l'aide du tableau de priorité standard ou du SLA du client . Après avoir déterminé la priorité appropriée de la demande, l'application de gestion des services l'utilise pour calculer la date et l'heure d'échéance.
