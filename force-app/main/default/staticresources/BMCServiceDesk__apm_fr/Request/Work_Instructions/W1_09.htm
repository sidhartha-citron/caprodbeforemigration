Instructions de travail pour les analystes du centre de services

1.9.1

Sélectionnez l'incident majeur lié dans le champ Diffusion.

Vous pouvez lier un incident existant à l'aide de l'option Outils d'agent > Incidents correspondants.

1.9.2

Complétez les autres champs requis et enregistrez la demande.

Remarque :

lorsqu'une demande est liée à un incident majeur, elle ne doit pas être modifiée directement. Les mises à jour de l'incident majeur sont copiées dans toutes les demandes enfant.

Une fois la diffusion sélectionnée et l'incident enregistré, les champs suivants sont remplis dans l'incident :

  • Description

  • EC

  • Impact

  • Priorité

  • Date de fin

  • Date d'échéance

  • Statut

  • Résolution
