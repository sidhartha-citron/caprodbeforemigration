Etape de procédure

Instructions de travail pour les analystes du centre de services

8.3.1

Si les informations incluses dans le champ Résolution ont été saisies par un spécialiste, assurez-vous qu'elles sont significatives (elles doivent par exemple expliquer comment la solution est implémentée). Sinon, contactez le spécialiste et étudiez avec lui comment améliorer les informations. Corrigez les éventuelles erreurs d'orthographe et de grammaire présentes dans le champ.

8.3.2

Informez l'utilisateur de la manière dont la demande a été exécutée et demandez-lui de vérifier cette solution. Vous pouvez le prévenir par téléphone pour qu'il vérifie la solution. Pour l'informer, vous pouvez également définir le champ Statut sur ClosedMail (E-MailFermeture) après avoir saisi le temps (en minutes) que vous avez consacré à l'examen final de la demande de support dans le champ Durée (HH:MM) de l'action Remarques.

Remarque :

lorsque la demande est enregistrée avec le statut ClosedMail (E-MailFermeture), un e-mail peut être automatiquement envoyé à l'utilisateur. Cet e-mail prévient l'utilisateur que la demande est terminée et lui demande de vérifier la solution. Il invite également l'utilisateur à contacter le centre de services si la solution ne le satisfait pas.
