Etape de procédure

Instructions de travail pour les analystes du centre de services

1.11.1

Sélectionnez le modèle de demande approprié dans le champ Modèle . L'application de gestion des services utilise les informations du modèle pour remplir les champs de la nouvelle demande.

1.11.2

Adaptez le texte dans le champ Description et ajoutez une action Remarques pour fournir toute information supplémentaire pouvant aider à résoudre la demande. Cela n'est possible qu'une fois la demande enregistrée.

1.11.3

Remplissez les champs Type de contact, Impact et Urgence si le modèle ne le fait pas.
