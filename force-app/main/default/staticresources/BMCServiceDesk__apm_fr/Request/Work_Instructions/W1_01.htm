Etape de procédure

Instructions de travail pour les analystes du centre de services

1.1.1

Ouvrez une nouvelle demande en sélectionnant l'onglet Console des incidents et en cliquant sur Nouveau.

1.1.2

Sélectionnez le type d'ID de client approprié (utilisateur du portail client pour les clients et utilisateur si le demandeur fait partie du personnel de support).

Sélectionnez l'utilisateur dans le champ ID de client . Pour ce faire, choisissez la recherche par ID de client et saisissez le nom de famille ou le prénom dans le champ Rechercher. Si les caractères saisis dans le champ Rechercher sont récurrents, l'application de gestion des services affiche une liste d'utilisateurs dont le nom de famille ou le prénom commence par ces caractères. Sélectionnez l'utilisateur dans la liste.

Remarque :

si l'utilisateur n'est pas enregistré, sélectionnez l'enregistrement d'utilisateur NORECORD et saisissez le nom et les coordonnées de l'utilisateur dans le champ Description. Cet enregistrement n'est peut-être pas disponible par défaut.
