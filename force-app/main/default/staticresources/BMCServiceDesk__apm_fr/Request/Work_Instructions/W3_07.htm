Etape de procédure

Instructions de travail pour les gestionnaires du centre de service

3.7.1

Décrivez comment la plainte a été exécutée dans le champ Résolution de la demande.

3.7.2

Vérifiez que le service N/A - Service does not (yet) exist or is out of scope (Le service n'existe pas (encore) ou est hors du champ d'application) a bien été sélectionné dans le champ Service. Ce service n'est peut-être pas disponible par défaut.

3.7.3

Assurez-vous que l'EC spécifique ayant le code N/A est sélectionné dans le champ EC.

3.7.4

Sélectionnez la catégorie de clôture appropriée dans le champ Catégorie de clôture et définissez le champ Statut de la plainte sur Terminé.

3.7.5

Dans le champ Durée (HH:MM) de l'action Remarques, saisissez le temps que vous avez consacré au traitement de la plainte ou de l'éloge.

3.7.6

Vérifiez une dernière fois que la demande de support est remplie correctement.
