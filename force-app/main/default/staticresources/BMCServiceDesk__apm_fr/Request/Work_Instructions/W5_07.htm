Instructions de travail

Etape de procédure

Instructions de travail pour les spécialistes

5.7.1

Résolvez la demande. Si nécessaire, demandez de l'aide auprès des fournisseurs externes.

Remarque :

si la demande ne peut être résolue avant la date et l'heure spécifiées dans le champ Date d'échéance, faites une estimation réaliste du moment où elle pourra être résolue et ajoutez une action Remarques pour saisir cette information dans la demande.

Remarque :

Définissez le champ Statut sur Waiting for (En attente de) ou Waiting for customer (En attente client) si le traitement ne peut pas se poursuivre, si vous attendez une pièce de rechange ou des informations de la part d'un client, par exemple. Ajoutez une action Remarques et spécifiez l'objet de l'attente. Si vous avez enregistré une demande auprès d'un fournisseur, saisissez le nom de ce fournisseur et le numéro de demande qu'il vous a fourni dans une action Remarques.

Remarque :

si vous avez fini votre travail sur la demande mais que des opérations supplémentaires sont nécessaires pour la résoudre, ajoutez une action Remarques pour indiquer ce que vous avez fait. Dans le champ Durée (HH:MM) de l'action Remarques, saisissez le temps que vous avez consacré au traitement de la demande. Si un membre du personnel particulier de votre groupe doit effectuer des opérations supplémentaires, utilisez l'icône de recherche située à côté du champ Propriétaire pour le sélectionner dans le champ Propriétaire. Si un autre groupe doit effectuer des opérations supplémentaires, utilisez l'icône de recherche située à côté du champ Propriétaire pour sélectionner le groupe dans le champ Propriétaire. Assurez-vous que le champ Statut est défini sur Assigned (Affecté).
