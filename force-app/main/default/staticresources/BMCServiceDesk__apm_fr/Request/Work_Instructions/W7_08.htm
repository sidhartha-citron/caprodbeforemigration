Etape de procédure

Instructions de travail pour les propriétaires de service

7.8.1

Examinez avec les spécialistes l'implémentation des changements destinés à résoudre l'incident. Identifiez les risques susceptibles d'entraîner un échec de l'implémentation et l'impact de celle-ci sur les clients.

7.8.2

Examinez avec les spécialistes les possibilités de retour arrière et comment limiter à un niveau acceptable les risques et l'impact de l'implémentation du changement.

7.8.3

Si le service affecté doit faire l'objet d'une reprise sur une infrastructure différente sur le site de production, déterminez si un plan de continuité est disponible. Si c'est le cas, sélectionnez les listes de contrôle du plan devant être suivi par les spécialistes pendant l'implémentation du changement afin de minimiser les risques d'échec.
