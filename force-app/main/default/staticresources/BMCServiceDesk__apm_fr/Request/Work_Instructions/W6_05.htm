Etape de procédure

Instructions de travail pour les analystes du centre de services / les spécialistes

6.5.1

Pour chaque tâche, déterminez le personnel de votre groupe le plus apte (en fonction des compétences, des disponibilités et des droits d'accès) à exécuter la tâche. Utilisez l'icône de recherche située à côté du champ Propriétaire pour sélectionner le membre du personnel ou le groupe adéquat dans le champ Propriétaire de la tâche.

Si vous pouvez exécuter la tâche, utilisez l'icône de recherche située à côté du champ Propriétaire pour vous sélectionner dans le champ Propriétaire de la tâche.

Si un autre groupe doit exécuter une tâche, utilisez l'icône de recherche située à côté du champ Propriétaire pour sélectionner le groupe dans le champ Propriétaire.

Si une tâche d'implémentation doit être exécutée par une organisation externe, examinez-la avec votre interlocuteur dans cette organisation. Déterminez avec cette personne un objectif d'implémentation réaliste. Utilisez l'icône de recherche située à côté du champ Propriétaire pour vous sélectionner dans le champ Propriétaire de la tâche. Ajoutez une action Remarques pour indiquer l'organisation externe chargée de traiter la tâche. Lorsque le traitement démarre, informez le fournisseur et, dans l'action Remarques, saisissez le numéro de référence sous lequel le fournisseur a enregistré cette affectation.

Remarque :

affectez uniquement les tâches ne nécessitant pas que d'autres tâches soient terminées au préalable.
