Description du processus


Le processus de gestion des connaissances est constitué des procédures suivantes :

  • La création de connaissances est utilisée par les spécialistes lors de la création de nouveaux articles de connaissance.

  • L'approbation des connaissances est utilisée par les responsables des connaissances lorsque des articles de connaissance doivent être approuvés.

  • La révision des connaissances est utilisée par les responsables des connaissances pour la révision d'articles existants.

Pour obtenir des informations sur ces procédures, cliquez sur Processus pour revenir à la représentation graphique et cliquez sur les encadrés représentant les procédures. La représentation graphique de la procédure s'affiche avec sa description dans le volet de droite.
