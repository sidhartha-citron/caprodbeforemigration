Etape de procédure

Instructions de travail pour les coordinateurs de problème

1.7.1

Pour ouvrir un nouveau problème, accédez à l'onglet Problèmes et cliquez sur Nouveau.

1.7.2

Assurez-vous d'être bien sélectionné dans le champ Propriétaire.

1.7.3

Dans le champ Source du problème, sélectionnez la source appropriée.

1.7.4

Dans le champ Catégorie, sélectionnez la catégorie appropriée.

1.7.5

Dans le champ Description, saisissez une brève description du problème.

1.7.6

Dans les champs Impact et Urgence, sélectionnez les niveaux d'impact et d'urgence appropriés. Cliquez sur Enregistrer pour enregistrer le problème.

1.7.7

Ajoutez une action Remarques et saisissez-y une description détaillée des symptômes du problème. Dans le champ Durée (HH:MM) de l'action Remarques, saisissez le temps que vous avez consacré à l'identification et l'enregistrement du problème.

1.7.8

Pour lier le service affecté par le problème, dans la section EC et services associés, cliquez sur Nouveau lien.

1.7.9

Pour lier l'EC affecté par le problème, dans la section EC et services associés, cliquez sur Nouveau lien.

1.7.10

Dans la section Demandes de changement liées, cliquez sur Lier des demandes de changement au problème. Vous pouvez également lier une demande précédente qui n'a pas été définie comme importante mais a causé une interruption de service.
