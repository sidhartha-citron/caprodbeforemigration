Etape de procédure

Instructions de travail pour les coordinateurs de problème

4.6.1

Définissez le champ Statut du problème sur Dead-End (Impasse) pour indiquer qu'aucune action supplémentaire n'est requise.

Remarque :

si le champ Statut du problème est déjà défini sur Dead-End (Impasse) car vous avez simplement procédé à une recherche périodique de nouvelles possibilités, indiquez-le dans l'action Remarques.

4.6.2

Dans le champ Catégorie de clôture, sélectionnez la catégorie appropriée.

4.6.3

Dans le champ Durée (HH:MM) de l'action Remarques, saisissez le temps que vous avez consacré à l'examen du problème.

4.6.4

Lisez l'article de connaissance et modifiez-le si nécessaire en y ajoutant les informations les plus récentes.
