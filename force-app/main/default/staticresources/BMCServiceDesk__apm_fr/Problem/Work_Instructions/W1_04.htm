Instructions de travail pour les coordinateurs de problème

1.4.1

Si la demande est importante, passez à l'étape 1.5.1. Si elle ne l'est pas, revenez à l'étape 1.2.1 pour examiner les détails de la demande suivante.

Remarque :

une demande est considérée comme importante si :

  • le niveau d'impact est défini sur Elevé ou Majeur ;

  • elle est survenue plusieurs fois ;

  • elle risque de se reproduire.
