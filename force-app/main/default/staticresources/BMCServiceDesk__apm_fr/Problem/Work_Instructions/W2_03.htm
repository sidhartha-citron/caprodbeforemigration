Etape de procédure

Instructions de travail pour les spécialistes

2.3.1

Essayez de trouver une solution de contournement temporaire permettant de résoudre les incidents provoqués par le problème. Vérifiez le champ Résolution des demandes associées au problème.

2.3.2

Si vous pouvez proposer une solution de contournement, décrivez-la (étape par étape) dans un nouvel article de connaissance et ajoutez un lien à cet article dans le champ Solution de contournement du problème.

Remarque :

les analystes et/ou spécialistes du centre de services peuvent utiliser cette solution de contournement pour résoudre les incidents provoqués par le problème jusqu'à ce qu'une solution structurelle soit proposée.

Remarque :

si une solution de contournement a déjà été spécifiée dans le champ Solution de contournement, prenez-en connaissance et améliorez-la si possible.

2.3.3

Si aucune solution de contournement concrète ne peut être proposée, indiquez-en les raisons dans le champ Solution de contournement.
