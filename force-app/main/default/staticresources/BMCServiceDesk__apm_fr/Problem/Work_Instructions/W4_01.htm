Etape de procédure

Instructions de travail pour les coordinateurs de problème

4.1.1

Examinez les informations récemment fournies dans les actions Remarques du problème.

4.1.2

Si une solution a été implémentée, vérifiez-la (en examinant les résultats du test de production, par exemple).

Remarque :

si vous ne disposez pas des droits d'accès vous permettant de vérifier la solution structurelle, vérifiez-la avec un utilisateur doté de ces droits d'accès.

Remarque :

si vous n'avez pas pu reproduire les symptômes des incidents provoqués par le problème avant l'implémentation de la solution, la seule façon de vérifier la solution est de contrôler les demandes de service afin de vous assurer que le problème ne cause aucun autre incident.
