Instructions de travail pour les coordinateurs de problème

1.8.1

Déterminez le spécialiste le plus à même de trouver la cause fondamentale et de proposer une solution structurelle. Utilisez l'option Changement à côté du champ Propriétaire pour sélectionner ce spécialiste dans le champ Propriétaire du problème.

Remarque :

si le problème est lié à un EC et que cet EC fait partie d'une instance de service intervenant dans le support du service dont vous êtes le coordinateur de problèmes, contactez le coordinateur de problèmes de ce service de support. Discutez du problème et, si l'autre coordinateur de problèmes et vous-même êtes d'accord, transférez-lui la responsabilité du problème en modifiant le nom indiqué dans le champ Propriétaire. Résumez les raisons de ce transfert de responsabilité dans le champ Informations.

1.8.2

Revenez à l'étape 1.2.1 pour examiner les détails de la prochaine demande de support.
