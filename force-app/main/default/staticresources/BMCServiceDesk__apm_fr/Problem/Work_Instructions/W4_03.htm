Etape de procédure

Instructions de travail pour les coordinateurs de problème

4.3.1

Dans le champ Résolution résumez la procédure d'implémentation de la solution structurelle.

4.3.2

Dans le champ Durée (HH:MM) de l'action Remarques, saisissez le temps que vous avez consacré à la vérification de la solution structurelle.

4.3.3

Définissez le champ Statut sur Solved (Résolu).

4.3.4

Dans le champ Catégorie de clôture, sélectionnez une catégorie appropriée.

Remarque :

si la solution a été fournie dans le champ Résolution du problème, le centre de services peut en être informé par e-mail.

4.3.5

Si la cause fondamentale a été supprimée et que l'article de connaissance créé pour le problème n'est pas nécessaire, supprimez-le de la base de connaissances.

4.3.6

Créez un article de connaissance décrivant la solution dans la base de connaissances, si nécessaire.
