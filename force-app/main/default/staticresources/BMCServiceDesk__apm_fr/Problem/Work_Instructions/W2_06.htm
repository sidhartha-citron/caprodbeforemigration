Etape de procédure

Instructions de travail pour les spécialistes

2.6.1

Maintenant que vous connaissez la cause fondamentale, recherchez un moyen de la supprimer ou de fournir une solution de contournement permanente.

Remarque :

si l'élément de configuration (EC) à l'origine du problème est toujours pris en charge par le support du fournisseur externe (c'est-à-dire si l'EC est toujours couvert par une garantie ou un contrat de support), vous pouvez envoyer une demande au fournisseur pour obtenir une solution au problème.

Pour savoir si le fournisseur propose toujours un support pour l'EC, ouvrez l'EC et cliquez sur l'onglet Caractéristiques pour afficher le champ Warranty expiry (Expiration de la garantie).

2.6.2

Ajoutez une action Remarques pour décrire chaque solution structurelle proposée.

2.6.3

Déterminez les meilleures solutions structurelles proposées (en fonction de critères techniques, financiers et de disponibilité). Ajoutez une action Remarques pour indiquer de manière argumentée quelle solution structurelle vous semble la plus adaptée.

2.6.4

Ajoutez une action Remarques pour décrire de façon détaillée la procédure d'implémentation de la solution structurelle la plus adaptée. Si une solution de contournement précédente doit être supprimée, spécifiez la procédure de suppression dans le même champ.

Remarque :

si aucune solution structurelle n'a été trouvée, ajoutez une action Remarques et indiquez-en les raisons.
