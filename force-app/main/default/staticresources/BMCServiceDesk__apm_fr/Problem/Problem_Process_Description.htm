Description du processus

Le processus de gestion des problèmes est constitué de quatre procédures.

  • La révision de demande est utilisée par les coordinateurs de problème lorsqu'ils examinent des demandes afin d'identifier les problèmes au sein des services dont ils sont responsables.

  • L'analyse de la cause fondamentale est utilisée par les spécialistes lors de l'analyse d'un problème.

  • La révision d'analyse est suivie par les coordinateurs de problème lorsqu'ils examinent les résultats d'une analyse de la cause fondamentale réalisée par un spécialiste.

  • La clôture du problème est utilisée par les coordinateurs de problème lors de la clôture d'un problème.

Pour obtenir des informations sur ces procédures, cliquez sur Processus pour revenir à la représentation graphique de ce processus et cliquez sur les encadrés représentant les procédures. La représentation graphique de la procédure s'affiche avec sa description dans le volet de droite.
