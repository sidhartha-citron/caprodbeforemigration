Description du processus

Le processus de gestion des mises en production est constitué des procédures suivantes :

  • La gestion des demandes de changement est utilisée par les coordinateurs de mise en production pour examiner les demandes et problèmes transmis à la gestion des mises en production.

  • La définition de mise en production est utilisée par les coordinateurs de mise en production pour organiser les réunions du CAB. Elle est également appliquée par les membres du CAB lorsqu'ils choisissent, parmi les demandes de changement soumises à leur examen, celles qui doivent être incluses dans la prochaine mise en production. Les coordinateurs de mise en production utilisent également cette procédure pour répartir les conditions des mises en production en groupes logiques pouvant être gérés de façon efficace par les coordinateurs de changement.

  • La justification métier est utilisée par les coordinateurs de mise en production lorsque des financements supplémentaires doivent être obtenus pour l'implémentation d'une mise en production.

  • La coordination de mise en production est utilisée par les coordinateurs de mise en production pour lancer l'implémentation des mises en production et décider des actions correctives à mener le cas échéant.

Pour obtenir des informations sur ces procédures, cliquez sur Processus pour revenir à la représentation graphique de ce processus et cliquez sur les encadrés représentant les procédures. La représentation graphique de la procédure s'affiche avec sa description dans le volet de droite.
