Etape de procédure

Instructions de travail pour les coordinateurs de mise en production

4.8.1

Ajoutez une action Remarques dans la mise en production pour résumer les résultats de l'implémentation de changement exécutée. Si l'implémentation des changements n'a pas totalement réussi, précisez les conditions qui n'ont pas été remplies lors de l'implémentation du changement, les raisons pour lesquelles elles n'ont pas été remplies, ainsi que celles pour lesquelles il a été décidé de ne pas ajouter de changement à la mise en production afin de respecter les conditions non remplies.

Remarque :

lorsqu'un coordinateur de changement vous informe qu'une autre demande de changement associée a été exécutée, revenez à l'étape 4.3.1.
