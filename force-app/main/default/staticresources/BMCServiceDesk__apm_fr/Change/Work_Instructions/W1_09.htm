Instructions de travail pour les coordinateurs de changement

1.9.1

Si la demande de changement concerne la prévention ou la résolution d'un problème, passez à l'étape 1.12.1, à condition que :

  • l'implémentation du changement puisse être coordonnée par un seul coordinateur de changement ;

  • l'implémentation n'altère pas les fonctionnalités du service ;

  • aucun financement supplémentaire ne soit nécessaire pour réaliser l'implémentation.

Pour tout autre type de demande de changement, poursuivez à l'étape 1.10.1.
