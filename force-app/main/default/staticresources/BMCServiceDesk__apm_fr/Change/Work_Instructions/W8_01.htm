Etape de procédure

Instructions de travail pour les spécialistes

8.1.1

Effectuez un dernier test dans l'environnement de production pour vérifier que le changement a été implémenté correctement.

Remarque :

si vous ne disposez pas des droits d'accès nécessaires pour effectuer un test dans l'environnement de production, travaillez avec un utilisateur qui en dispose.

8.1.2

Ajoutez une action Remarques dans la demande de support pour résumer le test effectué et ses résultats.
