Instructions de travail pour les spécialistes

6.4.1

Restaurez l'environnement à son état préalable à l'implémentation du changement.

6.4.2

Dans le champ Résolution de la tâche de test de production, décrivez comment l'environnement d'origine a été restauré.

6.4.3

Dans le champ Durée (HH:MM) de l'action Remarques, saisissez le temps que vous avez consacré à l'annulation du changement.
