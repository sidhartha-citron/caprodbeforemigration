Etape de procédure

Instructions de travail pour les membres du CAB

Remarque :

un e-mail est envoyé à tous les approbateurs définis dans le workflow d'approbation.

3.8.1

Ouvrez l'enregistrement d'approbation et examinez les informations sur le changement renseignées dans la section Demande de changement.

Ouvrez les conditions de changement, l'analyse de risque et d'impact et le plan d'implémentation.

3.8.2

Pour approuver la demande de changement, dans la section Historique des actions , cliquez sur Approuver/Refuser dans la colonne Action, ajoutez toute information supplémentaire dans le champ Commentaires et cliquez sur Approuver.

Pour refuser la demande de changement, sélectionnez l'option Approuver/Refuser dans la colonne Action, indiquez la raison du refus de la demande de changement dans le champ Commentaires et cliquez sur Refuser.

Remarque :

vous pouvez également approuver ou refuser la demande de changement en cliquant sur le lien dans la notification reçue par e-mail.
