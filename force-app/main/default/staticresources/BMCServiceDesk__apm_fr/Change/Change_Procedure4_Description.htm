Procédure 4 - Implémentation de changement d'infrastructure


Une fois le changement approuvé, les premières tâches d'implémentation sont affectées. S'il s'agit d'un changement d'application, l'implémentation se poursuit selon la Procédure 5 - Implémentation de changement d'application.

S'il s'agit d'un changement d'infrastructure, les spécialistes préparent l'implémentation afin de garantir la disponibilité opérationnelle. Cette préparation peut inclure la commande de matériel, la configuration d'un environnement de test et la réalisation de tests.

Une fois la préparation achevée, les spécialistes implémentent le changement conformément au plan de changement.
