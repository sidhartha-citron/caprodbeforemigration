Subcategory

Use this field to select the appropriate subcategory of the CI catalog part from the following list of options. Note that this field allows you to select only subcategories that are relevant for the selected category.

The following options are available for the Address category:

  • IP Address

  • MAC Address

The following options are available for the Card category:

  • LAN Card

  • Network Interface Card

  • Processor Card

  • Voice Card

  • WAN Card

  • Other Type of Card

Parameters (Y/N)

Track Asset/Tag #

Check this field to track the asset or tag number of the CIs that have been registered based on the CI catalog part.

Track Serial #

Check this field to track the serial number of the CIs that have been registered based on the CI catalog part.

Track Version #

Check this field to track the version number of the CIs that have been registered based on the CI catalog part.

Quantity

Total

This field automatically displays the number of unique CIs of the CI catalog part that are registered in the CMDB.
