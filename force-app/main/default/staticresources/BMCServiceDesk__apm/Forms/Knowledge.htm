Knowledge article


The following table lists the fields of the Knowledge Article form and provides utilization guidelines for each field.

Field

Utilization

Record Type of new record

Use this field to select the record type from the following options:

  • FAQ - Articles that provide answers to frequently asked questions.

  • How to - Articles that provide steps to perform a task.

  • Known Error - Articles that provide information about known errors and workaround, if any.

  • Problem Solution - Articles that provide solution to a problem.

Article Information

Owner

This field displays the name of the person who created the knowledge article.

Category

Use this field to select the category of the knowledge article.

Status

Use this field to select the status from the following options:

  • Draft

  • In Review

  • Published

  • Retired

Display in Self Service

Use this field to specify whether the knowledge article can be shown in Self Service.

Article Details

Title

Use this field to enter a short title for the knowledge article.

Keywords

Use this field to enter the keywords for the knowledge article. While searching, the search string is matched with keywords to show matching knowledge articles.

Record Type: FAQ and How To

Question

Use this field to enter the question for the FAQ or How To article.

Answer

Use this field to enter the answer for the FAQ or How To article.

Record Type: Known Error

Error

Use this field to enter the error description of the known error.

Root Cause

Use this field to enter the root cause of the known error.

Workaround

Use this field to enter the workaround for the known error, if available.

Record Type: Problem Solution

Problem

Use this field to enter the description of the problem.

Solution

Use this field to enter the solution to the problem.

Additional Information

Field

Utilization

Published Date

This field is set to the date and time at which the knowledge article is published.

Review Date

This field is set to the date and time at which the article is due for review.

Retired Date

This field is set to the date and time at which the article is marked as retired.

Viewed in Self Service #

This field is set to the number of times the article is viewed in Self Service.

Related Lists

Notes and Attachments

This section lists the documents that are related to the knowledge article. Use this section to attach the files or notes that provide additional information about the knowledge article.

Knowledge Feedback

This section lists the feedback for the knowledge article.

Linked Incidents

This section lists the incidents that are related to the knowledge article. Use this section to link incidents to the knowledge article.

Approval History

This section lists the approval records for the knowledge article.
