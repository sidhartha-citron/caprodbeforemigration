Service


The following table lists the fields of the Service form and provides utilization guidelines for each field.

Field

Utilization

Attributes > General

Instance name

Use this field to enter a unique name for the service.

Description

Use this field to enter the description of the service.

Short Description

Use this field to enter a short description of the service. This description is displayed in the CI Explorer.

Notes

Use this field to enter additional information about the service.

Manufacturer Name

This field is not applicable for a service.

Model

This field is not applicable for a service.

Version Number

This field is not applicable for a service.

Serial #

This field is not applicable for a service.

Asset #

This field is not applicable for a service.

Token Id

This field is not applicable for a service.

Priority

Use this field to select the priority of the service.

Created Date

This field is set to the date and time at which the service was registered.

Last Modified By

The login name of the user who last updated the service is saved in this field.

Last Modified Date

This field is set to the date and time at which the service was last updated.

Submitter

The login name of the person who registered the service is saved in this field.

Instance ID

This field is set to the instance of the element that was used to register the service.

Class ID

This field is set to the class ID of the element that was used to register the service.

Class Name

This field is set to the class name of the element that was used to register the service.

Supported

Use this field to specify whether technical support is provided for the service.

Warranty Exp Date

This field is not applicable for a service.

Site

This field is not applicable for a service.

Use this field to mark the service as soft-deleted.

Inactive

Relationships

Mark As Deleted

Use this field to mark the relationship as soft-deleted.

Use this field to select the appropriate impact direction from the following options:

  • Other

  • Unknown

  • Source - Destination

  • Destination - Source

  • Service - Subservice

  • Service - Offering

Related Lists or Supporting Information

History

This field provides the history notes that are created for the service.

Clients

This tab lists the clients that are related to the service. Use this tab to select the client who will use the service or for whom the service is reserved. If the service is not used by, or reserved for, a specific client, do not select a client.

Accounts

This tab lists the accounts that are related to the service. Use this tab to select the organizations that will use the service or for which the service is reserved. If the service is not used by, or reserved for, a specific organization, do not select an account.

Incidents

This section lists the incidents that are related to the service. Use this tab to link incidents to the service.

Change Requests

This section lists the change requests that are related to the service. Use this tab to link change requests to the service.

Problems

This section lists the problems that are related to the service. Use this tab to link problems to the service.

Releases

This section lists the releases that are related to the service. Use this tab to link releases to the service.

Documentation

This section lists the documents that are related to the service. Use this section to attach files that provide additional information about the service.

Auditing

This tab lists the modifications that have been made to the service information.
