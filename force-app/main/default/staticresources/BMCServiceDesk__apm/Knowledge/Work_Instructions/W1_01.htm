Work Instructions for Operators

1.1.1

Click the Knowledge Articles tab, and then click New. Select the correct record type. Select the appropriate category, and enter the title. Enter the details of the knowledge article in the available fields. Select Submit for Approval to send the article for approval.

Note:

If the article is created from the Request Management process, select the FAQ or How to record type. If the article is created from the Problem Management process, select the Known Error or Problem Solution record type. If the article is created from the Change Management process, select any of the record types.
