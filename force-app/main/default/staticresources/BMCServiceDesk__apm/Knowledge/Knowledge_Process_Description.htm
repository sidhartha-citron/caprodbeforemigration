Process description


The Knowledge Management process consists of the following procedures:

  • Knowledge Creation is used by specialists when new knowledge articles are created.

  • Knowledge Approval is used by knowledge managers when knowledge articles need to be approved.

  • Knowledge Review is used by knowledge managers to review existing articles.

For details about these procedures, click Process to return to the graphical representation, and click the boxes that represent the procedures. The graphical representation of a procedure appears with its description in the right-hand pane.
