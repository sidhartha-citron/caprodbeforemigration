Work Instructions for Service Level Administrators

1.4.1

You can request an approval for the new catalog item from the:

  • Service owner who is responsible for meeting the SLTs in the catalog item

  • Service level managers of the customer organizations for which the catalog item has been prepared

  • Financial manager of the service provider organization

Note:

Get the approval signatures on a printed version of the new catalog item.

Note:

If the catalog item is not prepared for a specific customer, it will be targeted at a certain type of customer. In that case, the service level managers should still approve the new catalog item.

1.4.2

Set the Status field of the task to Waiting for and specify in the Action History section that you are waiting for the catalog item to be approved.

1.4.3

In Notes action, Duration (HH:MM) field, enter the time that you spent creating the catalog item.
