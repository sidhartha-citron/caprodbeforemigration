Work Instructions for Service Level Administrators

1.8.1

Set the Status field of the task back to In Progress.

1.8.2

File the hardcopy of the new catalog item with the approval signatures.

1.8.3

Prepare a portable document format (PDF) of the new catalog item and add it to the service provider organization's web-based service catalog.

Note:

This digital version of the new catalog item should not include the page with the approval signatures.

Note:

If the catalog item was prepared to meet the specific SLRs of one customer, ensure that the published version of the new catalog item cannot be accessed by other customers.

1.8.4

Send an email to the approvers of the catalog item to inform them about where the new catalog item is published.
