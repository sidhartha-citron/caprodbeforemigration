Work Instructions for Service Desk Analysts

2.5.1

In the Duration (HH:MM) field, in the Notes action, enter the time that you spent registering or updating the request.

2.5.2

If this is a new request, select the appropriate assignment group using the lookup icon next to the Owner field.

2.5.3

If the request is rejected (the Status field is set to Rejected), or if the user is dissatisfied with its solution (the Status field is set to Completed or ClosedMail), change the status of the request to Assigned and select the appropriate assignment group using the lookup icon next to the Owner field.

Note:

Do not assign the request to a specific staff member. The group coordinator of the group decides the appropriate staff member (in terms of skills, availability and access rights) to work on the request.

2.5.4

If the user is on the telephone, provide the request number. This number is at top of the Incident form. By communicating the request number, the user is assured that the request is registered.

Note:

The user can also be informed by email of the request number after saving the support request.

2.5.5

If the request must be dealt with as soon as possible (for example in the case of a virus attack), telephone or page the coordinator of the assigned group to ensure that the coordinator is aware of the urgency. Do the same when it is currently after normal office hours. It is important that a specialist starts to work on the request.
