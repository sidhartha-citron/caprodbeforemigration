Work Instructions for Specialists

5.8.1

In the Resolution field, specify what is done to complete the request. Use this field to provide detailed solution instructions (for example, if the user must adjust a setting), to provide the answer to a request for information, or to describe why the request cannot be resolved if it is not possible to provide a solution or workaround.

5.8.2

Select the appropriate closure category in the Closure Category field and set the Status field of the request to Completed.

5.8.3

Select the appropriate CI in the Configuration Item field if this field is not populated or if the wrong CI is populated.

Note:

If the CI is not registered in the CMDB, but is in the scope of the Configuration Management, select the CI with the NORECORD code.

The CI with the NORECORD code might not be available by default.

Note:

If the request is not for specific CI or the CI is outside the scope of the Configuration Management process, select the CI with the N/A code.

5.8.4

If the Impact field is set to High or Major, populate the date and time when the service became unavailable in the Outage Start Date field and populate the date and time at which the service became available again in the Outage End date field.

5.8.5

If you resolved an incident and find that similar incidents are likely to occur if the root cause is not removed, check whether the problem that has caused the incident is already registered. If the problem is already registered, link the request that you resolved to this problem.

If the problem that caused the incident is not registered, notify the problem coordinator of the service that was affected, by email or telephone and ensure that you provide the request number of the incident to the problem coordinator and explain why you believe that the incident is likely to recur.

5.8.6

In the Duration (HH:MM) field, in the Notes action, enter the time that you spent resolving the request.

5.8.7

Perform a final review of the request to ensure that its information is accurate and complete.

5.8.8

If similar requests are likely to be submitted, create a knowledge article for the provided solution.
