Work Instructions for Specialists

5.7.1

Resolve the request. Request support from external suppliers, if necessary.

Note:

If the request cannot be resolved before the date and time specified in the Due Date field, estimate a realistic time by which the request can be resolved and add a Notes action to enter this information in the request.

Note:

Set the Status field to Waiting for or Waiting for customer if the work cannot continue (for example, because you are waiting for a spare part or for information from a customer). Add a Notes action and specify what you are waiting for. If you registered a request with a supplier, enter this supplier and the request number that you got from the supplier in Notes action.

Note:

If you have completed your work on the request, but further action is required to resolve the request, add a Notes action to specify what you have done. In the Duration (HH:MM) field, in the Notes action, enter the time that you spent working on the request. If a specific staff member in your group must do additional work, use the lookup icon next to the Owner field to select the staff member in the Owner field. If another group must do the additional work, use the lookup icon next to the Owner field to select the group in the Owner field. Ensure that the Status field is set to Assigned.
