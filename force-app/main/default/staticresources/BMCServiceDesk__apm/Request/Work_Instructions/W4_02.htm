Work Instructions for Service Desk Analysts

4.2.1

Check the status of the request periodically to ensure that the request progresses towards completion.

4.2.2

If the request is not progressing (for example, because, one day after it is registered, its Status field is still set to Assigned), contact the group coordinator to which the request is assigned (by phone or email) and ask the coordinator to ensure that the request is dealt with.
