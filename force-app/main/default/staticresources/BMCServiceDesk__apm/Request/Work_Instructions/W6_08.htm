Work Instructions for Group Coordinators

6.8.1

If the task is not assigned to the appropriate group or if it does not contain all information that must be provided to resolve the task, continue with 6.9.1. Otherwise, go to 6.11.1.

Note:

If you are the group coordinator of the service desk, ensure that the task contains all information that must be collected before the work can start on the task. Once this is done, assign the task to a service desk analyst who is currently available for resolving tasks. If the service desk cannot complete the task, assign it to the appropriate group.
