Work Instructions for Service Desk Analysts

2.9.1

Review the information in the request to understand what has been requested and find out why the request is rejected by a group coordinator.

2.9.2

If the group coordinator rejects the request because the information is incomplete, obtain the missing information, and add the information to the request (for example, select the configuration item for which the request is registered in the Configuration Item field). Add a Notes action and summarize what you have done.
