Work Instructions for Service Desk Analysts

1.12.1

Read the instructions in the Description field of the new request to determine the information that must be collected from the user and how to resolve the request.

1.12.2

Collect the required information from the user and add the information to the request as per the instructions in the Description field.

1.12.3

Save the request.

If the request definition is selected, that is, the request type is set to Service Request, the accompanying work orders are generated and a popup appears where you must answer the questions related to the service request.
