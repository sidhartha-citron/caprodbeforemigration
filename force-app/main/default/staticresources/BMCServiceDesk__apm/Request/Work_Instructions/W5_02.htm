Work Instructions for Group Coordinators

5.2.1

If Request Fulfillment can handle a request, go to 6.1.1. Otherwise, continue with 5.3.1.

Note:

Request Fulfillment can handle a request when the request can be implemented by following the workflow defined in a request template that is approved by the service owner and this template is used to create the request.

Change Management must handle the request if implementing the request requires risk and impact analysis or approvals other than financial approval.
