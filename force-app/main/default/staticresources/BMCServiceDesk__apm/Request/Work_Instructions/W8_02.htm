Work Instructions for Service Desk Analysts

8.2.1

Describe how the request is resolved in the Resolution field.

8.2.2

Select the appropriate configuration item (CI) in the Configuration Item field if this is not done yet, or the wrong CI is selected.

Note:

If the CI is not registered in the CMDB, but is in the scope of the Configuration Management process, select the CI with the NORECORD code.

The CI with the NORECORD code might not be available by default.

Note:

If the request is not for a specific CI, or if the CI is outside the scope of the Configuration Management process, select the CI with the N/A code.

8.2.3

In the Duration (HH:MM) field, in the Notes action, enter the time that you spent registering and resolving the request.

8.2.4

Select the appropriate closure category in the Closure Category field. Set the Status field of the request to Completed.

8.2.5

Ensure that you are selected in the Owner field.

8.2.6

Perform a final review to ensure that the request is filled correctly.

8.2.7

If the workaround of an open problem is applied to resolve the request, link the incident to the problem.
