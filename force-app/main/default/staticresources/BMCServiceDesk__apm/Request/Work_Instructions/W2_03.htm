Work Instructions for Group Coordinators

2.3.1

If Request Fulfillment can handle the request, go to 6.1.1. Otherwise, continue with 2.4.1.

Note:

If the request is an incident, a specialist determines in 5.2.1 how to resolve the incident. For an incident, go to 2.4.1.

Note:

Request Fulfillment can handle a request when the request can be implemented by following the workflow defined in a request template that is approved by the service owner and this template is used to create the request.
If the implementation of a request requires risk and impact analysis or approvals other than financial approvals, it must be handled by Change Management.
