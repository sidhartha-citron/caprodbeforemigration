Work Instructions for Service Desk Analysts

8.1.1

Resolve the request. If necessary, ask the user and your colleagues at the service desk for assistance.

Note:

If a template is used to register the request, the information in the request can explain how the request can be resolved.

Note:

If the request is caused by an open problem that offers a workaround in its Workaround field, follow the instructions to implement the workaround.
