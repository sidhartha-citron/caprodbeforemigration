Work Instructions for Service Desk Analysts / Specialists

6.5.1

For each task, determine the most appropriate staff member of your group (in terms of skills, availability and access rights) to execute the task. Use the lookup icon next to the Owner field to select the staff member or the correct group in the Owner field of the task.

If you can complete a task, use the lookup icon next to the Owner field to select yourself in the Owner field of the task.

If another group must complete a task, use the lookup icon next to the Owner field to select the group in the Owner field.

If an implementation task must be completed by an external organization, discuss the task with your contact person at this organization. Work with this person to determine a realistic implementation target. Use the lookup icon next to the Owner field to select yourself in the Owner field of the task. Add a Notes action to enter the external organization, that is asked to do the work. When the work starts, inform the supplier and enter the reference number, under which the supplier has registered this assignment, in the Notes action..

Note:

Only assign the task that do not have to wait for other tasks to be completed first.
