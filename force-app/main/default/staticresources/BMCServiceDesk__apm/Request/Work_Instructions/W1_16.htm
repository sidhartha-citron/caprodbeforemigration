Work Instructions for Service Desk Analysts

1.16.1

Use the lookup icon next to the Owner field to select the service owner of the failed Service.

Note:

Click Agent Tools > CI Explorer to view the service and relationships. Select Class Name to view the details of this service. The CMDB opens with the selected service. Select the specifications of the service to identity the service owner of the failed service.

1.16.2

Contact the service owner and inform the owner about the situation. Ensure that you inform the owner that the incident is now escalated to the owner.

Note:

Contact the on-duty manager if the service owner is not available.

1.16.3

Add a Notes action to specify that you have escalated the incident to the service owner or on-duty manager.

1.16.4

In the Duration (HH:MM) field, in the Notes action, enter the time that you spent reviewing the request.
