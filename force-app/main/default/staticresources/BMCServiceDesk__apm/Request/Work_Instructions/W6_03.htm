Work Instructions for Service Desk Analysts / Specialists

6.3.1

Add the required implementation and approval tasks to ensure that all request requirements are met. Ensure you select the correct category.

Note:

When multiple tasks must be assigned to the same person, and these tasks must be completed in sequence without interruption, a single task can be created for all these tasks.

6.3.2

Link each new implementation task to the CIs that are changed (if any) when the task is executed. Click Select and Link Services and Configuration Items to Change Request in the Linked Services and Configuration Items related list of such tasks.

6.3.3

Ensure that the date and time specified in the Due Date field of the new tasks is achievable. Update the Due Date field of tasks that are planned earlier, and must be completed after the new tasks have been executed.

6.3.4

Ensure that the Status field of the new tasks is set to Registered.
