Work Instructions for Service Desk Analysts

8.3.1

If the information in the Resolution field is entered by a specialist, ensure that it is meaningful (for example, the information can explain how the solution is implemented). Otherwise, contact the specialist and work with the specialist to improve the information. Correct any spelling or grammatical mistakes in this field.

8.3.2

Inform the user about how the request is completed and ask the user to verify this solution. The user might be informed by telephone to verify the solution. Alternatively, the user might be informed by setting the Status field to ClosedMail, after you have entered the number of minutes you spent on the final review of the support request in the Duration (HH:MM) field in the Notes action.

Note:

Saving the request with the ClosedMail status can automatically send an email to the user. This email informs the user that the request is completed and asks the user to verify the solution. It also invites the user to contact the service desk if the solution is not satisfactory.
