Work Instructions for Service Desk Analysts

1.2.1

If the user is on the telephone, or if the user visits the service desk in person, ask the user how you can provide assistance.

1.2.2

Use the provided information (regardless of whether the information is obtained over the telephone, in an email, or otherwise) to determine if the user has contacted the service desk about a previously reported request, or if the user wants to report a new request.

1.2.3

If the user is submitting a new request, determine the related service. Also, determine what the user is requesting, and ask for any additional information that could help resolve the request.

Note:

If you need more information to resolve the request, and you are not currently in contact with the user (for example, if you do not have the user on the telephone, or the user is not at the service desk), contact the user by phone or email, and request the additional required information. Do not register a request until this information is provided.
