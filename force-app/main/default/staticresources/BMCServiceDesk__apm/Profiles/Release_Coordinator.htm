Release Coordinator Profile

A release coordinator must have the following characteristics:

  • Capability to continuously improve the key performance indicators of the Release Management process

  • Detailed knowledge about all service instances used to provide the services for which they act as the release coordinator

  • Capability to understand to some extent the implications of requests to change the level of service or the functionality that a service provides

  • Excellent written communication skills

  • Service-oriented attitude

  • Detailed understanding of the Change and Release Management processes

  • Capability to use the service management application to fulfill the role of release coordinator

Recommendation:

The service owner or the change coordinator of each service
