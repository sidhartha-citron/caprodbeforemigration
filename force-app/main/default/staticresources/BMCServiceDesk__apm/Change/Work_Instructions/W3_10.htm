Work Instructions for Change Managers

3.10.1

Read the comments in the approval records where the status is Rejected to discover why the change request was rejected.

3.10.2

If the change request was rejected because one or more approvers considered the risk and impact analysis inadequate, or identified a timing conflict, return to 3.3.1. If the change request was rejected for any other reason, continue with 3.11.1.
