Work Instructions for Problem Coordinators

4.3.1

In the Resolution field, summarize how the structural solution was implemented.

4.3.2

In the Notes action, Duration (HH:MM) field, enter the time that you spent on verifying the structural solution

4.3.3

Set the Status field to Solved.

4.3.4

In the Closure Category field, select an appropriate category.

Note:

If solution is provided in the Resolution field of the problem, the service desk can be informed of the solution through email.

4.3.5

If the root cause was removed, and the knowledge article created for the problem is not required, remove this article from the knowledge base.

4.3.6

If needed, create a new knowledge article with the solution in the knowledge base.
