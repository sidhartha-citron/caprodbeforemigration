Work Instructions for Specialists

2.3.1

Try to find a temporary workaround that can be used to resolve incidents caused by the problem. Check the Resolution field of the requests that are linked to the problem.

2.3.2

If you are able to suggest a workaround, describe (step by step) it in a new knowledge article and provide a link to the article in the Workaround field of the problem.

Note:

Until you can provide a structural solution, the service desk analysts and/or specialists can use the workaround to resolve incidents that are caused by this problem.

Note:

If a workaround is already specified in the Workaround field, review and improve it if possible.

2.3.3

In the Workaround field, specify the reason if it is not possible to provide a practical workaround.
