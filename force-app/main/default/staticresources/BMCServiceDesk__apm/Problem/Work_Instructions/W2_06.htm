Work Instructions for Specialists

2.6.1

Now that you know the root cause, find ways to remove it or provide a permanent workaround.

Note:

If the configuration item (CI) in that has caused the problem is still supported by an external supplier (that is, the CI is still covered by warranty or a support contract), you can submit a solution request to the supplier for the problem.

To find out whether the CI is supported by an external supplier, open the CI and click the Specifications tab to check the Warranty Exp Date field.

2.6.2

Add a Notes action to describe each proposed structural solution.

2.6.3

Determine which of the proposed structural solutions is the most practical (based on technical, financial and availability considerations). Add a Notes Action to specify with reasoning which structural solution you consider most practical.

2.6.4

Add a Notes action to describe in detail how the preferred structural solution should be implemented. If an earlier workaround needs to be removed, specify how this should be done in the same field.

Note:

Add a Notes action with reasons if no practical structural solution was found.
