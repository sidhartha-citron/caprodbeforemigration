Work Instructions for Specialists

2.6.1

Knowing the root cause, find ways to remove it or to permanently work around it.

Note:

If the Configuration Item in which the problem resides is still supported by an external supplier (i.e. the Configuration Item is still covered by warranty or a support contract), you can submit a support request to the supplier to request a solution to the problem.

To find out whether the Configuration Item is still supported by an external supplier, open the Configuration Item and see the Warranty Expiration Date field.

2.6.2

Describe each structural solution option through Note actions.

2.6.3

Determine which of the proposed structural solutions is the most practical (based on technical, financial and availability considerations). Specify which structural solution you consider most practical, along with your reasoning, through Note action detail of the problem.

2.6.4

Use the Solution field of the problem to describe in detail how the preferred structural solution should be implemented. If a workaround that was implemented earlier needs to be removed when the structural solution is implemented, specify how this should be done in the same field.

Note:

If for any reason a practical structural solution could not be found, specify in the Note field of the problem why this is the case and update the action.
