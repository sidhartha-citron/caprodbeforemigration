Work Instructions for Problem Coordinators

4.6.1

Set the Status field of the problem to "No Resolution" to indicate that no further action is currently required.

Note:

If the Status field of the problem was already set to "No Resolution" because you just completed a periodic check for new possibilities, specify in the Notes action  that your periodic check was fruitless.

4.6.2

Enter the time you spent on the review of the problem in the Duration (HH:MM) field.
