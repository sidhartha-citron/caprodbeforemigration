Work Instructions for Specialists

2.3.1

Try to find a temporary workaround that can be used to resolve incidents that are caused by the problem. Check the Res olution field of the support request(s) that are linked to the problem. The information in this field could help you find a suitable workaround.

2.3.2

If you are able to propose a workaround, describe (step by step) in the Workaround field of the problem how it can implemented.

Note:

Until a structural solution has been implemented, service desk analysts and/or specialists can use this workaround to resolve incidents that were caused by this problem.

Note:

If a workaround was already specified in the Workaround field during a previous analysis, review it and improve it if possible.

2.3.3

If it is not possible to provide a practical workaround, ensure that this, along with the reason why, is specified in the Workaround field of the problem.
